# 비교계산소 SEO / 광고 준비 로드맵

## 문서 목적
- 비교계산소의 SEO, 검색 노출, 광고 준비 작업을 단계별로 정리한다.
- 실제 작업 진행 시 체크리스트와 기록 문서로 함께 사용한다.

## 프로젝트 목표

- 계산기 + 가이드 콘텐츠 구조를 정리해 검색 유입 기반을 만든다.
- Search Console로 인덱싱 상태를 관리한다.
- AdSense 심사 준비가 가능한 수준까지 사이트 품질과 콘텐츠를 확보한다.

## Phase 1 — 기술 SEO 기본 세팅

### 목표
- 크롤링 가능
- 인덱싱 가능
- 정식 도메인과 canonical 정리
- robots.txt / sitemap.xml 준비

### 작업 항목

#### 1. 정식 도메인 단일화
- 정식 주소를 `https://bigyocalc.com` 하나로 고정
- `www.bigyocalc.com` 사용 시 한쪽으로 301 리디렉션
- `pages.dev` 주소도 `bigyocalc.com`으로 리디렉션

#### 2. canonical 정리
- 모든 페이지에 canonical URL 삽입
- 계산기 상세, 가이드, 허브 페이지 모두 canonical 일관성 유지

#### 3. robots.txt 생성
- 최소 예시:

```txt
User-agent: *
Allow: /

Sitemap: https://bigyocalc.com/sitemap.xml
```

#### 4. sitemap.xml 생성
- 포함 대상
  - 홈
  - `/tools/`
  - `/guides/`
  - 계산기 상세 페이지
  - 가이드 페이지
  - 카테고리 / 허브 페이지

#### 5. 메타 태그 기본 세팅
- 모든 페이지에 최소 포함
  - `<title>`
  - `<meta name="description">`
  - canonical
  - Open Graph
  - Twitter Card

### 체크리스트
- [x] `https://bigyocalc.com/robots.txt` 접근 가능
- [x] `https://bigyocalc.com/sitemap.xml` 접근 가능
- [x] 확인한 주요 페이지 source에 canonical 존재
- [x] 대표 도메인 기준 중복 노출 이슈 없음
- [x] 404 페이지 별도 존재

### 진행 기록
- 상태: **완료** (2026-03-20)
- 메모:
  - `robots.txt` 실접속 확인 완료
  - `sitemap.xml` 실접속 확인 완료

## 네이버 vs 구글 트래픽 — 애드센스 수익화 전략 (2026-06-22 정리)

### 핵심 결론
**네이버 검색 유입만 있어도 애드센스 수익은 가능하다.** 애드센스는 "구글 검색 유입만 인정"하는 구조가 아니라, 정상적인 방문자에게 광고가 유효하게 노출·클릭되는지가 핵심이다. 네이버 자연검색 유입은 Google이 금지하는 비정상 트래픽(유료 클릭 유도, 소프트웨어 자동 노출 등)과 성격이 다르므로 불이익이 없다.

| 질문 | 답 |
| --- | --- |
| 네이버 유입만 있어도 애드센스 수익 가능? | 가능 |
| 구글 검색 유입이 꼭 필요함? | 아님 |
| 네이버 유입은 불이익 있음? | 정상 검색 유입이면 문제 없음 |
| 애드센스 승인은 트래픽보다 콘텐츠 품질이 중요? | 맞음 |
| 수익은 어디서 결정됨? | PV, 광고 노출수, CTR, CPC/RPM, 체류시간, 광고 위치 |

### 현재 사이트 구조상 네이버가 유리한 이유
비교계산소의 핵심 키워드(성과급 계산기, 세후 실수령액, 청년미래적금, 근로·자녀장려금 등)는 한국 생활형/금융 검색 의도가 강해 네이버 검색에서 더 빠르게 반응한다. 이런 키워드는 네이버 노출만 잘 잡아도 초기 수익화가 가능하다.

### 예상 수익 구조
```text
애드센스 수익 = 페이지뷰 ÷ 1,000 × 페이지 RPM
```
한국 정보성 사이트는 초반에 페이지 RPM 500원~3,000원 정도로 보수적으로 잡는 것이 현실적이다. 금융·보험·대출·세금·취업·연봉 주제는 RPM이 상대적으로 높고, 육아·일상 정보는 낮은 편이다.

| 일 방문자 | 월 페이지뷰 가정 | RPM 500원 | RPM 1,500원 | RPM 3,000원 |
| ------: | ---------: | -------: | ---------: | ---------: |
| 300명 | 9,000 PV | 4,500원 | 13,500원 | 27,000원 |
| 500명 | 15,000 PV | 7,500원 | 22,500원 | 45,000원 |
| 1,000명 | 30,000 PV | 15,000원 | 45,000원 | 90,000원 |
| 3,000명 | 90,000 PV | 45,000원 | 135,000원 | 270,000원 |
| 10,000명 | 300,000 PV | 150,000원 | 450,000원 | 900,000원 |

현재 네이버 일 클릭 400~500 수준 기준으로는 승인 후 초반 월 1만~5만 원대부터 보는 것이 현실적이며, 계산기·복지·연봉 키워드가 쌓이면 월 10만~30만 원 구간을 목표로 잡을 수 있다.

### 네이버 유입의 장단점

| 장점 | 설명 |
| --- | --- |
| 초기 노출 빠름 | 네이버 서치어드바이저 반응이 구글보다 빠른 편 |
| 한국 생활형 키워드 강함 | 연봉, 세후, 장려금, 적금, 복지 키워드와 잘 맞음 |
| 시즌 키워드 반응 좋음 | 청년미래적금, 근로장려금, 성과급 같은 이슈형 키워드 |
| CTR 높게 나올 수 있음 | 제목이 잘 맞으면 클릭률이 높음 |

| 단점 | 설명 |
| --- | --- |
| 체류시간 짧을 수 있음 | 답만 보고 바로 나가는 경우 많음 |
| 광고 단가가 구글 유입보다 낮을 수 있음 | 유입 품질/페이지 행동에 따라 차이 |
| 네이버 알고리즘 변동 리스크 | 노출이 갑자기 줄 수 있음 |
| 구글 색인 안 되면 장기 누적 약함 | 구글은 느리지만 오래 가는 트래픽이 됨 |

**전략 요약**: 초기 돈은 네이버, 장기 안정성은 구글.

### 승인 전/후 전략 차이
승인 전에는 트래픽량보다 "사이트가 독립적인 가치가 있는가"가 중요하다 (개인정보처리방침, 문의하기, 사이트 소개, 계산기 사용법, 광고/면책 고지, 얇은 페이지 정리, 자동 생성 느낌 줄이기, 계산기별 설명 콘텐츠 1,000~2,000자 이상, 모바일 레이아웃 안정화, 중복 페이지 정리) — 이 항목들은 위 "반려 이력과 학습" 섹션의 Low value content 대응과 동일한 맥락이다.

승인 후 추천 광고 배치:

| 위치 | 추천 |
| --- | --- |
| 본문 상단 | 1개 |
| 계산 결과 아래 | 1개 |
| FAQ 위 | 1개 |
| 하단 관련 계산기 위 | 1개 |
| 자동 광고 | 초반에는 켜되 과하면 조정 |

계산기 사이트는 특히 **계산 결과 아래 광고**가 효율적이다. 사용자가 결과를 확인한 다음 행동이 비는 구간이기 때문이다.

### bigyocalc.com에 맞는 수익화 순환 구조
```text
검색 유입 → 계산기 사용 → 결과 확인 → 관련 리포트 클릭 → 비교표 이동 → 광고 2~4회 노출
```
예시: 삼성전자 성과급 계산기 → 삼성전자 연봉 리포트 → 하이닉스 성과급 계산기 → 대기업 평균 연봉 비교표. 이런 내부 순환이 페이지뷰와 RPM을 함께 끌어올린다.

### 네이버만 믿으면 안 되는 이유

| 리스크 | 대응 |
| --- | --- |
| 네이버 노출 급락 | 구글 색인 개선 |
| 특정 키워드 시즌 종료 | 상시 키워드 확보 |
| 페이지 체류시간 낮음 | 계산 결과 + 관련 콘텐츠 강화 |
| 광고 클릭률 낮음 | 광고 위치 테스트 |
| 승인 지연 | 콘텐츠 품질/정책 페이지 보강 |

```text
단기: 네이버 검색으로 트래픽 확보
중기: 애드센스 승인 + 광고 배치 최적화
장기: 구글 색인 + 직접 방문 + 내부링크 허브화
```

### 카테고리별 애드센스 수익성 우선순위
새 콘텐츠 기획 시 이 우선순위를 참고해 어떤 주제를 먼저 강화할지 판단한다.

| 카테고리 | 수익성 | 이유 |
| --- | ---: | --- |
| 연봉/성과급 | 높음 | 직장인, 이직, 금융 광고 매칭 |
| 세후/실수령액 | 높음 | 세무·금융 광고 매칭 |
| 청년적금/ISA/연금 | 높음 | 금융 광고 매칭 |
| 소상공인 정책자금 | 높음 | 대출/사업자 금융 광고 가능 |
| 근로·자녀장려금 | 중상 | 복지+금융 혼합 |
| 육아수당/보육료 | 중간 | 검색량은 좋지만 단가는 낮을 수 있음 |
| 생활 계산기 | 낮음~중간 | 트래픽용 |

```text
우선순위: 1. 성과급/연봉 → 2. 세후/실수령액 → 3. 청년미래적금/ISA/연금
       → 4. 소상공인 정책자금 → 5. 근로장려금/자녀장려금 → 6. 육아수당/보육료
```

### 종합 결론
네이버 검색만 잘돼도 애드센스 수익은 발생한다. 오히려 한국형 생활 계산기 사이트는 초반에 네이버 유입이 더 빠르게 수익으로 연결될 가능성이 크다. 다만 역할 분담은 다음과 같이 본다.

```text
네이버 = 초기 트래픽/수익
구글 = 장기 누적/안정성
내부링크 = PV 증가
계산기 = 체류시간 증가
리포트 = SEO 확장
```

다음 단계는 **애드센스 승인용 품질 보강(2026-06-25 재신청 대비) + 네이버 상위 키워드 확장 + 구글 색인 문제 해결**이다.

# 한국인 평균 연봉·자산 2016→2026, 따라잡을 수 있나

## 1. 문서 개요

- **콘텐츠 유형**: 비교 리포트형 페이지
- **목적**: 2016년과 2026년 사이 한국인의 연봉·순자산 변화, 그리고 서울 집값 상승 속도를 함께 보여주며 “소득은 올랐는데 왜 체감은 더 어려운가”를 데이터로 설명하는 리포트
- **기준 레퍼런스**:
  - `https://bigyocalc.com/reports/wedding-cost-2016-vs-2026/` 와 유사한 비교형 구성
  - 업로드된 연봉·자산 HTML 리포트 구조 반영
- **핵심 질문**:
  - 한국인 평균/중위 연봉은 10년 동안 얼마나 올랐는가
  - 순자산은 얼마나 늘었는가
  - 서울 집값을 따라잡을 수 있었는가
  - 부동산 보유자와 미보유자의 격차는 얼마나 벌어졌는가

---

## 2. 페이지 목표

### 사용자 관점 목표
- 내 연봉이 지난 10년 흐름에서 어느 위치인지 이해한다.
- 단순 연봉 상승이 아니라 **자산 형성 가능성**까지 같이 본다.
- 서울 집값과 비교해 “따라잡을 수 있었는가”를 직관적으로 본다.
- 내 상황에서 저축·이직·자산관리 액션으로 이어질 수 있게 한다.

### 사이트 관점 목표
- `연봉`, `평균 자산`, `중위 연봉`, `서울 집값`, `자산 격차` 키워드 유입 확보
- 비교계산소 내 다른 계산기/리포트로 자연스럽게 내부 이동 유도
- ISA/IRP/자산관리/연봉 계산기 관련 제휴·연결 포인트 확보

## 3. 타겟 독자

## 6-3. 메인 차트 섹션 — 평균 연봉·평균 자산 추이

### 목적
중위값 중심 비교만으로는 부족하므로, 평균 연봉과 평균 순자산의 10년 흐름을 별도로 보여줘 사용자가 “전체 평균 체감”도 같이 읽게 만드는 섹션

### 차트 제목
`평균 연봉·평균 순자산 추이 (2016→2026)`

### 차트 설명
- 연도별 평균 연봉과 평균 순자산을 2개 라인으로 표시
- 단위가 다르므로 축을 분리
  - 좌측 Y축: 평균 연봉(만원)
  - 우측 Y축: 평균 순자산(억원)
- 평균값은 상위 구간의 영향이 커서, 중위값과 다를 수 있다는 설명을 같이 둠

### 추천 카피
`평균 연봉은 2016년 3,420만원에서 2026년 4,780만원으로 늘었지만, 평균 순자산은 2.8억에서 5.4억으로 더 크게 상승했다. 자산 가격 상승과 기존 보유자산 효과가 평균을 끌어올린 구조다.`

### UX 포인트
- hero 아래 KPI 다음 첫 차트로 배치 가능
- 차트 하단에 `평균은 상위 구간 영향이 크고, 중위는 가운데 사람의 값`이라는 안내 문구 배치
- 모바일에서는 line 2개 + dual axis 유지하되 tooltip에 단위를 명확히 표시

## 6-4. 메인 차트 섹션 — 연봉 vs 자산 vs 서울 집값 지수 비교

### 목적
이 페이지의 핵심 시각 장치. 2016년을 100으로 두고 2026년까지의 격차 확대를 직관적으로 보여줌

### 차트 제목
`중위 연봉·중위 순자산·서울 집값 지수 비교 (2016=100)`

### 차트 설명
- 2016년을 100으로 설정
- 선 3개를 동시에 노출
  - 중위 연봉 지수
  - 순자산 중위 지수
  - 서울 집값 지수
- 독자가 “연봉보다 자산이 더 올랐고, 자산보다 집값이 더 빨랐다”를 한눈에 이해할 수 있도록 구성

### 추천 카피
`2016년을 100으로 놓고 보면, 중위 연봉은 132, 순자산 중위는 173, 서울 집값은 252까지 올라갔다. 소득보다 자산이 빨리 늘었고, 자산보다 집값이 더 빨리 뛰었다.`

### UX 포인트
- 모바일에서는 단일 line chart 유지
- 범례는 직접 아래 설명 문구 또는 커스텀 legend로 처리
- 특정 연도 hover 시 세 수치가 같이 보이도록 tooltip 정리

## 6-7. 부동산 보유 여부에 따른 자산 분기 섹션

### 목적
이 페이지의 두 번째 핵심 메시지: **같은 10년이라도 보유 여부가 자산 결과를 크게 갈랐다**

### 구성
좌우 카드 2개
- 부동산 보유자 순자산
- 부동산 미보유자 순자산

### 카드 내용
#### 보유자
- 2026 순자산: 5.8억
- 2016 순자산: 2.6억
- 증가액: +3.2억 (+123%)
- 주요 자산: 부동산 평균 5.1억 / 금융자산 0.7억

#### 미보유자
- 2026 순자산: 0.6억
- 2016 순자산: 0.5억
- 증가액: +0.1억 (+20%)
- 주요 자산: 금융자산 0.4억
- 월세 부담: 연 1,404만원

### 강조 문구 박스
`2016년에는 자산 격차가 2.1억 수준이었지만, 2026년에는 5.2억까지 벌어졌다. 같은 기간 열심히 저축해도 부동산 가격 상승폭을 따라잡기는 매우 어려웠다.`

## 6-8. 연령대별 순자산 변화 섹션

### 목적
연령대별로 자산 형성 속도가 어떻게 달라졌는지 보여주는 블록

### 구성

- 20대
- 30대
- 40대
- 50대

### 카피 포인트
- 30대: 집을 사기 가장 힘들어진 세대
- 40~50대: 보유 자산 확대가 가장 크게 나타난 구간

### 설명 문구
`같은 10년이라도 연령대별 자산 증가 폭은 크게 달랐다. 특히 40~50대는 기존 보유자산의 상승 효과가 컸고, 30대는 자산 형성 출발선 자체가 늦어졌다.`

## 6-10. 체감 계산기 섹션

### 목적
리포트형 콘텐츠 안에서도 사용자의 연봉을 넣어 직접 체감하게 만드는 상호작용 블록

### 입력값
- 현재 연봉 (만원)

### 출력값
1. 연봉 분위 추정
2. 10년 저축 가능액 (월 30% 저축 가정)
3. 2026 서울 집값 대비 부족액
4. 서울 전세 보증금 마련 예상 기간

### 예시 문구
- `현재 연봉 4,000만원이면 중위 근로소득대와 유사한 수준입니다.`
- `월 30%를 10년간 저축해도 서울 평균 집값과는 큰 차이가 남습니다.`

### 연결 포인트
- `연봉 인상 계산기`
- `이직 계산기`
- `가구 소득 계산기`
- 향후 `자산 형성 계산기` 신설 가능

## 6-11. 관련 서비스 / 제휴 섹션

### 목적
자산 격차 대응 방법을 “실행 가능한 서비스”로 연결

### 추천 카드
1. **ISA 계좌**
   - 태그: `비과세`
   - 설명: 장기 자산 형성, 절세 포인트
2. **IRP / 연금저축**
   - 태그: `세액공제`
   - 설명: 소득이 있을수록 절세 체감 큼

## 12. 구현 체크포인트

- 평균 연봉·평균 순자산 추이 차트는 **중위 지수 차트와 분리**해서 보여주는 것이 이해하기 쉽다.
- 평균값은 상위 분위 영향을 크게 받으므로, 차트 아래에 `평균과 중위의 차이` 설명 문구를 반드시 넣는다.
- 숫자 표기는 아래 형식을 권장한다.
  - 연봉: `4,780만원`
  - 자산: `5.4억원`
  - 지수: `2016=100`
- 내부 데이터 파일은 평균 시계열 / 중위 지수 시계열 / 연봉 분위표 / 연령대 자산표로 나눠두면 재사용이 쉽다.

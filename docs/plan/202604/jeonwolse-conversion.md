## 2. 한 줄 정의

전세와 월세 조건을 **같은 기준으로 환산**해서
**전월세 전환율**, **전세 환산 보증금**, **월세 환산액**, **법정 상한 초과 여부**, **어느 쪽이 더 유리한지**를 한눈에 보여주는 양방향 계산기.

---

## 3. 기획 의도

전월세 계약에서는 숫자가 단순해 보이지만, 실제로는 다음이 섞여 있어 사용자가 헷갈립니다.

* 전세 보증금 차이
* 월세 금액 차이
* 전환율이 적정한지
* 법정 상한을 넘는지
* 내 자금 상황에서 무엇이 더 유리한지
* 대출금리 / 예금금리 / 기회비용까지 고려하면 결과가 달라지는지

이 계산기는 단순 환산을 넘어서 다음 3가지를 동시에 해결해야 합니다.

1. **계약서 숫자를 같은 기준으로 비교**
2. **법정 상한 기준으로 협상 가능성 판단**
3. **내 상황 기준 유불리 판정**

## 8. 계산 로직

## 8-1. 전월세 전환율 계산 공식

### 공식 1. 실제 전환율

```text
전환율(연) = ((월세 × 12) / (전세보증금 - 월세보증금)) × 100
```

### 공식 2. 전세보증금을 월세로 환산

```text
월세 환산액 = (전환 대상 보증금 × 전환율) / 12
```

### 공식 3. 월세를 전세 환산 보증금으로 계산

```text
전세 환산 보증금 = 월세보증금 + (월세 × 12 / 전환율)
```

### 공식 4. 법정 상한 전환율

```text
법정 상한 = min(연 10%, 한국은행 기준금리 + 연 2%)
```

주택임대차보호법은 월차임 전환 시 두 비율 중 낮은 비율을 상한으로 두고 있고, 시행령은 그 값들을 각각 연 10%, 연 2%로 규정합니다. 2026년 4월 한국은행 기준금리 2.50% 기준이면 상한은 4.50%입니다. ([법제처][1])

---

## 8-2. 유불리 판정 로직

### 판정 기준 A. 법정 상한 기준

* 실제 전환율 > 법정 상한
  → **세입자에게 불리 가능성 높음**
* 실제 전환율 = 법정 상한 근처
  → **시장 평균/법정 기준 수준**
* 실제 전환율 < 법정 상한
  → **상대적으로 양호**

### 판정 기준 B. 실질 총비용 기준

비교기간 동안 아래를 합산

```text
전세 총비용 = 전세대출이자 + 보증금 기회비용 + 이사비용 + 중개수수료
월세 총비용 = 월세 합계 + 월세보증금 기회비용 + 이사비용 + 중개수수료
```

### 판정 기준 C. 현금흐름 기준

* 월 고정지출 낮추는 것이 중요하면 전세 선호
* 초기 목돈이 부족하면 월세 선호
* 대출금리가 높으면 전세의 장점 약화
* 예금금리/투자수익률이 높으면 보증금을 묶는 비용 증가

## 11. 예시 시나리오

## 시나리오 1. 전세 3억 vs 보증금 5천/월세 90만원

```text
전환 대상 보증금 = 3억 - 5천만 = 2억5천만
실제 전환율 = (90만 × 12) / 2억5천만 = 약 4.32%
```

→ 법정 상한 4.50% 이하라면 **형식상 상한 이내**
→ 단, 대출금리 4% 이상이면 전세 실익이 줄어듦

## 시나리오 2. 전세 4억 제안 대신 보증금 1억 / 월세 130만원

```text
전환 대상 보증금 = 3억
실제 전환율 = (130만 × 12) / 3억 = 약 5.20%
```

→ 법정 상한 4.50%보다 높으면 **세입자에게 불리 가능성 경고**

## 시나리오 3. 사회초년생

* 목돈 부족
* 월 현금흐름 중요
* 대출 한도 제한

→ 총비용은 조금 높아도 월세 선택 가능
→ 계산기에서는 “비용”과 “현금흐름”을 분리해서 보여줘야 함

법정 상한은 **연 10%와 기준금리+연 2% 중 낮은 값**입니다. 2026년 4월 기준금리 2.50% 기준으로는 **연 4.50%**입니다. ([법제처][1])

**Q2. 법정 상한보다 높으면 무조건 불법인가요?**
실무에서는 계약 유형과 상황에 따라 검토가 필요하므로, 계산 결과는 참고 지표로 안내하고 법률 자문 문구는 별도로 분리합니다. 관련 분쟁조정 참고 서비스로 한국부동산원 계산/상담 맥락을 연결할 수 있습니다. ([한국부동산원 · LH 임대차분쟁조정위원회][2])

**Q3. 대출금리를 같이 넣어야 하나요?**
네. 전세는 월세가 없더라도 자금조달 비용이 있으므로 현실 비교를 위해 매우 중요합니다.

**Q4. 보증금을 예금이나 투자에 넣을 수 있다면?**
그 경우 보증금의 기회비용이 생기므로 월세 쪽이 상대적으로 덜 불리해질 수 있습니다.

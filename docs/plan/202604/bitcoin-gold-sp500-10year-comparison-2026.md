## 2. 이 콘텐츠가 강한 이유

### 검색 의도

이 키워드는 단순 정보 탐색이 아니라 **투자 판단형 검색**입니다.

사용자가 궁금한 것은 대체로 아래 4가지입니다.

1. 10년 동안 가장 많이 오른 자산이 무엇인지
2. 많이 오른 대신 얼마나 무서운 하락을 겪었는지
3. 지금 들어가도 되는지
4. 한국 투자자는 실제로 어떻게 사는지

### 확산 포인트

이 콘텐츠는 커뮤니티, 블로그, 카페, X, 카카오톡 공유에 강합니다.

* “2016년에 100만원 넣었으면 지금 얼마?”
* “비트코인은 수익률 1등인데 MDD는?”
* “금은 재미없어 보이는데 방어력은?”
* “S&P500이 의외로 가장 무난했네”

즉, **숫자 비교 + 감정 자극 + 투자 교훈**이 동시에 됩니다.

## 7. 핵심 데이터 정의

이 리포트의 신뢰도는 **데이터 정의를 얼마나 명확히 쓰느냐**에서 갈립니다.

### 비교 자산

* 비트코인: BTC/USD 기준
* 금: 국제 금 가격(USD 기준)
* S&P500: 가능하면 Total Return 기준 우선
* 나스닥100: 가능하면 Total Return 기준 우선
* 코스피: 가격지수 기준 또는 배당포함 대체지표 여부 명시

### 시작점 / 종료점

* 시작: 2016-01-01 또는 2016년 첫 거래일
* 종료: 2026-04 기준 최신 데이터 스냅샷
* 페이지 상단에 “기준일” 노출 필수

### 투자 가정

* 각 자산에 동일 금액 100만원 일시 투자
* 원화 기준 / 달러 기준 둘 중 하나를 메인으로 정하고, 나머지는 보조설명
* 미국 자산은 환율 영향 포함 여부를 반드시 명시
* 추가 버전에서는 “환헤지 제외 / 포함” 비교 가능

### 실무 권장

이 페이지는 **메인 계산 기준을 원화 투자자 기준**으로 잡는 게 좋습니다.

즉:

* BTC: 원화 환산
* 금: 원화 환산
* S&P500/나스닥100: 달러 자산 → 원화 환산
* 코스피: 원화 그대로

이렇게 해야 한국 사용자가 체감합니다.

현재 시장 레벨을 기준으로 “지금 들어가도 되나?”를 다루는 섹션입니다.

### 참고

* S&P 500과 Nasdaq-100은 FRED/Nasdaq 기반 현재·과거 데이터 접근이 가능하고, 금은 World Gold Council과 LBMA가 기준축이 됩니다. CPI는 BLS가 기준입니다. ([FRED][4])500 / 나스닥100은 가능하면 Total Return 기준**

* BTC는 종가 기준

## 2. 기획 배경

육아휴직과 육아기 근로시간 단축은 사용 조건, 잔여기간 계산, 급여 계산 방식이 서로 달라 사용자가 직접 계산하기 어렵습니다. 특히 2025년 제도 개편으로 육아휴직 기간이 최대 1년 6개월로 확대되고, 육아기 근로시간 단축은 최대 3년까지 가능해졌으며, 2026년부터 단축급여 상한도 다시 올라 계산 복잡도가 더 커졌습니다.

실제 고용노동부는 육아기 근로시간 단축 사용기간 및 잔여기간 계산용 자료를 별도로 제공할 정도로 제도 계산이 복잡하다고 안내합니다.

즉, 이 페이지는 단순 급여계산기가 아니라 아래 3개를 동시에 해결해야 합니다.

1. **육아휴직 총 가능 기간**
2. **육아기 근로시간 단축 총 가능 기간**
3. **휴직/단축 사용 시 월별 급여**

* 또는 **한부모 / 중증 장애아동 부모**는 별도 연장 대상임

이 기준은 고용노동부 공식 개정 안내에 명시돼 있습니다.

## 8. 핵심 계산 로직

### 8-1. 육아휴직 총 가능 기간 판정

```text id="lg1"
if one_parent_household or parent_of_severely_disabled_child:
    parental_leave_max_months = 18
elif both_parents_each_used_at_least_3_months:
    parental_leave_max_months = 18
else:
    parental_leave_max_months = 12
```

이 연장 조건은 고용노동부 공식 개정 기준입니다.

---

### 8-2. 육아휴직 잔여기간

```text id="lg2"
used_parental_leave_days = sum(all_parental_leave_periods)
remaining_parental_leave_days = max_parental_leave_days - used_parental_leave_days
```

분할 사용 시 각 회차를 합산합니다. 육아휴직은 분할 사용이 가능하므로 실제 사용 이력 합산 로직이 필요합니다.

---

### 8-3. 육아기 근로시간 단축 총 가능 기간

```text id="lg3"
base_short_work_months = 12
unused_parental_leave_months = parental_leave_max_months - actual_used_parental_leave_months
converted_short_work_months = unused_parental_leave_months * 2
short_work_total_months = min(36, base_short_work_months + converted_short_work_months)
```

공식 기준은 “기본 1년 + 미사용 육아휴직 기간의 2배 가산, 최대 3년”입니다. 계산기에서는 반드시 **36개월 상한**을 둬야 합니다.

---

### 8-4. 단축근무 잔여기간

```text id="lg4"
used_short_work_days = sum(all_short_work_periods)
remaining_short_work_days = short_work_total_days - used_short_work_days
```

---

### 8-5. 단축근무 급여 계산 (2026 상한 반영)

```text id="lg5"
reduced_hours = original_weekly_hours - reduced_weekly_hours

first_10_hours = min(reduced_hours, 10)
extra_hours = max(reduced_hours - 10, 0)

support_first_10 = min(
    monthly_ordinary_wage * (first_10_hours / original_weekly_hours),
    2500000 * (first_10_hours / 10)
)

support_extra = min(
    monthly_ordinary_wage * 0.8 * (extra_hours / original_weekly_hours),
    1600000 * (extra_hours / max(extra_hours, 1))
)

company_pay = monthly_ordinary_wage * (reduced_weekly_hours / original_weekly_hours)
estimated_total = company_pay + support_first_10 + support_extra
```

실제 지급은 회사 지급액, 감액 규정, 통상임금 산정 방식에 따라 달라질 수 있으므로 결과는 **모의계산**으로 안내해야 합니다. 2026년 상한은 최초 10시간 250만원, 나머지 160만원입니다.

> 구현 팁:
> 수학식은 내부적으로 계산하되, 사용자에게는
> **회사 급여 + 정부 지원금 = 예상 월 수령액**
> 형태로 보여주는 것이 가장 이해가 쉽습니다.

### 10-7. FAQ 섹션

**Q. 배우자 한 명만 3개월 쓰면 18개월인가요?**
A. 아닙니다. 부모가 **모두 각각 3개월 이상** 사용해야 합니다. 또는 한부모, 중증 장애아동 부모는 별도 예외입니다.

**Q. 단축근무는 최대 몇 년까지 가능한가요?**
A. 기본 1년이고, 미사용 육아휴직 기간의 2배를 가산해 최대 **3년**까지 가능합니다.

**Q. 단축근무 급여 상한은 250만원인가요?**
A. 2026년 1월 1일부터 **최초 10시간 단축분 상한 250만원**, 나머지 단축분 상한 160만원이 적용됩니다.

**Q. 단축근무는 하루 1시간만 줄여도 되나요?**
A. 핵심은 주당 기준입니다. 단축 후 근로시간이 **주 15시간 이상 35시간 이하**여야 합니다.

**Q. 사용 가능 자녀 나이는 어디까지인가요?**
A. 육아휴직은 8세 이하 또는 초2 이하, 육아기 근로시간 단축은 12세 이하 또는 초6 이하 자녀 기준입니다.

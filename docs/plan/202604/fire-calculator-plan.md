## 6. 계산기 핵심 정의

### 파이어족 계산기란

현재 자산과 매달 투자 가능한 금액, 생활비, 기대수익률을 바탕으로

### 이 계산기가 해결하는 문제

* 막연한 은퇴 준비를 숫자로 바꿔줌
* 목표 자산이 얼마인지 즉시 계산 가능
* 내 저축/투자 속도가 충분한지 판단 가능
* 수익률 가정을 대표 자산 CAGR과 비교 가능
* 생활비 입력값이 막막한 사용자를 평균값/프리셋으로 도와줌

## 11. 평균 생활비 프리셋

### 섹션 목적

사용자가 월 생활비 입력에서 가장 많이 막히므로,
통계 기준값과 실전형 프리셋을 함께 제공해 빠르게 계산을 시작하게 한다.

### 섹션 제목

**평균 생활비로 빠르게 시작하기**

### 섹션 설명

통계 기준 평균 소비지출과 비교계산소 실전형 프리셋을 함께 제공하여
월 생활비를 손쉽게 입력할 수 있도록 한다.
평균값은 참고용이며, 실제 FIRE 계산은 본인의 은퇴 후 예상 소비 수준에 맞게 수정하도록 유도한다.

### 노출 방식

* 생활비 입력 필드 하단 빠른 선택 칩
* 또는 별도 보조 카드/툴팁
* 클릭 시 값 자동 입력

### 공식 평균값 기준

| 기준       | 월평균 소비지출 | 설명            |
| -------- | -------: | ------------- |
| 1인 가구 평균 |  168.9만원 | 1인 FIRE 기준 참고 |
| 전체 가구 평균 |  293.9만원 | 일반 가구 기준 참고   |

### 서비스용 실전 프리셋

| 프리셋       | 월 생활비 | 설명          |
| --------- | ----: | ----------- |
| Lean FIRE | 180만원 | 최소 지출형      |
| 1인 기본형    | 220만원 | 1인 직장인 기준   |
| 2인 기본형    | 300만원 | 부부 기준       |
| 3~4인 기본형  | 400만원 | 자녀 포함 가구 기준 |
| 여유형       | 500만원 | 소비 여유 반영형   |

### UI 예시

* `169만원`
* `220만원`
* `294만원`
* `300만원`
* `400만원`
* `직접 입력`

### 보조 문구

* 평균 생활비는 참고용입니다.
* 서울 거주, 자녀 유무, 주거 형태, 차량 보유 여부에 따라 실제 소비는 크게 달라질 수 있습니다.
* FIRE 계산은 현재 소비보다 **은퇴 후 예상 소비 구조** 기준으로 입력하는 것이 더 적절합니다.

### 연계 메시지

생활비 입력값은 FIRE 목표 자산을 직접 바꾸는 핵심 변수입니다.
예를 들어 4% 룰 기준에서는 월 생활비 300만원이면 목표 자산이 약 9억원이 됩니다.

## 12. 계산 로직 정의

### 12-1. 연간 생활비

`월 생활비 × 12`

### 12-2. FIRE 목표 자산

`연간 생활비 ÷ 안전인출률`

예시

* 월 생활비 300만원
* 연간 생활비 3,600만원
* 안전인출률 4%
* 목표 자산 = 9억원

### 12-3. 미래 자산 추정

* 현재 순자산 복리 성장
* 월 투자 가능 금액 적립식 반영
* 물가상승률 반영 시 목표 자산도 시간에 따라 증가

### 12-4. 달성 시점 계산

* 매월 또는 매년 자산 증가 시뮬레이션
* 미래 자산이 목표 자산 이상이 되는 첫 시점을 반환

### 12-5. 추가 필요 월 투자금 계산

* 목표 은퇴 나이 고정 시
* 현재 입력값으로 부족한 경우
* 부족분을 채우기 위한 월 투자금 역산

## 14. 대표 자산 CAGR 비교 섹션

### 섹션 제목

**대표 자산 CAGR로 기대수익률 감 잡기**

### 섹션 설명

S&P500, 나스닥100, 애플, 삼성전자 같은 대표 자산의 과거 연평균 성장률(CAGR)을 참고하면
현재 입력한 기대수익률이 보수적인지, 중립적인지, 공격적인지 쉽게 판단할 수 있습니다.

### 표시 자산

* S&P500
* NASDAQ100
* AAPL
* 삼성전자

### 표시 방식

* 3년 / 5년 / 10년 CAGR 토글
* 기본값은 10년
* 카드형 또는 바 차트형 표시
* 사용자가 입력한 기대수익률 선 함께 표시

### 예시 UI 테이블

| 자산        | 3년 CAGR | 5년 CAGR | 10년 CAGR | 성격           |
| --------- | ------: | ------: | -------: | ------------ |
| S&P500    |       - |       - |        - | 미국 대형주 분산 기준 |
| NASDAQ100 |       - |       - |        - | 기술주 중심 성장    |
| AAPL      |       - |       - |        - | 개별 고성장주      |
| 삼성전자      |       - |       - |        - | 국내 대표 대형주    |

### 주의 문구

* 과거 수익률은 미래 수익을 보장하지 않습니다.
* 개별 종목 CAGR은 변동성이 크므로 장기 기대수익률로 그대로 적용하기 어렵습니다.
* FIRE 계산에서는 분산 포트폴리오 기준의 보수적 수익률 가정을 권장합니다.

## 16. FIRE 유형 설명 섹션

### 목적

| 유형           | 설명                         |
| ------------ | -------------------------- |
| Lean FIRE    | 생활비를 낮게 유지하는 최소 지출형 FIRE   |
| Regular FIRE | 일반적인 생활 수준을 유지하는 표준형 FIRE  |
| Fat FIRE     | 은퇴 후 여유 소비까지 고려한 고자산형 FIRE |
| Coast FIRE   | 추가 적립 없이 기존 자산 복리로 목표 가능   |
| Barista FIRE | 일부 소득 활동을 병행하는 현실형 FIRE    |

### 결과 연결 문구 예시

* 현재 자산 규모와 투자기간을 보면 **Coast FIRE 가능성도 참고할 수 있습니다.**

## 17. 계산 기준 / 설명 콘텐츠

### 포함 내용

* FIRE 개념 설명
* 4% 룰 설명
* 안전인출률의 의미
* 물가상승률 반영 이유
* 기대수익률을 너무 높게 잡으면 안 되는 이유
* 개별주와 인덱스 CAGR 해석 차이
* 평균 생활비를 참고하되 그대로 쓰면 안 되는 이유

### 설명 예시

#### 4% 룰이란?

은퇴 후 매년 전체 자산의 4% 수준을 인출해 생활하는 기준입니다.
예를 들어 연 생활비가 3,600만원이면 약 9억원의 자산이 필요하다고 계산할 수 있습니다.

#### 왜 CAGR 비교가 필요한가?

사용자는 기대수익률을 입력할 때 막연히 8%, 10% 같은 숫자를 넣기 쉽습니다.
대표 자산의 과거 CAGR을 참고하면 수익률 가정이 얼마나 보수적/공격적인지 감을 잡을 수 있습니다.

#### 왜 평균 생활비 프리셋이 필요한가?

많은 사용자가 월 생활비 입력에서 막힙니다.
평균 생활비는 빠른 시작을 도와주지만, 실제 FIRE 계산은 주거비·자녀비용·차량·의료비·여가비를 포함한 본인 소비 구조로 조정해야 더 현실적입니다.

## 18. FAQ 초안

### FAQ 1

**파이어족은 자산이 얼마 있어야 가능한가요?**
생활비에 따라 다릅니다. 보통 연간 생활비를 안전인출률로 나눈 값이 목표 자산이 됩니다.

### FAQ 2

**월 생활비 300만원이면 FIRE 목표 자산은 얼마인가요?**
연간 생활비 3,600만원 기준, 4% 룰이면 약 9억원 수준입니다.

### FAQ 3

**생활비를 얼마로 넣어야 하나요?**
평균 생활비 프리셋으로 시작할 수 있지만, 실제 계산은 은퇴 후 내 소비 구조에 맞춰 조정하는 것이 좋습니다.

### FAQ 4

**기대수익률은 몇 %로 넣는 게 현실적인가요?**
분산투자 기준으로 보수적으로는 5~6%, 중립적으로는 6~8%, 공격적으로는 8% 이상을 참고할 수 있습니다.

### FAQ 5

**애플이나 나스닥 수익률로 계산해도 되나요?**
참고는 가능하지만 개별 종목이나 특정 섹터의 높은 수익률을 장기 기대수익률로 그대로 적용하는 것은 위험할 수 있습니다.

### FAQ 6

**부부 합산으로 계산해도 되나요?**
네. 가구 기준 월 생활비와 순자산, 월 투자 가능액을 합산해서 계산하면 됩니다.

### FAQ 7

**집이 있으면 FIRE 목표 자산이 줄어드나요?**
거주비 부담이 줄어들면 은퇴 후 생활비가 낮아질 수 있어 목표 자산이 줄어들 수 있습니다.

### FAQ 8

**국민연금이나 배당소득도 포함해야 하나요?**
은퇴 후 안정적으로 들어오는 현금흐름이 있다면 보수적으로 반영할 수 있습니다.

## 22. 핵심 계산 함수 예시

```ts
export function calculateAnnualExpense(monthlyExpense: number): number {

}
```

```ts
export function calculateTargetFireAsset(
  annualExpense: number,
  withdrawalRate: number

  return annualExpense / withdrawalRate;
}
```

```ts
export function calculateCagr(
  startValue: number,
  endValue: number,
  years: number
): number {
  if (startValue <= 0 || endValue <= 0 || years <= 0) return 0;
  return Math.pow(endValue / startValue, 1 / years) - 1;
}
```

```ts
export function projectAssetGrowth(
  currentNetWorth: number,
  monthlyInvestment: number,
  annualReturnRate: number,
  years: number
): number {
  const monthlyRate = annualReturnRate / 12;
  let asset = currentNetWorth;

  for (let i = 0; i < years * 12; i++) {
    asset = asset * (1 + monthlyRate) + monthlyInvestment;
  }

  return asset;
}
```

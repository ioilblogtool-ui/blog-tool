## 2. 기획 배경

소방관 콘텐츠는 비교계산소와 궁합이 좋습니다. 이유는 명확합니다.

* **계급 체계가 분명함**
* **호봉 데이터가 구조화 가능함**
* **기본급보다 수당 영향이 커서 “숫자로 비교” 포맷이 잘 먹힘**
* **경찰 리포트와 묶어서 시리즈화 가능함**

2026년에는 전체 공무원 보수가 **3.5% 인상**됐고, 7~9급 상당 저연차 초임은 **6.6% 인상**됐습니다. 또 현장 공무원 처우 개선으로 경찰·소방 **위험근무수당은 월 8만 원**, 소방의 **화재진화·구조·구급 출동가산금 1일 상한은 4만 원**, **비상근무수당은 1일 1만6천 원**, 월 상한은 **18만 원**으로 상향됐습니다. ([인사혁신처][1])

즉 이 콘텐츠는 단순 “소방사 초봉 소개”가 아니라,
**“계급 → 호봉 → 교대근무 → 수당 → 체감 실수령”** 구조로 설계해야 강합니다.

## 3. 업로드 HTML 진단 요약

업로드한 HTML은 디자인과 상호작용 측면에서 상당히 좋습니다. 

### 강점

| 항목     | 평가    | 메모                                                   |
| ------ | ----- | ---------------------------------------------------- |
| 비주얼 컨셉 | 매우 좋음 | 소방 리포트답게 불꽃/다크 톤이 명확함                                |
| 인터랙션   | 좋음    | 계급 카드 클릭 → 상세 패널 구조가 체류시간에 유리                        |
| 차별화    | 좋음    | 경찰 vs 소방 비교 배너, 교대근무 강조가 눈에 띔                        |
| 정보 구조  | 좋음    | Hero → 계급 → 연봉 차트 → 수당 → 승진 → 생애소득 → 입직 비교 흐름이 자연스러움 |

### 보정 필요

| 항목              | 조치                       |
| --------------- | ------------------------ |
| 일부 계급 1호봉 기본급   | 2026 공식 봉급표 기준으로 교체 필요   |
| 소방총감 기본급/호봉형 표현 | 별도 보수 체계 오해 없게 보수적 표기 필요 |
| 일부 수당 금액        | 공식 확인 가능한 항목 중심으로 정리 필요  |
| 승진 평균연수/생애소득    | “추정”, “예시”, “참고용” 배지 필요  |

업로드 HTML에서 메타 설명은 “소방사부터 소방총감까지 11계급 완전 정리”로 잘 잡혀 있고, 카드형 계급 UI와 교대근무 섹션은 그대로 가져가도 좋습니다.

## 4. 핵심 메시지

### 메인 메시지

**소방관 월급은 기본급보다 교대근무·현장수당 구조까지 봐야 정확하다.**

### 서브 메시지

* 소방은 총 11계급 구조다. ([대전광역시][2])
* 기본급은 공무원보수규정 별표 10 기준이다. ([법제처][3])
* 실수령은 야간, 시간외, 출동, 위험근무수당에 따라 달라진다. ([인사혁신처][1])
* 경찰과 기본 봉급표는 같지만, 체감 보수 구조는 다르다. ([법제처][3])

### 차트 유지

업로드 HTML의 바 차트 구조 유지 추천 

### 산정 기준 문구

> 기본급(대표 호봉) × 12 + 정액급식비 + 위험근무수당 + 정근수당 + 명절휴가비 + 직급보조비 + 현장 교대근무 평균 수당을 반영한 추정치입니다. 근무지, 교대형태, 출동 빈도, 가족수당에 따라 실제 연봉은 달라질 수 있습니다.

### 공식 근거

정액급식비는 공무원 보수 항목이고, 위험근무수당·출동가산금·비상근무수당은 2026년 현장 처우개선 항목입니다. ([인사혁신처][1])

### 주의

업로드 HTML의 평균 근속 값은 실무 감각상 자연스럽지만, 이번 검토에서는 공식 통계 확인을 하지 않았습니다. 따라서 최종 공개본에서는 다음처럼 쓰는 것이 좋습니다.

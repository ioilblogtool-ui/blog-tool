# 3. 기획 목적

## 3-1. 사용자 목적

* 아기 1년 육아비가 실제로 얼마나 드는지 감 잡기
* 어떤 항목에서 비용 차이가 크게 나는지 이해하기
* 자기 소비 성향이 가성비형인지 평균형인지 프리미엄형인지 파악하기
* 정부 지원금 반영 후 실부담 예산 가늠하기

## 3-2. 서비스 목적

* 육아/출산 카테고리의 검색 유입형 대표 리포트 확보
* 출산지원금, 부모급여, 육아휴직 계산기와 내부 링크 연결
* 향후 육아비 계산기 개발 전 사전 수요 검증
* 제휴/광고 친화형 정보 콘텐츠 기반 마련

## 11-3. 가성비 / 평균 / 프리미엄 기준 설명

### 섹션 제목

**우리 집은 어떤 타입에 가까울까?**

### 설명 표

| 구분   | 소비 특징                         | 추천 대상                  |
| ---- | ----------------------------- | ---------------------- |
| 가성비  | 필수 위주, 중고/선물 적극 활용, 실용성 우선    | 초반 지출을 줄이고 싶은 가정       |
| 평균   | 많이 쓰는 제품군 중심, 신품 위주, 가격·품질 균형 | 가장 일반적인 육아 준비          |
| 프리미엄 | 브랜드/편의성/고급 사양 중심, 신품 선호       | 예산 여유가 있고 편의성을 중시하는 가정 |

### 요약 문구

아기 1년 육아비는 집집마다 다르지만, 대부분은 이 3가지 기준 중 하나에 가깝게 움직입니다.
핵심은 “모든 항목을 비싸게 사느냐”가 아니라, **어떤 항목에서 예산 차이가 벌어지느냐**를 보는 것입니다.

## 11-4. 항목별 비용 비교

### 섹션 제목

**어디서 비용 차이가 크게 벌어질까**

### 표 예시

| 항목       | 가성비        | 평균          | 프리미엄          |
| -------- | ---------- | ----------- | ------------- |
| 유모차      | 중고/보급형     | 절충형 인기 모델   | 프리미엄 브랜드      |
| 카시트      | 기본형        | 안전성+편의성 균형형 | 회전형/고급형       |
| 아기띠      | 기본 기능형     | 착용감 좋은 대중형  | 프리미엄 소재/브랜드형  |
| 젖병/소독기   | 최소 구성      | 무난한 브랜드 세트  | 자동화/고급형       |
| 기저귀      | 행사/가성비 브랜드 | 대중 브랜드      | 프리미엄 라인       |
| 분유       | 표준 제품      | 인기 제품       | 프리미엄/특수 라인    |
| 의류       | 선물+기본 구매   | 계절별 적정 구매   | 브랜드/외출복 비중 높음 |
| 장난감/놀이매트 | 최소 구성      | 발달 단계별 구매   | 대형 매트/고급 교구   |

### 하단 설명

총비용 차이는 주로 **초기 대형 용품 + 반복 구매 소모품 + 후반부 이유식/놀이용품**에서 크게 벌어집니다.

## 11-5. 월령별 비용 흐름

### 섹션 제목

**언제 어떤 비용이 많이 드나**

### 표 예시

| ------- | ------------------ | ---------------- |

| 2~3개월   | 소모품, 예방접종, 의류 교체   | 월 고정비 패턴이 보이기 시작 |
| 4~6개월   | 장난감, 놀이매트, 외출용품    | 발달 관련 소비가 늘어남    |

| 10~12개월 | 활동용품, 신발, 추가 장난감   | 활동량 증가로 품목 다양화   |

### 요약 문구

초반에는 **준비비**, 중반에는 **소모품**, 후반에는 **이유식·놀이·활동용품** 비중이 커지는 흐름으로 보는 것이 이해하기 쉽습니다.

* 실제 적용 금액은 관련 계산기에서 별도 확인할 수 있도록 연결합니다.

### 섹션 제목

**육아비를 줄이는 현실적인 방법**

### 체크리스트

* [ ] 초반에 모든 용품을 한 번에 사지 않기
* [ ] 큰 용품은 사용 시점에 맞춰 구매하기
* [ ] 선물/중고/지인 물려받기 적극 활용하기
* [ ] 행사성 구매가 가능한 소모품은 월단위로 관리하기
* [ ] 의류는 계절과 성장속도를 고려해 최소 구매하기
* [ ] 브랜드보다 사용 빈도 높은 품목부터 예산 배분하기

### 한 줄 요약

가성비 육아의 핵심은 무조건 싼 제품이 아니라, **오래 쓰는 품목과 소모품을 구분해서 돈을 쓰는 것**입니다.

# 19. 에디터 최종 작성용 본문 오프닝 초안

신생아부터 돌까지의 육아 비용은 집마다 크게 다릅니다.
하지만 막연하게 “돈이 많이 든다”는 수준으로만 보면 실제 준비에 도움이 되지 않습니다.

중요한 것은 **어떤 방식으로 준비하느냐**입니다.
필수 위주로 실용적으로 준비하는지, 평균적인 신품 기준으로 준비하는지, 아니면 편의성과 브랜드를 고려한 프리미엄 기준으로 준비하는지에 따라 총비용은 꽤 달라집니다.

이 글에서는 아기 1년 육아비를 **가성비 / 평균 / 프리미엄** 3가지 기준으로 나눠 비교하고, 지원금 반영 후 실제 체감 부담까지 함께 정리해봅니다.

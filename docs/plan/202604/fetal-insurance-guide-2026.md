## 8. 섹션별 상세 설계

### 섹션 1. 태아보험이란 무엇인가

**핵심 메시지**

* 태아보험은 보통 어린이보험에 **태아 시기 관련 특약**을 더한 구조로 이해시키는 편이 실무적
* 출생 전에는 가입 절차와 특약 선택이 중요하고, 출생 후에는 어린이 보장 구조로 이어지는 흐름을 설명

**문구 예시**

* 태아보험은 별도 독립 상품이라기보다, 어린이보험에 태아 관련 특약을 더해 임신 중 가입하는 형태로 이해하면 쉽습니다.
* 핵심은 “몇 주에 가입하느냐” 하나보다, **그 시점에 어떤 특약을 넣을 수 있고 어떤 심사 변수가 생기는지**를 아는 것입니다.

---

### 섹션 2. 태아보험, 언제부터 알아보면 좋을까

삼성화재는 공개 FAQ에서 **태아보험 가입 가능 시기를 임신 22주까지**로 안내하고, **가능하면 기형아 검사 전에, 통상 12주 전후 이전 가입 검토**를 권합니다. DB손해보험은 2024년 보도자료에서 **대다수 인터넷 태아보험은 22주 이내에만 태아보장 특약 가입이 가능하지만 자사 일부 상품은 22주 이후에도 가능**하다고 설명했습니다. ([한화손해보험][4])

**실무 해석**

* “무조건 12주”가 정답은 아님
* 다만 **검사 결과, 고지사항, 특약 제한 가능성**을 생각하면 너무 늦게 보는 것은 불리할 수 있음
* 따라서 이 리포트는 “최저가 시기”보다 “확인 시기”를 알려주는 구조가 맞음

---

### 섹션 3. 임신 주수별 체크포인트 비교표

보험료 그래프 대신 아래 표가 더 현실적입니다.

| 구간           | 핵심 포인트         | 추천 메시지                   |
| ------------ | -------------- | ------------------------ |
| 임신 확인~12주 전후 | 비교 시작 적기       | 보장 구조·특약 범위 확인 시작        |
| 13~16주       | 본격 비교 구간       | 태아특약/고지사항 체크             |
| 17~20주       | 늦지 않았지만 서둘러야 함 | 필요 특약 누락 없는지 확인          |
| 21~22주 전후    | 중요 분기점         | 일부 상품·특약 가능 여부 우선 확인     |
| 22주 이후       | 선택 폭 달라질 수 있음  | 가능한 상품 구조와 출생 후 보장 중심 검토 |

이 표는 공식 보험사 안내와 맞닿아 있으면서도, 과장 없이 정보 전달에 집중할 수 있습니다. ([한화손해보험][4])

### 섹션 5. 실손 연동과 중복 보상 주의

손해보험협회 소비자포털은 **의료실비는 실손보장의 원칙에 따라 비례보상**, 반면 **암 진단비 같은 정액형 담보는 중복보상 가능**하다고 설명합니다. 또 설명서 표준안에서도 실손담보는 다수보험일 때 **실제 손해를 초과해 지급되지 않고 비례보상**된다고 안내합니다. ([소비자포털][3])

**이 리포트에서 강조할 메시지**

* “실손은 여러 개 있어도 실제 쓴 돈 이상으로 받는 구조가 아님”
* “진단비/정액보장과 실손은 성격이 다름”
* “이미 가입한 보험이 있다면 중복 여부를 먼저 확인”

### 인트로

특히 12주 전후 검사 시점, 22주 전후 태아특약 가능 여부, 실손 중복보상 여부는 상담 전에 미리 알고 가는 편이 훨씬 유리합니다. 일부 보험사는 임신 22주까지 태아보험 가입 또는 태아 관련 특약 가입을 안내하고 있으며, 검사 결과나 고지사항에 따라 조건이 달라질 수 있습니다. ([한화손해보험][4])

### 요약 박스

* 태아보험은 “무조건 빨리”보다 **주수별 확인 포인트를 놓치지 않는 것**이 중요합니다. ([한화손해보험][4])
* 실손은 여러 개 가입해도 실제 손해 범위 안에서 비례보상됩니다. ([소비자포털][3])
* 정액형 진단비 특약은 실손과 달리 중복보상이 가능할 수 있습니다. ([소비자포털][3])

## 2. 기획 배경

사교육비는 부모 입장에서 매우 자주 반복 확인하는 생활형 지출입니다.
하지만 실제로는 아래 문제가 있습니다.

* 과목별로 흩어져 있어 총액이 한눈에 안 보임
* 아이 1명 기준은 계산해도 형제자매 합산은 어렵고 누적 부담은 체감이 안 됨
* “많이 쓰는 편인지 적당한지” 비교 기준이 없음
* 교육비를 줄였을 때 장기적으로 얼마나 차이 나는지 감이 안 옴

이 계산기는 단순 합산기가 아니라 다음 3가지를 동시에 해결합니다.

1. **현재 비용 파악**
2. **또래 평균 대비 비교**
3. **장기 누적 및 기회비용 체감**

## 3. 한 줄 서비스 정의

**우리 집 사교육비를 계산하고, 또래 평균 대비 수준과 장기 부담까지 함께 보여주는 교육비 계산기**

## 4. 핵심 사용자 문제

### 사용자 질문

* 우리 아이 학원비는 많은 편일까?
* 월 사교육비가 실제로 얼마인지 정확히 알고 싶다
* 초등~고등까지 이어지면 총 얼마가 들까?
* 둘째까지 합치면 교육비 부담이 얼마나 커질까?
* 이 돈을 투자했다면 얼마나 차이가 났을까?

### 해결 방식

* 자녀별 입력
* 과목별 합산
* 학교급/지역 기준 평균 비교
* 연간/누적 금액 표시
* 기회비용 시뮬레이션 연결

## 6. 주요 타겟 페르소나

### 페르소나 A. 초등 저학년 부모

* 영어, 수학, 예체능을 병행 중
* 한 달 총 교육비가 감으로만 있음
* 또래 대비 많은지 궁금함

### 페르소나 B. 둘 이상 자녀 부모

* 첫째/둘째 교육비 합산이 필요
* 장기 가계 부담을 보고 싶음
* 연간 총액과 절감 포인트가 중요함

### 페르소나 C. 예비 학부모

* 아직 본격 사교육 전
* 향후 지출 규모를 미리 보고 싶음
* 학교급 전환에 따른 예상 비용이 궁금함

## 9-3. 결과 대시보드 섹션

### 목적

입력한 정보가 어떤 규모의 지출인지 바로 체감하게 한다.

### 결과 카드

| 카드명      | 설명                   |
| -------- | -------------------- |
| 월 사교육비   | 현재 월 총 교육비           |
| 연간 사교육비  | 월 비용 × 12            |
| 누적 예상 비용 | 현재 구조 유지 시 학교급 기간 누적 |

### 강조 UI

* 숫자 크게
* 월 / 연 / 누적 단위 분리
* 자녀별 소계 + 전체 합계 함께 노출

### 예시 문구

* 이번 달 예상 사교육비는 **78만 원**
* 1년이면 약 **936만 원**
* 현재 구조가 유지되면 초등 6년 누적 약 **5,616만 원**

## 9-4. 또래 비교 섹션

### 목적

“우리 집이 어느 수준인가”를 쉽게 판단할 수 있게 한다.

### 비교 방식

정밀 백분위보다 **평균 대비 티어형 비교**로 제공

### 계산식

```text id="i7bv5h"
비교지수 = 우리 집 월 사교육비 ÷ 동일 학교급·지역 평균 사교육비
```

### 티어 정의

| 비교지수      | 티어     | 배지 문구      |
| --------- | ------ | ---------- |
| 0.7 미만    | 절약형    | 평균보다 낮은 수준 |
| 0.7 ~ 1.1 | 평균권    | 또래 평균 수준   |
| 1.1 ~ 1.5 | 적극 투자형 | 평균보다 높은 수준 |
| 1.5 이상    | 고지출형   | 높은 지출 구간   |

### 표시 요소

* 가로 막대 그래프
* 평균 위치 마커
* 현재 위치 뱃지
* “평균보다 월 12만 원 높음” 같은 차액 문구

### 카피 원칙

* 비교는 참고용으로 표현
* 경쟁 심리를 자극하는 문구는 지양
* “상위 몇 %”보다는 “평균 대비 어느 수준” 중심

---

## 9-5. 누적 비용 시뮬레이션 섹션

### 목적

교육비의 장기 부담을 체감하게 한다.

### 제공 결과

* 초등 6년 누적
* 중등 3년 누적
* 고등 3년 누적
* 초등~고등 전체 누적
* 자녀 2명/3명 합산 누적

### 시나리오 예시

* 현재 초등 2학년, 월 70만 원이면 남은 초등 기간 누적
* 중등 진학 시 비용 증가 가정 옵션
* 고등 구간 진입 시 상향 시뮬레이션 옵션

### 추천 옵션

* 현재 수준 유지
* 학교급 상승 시 10% 증가
* 학교급 상승 시 20% 증가

## 9-6. 기회비용 섹션

### 목적

교육비를 소비 지출이 아닌 장기 자금 관점에서도 생각해보게 한다.

### 핵심 질문

**이 돈을 매달 투자했다면 얼마나 되었을까?**

### 계산 로직

```text id="m3l4t3"
미래가치 = 월 사교육비를 일정 수익률로 매월 적립 투자한다고 가정한 금액
```

### 입력 옵션

* 예상 기간: 5년 / 10년 / 15년
* 기대수익률: 3% / 5% / 7%

### 결과 예시

* 월 60만 원을 연 5%로 10년 적립 시 약 OOO만 원
* 형제 합산 월 130만 원이면 10년 후 약 OOO억 원 가능

### 연결 콘텐츠

* 적립식 투자 계산기
* 복리 계산기
* 노후자금 계산기

## 9-7. 절감 팁 가이드 섹션

### 목적

계산 결과 이후 실질 행동으로 이어지게 한다.

### 추천 구성

1. 과목 중복 여부 점검
2. 오프라인/온라인 혼합 학습 비교
3. 학년 전환 시 필수 과목 중심 재편
4. 형제자매 예산 상한선 설정
5. 월 고정지출 기준으로 교육비 비율 관리

### 문구 예시

* 과목 수보다 실제 학습 효과를 먼저 점검해보세요
* 월 교육비 총액보다 연간 누적 부담을 같이 보세요
* 형제자매가 있다면 자녀별이 아닌 가계 전체 기준으로 관리해보세요

## 9-8. 데이터 출처 및 유의사항

### 목적

신뢰성 확보와 과도한 해석 방지

### 표기 원칙

* 평균값은 공식 조사 기반 참고치
* 실제 비용은 지역, 브랜드, 수업 횟수, 레벨에 따라 차이 발생
* 비교 결과는 참고용 추정치

### 추천 문구

본 계산기의 평균 비교 결과는 공개 통계 및 내부 기준값을 바탕으로 제공하는 참고용 정보입니다. 실제 사교육비는 지역, 과목, 기관 유형, 수강 횟수, 개인 상황에 따라 달라질 수 있습니다.

## 10. 계산 로직 상세

## 10-1. 기본 계산

```text id="o9k1d5"
월 총 사교육비 = 자녀별(과목 월 비용 합계 + 교재비 + 기타비용)의 총합
연간 사교육비 = 월 총 사교육비 × 12
누적 사교육비 = 연간 사교육비 × 남은 적용 연수
```

## 10-2. 자녀별 합산 예시

```text id="we0glr"
첫째 월 비용 + 둘째 월 비용 + 셋째 월 비용 = 전체 월 사교육비
```

## 10-3. 비교 차액 계산

```text id="32l0p4"
차액 = 우리 집 월 사교육비 - 평균 사교육비
```

## 10-4. 기회비용 계산

```text id="m4acgk"
매월 동일 금액을 적립 투자한다고 가정한 미래가치 공식 사용
```

## 20. 최종 정리

### 이 계산기가 좋은 이유

* 실생활에서 바로 써먹는 문제를 다룸
* 부모의 실제 고민인 “합계 + 비교 + 장기 부담”을 한 번에 해결
* 육아 카테고리 안에서 강한 검색 의도를 가짐
* 투자/가계관리 콘텐츠와 연결이 매우 좋음
* 형제자매 기능으로 반복 사용 가능성이 높음

### 운영 시 주의점

* 비교 결과를 지나치게 경쟁형으로 보이게 하지 말 것
* 공식 통계가 갱신되면 기준값도 업데이트할 것
* 사용자 입력과 평균값 괴리를 고려해 면책 문구를 분명히 둘 것

### 최종 결론

**비교계산소에 매우 잘 맞는 생활밀착형 계산기 주제이며, 육아 카테고리 내 핵심 유입 툴이 될 가능성이 높다.**
특히 **평균 비교 + 누적 비용 + 기회비용** 조합이 강점이다.

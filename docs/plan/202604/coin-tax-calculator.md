## 7. 출력값 설계

| 출력 항목          | 계산 방식                        |
| -------------- | ---------------------------- |
| 총 매수금액         | 평균 매수단가 × 수량                 |
| 총 매도금액         | 매도가 × 수량                     |
| 총 매수 수수료       | 총 매수금액 × 매수 수수료율             |
| 총 매도 수수료       | 총 매도금액 × 매도 수수료율             |
| 양도차익           | 총 매도금액 - 총 매수금액 - 수수료 - 기타비용 |
| 기본공제 적용 후 과세표준 | max(0, 양도차익 - 2,500,000)     |
| 예상 세금          | 과세표준 × 22%                   |
| 세후 순이익         | 양도차익 - 예상 세금                 |
| 총 실수령 금액       | 총 매도금액 - 총 매도 수수료 - 예상 세금    |

## 8. 계산 로직

### 단일 코인 기준

```text
총 매수금액 = 평균 매수단가 × 수량
총 매도금액 = 매도가 × 수량

총 수수료 = (총 매수금액 × 매수수수료율) + (총 매도금액 × 매도수수료율)

양도차익 = 총 매도금액 - 총 매수금액 - 총 수수료 - 기타 필요경비

과세표준 = max(0, 양도차익 - 2,500,000)

예상 세금 = 과세표준 × 0.22

세후 순이익 = 양도차익 - 예상 세금
```

### 복수 코인 확장 시

```text
연간 총 손익 = 각 코인별 양도차익 합계
과세표준 = max(0, 연간 총 손익 - 2,500,000)
예상 세금 = 과세표준 × 0.22
```

## 2. 페이지 목표

### 사용자 목표

- 연금저축과 IRP가 어떻게 다른지 쉽게 이해한다.
- 세액공제 관점에서 어떤 조합이 유리한지 판단한다.
- 운용 자유도, 중도 인출, 수령 방식까지 포함해 내 상황에 맞는 선택을 찾는다.

### 서비스 목표

- 은퇴/연금 카테고리의 대표 비교 리포트로 검색 유입을 확보한다.

## 3. 핵심 메시지

- 대부분의 직장인에게는 `연금저축 우선 + IRP 보완` 구조가 가장 설명하기 쉽다.
- IRP는 세액공제와 퇴직금 관리에 강하지만, 운용 및 인출 제약도 함께 본다.
- 연금보험은 세액공제 상품이 아니라 `비과세형 장기 보험` 관점에서 비교해야 한다.

## 7. 본문 설계

### 섹션 1. 30초 요약

목적: 검색 유입 사용자가 스크롤 초반에 핵심을 즉시 파악

포맷:

- 요약 카드 3개
- “누구에게 어떤 상품이 맞는지” 한 줄 정리

예시 메시지:

- 세액공제 중심이면 연금저축 + IRP 조합이 핵심
- 퇴직금 관리까지 보면 IRP 비중이 커진다
- 비과세와 보장 중심이면 연금보험은 다른 축에서 봐야 한다

### 섹션 2. 핵심 비교표

| 항목 | 연금저축 | IRP | 연금보험 |
| --- | --- | --- | --- |
| 가입 대상 | 개인 | 개인, 퇴직금 수령자 | 개인 |
| 세액공제 | 가능 | 가능 | 일반적으로 없음 |
| 퇴직금 이전 | 불가 | 가능 | 불가 |
| 운용 자유도 | 높음 | 중간 | 낮음 |
| 중도 인출 | 상대적으로 유연 | 제한적 | 상품별 상이 |
| 수령 구조 | 연금/해지 | 연금/일시금 | 연금 중심 |
| 추천 성향 | 투자형 | 절세+퇴직금 관리형 | 안정/보장형 |

설계 포인트:

- 표는 모바일에서 카드형으로 전환
- “무조건 우위”가 아니라 “강점/약점” 방식으로 표현

### 섹션 3. 세액공제 구조 설명

목적: 가장 강한 검색 의도 해결

포함 내용:

- 연금계좌 세액공제 기본 구조
- 연금저축 단독 활용 시
- 연금저축 + IRP 조합 시
- 소득 구간별 체감 차이 예시

주의:

- 세법 세부 수치는 “기준/예시”로 표현
- 페이지 내 숫자는 설명 목적의 범용 예시로 제시

### 섹션 4. 운용 자유도·중도 인출 비교

목적: 세액공제만 보고 가입했다가 후회하는 사용자를 위한 보완

포함 내용:

- ETF/펀드/예금 선택 폭
- 중도 인출 제약
- 계좌 관리 난이도

핵심 메시지:

- 연금저축은 상대적으로 운용 유연성 설명에 유리
- IRP는 세제와 퇴직금 관리 강점 대신 제약 설명 필요

## 15. 최종 판단

이 리포트는 검색 의도 폭이 넓고 계산기 연결 효율이 높다. `연금저축 vs IRP`는 단일 키워드 경쟁이 강하지만, 사용자가 실제로 원하는 것은 “차이 요약 + 내 상황에 맞는 선택 + 계산기로 이어지는 다음 행동”이다. 따라서 이 문서는 단순 설명형 글이 아니라 `비교 리포트 + 선택 가이드 + 계산기 허브` 구조로 설계하는 것이 맞다.

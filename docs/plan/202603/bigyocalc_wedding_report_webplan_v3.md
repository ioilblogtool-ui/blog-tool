## 2. 페이지 한 줄 정의

**“2016년과 2026년 결혼비용을 비교해, 지금 결혼이 왜 더 부담스러워졌는지 수치와 체감으로 보여주는 리포트 페이지”**

---

## 3. 페이지 목표

### 3.1 사용자 관점 목표
- 결혼비용이 10년 사이 얼마나 상승했는지 빠르게 이해
- 웨딩홀, 식대, 스드메, 신혼여행 등 항목별 부담 증가 확인
- 내 연봉 기준으로 현재 결혼비용이 얼마나 무거운지 체감
- 실제 예산 계산기로 이어져 직접 시뮬레이션

### 3.2 비즈니스 관점 목표
- 검색 유입 확보: `결혼비용 평균`, `결혼 준비 비용`, `스드메 비용`, `웨딩홀 식대`
- 리포트 → 계산기 → 제휴 링크 → 연관 리포트 흐름 설계
- “10년 비교 시리즈”의 대표 템플릿으로 재사용
- 광고/제휴 배치가 과하지 않으면서도 클릭 지점 확보

## 4. 핵심 메시지

### 메인 메시지
> 10년 전에는 연봉의 일부로 감당되던 결혼비용이, 이제는 연봉 1년치를 넘길 수 있는 수준까지 올라왔다.

### 서브 메시지
- 문제는 단순 물가 상승이 아니라, 웨딩홀/식대/스드메 같은 핵심 항목이 더 빠르게 올랐다는 점
- 결혼 건수는 줄었지만, 1건당 비용 부담은 더 커졌다
- 현재는 “평균 비용”보다 “내 상황에 맞는 예산 설계”가 더 중요하다

**보조 카피**

### 7.6 체감 계산 섹션

#### 목적
- 정적인 리포트를 인터랙티브하게 전환
- 계산기 클릭 전 예열 구간

#### 입력값
| 입력 항목 | 타입 | 기본값 |
|---|---|---|
| 연봉(만원) | 숫자 입력 | 4000 |

#### 출력값
| 출력 항목 | 설명 |
|---|---|
| 2016 결혼비용 부담률 | 총비용 / 연봉 |
| 2026 결혼비용 부담률 | 총비용 / 연봉 |
| 10년간 추가 부담액 | 2026 - 2016 |
| 절약 목표 추가 기간 | 월 50만원 저축 가정 |

#### 안내 문구
- `내 연봉 기준으로 보면 결혼비용 부담이 어느 정도인지 바로 확인할 수 있습니다.`
- `실제 계약 금액은 다를 수 있지만, 현재 평균 수준이 내 소득과 비교해 얼마나 무거운지 체감하는 데 유용합니다.`

#### 결과 문구 예시
- `연봉 4,000만원 기준 2026년 평균 결혼비용은 연봉의 110.3% 수준입니다.`
- `월 50만원씩 저축한다면 10년 전보다 약 39개월을 더 모아야 하는 셈입니다.`

### 7.8 예산 절약 가이드 섹션 (v3 신규 강화)

#### 카드/리스트 구성
1. 평일·오전 예식 할인 활용
2. 스드메 패키지 분리 견적
3. 모바일 청첩장 적극 활용
4. 답례품 단가 기준 먼저 설정
5. 신혼여행은 얼리버드/비성수기 활용

#### 문구 예시

- `결혼비용 줄이는 현실적인 방법`

**리드 문구**

**아이템 예시**
- `식대는 하객 수와 단가가 동시에 반영되므로 총비용에 가장 큰 영향을 줍니다. 초대 기준을 먼저 정리하는 것이 중요합니다.`
- `스드메는 패키지 전체가 저렴해 보여도 옵션이 붙으면 실제 계약가가 올라갈 수 있어, 기본가와 추가금 구조를 분리해서 봐야 합니다.`

## 16. 최종 요약

이 페이지는 단순히 “결혼비용이 올랐다”를 말하는 페이지가 아니라, 아래 흐름으로 설계해야 한다.

1. **숫자로 충격을 주고**
2. **항목별로 이유를 보여주고**
3. **내 연봉 기준으로 체감하게 만든 뒤**
4. **실제 계산기와 관련 서비스로 자연스럽게 연결한다**

즉,  
**리포트형 콘텐츠 + 인터랙션 + 전환 구조**를 동시에 가져가는 비교계산소 대표형 페이지로 설계한다.

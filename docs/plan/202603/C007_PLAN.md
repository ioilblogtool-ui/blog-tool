| 구분 | 일반 기사형 콘텐츠 | 비교계산소 기획 방향 |

3. “누가 많다”보다 **어떻게 보유했는지**를 보여주는 구조  
4. 대통령실 / 총리실 / 내각으로 필터링 가능한 데이터형 화면

### 비교 대상
| 직위군 | 연봉 |
|---|---:|
| 대통령 | 27,177.2만원 |
| 국무총리 | 21,069만원 |

### 활용 방식
- 공직 연봉 자체 비교
- 총재산 ÷ 공직 연봉
- 순자산 ÷ 공직 연봉
- 금융자산 비중 비교

> 위 연봉은 2026년 공무원보수규정 별표 기준입니다.

- “장관급은 재산 총액보다 자산 편중 구조 차이를 함께 보는 것이 유용합니다.”

## S6. 인물 비교 도구

### 목적
- 비교계산소의 핵심 UX 요소
- 2명 선택만으로 체류시간과 탐색성을 높임

### 기능명
**공직자 자산 구조 비교기**

### 입력값
| 입력 항목 | 타입 | 설명 |
|---|---|---|
| 인물 A 선택 | select | 비교 대상 1 |
| 인물 B 선택 | select | 비교 대상 2 |
| 비교 기준 | select | 총재산 / 순자산 / 부동산 / 예금 / 증권 / 연봉 대비 배수 |
| 보기 방식 | select | 금액 / 비율 / 배수 |

### 출력값
| 출력 항목 | 설명 |
|---|---|
| 총액 차이 | 누가 얼마나 더 많은지 |
| 자산군 차이 | 어떤 자산이 큰지 |
| 자산 편중 | 부동산 중심인지 금융자산 중심인지 |
| 순자산 차이 | 채무 제외 기준 비교 |
| 코멘트 | 한 줄 요약 |

### 계산식
```text
순자산 = 총재산 - 채무
연봉 대비 자산배수 = 총재산 / 공직 연봉
금융자산비중 = (예금 + 증권) / 총재산
부동산비중 = (건물 + 토지 + 전세권) / 총재산
````

### 결과 메시지 예시

* “이재명 대통령은 김민석 총리보다 총재산 규모가 크게 높고, 금융자산 비중도 높은 편입니다.”
* “두 인물 모두 총재산보다 자산 구조 차이를 보는 것이 더 의미 있습니다.”

1. 이재명 대통령 재산은 얼마인가요?
2. 국무총리와 대통령비서실장 재산은 얼마인가요?

4. 재산공개 금액은 시세인가요?

6. 장관 연봉은 얼마인가요?

### FAQ 하단 설명 문구 예시

* 재산공개 금액은 시세와 다를 수 있음
* 공시가격, 취득가, 신고 기준 차이 존재
* 전세권·채무 포함 여부를 함께 봐야 해석이 가능함

> 공개 재산은 시장가가 아니라 신고 기준 금액일 수 있어 실제 체감 자산과 차이가 날 수 있습니다.

## S9. 안내/면책 문구

### 목적

* 수치 해석 기준 명확화
* 정치 편향 및 해석 오류 리스크 완화

### 문구 예시

* 본 페이지의 재산 정보는 공직자 정기 재산변동 공개 자료와 보도자료를 기반으로 정리한 비교용 참고 데이터입니다.
* 재산공개 금액은 시세가 아니라 공시가격, 취득가, 신고 기준 금액일 수 있습니다.
* 공직 보수는 법령 기준이며, 실제 수령액과는 차이가 있을 수 있습니다.
* 자산 규모에 대한 평가는 배제하고, 구조 비교 중심으로 해석하는 것을 권장합니다.
* 일부 직위는 장관급/차관급 등 보수 기준군으로 환산해 표기할 수 있습니다.

## 17. 최종 요약

### 이 페이지의 핵심

* 정치 기사형 소비가 아니라 **공식 재산공개 데이터를 비교하는 데이터형 페이지**로 만든다.
* 사용자는 **총재산, 자산 구성, 공직 보수**를 함께 이해할 수 있어야 한다.
* 비교계산소 성격에 맞게 **데이터 + 비교 + 해설** 구조로 설계한다.

### v1 구현 기준

* 핵심 공직자 10명 내외
* 총재산 / 부동산 / 예금 / 증권 / 채무 / 연봉
* 비교표 + 차트 + 2인 비교 + FAQ

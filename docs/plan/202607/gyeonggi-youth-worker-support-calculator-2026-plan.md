## 1. 배경 및 목적

경기도는 도내 중소기업 등에 재직 중인 청년을 위해 두 가지 지원사업을 운영한다.

1. **청년 복지포인트** — 연 최대 120만 원(반기 60만 원), 경기도 거주 만 19~39세, 도내 중소·중견기업·소상공인·비영리법인(공공기관 제외) 주 36시간 이상 재직자, 월급여 385만 원(1인 중위소득 150%) 이하. 경기청년몰 포인트로 지급.
2. **중소기업 청년노동자 지원사업** — 2년간 최대 480만 원(반기 120만 원), 경기도 거주 만 19~39세, 도내 중소기업 주 36시간 이상 재직자.

두 사업은 **나이·거주·재직 형태·근무시간 조건이 거의 동일**하고 금액·지급주기만 다르다. 재직 청년 입장에서는 "나는 어떤 사업에 해당되고 합쳐서 얼마를 받을 수 있는지"가 핵심 질문인데, 경기도청 공식 페이지는 사업별로 안내가 분리돼 있어 한 번에 비교하기 어렵다.

**목표**
- 나이·재직 기업 형태·근무시간·월급여를 입력하면 → 두 사업 각각의 대상 여부와 예상 지원금을 **동시에** 계산
- 두 사업의 병행(중복) 가능 여부를 안내해 "합산하면 얼마"까지 확인
- 기존 청년 정책 계산기(청년월세지원, 청년 적금, 복지급여 수급자격)와 내부 링크로 연결해 청년 복지 콘텐츠 클러스터를 강화

### 데이터 신뢰도 메모
사용자가 제공한 원본 텍스트가 "경기도청 +6" 표시와 함께 잘려 있어 **중소기업 청년노동자 지원사업의 소득 기준·최신 공고 여부가 확인되지 않은 상태**다. 구현 단계에서 경기도일자리재단(잡아바) 공식 공고를 반드시 재확인하고, 확인 전까지는 해당 항목에 `확인 필요` 배지를 명시한다. (이 사업은 과거 예산 상황에 따라 모집이 중단·재개된 이력이 있어 특히 주의)

### 1인 중위소득 150% 산출 근거
- `WBE_2026_THRESHOLDS`(생계급여 계산기 기존 데이터)의 1인 가구 기준 중위소득 2,564,238원을 재사용
- 150% = 3,846,357원 ≈ 사용자 제공 "월 385만 원"과 일치 → 기존 데이터 파일을 import해 중복 정의 방지

```ts
import { WBE_2026_THRESHOLDS } from "./welfareBenefitEligibility";
export const GYWS_INCOME_LIMIT = Math.round(
  WBE_2026_THRESHOLDS.find(t => t.householdSize === 1)!.medianIncome * 1.5
); // 3,846,357원 ≈ 385만원
```

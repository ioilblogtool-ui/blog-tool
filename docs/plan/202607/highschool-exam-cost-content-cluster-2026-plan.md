## 1. 배경 및 목적

"재수 비용", "재수학원 비용", "수시 원서비", "고3 사교육비"는 실검색 볼륨이 크고 학부모·수험생 양쪽이 동시에 검색하는 키워드다. 특히 재수 비용은 "등록금보다 비싼 재수학원비"라는 화제성 있는 프레임이 언론에도 자주 등장해 콘텐츠 소구력이 높다.

핵심 컨셉은 대학생 클러스터와 동일하게 **"내가 실제로 얼마를 쓰게 되는가"**를 계산해주는 것이다. 재수 비용·원서비는 정보 나열보다 계산기로 만들어야 체류시간과 공유가 나온다.

### 목표
- "재수 비용 계산기", "수시 원서비 계산기" 검색 유입 확보
- 대학생 클러스터와 양방향 연결(재수 비용 vs 대학 등록금 비교 등)로 두 클러스터가 서로 트래픽을 주고받게 함
- 시즌 대응: 정시 원서접수는 매년 12월 말, 수시는 9월 초 — 재수학원 등록은 1~2월(신학기) 성수기. **9월 이전 배포를 목표로 한다** (수시 시즌 직전 노출 확보)

## 4. 페이지 1 — `retake-exam-cost-calculator-2026` (1차 최우선)

### 4-1. URL·메타

```
슬러그: /tools/retake-exam-cost-calculator-2026/
타이틀: 재수 비용 계산기 2026 | 학원비·생활비 1년 총액 바로 계산
디스크립션: 재수 유형(통학 종합반/기숙학원)과 기간을 입력하면 학원비·특강비·생활비를 합산한 1년 총비용을 바로 계산합니다. 대학 등록금과 비교도 가능합니다.
```

### 4-2. 입력값

| 입력 | 설명 |
|---|---|
| 재수 유형 | 통학 종합반 / 기숙학원 (mode-chip 2종, 항목 구성이 달라짐) |
| 월 수강료 | 통학형 기본값 200만 원 / 기숙형 기본값 350만 원(전국 평균) |
| 재수 기간 | 개월 수(기본 10개월, 2~11월 프리셋) |
| 특강·교재비 | 자유 입력(통학형 기본값 반영) |
| 기숙사·생활비(기숙형만) | 기숙형 선택 시에만 노출 |
| 급식비(기숙형만) | 기숙형 선택 시에만 노출 |

### 4-3. 계산 로직

```
1년 총비용 = 월 수강료 × 개월 수 + 특강·교재비 + (기숙형인 경우: 생활비·급식비 등 추가 항목)
```

두 유형은 별도 항목 구성이므로 계산 함수도 유형별로 분기한다(4-1 데이터의 2-2/2-1 근거값을 각각 기본값으로 사용).

### 4-4. 결과 및 연결
- KPI: 학원비 총액 / 특강·부대비용 총액 / (기숙형만) 생활비 총액 / 1년 총비용
- CTA: "이 비용, 대학 등록금이랑 비교하면?" → `retake-vs-college-tuition-2026`(2차) 및 대학생 클러스터 `university-cost-calculator-2026` 연결

## 5. 페이지 2 — `college-application-fee-calculator-2026` (1차)

### 5-1. URL·메타

```
슬러그: /tools/college-application-fee-calculator-2026/
타이틀: 대입 원서비 계산기 2026 | 수시·정시 전형료 총액 바로 계산
디스크립션: 수시·정시 지원 개수와 대학별 전형료 구간을 입력하면 원서비 총액을 바로 계산합니다. 최대 수시 6개·정시 3개 기준.
```

### 5-2. 입력값

| 입력 | 설명 |
|---|---|
| 수시 지원 개수 | 0~6개 (슬라이더) |
| 수시 평균 전형료 구간 | 저가(3만 원대)/평균(5만 원대)/고가(10만 원대) 프리셋 + 자유 입력 |
| 정시 지원 개수 | 0~3개 |
| 정시 평균 전형료 구간 | 수시와 동일 프리셋 구조 |

**착수 전 확인 필요**: 원서 1건당 평균값(4만 7,806원)이 2020년 데이터라, 착수 직전 최신 연도(2025~2026학년도) 평균 전형료를 웹검색으로 재확인해 프리셋 금액을 업데이트한다.

### 5-3. 계산 로직

```
원서비 총액 = (수시 지원 개수 × 수시 평균 전형료) + (정시 지원 개수 × 정시 평균 전형료)
```

### 5-4. FAQ 초안 (5개)
1. 수시는 몇 개까지 지원할 수 있나요? (→ 최대 6개)
2. 정시는 몇 개까지 지원할 수 있나요? (→ 가/나/다군 최대 3개)
3. 전형료가 가장 비싼 대학은 얼마나 하나요?
4. 원서비도 카드 할부가 되나요? (→ 대학별 상이, 원서접수 사이트에서 확인 안내)
5. 수시 6개 다 최상위권 대학에 넣으면 얼마나 드나요?

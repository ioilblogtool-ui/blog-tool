## 2. 기획 배경

비교계산소에는 이미 육아 비용·분유·출산지원금 관련 콘텐츠가 있고, 네이버 서치어드바이저에서도 `분유 가격 비교표`가 반복적으로 TOP30에 들어오고 있다. 육아 클러스터에 비용형 콘텐츠만 쌓여 있는 상태에서, 부모의 긴급 검색 의도인 **아기 열, 해열제, 상비약, 병원 기준**을 흡수할 도구형 페이지를 추가하면 카테고리 확장성이 좋다.

다만 의료 콘텐츠는 YMYL 성격이 강하다. 따라서 이 페이지는 **"몇 ml 먹이면 된다"**를 계산하는 도구가 아니라, **준비물 체크리스트·복약 시간 기록·진료 신호 확인**에 집중한다. 핵심은 사용자가 급한 상황에서 정보를 정리하도록 돕되, 최종 판단은 의사·약사·응급의료 상담으로 연결하는 것이다.

**목표**
- 집에 준비한 아기 상비약·상비용품을 체크하면 준비율과 부족 항목을 보여준다.
- 해열제·감기약·설사·구토·피부·상처 항목을 안전 주의사항과 함께 정리한다.
- 복용 시간·성분·체온을 기록해 같은 성분 중복 복용을 피하도록 돕는다.
- 응급 신호 체크 시 "즉시 진료/상담 권고" 문구를 강하게 노출한다.

## 5. 핵심 기능

### 5-1. 준비율 체크리스트

입력 방식: 보호자가 보유한 항목을 체크하면 준비 점수와 부족 항목을 보여준다.

| 카테고리 | 항목 | 점수 | 주의 문구 |
|---|---|---:|---|
| 체온 확인 | 전자 체온계 | 12 | 같은 방식으로 반복 측정해 추이를 본다 |
| 해열·통증 | 아세트아미노펜 계열 해열제 | 12 | 제품 설명서·성분 중복 확인 필수 |
| 해열·통증 | 이부프로펜 또는 덱시부프로펜 계열 | 8 | 탈수·구토·신장질환 의심 시 임의 사용 주의 |
| 코막힘 | 생리식염수 스프레이/앰플 | 8 | 약 성분 없는 제품 중심 |
| 설사·구토 | 경구수분보충액(ORS) | 10 | 소변 감소·처짐 동반 시 진료 |
| 피부 | 보습제·기저귀 발진 보호크림 | 8 | 스테로이드 연고는 상담 후 사용 |
| 상처 | 멸균 거즈·밴드·소독용품 | 8 | 깊은 상처·출혈 지속 시 진료 |
| 투약 도구 | 약 먹이는 주사기·투약 스푼 | 10 | 밥숟가락 사용 금지 안내 |
| 기록 | 복약 기록표 | 12 | 시간·성분·용량·체온 기록 |
| 응급 정보 | 달빛어린이병원·119·주말진료 병원 메모 | 12 | 야간·휴일 대응 |

### 결과 출력

| 점수 | 등급 | 출력 문구 |
|---:|---|---|
| 80~100 | 준비 양호 | "기본 상비약·기록 도구가 대부분 준비되어 있습니다." |
| 50~79 | 보완 필요 | "응급 연락처와 복약 기록표, ORS 등 부족 항목을 보완하세요." |
| 0~49 | 우선 준비 | "체온계·해열제 성분 확인·투약 도구부터 먼저 준비하세요." |

결과 카드:
- 우리 집 준비율: `72점`
- 부족한 항목: `ORS`, `복약 기록표`, `달빛어린이병원 정보`
- 바로 할 일: `유효기간 확인`, `제품 성분명 메모`, `약사에게 사용 가능 연령 확인`

### 5-2. 복약 기록표

약 용량을 자동 산출하지 않고, 보호자가 입력한 복약 내역을 정리한다.

| 입력값 | 유형 | 예시 |
|---|---|---|
| 아이 이름 | text | 지우 |
| 현재 체중 | number | 9kg |
| 체온 | number | 38.5도 |
| 복용 성분 | select | 아세트아미노펜 / 이부프로펜 / 덱시부프로펜 / 기타 |
| 제품명 | text | 보호자 입력 |
| 복용 시간 | time | 21:30 |
| 복용량 | text | 3mL |
| 증상 메모 | textarea | 콧물, 기침, 처짐 없음 |

출력:
- 마지막 복용 기록
- 같은 성분 중복 주의 문구
- "다음 복용 가능 시간은 제품 설명서·약사 안내 기준으로 확인" 고지
- 기록 복사 버튼 또는 프린트용 표

### 5-3. 병원 상담 신호 체크

결과를 진단하지 않고 위험 신호가 있으면 강하게 진료 권고한다.

## 12. 수익화·광고 관점

| 항목 | 판단 |
|---|---|
| 검색 수요 | 높음. 부모 긴급 검색 의도 |
| CTR 가능성 | 높음. `아기`, `상비약`, `해열제`, `병원 기준` 조합은 제목 적합도가 중요 |
| 체류시간 | 높음. 체크리스트·기록표·FAQ로 스크롤 유도 가능 |
| 광고 단가 | 중간. 육아·건강·보험 광고와 매칭 가능 |
| 리스크 | 중상. 의료 조언 오인 방지 필수 |

광고 승인 전후 모두 과한 광고 배치는 피한다. 의료 긴급 상황 페이지처럼 보일 수 있으므로, 본문 상단보다 체크리스트 결과 이후·FAQ 위 정도가 적절하다.

## 13. 품질/주의사항

- 용량 계산, 복용 가능 여부, 진단, 응급실 필요 여부를 확정하지 않는다.
- "교차복용 방법" 콘텐츠처럼 보이지 않게, "기록·중복 주의" 중심으로 작성한다.
- 상비약 브랜드명은 가급적 피하고 성분명 중심으로 쓴다.
- 아기 상태가 나쁘면 사이트 결과보다 의사·약사·119·응급의료 안내가 우선이라는 문구를 반복한다.
- 공식 출처는 식약처·의약품안전나라·응급의료포털 중심으로 둔다.
- 아스피린, 2세 미만 감기약, 항생제 재사용 금지, 유효기간 지난 약 폐기 문구는 반드시 포함한다.

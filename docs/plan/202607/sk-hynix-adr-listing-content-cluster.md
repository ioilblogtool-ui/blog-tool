# SK하이닉스 미국 ADR 상장 콘텐츠 클러스터 기획서

## 0. 사실관계 확정 (2026-07-10 기준, 확인 완료)

SK하이닉스 미국 ADR 상장은 **검토설이 아니라 완료된 사실**이다. 아래 조건이 확정 팩트이며, 이후 모든 문서·페이지는 이 기준으로 확정형 표현을 사용한다.

| 항목 | 확정 내용 |
|---|---|
| 상장일 | 2026년 7월 10일 나스닥 거래 개시 |
| 거래소 | Nasdaq Global Select Market |
| 티커 | SKHY |
| 공모가 | ADS 1주당 149달러 |
| ADR 비율 | ADS 10주 = 한국 보통주 1주 (즉 1 ADS = 보통주 0.1주) |
| 발행 구조 | 한국 보통주 약 1,779만 주 규모 신주 발행 기반 ADR 조성 |
| 조달 규모 | 약 265억 달러(원화 약 36~40조 원, 로이터 등 외신은 약 280억 달러로도 보도) |
| 주요 목적 | HBM·첨단 메모리 생산능력 확대, 국내 신규 팹 건설·장비 투자 재원 확보 |
| 시장 반응 | 기관 주문 수요 공모 물량 대비 약 7배 이상 초과 청약 |
| 출처 | Reuters, "South Korea's SK Hynix launches $28 billion US listing" (2026-07-06) 등 |

**표현 원칙:**
- 상장 자체, 나스닥/SKHY, 공모가, ADR 비율, 조달 규모, 발행 목적 → **확정형**으로 서술 ("~했다", "~상장했다")
- 향후 효과(코리아 디스카운트 완화, 마이크론 대비 밸류에이션 격차 축소, 글로벌 AI·HBM 대표주 재평가) → **전망형** 유지 ("~기대된다", "~가능성이 있다", "미국 시장 유동성·HBM 실적·주주환원에 따라 결정될 전망")
- 신주 발행에 따른 기존 주주 지분 희석, 단기 수급 부담 등 리스크 요인도 균형 있게 명시
- 핵심 프레임: "ADR 상장으로 국장 디스카운트를 줄일 통로가 생겼다. 다만 실제 재평가는 미국 기관 수급과 HBM 실적이 결정한다."

## 2. 기획 배경

성과급·부동산 축 콘텐츠는 이미 다수 보유. ADR 상장은 "주가·투자" 축의 신규 트리거 이벤트로, 국내 개인 투자자 입장에서 실질적 액션(환전, 세금, 매매 전략)이 걸린 주제라 체류시간·전환 모두 기대할 수 있다. 계산기를 트래픽 유입구로, 리포트 2종을 배경 설명·내부링크 허브로 쓰는 구조는 `sk-hynix-bonus` 계산기 + `sk-hynix-bonus-2027` 리포트 조합에서 이미 검증된 패턴이다.

## 5. 표기 원칙 (CONTENT_GUIDE.md / CLAUDE.md 준수)

- 상장 사실·거래소·티커·공모가·ADR 비율·조달 규모·발행 목적은 **확정형**으로 표기 (§0 표 참고)

## 2. 기획 의도

`청년수당`은 검색 수요가 꾸준하지만, 지역마다 실제 제도 구조가 다르다. 서울시의 청년수당은 대체로 **미취업 청년의 진로탐색·구직활동을 지원하는 현금성 활동지원금**으로 검색되고, 경기도는 `청년수당`이라는 표현보다 **청년기본소득**이 대표 키워드로 잡힌다. 사용자는 둘을 같은 선상에서 비교하려고 하지만, 실제로는 연령, 거주 요건, 소득 요건, 사용처, 지급 방식이 크게 다르다.

비교계산소에는 청년월세지원·청년미래적금·정부 복지 리포트가 이미 있으므로, 이 리포트는 `지역별 청년지원 비교 허브` 역할을 할 수 있다. 단순히 “어디가 더 많이 주나”가 아니라 **내 상황이면 어느 제도를 먼저 확인해야 하는지**를 알려주는 것이 핵심이다.

특히 서울과 경기 사이에서 이사·전입을 고민하는 청년은 `주민등록 기준`, `거주기간`, `신청일 기준`, `사용처 제한`을 놓치기 쉽다. 이 페이지는 금액 비교표보다 **조건 체크와 신청 타이밍 비교**를 우선한다.

## 8. UX·인터랙션 설계

### MVP

정적 비교 리포트로 시작한다.

- 상단 2열 비교 카드
- 메인 비교표
- 상황별 추천 카드
- 지역 이동 시나리오
- 공식 확인 체크리스트
- FAQ

### 2차 확장

간단한 선택형 인터랙션을 추가할 수 있다.

입력:
- 현재 거주지: 서울 / 경기 / 기타
- 나이: 만 나이
- 취업 상태: 미취업 / 단기근로 / 취업
- 경기도 거주기간: 3년 이상 계속 / 10년 이상 합산 / 미충족 / 모름
- 소득기준: 서울 기준 충족 가능 / 모름 / 어려움

출력:
- 서울 청년수당 먼저 확인
- 경기 청년기본소득 먼저 확인
- 둘 다 공식 공고 확인 필요
- 현재 조건으로는 가능성 낮음

단, 정확한 판정 계산기는 공식 공고 기준이 매년 바뀌므로 MVP에서는 `자가점검` 수준으로 제한한다.

## 13. 리스크 및 주의사항

- `경기 청년수당`이라는 검색어는 실제 공식 제도명과 다를 수 있다. 페이지에서는 `경기도 청년기본소득`을 공식명으로 쓰고, `경기 청년수당`은 검색어 맥락으로만 다룬다.
- 2026년 서울 청년수당 모집 규모, 신청기간, 소득 기준은 공고마다 바뀔 수 있다.
- 경기도 청년기본소득은 시군별 지급 여부나 예산 상황에 따라 예외가 생길 수 있다.
- 금액만 비교하면 서울이 유리해 보이지만, 대상 조건이 다르므로 “서울이 더 좋다” 식 단정 금지.
- 지역화폐 사용처 제한, 활동지원금 사용 제한을 반드시 분리해서 설명한다.
- 중복수급은 제도별로 달라 “가능/불가능”을 단정하지 않고 공식 공고 확인을 안내한다.

## 14. 우선순위 판단

| 평가 항목 | 점수 | 근거 |
|---|---|---|
| 검색 수요 | 상 | 서울 청년수당, 경기 청년기본소득 모두 반복 검색 |
| 시즌성 | 중상 | 서울 모집 공고, 경기 분기 신청 때 피크 발생 |
| 구현 난이도 | 중 | 계산 로직은 가볍지만 공식 기준 최신화가 중요 |
| 내부링크 효과 | 상 | 청년월세지원·청년적금·정부복지 리포트로 연결 가능 |
| 데이터 리스크 | 중상 | 연도별 공고와 시군별 차이 확인 필요 |

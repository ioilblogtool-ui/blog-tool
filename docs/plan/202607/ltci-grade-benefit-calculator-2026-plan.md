## 1. 배경 및 목적

65세 이상 인구가 2026년 기준 전체의 20%를 넘어섰다. 부모님 요양을 처음 준비하는 40~50대 자녀들이 "장기요양 몇 등급이면 얼마 나오나"를 검색하지만, 건강보험공단 공식 페이지는 복잡하고 시뮬레이션이 없다.

**목표**
- 등급(1~5등급 + 인지지원등급)과 소득 수준을 입력하면 → 월 급여 한도·본인부담금·실수령 혜택을 즉시 계산
- 요양원(시설급여) vs 재가요양(재가급여) 비용을 나란히 비교해 의사결정 지원
- `/reports/` 요양원 vs 재가요양 비교 리포트(별도 기획)로의 내부 링크 허브

## 2. SEO 타겟

### 메인 키워드

| 키워드 | 검색 의도 |
|---|---|
| 장기요양등급 혜택 | 등급별 받을 수 있는 것 확인 |

| 노인요양 비용 계산기 | 월 비용 시뮬레이션 |

### 타겟 독자
- 부모님 장기요양 판정을 앞두거나 막 받은 40~60대 자녀
- 본인 또는 배우자 노후 준비를 시작한 50~60대
- 사회복지사·요양보호사 실무 참고용

## 3. 핵심 입력값 / 출력값

### 입력
| 항목 | 유형 | 비고 |
|---|---|---|
| 장기요양등급 | 셀렉트 (1~5등급, 인지지원등급) | |
| 급여 유형 | 토글 (재가 / 시설 / 비교) | |
| 건강보험료 납부 유형 | 셀렉트 (일반·감경 40%·감경 60%·산정특례) | 본인부담 비율 결정 |
| 월 이용 일수 (재가) | 슬라이더 15~30일 | 시설은 30일 고정 |

### 출력 (결과 카드 3개)
| 카드 | 내용 |
|---|---|
| 월 급여 한도 | 등급별 공단 인정 한도액 |
| 본인부담금 | 일반 15% / 감경 7.5%·6% / 기초 면제 |
| 실제 혜택 금액 | 공단 부담분 (한도 × (1-본인부담률)) |

비교 모드: 재가 vs 시설 월비용 나란히 표시

## 1. 배경 및 목적

"국가장학금", "대학 등록금", "기숙사 자취" 등은 대학생·학부모 양쪽에서 검색하는 실검색 볼륨이 큰 롱테일 키워드 묶음이다. 비교계산소 타깃(2030 직장인)보다 어린 세그먼트지만, 곧 "취업 후 학자금대출 상환", "첫 연봉 실수령" 검색으로 자연 유입되는 퍼널 관계라 사이트 포지션과 궁합이 좋다.

핵심 컨셉은 단일 정보 나열이 아니라 **"내가 실제로 얼마 부담하고, 어떤 지원을 받는가"**를 계산해주는 것 — 등록금만 보여주는 페이지가 아니라 등록금 + 주거·생활비 - 장학금 구조를 계산기로 제공한다.

### 목표
- "국가장학금 계산기", "대학 등록금 계산기" 검색 유입 확보 (구글 색인 + 네이버 스마트블록/파워링크 동시 대응)
- 계산기끼리 자연스럽게 연결되는 클러스터 구조 (총비용 계산기 ↔ 장학금 계산기 ↔ 비교 리포트)
- 시즌성 대응: 국가장학금은 매 학기 신청 시즌(통상 학기 시작 1~2개월 전) 직전에 검색이 급증하는 패턴이 반복되는 제도이므로, 신청 시즌 전 게시·리프레시를 우선한다 (정확한 신청 일정은 페이지 게시 시점에 한국장학재단 공지로 재확인)

### 페이지 수 결정

원안(10개 후보) 중 세 건은 별도 페이지로 만들지 않는다.

- **"부모가 준비해야 할 대학 4년 비용" 페이지는 별도로 만들지 않는다.** `university-cost-calculator-2026`(등록금·생활비 총비용 계산기)에 "학생 관점 / 부모 관점" 결과 토글만 추가하면 동일 계산 로직으로 커버된다. 별도 페이지를 만들면 계산 엔진이 100% 중복된다.
- **"대학생 월 생활비 평균 계산기"도 별도 페이지로 만들지 않는다.** `university-cost-calculator-2026`의 입력값(월 생활비)과 `dorm-vs-commute-cost-comparison-2026`의 거주형태별 생활비 비교가 이미 이 수요를 커버한다.
- **"대학생 알바 월급 계산기"는 만들지 않기로 결정** (2026-07-15, 승스 판단). 기존 `bonus-after-tax-calculator`·`overtime-pay-calculator` 등과 겹치는 시급×시간 계산이라 이 클러스터에서는 우선순위가 낮다고 판단. 대신 `work-study-vs-part-time-comparison-2026`(2차, 국가근로 vs 알바 비교 리포트)에서 알바 조건을 참고치로만 다룬다.

이렇게 10개 원안을 **7개**로 정리한다.

### 2-2. 구글·네이버 노출 대응 원칙

- **구글**: `SeoContent`의 FAQ가 `BaseLayout`에서 FAQ 스키마로 자동 변환되므로, FAQ 질문 문구를 실제 검색 롱테일 쿼리 형태("~인가요?", "~얼마나 되나요?")로 작성해 스니펫 노출을 노린다.
- **네이버**: 네이버는 스마트블록·인플루언서 콘텐츠 비중이 높아 표·숫자 기반 요약이 상단 노출에 유리하다. 각 페이지 상단(KPI 요약 직후)에 "핵심 숫자 3~4개 표"를 배치해 네이버가 긁어가기 쉬운 구조로 만든다 (기존 `docs/NAVER_SEARCH_STATS.md` 트래킹 결과, 표·숫자형 URL이 CTR이 높았던 패턴과 일치).
- 클러스터 내부링크: 허브 페이지(`university-cost-calculator-2026`)에서 나머지 6개 페이지로, 나머지 페이지에서 허브로 양방향 링크 — 체류시간·크롤 예산 분산 방지.
- `public/sitemap.xml`에 7개 URL 모두 트레일링 슬래시 포함으로 등록 (`docs/GOOGLE_SEO_RULES.md` 3장 기준), `changefreq`는 `monthly` 사용 (등록금·장학금 데이터는 학기·연도 단위로만 갱신).

3. **해석 가이드**: 구간이 낮을수록(1~3구간) 지원금이 크다는 점이 핵심입니다. 1~3구간은 연 최대 600만 원, 4~6구간은 440만 원, 7~8구간은 360만 원, 9구간은 100만 원이 지원되며 10구간은 지원 대상에서 제외됩니다. 다자녀 가구는 셋째 이상 자녀부터 8구간까지 등록금 전액이 지원되므로, 형제자매 수가 많다면 예상보다 훨씬 많은 지원을 받을 수 있습니다.
4. **한계**: 이 계산기는 월 소득만으로 구간을 추정하는 단순화된 계산이며, 실제 학자금 지원구간은 금융재산·부채·자동차 가액 등을 종합한 소득인정액으로 한국장학재단이 별도 산정합니다. 정확한 구간과 지원금은 한국장학재단 홈페이지에서 학자금 지원구간 산정 신청 후 확인해야 하며, 이 결과는 신청 전 대략적인 예상치로만 활용하세요.

### 6-5. 섹션 구성
1. Hero + InfoNotice(3-8의 재검증 필요 경고 반영)
2. 입력 폼
3. 결과: 예상 지원구간 / 예상 지원금 / 등록금 대비 지원 비율
4. 소득구간 경곗값 표(3-7 데이터 그대로 노출 — 사용자가 직접 확인 가능하게, 네이버 스마트블록 대응 표)
5. 관련 CTA → 대학 등록금 계산기, 학자금대출 계산기(3차 완성 후)
6. SeoContent (FAQ 8개)

### 6-6. FAQ (8개)

1. **국가장학금 소득분위는 어떻게 계산되나요?** — 부모(가구)의 소득과 재산을 합산한 '소득인정액'을 기준중위소득과 비교해 1~10구간으로 나눕니다. 2026년 기준 1구간은 월 194만 8,421원 이하, 5구간은 649만 4,738원 이하입니다.
2. **신입생과 재학생 신청 방법이 다른가요?** — 신청 시기와 절차는 유사하지만 신입생은 입학 전 소득 자료가 확정되지 않아 가결정으로 우선 지원받고, 이후 재확정 절차를 거치는 경우가 많습니다. 정확한 절차는 한국장학재단 공지를 확인하세요.
3. **다자녀 가구는 얼마나 더 받나요?** — 2026년부터 다자녀 가구는 첫째·둘째 1~3구간 610만 원, 4~8구간 505만 원까지 지원되며 셋째 이상 자녀는 8구간까지 등록금 전액이 지원됩니다.
4. **국가장학금 I유형과 II유형은 뭐가 다른가요?** — I유형은 소득구간에 따라 학생 개인에게 정액 지급되고, II유형은 대학이 등록금 인하·장학금 확충 노력과 연계해 추가로 지원하는 제도로 대학별 지급액이 다릅니다.
5. **9구간, 10구간도 받을 수 있는 지원이 있나요?** — 9구간은 연 100만 원(학기당 50만 원)이 지원되며, 다자녀 가구는 8구간까지 확대 지원됩니다. 10구간은 국가장학금 I유형 지원 대상에서 제외됩니다.
6. **2026년 국가장학금 규모는 얼마나 되나요?** — 2026년 전체 예산은 역대 최대인 약 5조 1,161억 원, 수혜 인원은 약 150만 명 규모로 확대됐습니다. 1학기에만 108만 명이 2조 2,267억 원을 지원받았습니다.
7. **소득인정액에 재산도 포함되나요?** — 네, 부동산·금융재산·자동차 등을 소득으로 환산한 금액이 소득인정액에 포함됩니다. 이 계산기는 월 소득 위주의 단순 추정이라 실제 구간과 차이가 날 수 있습니다.
8. **계산된 예상 지원금으로 등록금 실부담금도 알 수 있나요?** — 네, 이 페이지의 예상 지원금을 대학 등록금 계산기의 '연간 장학금' 입력값에 넣으면 4년 실부담금을 함께 확인할 수 있습니다.

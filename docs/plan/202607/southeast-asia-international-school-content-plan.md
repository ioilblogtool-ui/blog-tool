## 1. 배경 및 목적

국내 국제학교 클러스터(2026-07-08 배포)가 "학비 부담" 각도로 학부모층을 흡수했다면, 이번 확장은 **"한국 국제학교는 너무 비싸서 동남아로 눈 돌리는 부모들"** 흐름을 잡는다. 사용자가 가져온 초안(ChatGPT 브레인스토밍, 60개 이상 제목 후보)을 이 사이트 컨벤션(`CLAUDE.md` 타이틀 공식, `CONTENT_GUIDE.md` 수익화·SEO 기준)에 맞춰 정리하고, 겹치는 페이지를 통합해 실속 있는 개수로 재구성한다.

### 목표
- "말레이시아 국제학교", "태국 국제학교 학비", "동남아 조기유학 비용" 등 핵심 검색 유입 확보
- **"한국 vs 동남아 총비용 비교"**를 메인 후킹으로 삼아 기존 국내 클러스터와 상호 링크
- 계산기 1개 + 국가별 리포트로 구성, 국내 클러스터의 학교 마스터 데이터 구조를 재사용

### 페이지 수 결정 — 원안 60개 이상 → 5개로 압축 (국내 클러스터보다 더 보수적으로)

원안은 국가 6개(말레이시아·태국·베트남·필리핀·싱가포르·인도네시아) × 도시별·주제별 제목 8개씩을 제안했지만, 국내 클러스터에서 확인한 원칙("입력값과 로직이 같은 계산기를 여러 페이지로 쪼개지 않는다")에 더해 이번 주제는 **국가마다 통화·학비 체계·생활비 데이터를 전부 새로 확보해야 하는 리서치 비용**이 있다. 따라서:

- 계산기는 국가·도시를 입력 옵션으로 흡수해 **1개로 통합**
- 국가별 리포트는 **검색 의도가 뚜렷하고 "한국보다 싸다" 내러티브가 성립하는 3개국(말레이시아·태국·베트남)만 1차 launch**
- 필리핀(영어캠프 성격이 강해 국제학교 검색 의도와 다름)·싱가포르(비싸서 "가성비" 내러티브가 안 맞음)·인도네시아/발리(니치)는 확장 후보로만 남긴다

## 2. ⚠️ 구현 전 필수 팩트체크 원칙 (국내 클러스터보다 리스크 높음)

국내 클러스터는 학교 공식 홈페이지 하나만 확인하면 됐지만, 이번 확장은 **① 학교 학비(현지 통화) ② 생활비(주거·식비·교통) ③ 환율** 세 가지 변수가 겹쳐 오차가 누적된다. 이 문서에 실린 5번 표(국가별 대략 비용 감각)는 사용자가 가져온 초안의 예시 구간이며, **설계 문서 작성 전 반드시 재조사**해 대체한다.

| 확인 필요 항목 | 확인 방법 |
|---|---|
| 국가별 대표 국제학교(쿠알라룸푸르·조호바루 / 방콕·치앙마이 / 호치민·하노이) 연간 학비 | 각 학교 공식 홈페이지 Tuition/Fees 페이지, 통화 단위 명시 |
| 도시별 생활비(주거·식비·교통·의료보험) | Numbeo, 각국 이주·주재원 커뮤니티 공식 통계, 현지 부동산 플랫폼 — 복수 출처 교차 확인 |
| 환율 | 계산 시점의 실시간 환율 API 또는 최근 고시환율, "OO년 O월 기준" 명시 |
| 학교 등급(저가/중간/프리미엄) 분류 기준 | 학교별 커리큘럼(IB/A-Level 등)·연간 학비 구간으로 객관적 분류, 주관적 "네임밸류" 배제 |
| 국가별 장기체류·교육 비자 요건 (MM2H 등) | 각국 이민청 공식 자료 — **이 클러스터는 비자/이민 자격을 단정하지 않고 "확인 필요" 안내만 제공** (법적 자격 판단은 스코프 밖) |

**생활비·환산 비용은 전부 `추정` 라벨을 명시**한다(`CLAUDE.md` "Estimate/scenario values must be labeled 추정, 참고" 원칙). 특히 "부모 동반 거주비"는 가족 구성·라이프스타일에 따라 편차가 가장 큰 항목이므로, 계산기 결과 화면에 "실제 이주 전 반드시 현지 답사·최신 학비 재확인" 문구를 InfoNotice로 고정 노출한다.

| 자녀 수 | 1명 / 2명 |
| 학교 등급 | 저가형 / 중간형 / 프리미엄 (2번 팩트체크 기준으로 분류) |

## 5. 페이지 2 — `korea-vs-southeast-asia-international-school-2026` (메인 후킹 허브)

```
슬러그: /reports/korea-vs-southeast-asia-international-school-2026/
타이틀: 한국 vs 동남아 국제학교 2026 | 1년 총비용 한눈에 비교
디스크립션: 한국과 말레이시아·태국·베트남 국제학교의 학비, 생활비, 5년 누적 비용을 비교합니다. 계산기로 우리 가족 비용도 바로 확인하세요.
```

전체 클러스터의 메인 후킹 — "학비만 보면 동남아가 싸 보이지만, 부모 동반 거주비까지 넣으면 달라진다"는 핵심 포인트를 국가별 비교 카드로 제시. 계산기·국가별 리포트·기존 국내 클러스터(허브 리포트 포함)로 내부링크를 뿌리는 역할.

**섹션 구성**
1. 한국 국제학교 평균 비용 요약 (기존 `international-school-overview-2026` 데이터 재사용)
2. 국가별(말레이시아·태국·베트남) 비용 비교 카드
3. "학비만 비교하면 안 되는 이유" — 생활비 포함 총비용 강조
4. 자녀 1명/2명 비용 비교
5. 5년 누적 비용 비교 차트
6. 나라별 추천 대상 (원안 8번 표 활용)
7. 계산기·국가별 리포트 링크

## 9. 공통 데이터·컴포넌트 재사용 원칙

- 국내 클러스터의 `InternationalSchoolProfile` 타입(`src/data/internationalSchoolTuitionCalculator2026.ts`)은 `country: "KR"` 리터럴 유니언이라 그대로 확장하면 국가별 특수 필드(통화, 도시, 생활비 구간, 학교 등급)를 담기 어렵다 — **설계 문서 단계에서 별도 타입(`SoutheastAsiaSchoolProfile`, 가칭 `src/data/southeastAsiaInternationalSchool2026.ts`)으로 분리할지, 공통 필드만 뽑아 공유 타입으로 리팩터링할지 결정**한다. 최소한 아래는 신규 타입에도 동일하게 유지:
  - `sourceUrl` + `asOfDate` 필드로 학비 출처·확인일자 관리 (국내 클러스터와 동일 원칙)
  - `country` 필드는 ISO 국가 코드(`"MY"` `"TH"` `"VN"`)로 확장
- 생활비 데이터는 학비보다 출처 신뢰도가 낮으므로 `costOfLivingSource` 필드를 별도로 두고, 페이지에 학비와 다른 출처 표기를 노출
- 환율은 계산기 컴포넌트에서 기본값 상수로 관리하되 사용자가 직접 수정 가능하도록 국내 계산기(달러 납부 학교용 환율 입력)와 동일한 패턴 재사용
- 한국 대비 비교 로직은 계산기 페이지에서만 계산, 리포트는 계산값을 인용만 함 (국내 클러스터 원칙과 동일)

## 12. 이번 스코프에서 제외한 것 (확장 후보로만 보류)

- **필리핀**: "국제학교"보다 "영어캠프·어학연수" 검색 의도가 강해 이 클러스터의 "정규 국제학교 학비 비교" 톤과 어긋남 — 별도 콘텐츠(영어캠프 비용 비교)로 분리 검토
- **싱가포르**: "저렴하다" 내러티브가 성립하지 않아(한국보다 오히려 비쌈) 이번 클러스터의 핵심 후킹과 맞지 않음 — "프리미엄 교육" 앵글로 별도 기획 필요
- **인도네시아/발리**: 검색 볼륨 확인 전까지 보류, 니치 키워드로 판단
- 국가별 개별 학교 단독 페이지(채드윅 송도처럼 검색량 상위 학교 1곳 단독화)는 국가 리포트 배포 후 검색 데이터로 수요 확인 후 결정

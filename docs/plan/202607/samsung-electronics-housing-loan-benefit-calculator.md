# 삼성전자 주택대출 복지 계산기 기획

> `5억 원을 연 1.5%로 빌릴 수 있다면 실제 이득은 얼마인가?`라는 질문에 바로 답하는 복지 환산형 계산기 기획입니다. 삼성전자 임직원 주택자금 대출 조건은 공개 자료만으로 확정하기 어려우므로, 본 기획에서는 `5억 원·연 1.5%`를 사용자 제안 시나리오로 두고 모든 수치에 `확인 필요` 또는 `시뮬레이션` 배지를 붙입니다.

## 2. 왜 지금 만들 만한가

- 삼성전자 복지는 취업·이직·연봉 비교 검색에서 관심도가 높다.
- 단순히 `삼성전자 복지 좋다`가 아니라, 복지를 돈으로 환산하면 얼마인지 보여주면 비교계산소의 정체성과 정확히 맞는다.
- 주택자금 대출은 체감 금액이 크다. 5억 원 기준 금리가 1%p만 낮아져도 연간 500만 원 차이가 난다.
- 연봉 콘텐츠와 자연스럽게 연결된다. 예를 들어 `삼성전자 연봉`, `삼성전자 성과급`, `삼성전자 복지`, `대기업 복지 비교`와 내부 링크를 만들 수 있다.
- 사용자는 계산기에서 본인 조건을 입력해볼 수 있어 체류 시간이 길어질 가능성이 높다.

- 시중금리, 복지금리, 대출한도, 상환방식을 사용자가 바꿀 수 있다.
- 세금과 과세 가능성은 단정하지 않고 별도 안내로 분리한다.
- 삼성전자 단독 계산에서 시작하되, 향후 SK하이닉스·현대차·LG·공공기관 복지 비교로 확장할 수 있다.

## 7. 핵심 계산 로직

### 7.1 단순 연간 이자 절감액

가장 직관적인 첫 화면 계산이다.

```text
연간 이자 절감액 = 대출원금 × (시중금리 - 복지금리)
월 이자 절감액 = 연간 이자 절감액 ÷ 12
```

대표 시나리오:

| 대출원금 | 복지금리 | 비교 시중금리 | 연간 절감액 | 월 절감액 |
|---:|---:|---:|---:|---:|
| 5억 원 | 1.5% | 4.0% | 1,250만 원 | 약 104만 원 |
| 5억 원 | 1.5% | 4.5% | 1,500만 원 | 125만 원 |
| 5억 원 | 1.5% | 5.0% | 1,750만 원 | 약 146만 원 |

주의:

- 위 표는 원금이 그대로 유지되는 단순 이자 기준이다.
- 원리금균등·원금균등 상환에서는 시간이 갈수록 원금이 줄어 절감액도 달라진다.
- 실제 이득은 세금, 대출 실행 비용, 중도상환수수료, 기존 대출 갈아타기 조건에 따라 달라질 수 있다.

### 7.2 세전 연봉 환산

복지 이득을 연봉 인상분처럼 이해하기 위한 지표다.

```text
세전 연봉 환산액 = 연간 순이득 ÷ (1 - 추정 세율)
```

기본 세율 옵션:

| 세율 프리셋 | 설명 |
|---|---|
| 15% | 낮은 세율 가정 |
| 24% | 중간 소득 구간 가정 |
| 35% | 고소득 구간 가정 |
| 직접 입력 | 사용자가 본인 한계세율 입력 |

예시:

```text
연간 절감액 1,250만 원, 추정 세율 24%라면
세전 연봉 환산액 = 1,250만 원 ÷ 0.76 = 약 1,645만 원
```

표현 원칙:

- `연봉이 1,645만 원 오른 것과 같다`라고 단정하지 않는다.
- `세후 현금흐름 기준으로 보면 세전 연봉 약 1,645만 원의 가치로 환산할 수 있다`라고 표현한다.

## 10. 본문 콘텐츠 초안

### 10.1 5억 원을 1.5%로 빌리는 게 왜 큰가

대출금리가 1%p 낮아지는 효과는 생각보다 크다. 원금 5억 원 기준으로 금리 1%p는 연간 500만 원, 월 약 42만 원의 이자 차이다. 복지금리가 1.5%이고 시중금리가 4.0%라면 차이는 2.5%p이므로 연간 약 1,250만 원, 월 약 104만 원의 현금흐름 차이가 생긴다.

이 금액은 단순히 `이자가 조금 싸다` 수준이 아니다. 세후로 매년 1,250만 원을 아끼는 효과라면, 세율 24% 가정에서는 세전 연봉 약 1,645만 원의 가치로 환산할 수 있다. 그래서 대기업 복지는 연봉만 보고 판단하기 어렵다.

### 10.2 복지는 연봉표에 안 보이는 총보상이다

연봉 비교에서 기본급, 성과급, 주식 보상은 자주 비교되지만 주택자금 대출 같은 금융 복지는 빠지는 경우가 많다. 하지만 대출금리 차이는 실제 생활비에 바로 영향을 준다. 특히 주거비 부담이 큰 수도권 직장인에게는 낮은 금리의 사내대출이 성과급 못지않게 큰 복지일 수 있다.

다만 모든 임직원이 같은 조건을 적용받는다고 단정하면 안 된다. 재직기간, 직급, 주택 보유 여부, 용도, 보증 조건, 내부 한도, 실행 시점의 규정에 따라 실제 가능 금액은 달라질 수 있다. 이 페이지는 특정 회사 복지의 경제적 가치를 이해하기 위한 계산 도구로 설계한다.

### 10.3 세금까지 보면 순이득은 달라질 수 있다

사내 저리 대출은 조건에 따라 과세 이슈가 생길 수 있다. 회사가 시중보다 낮은 금리로 임직원에게 자금을 빌려주는 경우, 그 차익이 근로소득 또는 인정이자 성격으로 판단될 여지가 있다. 구체적인 과세 여부는 세법, 회사 운영 방식, 대출 목적, 적용 금리, 개인 상황에 따라 달라질 수 있으므로 계산기에서는 `세전 환산`과 `세후 체감`을 분리해서 보여준다.

초기 버전에서는 세금 계산을 확정적으로 처리하지 않는다. 대신 사용자가 추정 세율을 입력하면 `이 정도 현금흐름은 세전 연봉으로 얼마에 해당하는가`를 보여준다.

   - 이 페이지의 1.5%는 대표 시나리오 입력값입니다. 실제 금리와 한도는 삼성전자 내부 규정, 재직 조건, 신청 시점에 따라 달라질 수 있으므로 공식 확인이 필요합니다.
2. 5억 원을 1.5%로 빌리면 얼마를 아끼나요?
   - 시중금리 4.0%와 비교하면 단순 이자 기준 연간 약 1,250만 원, 월 약 104만 원을 아끼는 계산이 나옵니다.
3. 이 금액을 연봉으로 환산하면 얼마인가요?
   - 연간 절감액 1,250만 원을 세율 24%로 환산하면 세전 연봉 약 1,645만 원 가치로 볼 수 있습니다. 단, 실제 세율과 과세 여부에 따라 달라집니다.
4. 사내대출 이득도 세금을 내야 하나요?
   - 저리 대출은 조건에 따라 과세 이슈가 생길 수 있습니다. 이 계산기는 세무 판단을 대신하지 않으며, 실제 적용은 회사와 세무 전문가 확인이 필요합니다.
5. 원리금균등 상환이면 결과가 달라지나요?
   - 달라집니다. 원금이 줄어들면 금리 차이로 절감되는 이자도 줄어듭니다. 초기 버전은 단순 이자 기준을 기본으로 하고, 확장 버전에서 상환방식별 계산을 제공할 수 있습니다.
6. 삼성전자 말고 다른 회사 복지도 비교할 수 있나요?
   - 구조상 가능합니다. 동일한 계산식을 사용해 SK하이닉스, 현대차, LG, 공공기관 등 회사별 주택자금 대출 복지를 비교 콘텐츠로 확장할 수 있습니다.
7. 주택담보대출 갈아타기와 비교해도 되나요?
   - 가능합니다. 기존 대출금리와 사내대출 금리를 입력하면 갈아탔을 때의 이자 차이를 계산할 수 있습니다. 다만 중도상환수수료와 담보 조건은 별도 확인해야 합니다.

## 13. 계산 함수 제안

```ts
export function calculateSimpleInterestBenefit(
  principalKrw: number,
  benefitRatePercent: number,
  marketRatePercent: number,
  years: number,
  taxRatePercent: number
) {
  const benefitRate = benefitRatePercent / 100;
  const marketRate = marketRatePercent / 100;
  const taxRate = taxRatePercent / 100;
  const annualBenefitInterestKrw = principalKrw * benefitRate;
  const annualMarketInterestKrw = principalKrw * marketRate;
  const annualSavingKrw = annualMarketInterestKrw - annualBenefitInterestKrw;
  const monthlySavingKrw = annualSavingKrw / 12;
  const totalSavingKrw = annualSavingKrw * years;
  const grossSalaryEquivalentKrw = taxRate >= 1 ? annualSavingKrw : annualSavingKrw / (1 - taxRate);

  return {
    annualBenefitInterestKrw,
    annualMarketInterestKrw,
    annualSavingKrw,
    monthlySavingKrw,
    totalSavingKrw,
    grossSalaryEquivalentKrw,
  };
}
```

## 17. 결론

이 콘텐츠는 삼성전자 복지를 단순 소개가 아니라 `돈으로 환산되는 혜택`으로 보여줄 수 있어 비교계산소와 잘 맞는다. 특히 `5억 연 1.5%면 실제 이득 얼마?`라는 제목은 클릭 유도가 강하고, 계산 결과도 직관적이다.

다만 삼성전자 실제 주택자금 대출 조건은 공개 자료만으로 확정하기 어렵기 때문에, 페이지는 반드시 입력형 계산기로 만들어야 한다. 상단에서는 대표 시나리오로 빠르게 답하고, 본문에서는 실제 조건 확인 필요성과 세금 가능성을 분리해 설명하는 방식이 가장 안전하다.

## 1. 배경 및 목적

국제학교는 학비·입학조건 검색 의도가 뚜렷하고("국제학교 학비", "제주 국제학교 비교", "외국인학교 내국인 입학" 등), 자녀 교육비를 미리 계산해보려는 2030~4050 학부모층이 비교계산소의 핵심 타깃과 겹친다. 사용자가 참고용으로 가져온 초안(ChatGPT 브레인스토밍, 약 60개 제목 후보)을 이 사이트의 실제 컨벤션(`CLAUDE.md` 타이틀 공식, `CONTENT_GUIDE.md` 페이지 구조·수익화 규칙)에 맞춰 정리하고, 로직이 겹치는 페이지를 통합해 실속 있는 개수로 재구성한다.

### 목표
- "국제학교 학비", "제주 국제학교", "외국인학교 입학조건" 등 핵심 검색 유입 확보
- 계산기 1개 + 비교/정보 리포트로 구성된 클러스터, 서로 내부링크로 연결
- **정확성 최우선** — 실제 존재하는 학교명(NLCS Jeju, Chadwick 등)과 구체적 학비 숫자를 다루므로, 잘못된 수치는 실사용자의 진학 판단에 직접 영향을 준다

### 페이지 수 결정 — 원안 60개 → 7~8개로 압축

사용자가 가져온 초안은 "학비 계산기"류만 10개(자녀 2명용, 환율용, 첫해 비용용, 12년 누적용 등)를 별도 페이지로 제안했지만, `senior-job-content-cluster-2026`에서 이미 확인했듯이 **입력값과 로직이 사실상 같은 계산기를 여러 페이지로 쪼개면 유지보수 부담만 커지고 콘텐츠 품질(AdSense 심사 기준)도 흐려진다.** 대신 **입력 옵션(학년·자녀 수·지역·통학/기숙)을 하나의 계산기에 다 넣고**, 진짜 관점이 다른 것(지역별 비교표, 제도 차이 설명, 대안 비교)만 리포트로 분리한다.

## 2. ⚠️ 구현 전 필수 팩트체크 원칙 (가장 중요)

이 클러스터는 **실제 학교명 + 실제 학비 숫자**를 다룬다. 사용자가 가져온 초안의 학비 관련 숫자는 전부 예시/가정치이며, **이 문서에는 학비 확정 수치를 하나도 기재하지 않는다** — 잘못된 추정치를 그대로 베끼는 것이 이 카테고리에서 가장 위험한 실수다.

**설계 문서 작성 및 구현 착수 전, 아래 항목을 학교 공식 홈페이지·최신 입학요강 기준으로 반드시 재조사한다:**

| 확인 필요 항목 | 확인 방법 |
|---|---|
| 제주 국제학교 4개교(NLCS Jeju, Branksome Hall Asia, SJA, KIS Jeju) 연간 학비 | 각 학교 공식 홈페이지 Tuition/Fees 페이지 |
| 서울·경기·송도권 학교(SFS, Dulwich Seoul, Dwight School Seoul, Chadwick Songdo, YISS, KIS Pangyo, SIS 등) 연간 학비 | 각 학교 공식 홈페이지 |
| 외국인학교 입학자격 (내국인 3년 이상 해외거주 요건 등) 법적 근거 | 교육부·시도교육청 「외국인학교 및 외국인유학생을 위한 초·중·고등학교의 설립·운영에 관한 규정」 |
| 제주국제학교(제주영어교육도시) 내국인 입학자격이 외국인학교와 다른 근거 | 제주특별자치도·JDC(제주국제자유도시개발센터) 공식 자료 |
| 미인가 국제학교의 학력 인정 여부 | 교육부 학력인정 관련 공식 안내 |
| 각 학교 커리큘럼(IB/AP/A-Level 등) | 학교 공식 페이지 |

각 수치는 반드시 **출처 링크 + 확인일자**를 데이터 파일에 함께 기록하고, 학교별로 학비 개정 시점이 다르므로 InfoNotice에 "학교별 최신 학비는 공식 입학처에서 재확인 필요"를 명시한다. `senior-job-content-cluster-2026-plan.md`의 "확인 필요를 최소화하되, 근거 없는 단일 숫자를 확정 발표하지 않는다" 원칙을 그대로 따른다.

## 4. 페이지 1 — `international-school-tuition-calculator-2026` (메인 계산기)

### 4-1. URL·메타 (타이틀 공식 적용)

```
슬러그: /tools/international-school-tuition-calculator-2026/
타이틀: 국제학교 학비 계산기 2026 | 연간 비용 바로 계산
디스크립션: 지역·학년·자녀 수 입력하면 국제학교 연간 학비와 월 부담액, 12년 누적 비용까지 바로 계산. 소득 대비 비율 진단 포함.
```

### 4-2. 입력값 (원안의 계산기 10개를 여기로 통합)

| 입력 | 선택지 |
|---|---|
| 지역/학교 유형 | 서울·경기 / 송도 / 제주(통학) / 제주(기숙) |
| 학년 | 유치부 / 초등 / 중등 / 고등 |
| 자녀 수 | 1명 / 2명 / 3명 |
| 기숙사 여부 | 통학 / 기숙 (제주권 선택 시 노출) |
| 가구 연소득 | 자유 입력 (소득 대비 비율 계산용) |
| 환율 | 기본값 제공, 수정 가능 (달러 납부 학교 대상) |

### 4-3. 출력값

| 결과 | 설명 |
|---|---|
| 연간 총비용 | 학비 + 입학금(1년차) + 기숙사비(해당 시) + 버스비 등 |
| 월 환산 비용 | 연간 총비용 ÷ 12 |
| 소득 대비 비율 | 연간 총비용 ÷ 가구 연소득 |
| 12년 누적 비용 | 유치부~고3까지 단순 누적 추정 (물가상승률 미반영, 참고용 명시) |
| 자녀 수별 총비용 | 형제 할인 적용 여부 학교별 상이 — 별도 확인 필요 항목으로 표기 |

### 4-4. 페이지 섹션 구성 (CONTENT_GUIDE.md 계산기 골격)

1. CalculatorHero
2. InfoNotice — "학비는 학교·연도별로 실제 인상되며, 이 계산은 공식 발표 학비 기준 추정치입니다. 최신 학비는 각 학교 입학처에서 재확인하세요."
3. 입력 영역 (4-2)
4. 결과 카드 (4-3)
5. 지역별 참고 학비 비교 미니 테이블 (제주/서울/송도 대표 학교, 상세는 리포트로 링크)
6. SeoContent (intro 4단락 이상, FAQ 5개 이상 — `CONTENT_GUIDE.md` AdSense 기준 준수)

### 4-5. FAQ (초안 — 답변은 실제 수치 확정 후 작성)
1. 국제학교 학비에 입학금도 포함되나요?

3. 제주 국제학교는 기숙사비까지 포함하면 얼마나 더 드나요?

- 학교 마스터 데이터(`InternationalSchoolProfile[]`: 학교명, **국가/지역**, 학비, 통학/기숙, 커리큘럼, 입학자격, 출처)는 **한 파일에서 정의**해 계산기·허브·지역별 리포트가 공통 참조 (`src/data/internationalSchoolShared.ts` 검토)
  - `country` 필드를 지금부터 포함해 `"KR"`로 고정 사용 — 13번 향후 확장 고려사항의 동남아 확장 시 필드 재정의 없이 값만 추가할 수 있게 미리 대비
- 학비 숫자는 전부 `sourceUrl` + `asOfDate` 필드를 가진 구조로 관리 — 페이지에 "OO년 O월 기준, 출처: 학교 공식 홈페이지" 형태로 항상 노출
- 소득 대비 비율 계산 로직은 계산기 페이지에서만 사용, 리포트는 계산값을 인용만 하고 재계산하지 않음

## 14. 향후 확장 고려사항 — 동남아 국제학교 (스코프 아님, 대비만)

사용자 방향: **1차는 국내 국제학교만.** 이후 싱가포르·말레이시아·베트남 등 동남아 국제학교(조기유학·이주 검토 가정 타깃)로 확장 예정이나, 이번 클러스터의 범위는 아니다.

지금 단계에서 반영해둘 것:
- 슬러그를 `korea-international-school-*`가 아니라 **국가 접두어 없는 `international-school-*`**로 설계 — 나중에 국가별로 쪼개지 않고 이 이름 그대로 국내 콘텐츠의 정식 슬러그로 유지 가능
- 학교 마스터 데이터에 `country` 필드 지금부터 포함 (위 11번 반영)
- `international-school-overview-2026`(허브) 설계 시, 나중에 국가 탭(예: "한국 | 싱가포르 | 말레이시아")을 얹을 수 있는 구조로 섹션을 짜되, **지금은 국내 탭 하나만 노출**하고 동남아 확장은 별도 기획 문서로 나중에 새로 잡는다
- 동남아 확장은 학비 통화(SGD, MYR, VND 등)·환율 변수가 추가되므로, 메인 계산기의 환율 입력 필드를 지금부터 범용적으로 설계해두면 재사용 여지가 생긴다

### 계산 로직

```
월 사용량 = 해당 월령 하루 평균 사용량 × 해당 월 일수(30)
총 사용 개수 = 각 월령 구간 월 사용량 합산
총 비용 = 총 사용 개수 × 선택 브랜드 장당가
브랜드 간 비용 차이 = 최고가 브랜드 총비용 - 최저가 브랜드 총비용
```

**입력값**

| 입력 항목 | 타입 | 기본값 | 비고 |
|---|---|---|---|
| 계산 기간 | 슬라이더 (1~24개월) | 12개월 | "돌까지" 기본 |
| 브랜드 선택 | 멀티 체크박스 | 하기스/보솜이 기본 선택 | 최대 3개 비교 |
| 장당 가격 직접 입력 | 숫자 입력 | 기본값 자동 채움 | 쿠팡 최저가 직접 입력 가능 |

**출력값**

| 출력 항목 | 표시 방식 |
|---|---|
| 총 기저귀 사용 개수 | 요약 카드 (큰 숫자) |
| 브랜드별 총비용 | 비교 카드 3열 |
| 브랜드 간 절약 가능 금액 | 강조 카드 ("보솜이 선택 시 X원 절약") |
| 월별 비용 추이 | 막대 차트 (Chart.js) |
| 월령별 사이즈 전환 시점 | 인포 박스 |

### FAQ 기획

| 질문 | 핵심 답변 |
|---|---|
| 기저귀는 언제까지 써야 하나요? | 보통 만 2~3세, 배변 훈련 완성 시 졸업. 계산기는 돌까지 기준. |
| 신생아 기저귀 하루에 몇 개 써야 하나요? | 신생아 초기 10~12개, 생후 3개월부터 6~8개로 줄어듦. |

#### 페이지 구조 (§2-A 계산기)

```
① CalculatorHero      — "돌까지 기저귀값 계산기" + 한 줄 설명
② InfoNotice          — "월령별 사용량·브랜드 가격은 추정값입니다. 참고용으로만 활용하세요."
③ 입력 영역           — 기간 슬라이더 + 브랜드 체크박스 + 장당가 수정
④ KpiCard / SummaryCards — 총 사용 개수 / 브랜드별 총비용 / 절약 가능액
⑤ 상세 비교           — 월별 막대 차트 + 브랜드 비교표
⑥ 제휴 마케팅 블록    — affiliate-box 1개 (SeoContent 앞)
⑦ SeoContent          — FAQ 6개 + 관련 계산기 링크
```

#### 디자인·UX 규칙 (§4)

| 항목 | 이 페이지 적용 기준 |
|---|---|
| 레이아웃 순서 | 슬라이더 입력 → 요약 카드 → 브랜드 비교표 → 차트 순 (모바일 퍼스트) |

### 계산 로직

```
월 수유량(ml) = 해당 월령 하루 수유량 × 해당 월 일수(30)
총 수유량(ml) = 각 월령 구간 월 수유량 합산
분유 총 소비량(g) = 총 수유량(ml) × 0.14  (100ml당 14g 기준)
필요 캔 수 = 분유 총 소비량(g) ÷ 캔 용량(g), 올림
총 비용 = 필요 캔 수 × 선택 브랜드 캔 가격
브랜드 간 비용 차이 = 최고가 브랜드 총비용 - 최저가 브랜드 총비용
```

### 데이터 기준 — LTV 비율표 (2026년 3월 기준, 참고용)

> LTV 규정과 주담대 한도는 정책에 따라 수시 변경된다.

LTV기준_대출가능액 = 매매가 × LTV비율(지역유형·주택수 기준)

// 2) 정책상 주담대 한도
정책상_한도 =
if (수도권·규제지역 && 매매가 <= 15억) → 6억
else if (수도권·규제지역 && 매매가 <= 25억) → 4억
else if (수도권·규제지역 && 매매가 > 25억) → 2억
else → 매우 큰 값(사실상 제한 없음)

// 3) 최종 대출 가능액
최대 대출 가능액 = min(LTV기준_대출가능액, 정책상_한도, DSR기준_한도)

// 취득세 (비례구간 처리)
취득세_기본 =
if (매매가 <= 6억) → 매매가 × 1%
else if (매매가 <= 9억) → 매매가 × (1% + (매매가-6억)/(9억-6억) × 2%)
else → 매매가 × 3%

다주택·조정지역 중과 시 해당 요율 직접 적용
취득세_합산 = 취득세_기본 × 1.1

// 중개보수
중개보수 = min(매매가 × 상한요율, 구간별 한도)

// 총 필요 현금
총 부대비용 = 취득세_합산 + 중개보수 + 등기비용
최소 필요 현금 = (매매가 - 최대 대출 가능액) + 총 부대비용
현금 여유·부족액 = 보유현금 - 최소 필요 현금

// 원리금균등상환
월이율 = 연금리 / 12
월 상환액 = 대출금 × [월이율 × (1+월이율)^(기간×12)] / [(1+월이율)^(기간×12) - 1]

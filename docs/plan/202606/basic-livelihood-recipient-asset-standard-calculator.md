## 1. 왜 새로 만들까

기존 `/tools/welfare-benefit-eligibility/`는 기초생활보장 급여 전체 가능성을 계산한다. 하지만 검색 사용자의 실제 질문은 더 좁고 불안이 선명하다.

```text
재산이 있는데 기초생활수급자가 될 수 있나?
전세보증금이 있으면 탈락하나?
자동차가 있으면 무조건 안 되나?
예금 얼마까지 괜찮나?
```

이 질문은 `생계급여 소득인정액 계산기`와도 다르다. 생계급여 페이지는 “월 얼마 받을 수 있나”가 중심이고, 이 페이지는 **재산 항목이 수급 가능성에 어떤 영향을 주는지**가 중심이다.

따라서 신규 페이지는 금액 결과보다 **재산 위험도, 기본재산액 차감 후 남는 금액, 재산의 월 소득환산액, 자동차 확인 필요 여부**를 전면에 배치한다.

## 6. 입력 설계

재산 기준 검색 사용자는 월소득보다 재산 항목에 집중한다. 그래서 입력 순서를 기존 통합 계산기와 다르게 둔다.

### Step 1. 가구·지역

| 입력 | UI | 기본값 | 설명 |
|---|---|---:|---|
| 가구원 수 | select | 4 | 결과 기준선 계산 |
| 거주 지역 | segmented control | 대도시 | 기본재산액 차감 기준 |
| 주거 형태 | select | 전세 | 주거재산 설명 문구 변경 |

### Step 2. 재산

| 입력 | UI | 기본값 | 설명 |
|---|---|---:|---|
| 전세보증금/주택가액 | number | 50,000,000 | 주거재산 |
| 일반 부동산·토지 | number | 0 | 일반재산 |
| 예금·적금·주식 | number | 10,000,000 | 금융재산 |
| 보험 해지환급금 | number | 0 | 금융재산에 합산 |
| 인정 가능 부채 | number | 0 | 차감 가능 여부 안내 |

### Step 3. 자동차

| 입력 | UI | 기본값 | 설명 |
|---|---|---:|---|
| 자동차 보유 | toggle | false | 없으면 상세 숨김 |
| 차량가액 | number | 0 | 확인 필요 플래그 |
| 차량 용도 | select | 일반 | 일반, 생업용, 장애인용, 보호용 |
| 배기량/연식 | select/input | 모름 | 추후 확장 |

### Step 4. 월소득

| 입력 | UI | 기본값 | 설명 |
|---|---|---:|---|
| 월 근로·사업소득 | number | 1,200,000 | 소득인정액 전체 확인용 |
| 공적 이전소득 | number | 0 | 국민연금 등 |
| 근로소득 공제 적용 | toggle | true | 간이 공제 |

## 7. 계산 로직

### 재산 기준 핵심 출력

```text
총재산 = 주거재산 + 일반재산 + 금융재산 + 보험해지환급금
순재산 = max(총재산 - 부채, 0)
기본재산액 차감 후 재산 = max(순재산 - 지역별 기본재산액, 0)
재산의 월 소득환산액 = 재산 종류별 환산액 합계
```

### 지역별 기본재산액

기존 `WBE_ASSET_DEDUCTION_BY_REGION`을 재사용한다.

| 지역 | 기본재산액 |
|---|---:|
| 대도시 | 9,900만 원 |
| 중소도시 | 7,700만 원 |
| 농어촌 | 5,300만 원 |

### 재산 항목별 해석

| 항목 | MVP 처리 | 결과 메시지 |
|---|---|---|
| 주거재산 | 기본재산액 차감 대상 | 전세보증금은 재산에 포함되지만 지역별 기본재산액 차감 후 반영됩니다. |
| 일반재산 | 환산액 계산 | 토지·비주거 부동산은 결과에 민감하게 반영될 수 있습니다. |
| 금융재산 | 별도 환산율 | 예금·적금·주식은 생활준비금 등 세부 공제 확인이 필요합니다. |
| 부채 | 차감 | 금융기관 부채 등 인정 가능한 부채인지 확인이 필요합니다. |
| 자동차 | 위험 플래그 중심 | 일반 자동차는 조사에서 중요하게 확인될 수 있습니다. |

### 판정 단계

| 단계 | 조건 | 메시지 |
|---|---|---|
| 재산 부담 낮음 | 기본재산액 차감 후 재산이 거의 없음 | 재산 기준만 보면 부담이 낮은 편입니다. |
| 재산 경계 | 재산환산액이 생계급여 기준의 10~25% | 소득과 함께 보면 결과가 달라질 수 있습니다. |
| 재산 부담 높음 | 재산환산액이 생계급여 기준의 25% 초과 | 재산 때문에 기준을 초과할 가능성이 있습니다. |
| 자동차 확인 필요 | 일반 차량 보유 또는 차량가액 입력 | 차량 용도와 가액 확인이 필요합니다. |

## 12. 주의사항

이 페이지는 복지 신청 여부를 확정하면 안 된다. 특히 자동차, 부채, 금융재산, 주거재산 인정 방식은 실제 조사에서 세부 조건이 많다. 모든 결과에는 다음 표현을 유지한다.

```text
자가 점검용 추정입니다.
실제 수급 여부는 주민센터 조사와 공적자료 확인을 통해 결정됩니다.
자동차·부채·재산 특례는 반드시 별도 확인이 필요합니다.
```

## 14. 결론

이 페이지는 “기초생활수급자 자격 계산기”를 다시 만드는 것이 아니다. 사용자가 가장 불안해하는 **재산 때문에 탈락하나**라는 질문에 집중하는 보조 계산기다.

기존 통합 계산기는 최종 자격 가능성, 생계급여 페이지는 예상 지급액, 이 페이지는 재산 기준 이해를 담당하게 하면 복지 클러스터가 서로 겹치지 않고 넓어진다.

## 1. 배경 및 목적

통신비는 매달 나가는 고정비 중 체감이 큰 항목이다. "알뜰폰으로 갈아타면 얼마 아낄까?" 라는 검색은 **연령·소득에 관계없이 발생하는 생활형 검색 수요**다. 연봉·투자 계산기보다 훨씬 넓은 타겟층을 커버한다.

### 핵심 컨셉: "갈아타면 2년 동안 얼마 아낄 수 있어?"

단순 월 절약액이 아니라 **1년·2년 누적 절감액을 크게 보여주는 것**이 핵심이다. 월 3만원 절약이 2년이면 72만원이라는 체감 수치가 행동을 유발한다.

## 2. 타겟 사용자

- 이통3사(SKT·KT·LGU+) 요금제를 쓰면서 "비싸다"고 느끼는 사람
- 자녀 스마트폰을 개통해줘야 하는 부모
- 단말기 할부가 끝나거나 약정 해지 시점이 다가온 사람
- 선택약정 24개월이 끝난 직후인 사람
- 시니어 부모님 요금제를 대신 알아보는 자녀
- 1인 가구 / 사회초년생 (고정비 절감 민감)

## 8. 출력값 설계

### 8-1. 핵심 KPI 카드 (결과 우선 panel)

| 카드 | 표시 내용 | 강조 |
|------|----------|------|
| 월 절감액 | 현재 실납부 - 알뜰폰 월비용 | 보조 |
| 1년 절감액 | 월 절감액 × 12 | 보조 |
| 2년 절감액 | 월 절감액 × 24 | **메인 강조** (accent) |
| 위약금 회수 기간 | 위약금 ÷ 월 절감액 (개월) | 보조 |

> 위약금이 0이면 "위약금 없음 · 즉시 갈아타기 가능" 표시  
> 절감액이 음수(알뜰폰이 오히려 더 비쌀 때): "현재 요금제가 더 저렴합니다" 경고 표시

### 8-2. 누적 절감액 그래프

- X축: 월(1~24개월 또는 1~36개월)
- Y축: 누적 절감액 (원)
- 위약금 회수 기점에 수직선 + 라벨 표시
- Line chart (Chart.js) — 월별 누적 계산

### 8-3. 현재 vs 알뜰폰 비교 카드

```
[현재 요금제]                    [알뜰폰]
기본료: 65,000원                 월 요금: 25,000원
가족결합 할인: -5,000원           가족결합 상실 추가: +3,000원
선택약정 할인: -10,000원
─────────────────               ─────────────
실 납부: 50,000원               실질 월 비용: 28,000원
```

→ 차이: **월 22,000원 절약**

### 8-4. 손익분기점

- 위약금이 있는 경우: "위약금 ○만원 ÷ 월 절감액 ○만원 = N개월 후 이득"
- 막대 or 카드로 시각화

갈아타기 망설임을 해소하는 섹션. 체크리스트 형식으로 "이 항목이 괜찮으면 갈아타도 됩니다"를 보여준다.

| 항목 | 실제 상황 | 중요도 |
|------|----------|--------|
| 망 품질 | SKT·KT·LGU+ 망 임차해 사용. 체감 차이 없는 경우 多 | ★★★ |
| 고객센터 | 대형 이통사보다 불편. 챗봇·앱 위주 | ★★ |
| 단말기 지원금 | 없거나 미미함. 자급제 폰 사는 편이 유리 | ★★★ |
| 가족결합 불가 | 이통3사 가족결합 불가 (알뜰폰끼리는 가능한 경우 있음) | ★★★ |
| 5G 지원 | 일부 알뜰폰은 5G 지원 LTE 전용 요금 많음 | ★★ |

## 11. 계산 로직 (수식 정리)

```
// 현재 실납부액 (월)
currentMonthly = baseFee - familyDiscount - contractDiscount

// 알뜰폰 실질 월 비용
mvnoMonthly = mvnoFee + (familyLost ? familyLossAmount : 0)

// 월 절감액
monthlySaving = currentMonthly - mvnoMonthly

// 누적 절감액 (위약금 차감)
cumulativeSaving(month) = monthlySaving * month - earlyTermFee

// 손익분기점 (개월)
breakEvenMonths = earlyTermFee > 0 ? Math.ceil(earlyTermFee / monthlySaving) : 0

// 누적 그래프 데이터
chartData = Array.from({length: period}, (_, i) => ({
  month: i + 1,
  saving: monthlySaving * (i + 1) - earlyTermFee   // 음수 → 0 처리 표시용
}))
```

# 강아지·고양이 월 양육비 계산기 기획

## 1. 배경 및 목적

- **계기**: 국내 반려동물 양육 가구 약 600만(2024 기준). "강아지 한 달 비용", "고양이 양육비" 등 생활밀착형 검색 수요 꾸준히 존재. 현재 사이트에 반려동물 관련 콘텐츠 전무
- **목적**:
  1. "강아지 월 양육비", "고양이 한 달 비용" 키워드 SEO 확보
  2. 감성 콘텐츠 특성상 SNS 공유·재방문율 높음 → 페이지뷰 상승
  3. 반려동물 관련 애드센스 단가는 생활용품·펫푸드 광고로 일반 대비 높은 편
- **타겟 사용자**: 반려동물 입양 예정자, 현재 반려인(비용 점검), 예산 계획 중인 새내기 집사

---

## 2. 콘텐츠 성격

- 계산기형 도구: 항목별 월 지출 입력 → 총 월 양육비 + 연간 비용 + 10년 누적비 출력
- 강아지 / 고양이 탭 전환
- 기본값은 실제 평균 지출 데이터 기반 (한국펫산업리서치, 농림축산식품부 반려동물 보고서)

## 4. 입력 항목 및 기본값

### 공통 구조

```
탭: 강아지 / 고양이

[필수 항목]
- 사료비 (월)          강아지: 60,000원 / 고양이: 50,000원
- 간식비 (월)          강아지: 20,000원 / 고양이: 15,000원
- 모래·패드 (월)       강아지: 10,000원(패드) / 고양이: 25,000원(모래)
- 병원·예방접종 (월 평균)  강아지: 30,000원 / 고양이: 20,000원
- 미용비 (월 평균)     강아지: 50,000원 / 고양이: 10,000원
- 장난감·용품 (월)     공통: 10,000원
- 펫보험 (월)          선택: 0원 (입력 시 반영)

[선택 항목 — 토글로 열기]
- 호텔·돌봄비 (월 평균)
- 훈련·교육비 (월)
- 의류·악세서리 (월)
```

### 데이터 근거

| 항목 | 근거 |
|---|---|
| 사료비 | 농림축산식품부 2023 반려동물 보고서 월평균 사료 지출 |
| 병원비 | 한국펫산업리서치 2024, 연간 병원비 평균 ÷ 12 |
| 미용비 | 소형견 기준 6~8주 주기 2~5만원 평균 |

- "연간 해외여행 1회 비용과 비슷합니다"

## 6. UI/IA 상세

1. CalculatorHero — "강아지·고양이 월 양육비 계산기 2026"
2. 탭: 강아지 / 고양이 전환 (탭 전환 시 기본값 교체)
3. 입력 패널 — 슬라이더 + 숫자 입력 혼합
4. KPI 카드 4개 (월·연·10년·하루)
5. 도넛 차트 — 항목별 비율
6. 비교 인사이트 텍스트 박스
7. 관련 링크 → 품종별 10년 비용 비교, 펫보험 손익 계산기
8. SeoContent (FAQ 포함)

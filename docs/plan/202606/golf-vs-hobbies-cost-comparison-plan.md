# 골프 vs 다른 취미 비용 비교 기획

## 1. 배경 및 목적

- **계기**: "골프가 비싼 취미인가?", "골프 vs 낚시 비용", "골프 vs 헬스 비교" 등 감성 비교 콘텐츠 수요 높음. 비교형 인포그래픽은 소셜 공유율이 높아 유입 다양화에 유리
- **목적**:
  1. "취미 비용 비교", "골프 비용 비싼가" 키워드 SEO 확보
  2. 비교 콘텐츠 → 다양한 타겟 유입 (골프 검토자 + 타 취미 사용자)
  3. 골프 월 유지비 계산기·회원권 손익과 3종 내부링크 허브
- **타겟 사용자**: 새 취미 고민 중인 30~50대, 현재 다른 취미에서 골프 전환 검토자, 골퍼의 지출 정당화 니즈

## 4. 데이터 스키마

```ts
export type HobbyCategory = "outdoor" | "indoor" | "sport" | "leisure";

export interface HobbyCostEntry {
  id: string;
  name: string;               // "골프", "테니스"
  category: HobbyCategory;
  monthlyMin: number;         // 월 최소 비용
  monthlyMax: number;         // 월 최대 비용
  monthlyAvg: number;         // 월 평균 비용
  initialCost: number;        // 초기 투자비 (장비+가입비)
  annualCost: number;         // 연간 비용 (monthlyAvg × 12)
  fiveYearTotal: number;      // 5년 총비용 (initialCost + annualCost × 5)
  tenYearTotal: number;       // 10년 총비용
  socialScore: number;        // 인맥 형성 지수 1~5
  healthScore: number;        // 건강 효과 지수 1~5
  entryBarrier: "낮음" | "보통" | "높음"; // 진입 장벽
  peakSeason: string;         // 주요 시즌 ("연중" / "3~11월" 등)
  note: string;               // 특이사항
}
```

## 10. QA/리스크

- 비용 데이터는 "전국 평균" 기준임을 명시 (지역·수준별 편차 큼)
- 골프 비용이 다른 취미보다 항상 높게 나올 수 있어 골퍼 이탈 방지 위해 "그래도 골프를 하는 이유" 섹션 균형있게 구성
- 취미 간 비교는 순위 다툼 느낌 없이 중립적으로 ("각자의 가치에 따른 선택")
- 비용 데이터 출처 명시 → "2026년 시중 평균가 기준"

## 2. 기획 배경

아파트 관리비는 여름 전기요금, 겨울 난방비, 공용관리비 인상 이슈가 겹칠 때 검색 수요가 반복적으로 올라간다. 특히 사용자는 "우리집 관리비가 비싼 건지", "24평이면 보통 얼마가 나오는지", "전기요금이 많이 나온 건지 공용관리비가 문제인지"를 빠르게 알고 싶어 한다.

이 계산기는 관리비 고지서의 주요 항목을 입력하면 `평당 관리비`, `평균 대비 높음/보통/낮음`, `어느 항목이 높은지`를 보여주는 생활비 진단형 도구다. 정책성 데이터보다 사용자 입력 의존도가 높으므로 모든 비교 결과는 `추정`, `참고`로 표시한다.

기존 `/tools/aircon-electricity-cost/`와 연결성이 좋다. 여름에는 "에어컨 때문에 관리비가 올랐나?"라는 흐름으로 유입되고, 겨울에는 "난방비 때문에 관리비가 비싼가?"라는 흐름으로 확장할 수 있다.

## 6. 계산 로직

### 6-1. 핵심 계산식

```text
평당 총 관리비 = 총 관리비 / 평수
평당 공용관리비 = 공용관리비 / 평수
전기요금 비중 = 전기요금 / 총 관리비
수도요금 비중 = 수도요금 / 총 관리비
난방비 비중 = 난방비 / 총 관리비
공용관리비 비중 = 공용관리비 / 총 관리비
```

### 6-2. 적정성 판정

기준값은 구현 단계에서 최신 공개 통계와 내부 참고 구간으로 보정한다. 초안 판정 기준은 아래처럼 둔다.

| 평당 총 관리비 | 판정 | 문구 |
| ---: | --- | --- |
| 8,000원 미만 | 낮음 | 같은 평형대 대비 낮은 편입니다. 빠진 항목이 없는지 확인하세요. |
| 8,000~12,000원 | 보통 | 일반적인 참고 구간에 가깝습니다. |
| 12,000~16,000원 | 높음 | 같은 평형대 대비 높은 편일 수 있습니다. 항목별 비중을 확인하세요. |
| 16,000원 초과 | 매우 높음 | 계절 요금, 공용관리비, 장기수선충당금, 커뮤니티 비용을 나눠 확인해야 합니다. |

> 위 기준은 공식 평균이 아니라 비교계산소의 참고 판정 구간이다. 페이지에서는 반드시 `추정`, `참고` 배지를 표시한다.

### 6-3. 세대수 보정

| 세대수 | 해석 |
| --- | --- |
| 100세대 미만 | 경비·청소·시설비가 세대별로 크게 나뉘어 평당 공용비가 높아질 수 있음 |

## 10. UI/UX 방향

- 모바일 우선. 관리비 고지서를 보면서 바로 입력할 수 있게 숫자 입력을 크게 배치한다.
- 평형 프리셋 버튼: `18평`, `24평`, `32평`, `40평`
- 난방방식은 segmented control
- 계절은 segmented control
- 비용 입력은 만원 단위로 받되 내부 계산은 원 단위로 변환한다.
- 결과 판정은 색상만 의존하지 않고 `낮음`, `보통`, `높음`, `매우 높음` 텍스트를 함께 표시한다.
- 항목별 비중은 막대 그래프로 보여준다. Chart.js 없이 CSS bar로 충분하다.
- URL state를 지원해 고지서 조건을 공유할 수 있게 한다.

## 11. FAQ 초안

1. 아파트 관리비 평균은 어디서 확인하나요?
   - K-apt 공동주택관리정보시스템에서 단지별 관리비 공개 자료를 확인할 수 있습니다. 이 계산기는 사용자가 입력한 고지서 금액을 평수 기준으로 환산해 참고 구간과 비교합니다.

2. 24평 아파트 관리비는 얼마가 보통인가요?
   - 단지 세대수, 난방방식, 계절, 커뮤니티 시설, 주차비 포함 여부에 따라 차이가 큽니다. 그래서 이 계산기는 24평 기준 금액을 단정하지 않고 평당 관리비와 항목별 비중으로 판단합니다.

3. 관리비가 많이 나오는 가장 흔한 이유는 무엇인가요?
   - 여름에는 전기요금, 겨울에는 난방비 영향이 큽니다. 계절 요인이 아닌데도 높다면 공용관리비, 경비·청소비, 장기수선충당금, 커뮤니티 시설 비용을 확인해야 합니다.

4. 공용관리비는 어느 정도면 높은 편인가요?
   - 공용관리비는 세대수가 적거나 경비·청소 인력이 많거나 커뮤니티 시설이 큰 단지에서 높아질 수 있습니다. 계산기는 평당 공용관리비와 총 관리비 대비 비중을 함께 보여줍니다.

5. 장기수선충당금도 관리비에 포함해서 봐야 하나요?
   - 고지서 총액에는 포함되지만, 세입자는 퇴거 시 임대인에게 정산받을 수 있는 항목일 수 있습니다. 그래서 총 관리비 판정과 별도로 장기수선충당금 비중을 안내하는 것이 좋습니다.

6. 전기요금이 높게 나오면 어떻게 확인하나요?
   - 이 계산기에서 전기요금 비중이 높게 나오면 에어컨 전기요금 계산기로 연결해 사용 시간, 냉방 설정, 누진 구간을 따로 확인하도록 안내합니다.

# 삼성전자 vs SK하이닉스 2026~2028 실적·성과급 비교 — 기획 문서

## 1. 배경

- "TOP10 순위표는 뻔하다, 진짜 엣지는 돈 버는 구조를 계산해주는 콘텐츠다"라는 방향성 합의에 따라, 1순위 추천 콘텐츠로 **"삼성전자 vs SK하이닉스 2026 실적·성과급 비교"**를 기획.
- ⚠️ **중복 확인 결과**: `/reports/semiconductor-stocks-forecast-2026-2028/`(데이터: `src/data/semiconductorStocksForecast2026_2028.ts`)가 이미 삼성전자·SK하이닉스·마이크론·TSMC의 2026~2028년 매출·영업이익·영업이익률·밸류에이션 시나리오를 다루고 있음. **매출/영업이익 숫자 자체를 또 만들면 카니발라이제이션**이 발생하므로, 이 리포트는 기존 데이터를 그대로 재사용(import)하고 **"실적이 이러면 성과급은 얼마인가"라는 새로운 레이어**만 얹는 것으로 범위를 좁힌다.
- 이미 `samsungCompensation.ts`(OPI/TAI 시나리오, 잠정합의안 12% 구조), `skHynixCompensation.ts`(PS 멀티플라이어 2026/2027/2028 보수·기준·공격적 시나리오)가 존재 — 이 두 파일의 성과급 로직과 위 실적 데이터를 **연결하는 것**이 이번 리포트의 핵심 차별점.

## 2. 기존 자산 재사용 매핑

| 재사용 대상 | 위치 | 활용 방식 |
|---|---|---|
| 2026~2028 매출·영업이익·영업이익률(삼성전자, SK하이닉스) | `src/data/semiconductorStocksForecast2026_2028.ts` (`earningsRows`, companySlug "samsung-electronics"/"sk-hynix") | import해서 그대로 표시, 숫자 재생성 안 함 |
| 삼성전자 OPI/TAI 시나리오 (FY2026 컨센서스 297.5조, 2026 300조 시나리오, FY2027 슈퍼사이클 317조 / 잠정합의안 12%) | `src/data/samsungCompensation.ts` (`operatingProfitScenarios`, `unionDemandScenarios`, `scenarioPresets`) | 영업이익 시나리오별 OPI/TAI 환산액 그대로 인용 |
| SK하이닉스 PS 멀티플라이어 (2026 24~33%, 2027 54~66%, 2028 58~70%) | `src/data/skHynixCompensation.ts` (`psMultipliersByYear`, `scenarioOptions`, `factAnchors`) | 영업이익 시나리오별 PS 환산액 그대로 인용 |
| 직급별 기준급 프리셋(사원~부장) | 양쪽 compensation 파일의 `rankPresets` | 직급 선택 시 실수령 환산에 재사용 |

## 3. 컨셉

- "매출이 더 큰 회사"(삼성전자)와 "이익률이 더 높은 회사"(SK하이닉스)를 같은 화면에서 비교하되, 핵심은 **"내 직급이면 올해·내년·내후년 성과급이 얼마나 차이 나는가"**로 귀결시키는 것.
- 단순 실적 비교 리포트가 아니라, 기존 `samsung-bonus`, `sk-hynix-bonus` 계산기로 자연스럽게 이어지는 **허브+계산 다리** 역할.
- "그래서 내 성과급 얼마?"라는 사용자의 진짜 질문에 답하는 구조로 설계 — 실적 표는 근거 자료로, 성과급 환산은 메인 콘텐츠로.

## 4. 타겟 검색 의도

| 키워드 그룹 | 예시 |

| 성과급 연동 | 영업이익 늘면 성과급 얼마, 삼성전자 하이닉스 성과급 비교 2026 |
| 미래 전망 | 2027 반도체 성과급 전망, 2028 삼성전자 성과급 전망 |
| 직장인 관심 | 삼성전자 SK하이닉스 어디가 성과급 더 줄까 |

## 5. 입력값 / 출력값

### 입력값
- 회사 선택(삼성전자 / SK하이닉스, 또는 동시 비교)
- 연도 선택(2026 / 2027 / 2028)
- 시나리오 선택(보수적 / 기준 / 공격적) — 기존 두 compensation 파일의 시나리오 코드 그대로 사용
- 직급 선택(사원~부장) — 기준급 환산용

### 출력값
- 연도별 매출·영업이익·영업이익률 비교 카드 (기존 forecast 데이터 인용)
- 시나리오별 성과급 환산표 (OPI/TAI vs PS, 직급별 예상 수령액)
- "올해 대비 증가액" 비교 (예: 2026 기준급 1억원 사원이 2027년에 받는 성과급 변화)
- 두 회사 동시 비교 패널 ("이 직급·이 시나리오라면 어디가 더 받을까")

## 6. 올해·내년·내후년 목표치 (기존 데이터 기준, 신규 추정 없음)

### 매출·영업이익 (출처: `semiconductorStocksForecast2026_2028.ts`, 2026-05-31 기준 컨센서스·시나리오 재구성)

| 연도 | 삼성전자 매출 | 삼성전자 영업이익(영업이익률) | SK하이닉스 매출 | SK하이닉스 영업이익(영업이익률) |
|---|---|---|---|---|
| 2026E | 585조원 | 145조원 (24.8%) | 300조원 | 228조원 (76.0%) |
| 2027E | 635조원 | 165조원 (26.0%) | 365조원 | 284조원 (77.8%) |
| 2028E | 660조원 | 150조원 (22.7%) | 370조원 | 265조원 (71.6%) |

> 주의: 위 표는 `semiconductorStocksForecast2026_2028.ts`의 컨센서스·시나리오 값을 그대로 인용한 것이며, 별도 웹 검색으로 새 숫자를 추정하지 않았다(중복 방지 원칙). 페이지 구현 시 반드시 원본 파일 값을 import해서 표시하고, 이 문서의 표는 참고용 캡처본으로만 취급한다.

### 성과급 시나리오

**삼성전자 OPI/TAI** (출처: `samsungCompensation.ts`)
- 2026 컨센서스 영업이익 297.5조원 기준 / 2026 300조 시나리오 / 2027 슈퍼사이클 317조원 시나리오
- 2026년 잠정합의안: 총 재원 최대 12%(기존 OPI 1.5% + DS 특별경영성과급 10.5%), DS 배분 구조 40:60

**SK하이닉스 PS** (출처: `skHynixCompensation.ts`)
- 2026: 보수 24% / 기준 29.64%(2026 실제 지급률) / 공격적 33%
- 2027: 보수 54% / 기준 60% / 공격적 66%
- 2028: 보수 58% / 기준 64% / 공격적 70%

이 두 시나리오 세트를 그대로 가져와서 "영업이익 컨센서스가 맞으면 PS는 기준 시나리오에 가깝게 움직일 가능성이 크다"는 식의 **연동 해석**을 추가하는 것이 이번 리포트의 신규 콘텐츠다.

## 8. 리스크

- **중복 콘텐츠 리스크(가장 중요)**: 매출/영업이익 표를 독자적으로 다시 만들면 기존 `semiconductor-stocks-forecast-2026-2028`와 직접 경쟁하게 됨 → 반드시 import로 재사용, 신규 수치 생성 금지.
- **성과급 시나리오의 확정값 오인**: PS/OPI 시나리오는 추정이며, 실제 노사 협의 결과와 다를 수 있음 — 매 시나리오 카드에 "추정/시나리오" 라벨 필수.
- **두 회사 비교가 "어디가 더 좋다"는 투자/이직 추천처럼 읽힐 위험**: 성과급 체감 비교는 사실 기반 정보 제공이며 이직·투자 추천이 아니라는 점을 반복 고지.
- **데이터 갱신 동기화**: 원본 두 compensation 파일이나 forecast 파일이 갱신되면 이 리포트도 함께 갱신해야 함 — 단순 표시이므로 import 구조 유지 시 자동 동기화되나, 텍스트 해석 문구는 별도 점검 필요.

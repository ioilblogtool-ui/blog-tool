# 신생아 초기 용품 풀세트 비용 계산기 기획 문서

작성일: 2026-06-27

## 1. 배경 및 목적

신생아 초기 용품은 출산 전 한 번에 큰돈이 나가는 대표 지출입니다. 유모차, 카시트, 아기침대, 젖병 세척기, 젖병소독기, 아기띠처럼 가격 차이가 큰 품목이 많고, 가성비 제품을 고르는지 프리미엄 브랜드를 고르는지에 따라 총액이 수백만 원 이상 벌어질 수 있습니다.

기존 비교계산소의 육아 콘텐츠는 분유비, 기저귀비, 출산지원금, 첫 1년 육아비처럼 반복 지출과 지원금 중심의 계산기가 많습니다. 이번 콘텐츠는 출산 전후 초기 장비 구매 비용을 계산하는 입구 페이지로 만들고, 결과 하단에서 자연스럽게 "첫 1년 육아비" 가이드와 분유비/기저귀비 계산기로 이어지게 하는 것이 핵심입니다.

이 페이지의 목적은 다음과 같습니다.

- 출산준비물 비용을 한 번에 계산하는 검색 진입 페이지 확보
- 가성비/중급/프리미엄 등급별 총액 차이를 직관적으로 보여주기
- 필수템과 선택템을 분리해 과소비를 줄이는 체크리스트 제공
- 첫 1년 육아비 리포트, 분유비 계산기, 기저귀비 계산기로 내부 이동 유도
- 가격 변동이 큰 쇼핑 정보는 "참고 추정값"으로 표기하고 사용자가 직접 조정하게 설계

## 2. 콘텐츠 한 줄 정의

신생아 출산 전 준비해야 할 카시트, 유모차, 아기침대, 젖병 세척기, 수유용품, 목욕용품, 의류·침구 스타터 세트의 비용을 가성비/중급/프리미엄 등급별로 합산해 보여주는 계산기입니다.

## 3. 타깃 사용자

### 3.1 핵심 사용자

- 출산 예정일을 앞두고 신생아 용품 예산을 잡는 예비 부모
- 첫째 출산이라 어떤 물건을 반드시 사야 하는지 모르는 사용자
- 유모차와 카시트 같은 고가 품목의 등급별 예산 차이를 알고 싶은 사용자
- 출산지원금이나 부모급여를 초기 용품 구매에 얼마나 쓸 수 있는지 계산하려는 사용자

### 3.2 확장 사용자

- 둘째 이상 출산으로 기존 용품 재사용률을 반영하고 싶은 사용자
- 중고/렌탈/선물 수령을 반영해 실부담액을 줄이고 싶은 사용자
- 첫 1년 육아비를 보기 전에 초기 준비 비용만 먼저 알고 싶은 사용자

## 9. 계산 로직

### 9.1 기본 계산식

```text
품목별 금액 = 선택 여부 × 입력 가격
기본 합계 = 필수템 합계 + 선택템 합계
재사용 절감액 = 재사용 가능 품목 합계 × 재사용률
중고·렌탈 절감액 = 중고·렌탈 선택 품목 합계 × 절감률
총 준비 비용 = 기본 합계 - 재사용 절감액 - 중고·렌탈 절감액
실부담액 = 총 준비 비용 - 선물·축하금 반영액 - 지원금 사용액
```

### 9.2 구매 방식별 절감률

| 구매 방식 | 기본 절감률 | 적용 방향 |
|---|---:|---|
| 새 제품 위주 | 0% | 모든 품목을 입력 가격 그대로 반영 |
| 새 제품+중고 혼합 | 15% | 중고 가능 품목에만 절감률 적용 |
| 렌탈 일부 활용 | 20% | 단기 사용 품목 중심으로 절감률 적용 |
| 직접 입력 | 사용자 입력 | 사용자가 직접 절감액 조정 |

### 9.3 둘째 이상 재사용 기본값

| 출산 순서 | 재사용률 기본값 | 설명 |
|---|---:|---|
| 첫째 | 0% | 대부분 신규 구매로 가정 |
| 둘째 이상 | 30% | 침구, 의류, 수납, 일부 편의용품 재사용 가정 |

카시트는 안전 이슈가 있으므로 재사용 가능 품목으로 넣더라도 안내 문구를 별도로 둡니다. 사고 이력, 사용기한, 인증 상태를 확인하지 못하면 비용 절감보다 안전 리스크가 더 큽니다.

## 13. FAQ 기획

### Q1. 신생아 용품 준비 비용은 보통 얼마 정도 잡아야 하나요?

가성비 필수형은 약 120만~220만 원, 중급 표준형은 약 350만~600만 원, 프리미엄 풀세트는 800만 원 이상까지도 올라갈 수 있습니다. 다만 이는 예산 시뮬레이션용 참고 범위이며 실제 금액은 브랜드, 행사, 중고·선물 활용 여부에 따라 달라집니다.

### Q2. 출산 전에 반드시 사야 하는 품목은 무엇인가요?

퇴원과 이동에 필요한 카시트, 기본 의류, 기저귀, 체온계, 목욕용품, 수유 방식에 맞는 최소한의 젖병·수유용품은 출산 전에 준비하는 편이 좋습니다. 유모차, 바운서, 아기모니터 등은 생활 패턴에 따라 출산 후 구매해도 됩니다.

### Q3. 카시트와 유모차는 꼭 프리미엄으로 사야 하나요?

반드시 프리미엄이어야 하는 것은 아닙니다. 카시트는 가격보다 안전 인증, 설치 방식, 아이 체형 적합성이 중요하고, 유모차는 주거 환경, 엘리베이터, 차량 트렁크, 외출 빈도에 따라 적합한 등급이 달라집니다.

### Q4. 중고로 사도 되는 품목과 피해야 하는 품목은 무엇인가요?

바운서, 모빌, 수납용품, 일부 의류처럼 사용 기간이 짧고 상태 확인이 쉬운 품목은 중고 활용 여지가 있습니다. 카시트는 사고 이력, 사용기한, 인증 상태를 확인할 수 없다면 중고 구매에 신중해야 합니다.

### Q5. 젖병세척기와 젖병소독기는 둘 다 필요한가요?

가정의 수유 방식과 생활 패턴에 따라 다릅니다. 분유 수유 비중이 높고 젖병 사용량이 많다면 편의성이 크지만, 모유 수유 중심이거나 젖병 사용량이 적다면 우선 최소 구성으로 시작한 뒤 필요할 때 추가해도 됩니다.

### Q6. 둘째 출산이면 준비 비용이 얼마나 줄어드나요?

기존 용품 상태가 좋다면 의류, 침구, 수납, 일부 편의용품을 재사용해 20~50% 정도 줄일 수 있습니다. 다만 안전이나 위생이 중요한 품목은 상태를 확인하고 교체 여부를 판단해야 합니다.

### Q7. 이 계산기와 첫 1년 육아비 가이드는 어떻게 다른가요?

이 계산기는 출산 전후 한 번에 구매하는 초기 용품비를 계산합니다. 첫 1년 육아비 가이드는 분유, 기저귀, 병원비, 이유식, 돌잔치처럼 매달 또는 시기별로 나가는 비용까지 포함합니다.

## 19. 구현 요청 프롬프트 초안

신생아 초기 용품 풀세트 비용 계산기를 구현해줘.

추천 URL은 `/tools/newborn-starter-kit-cost/`이고, 가성비/중급/프리미엄 프리셋으로 카시트, 유모차, 아기침대, 젖병·수유용품, 젖병세척기·소독기, 아기띠, 아기욕조, 의류·침구, 체온계·위생용품, 바운서·모빌, 아기모니터·편의가전 비용을 합산하는 구조야.

필수 요구사항:

- 모든 가격은 참고 추정값으로 표기
- 사용자가 품목 포함 여부와 가격을 직접 수정 가능
- 출산 순서, 구매 방식, 재사용률, 선물·지원금 반영액 입력 가능
- 결과는 예상 초기 준비 비용, 실부담액, 필수템 합계, 선택템 합계, 절감액으로 표시
- 구매 우선순위를 출산 전 준비/출산 후 구매/중고·렌탈 검토/새 제품 권장으로 분류
- 첫 1년 육아비 가이드, 분유비 계산기, 기저귀비 계산기, 출산지원금 계산기로 내부 링크 연결
- `src/data/tools.ts`, `public/sitemap.xml`, `src/styles/app.scss` 등록
- `npm run build` 성공 확인

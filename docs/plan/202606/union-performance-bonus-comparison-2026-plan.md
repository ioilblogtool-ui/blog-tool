# 2026 노조 성과급 요구 비교 리포트 기획 (현대차·삼성바이오로직스·카카오)

> 2026-06-26 작성. 사용자 제안 기획안을 비교계산소 리포트 형식으로 정리. **수치 미확정 — 작성 전 실제 보도자료 기준 검증 필요.**

## 1. 배경 및 목적

- **계기**: 현대차·삼성바이오로직스·카카오 3사에서 2026년 임단협/성과급 요구 이슈가 동시에 부각. 단순 뉴스 요약이 아니라 "요구액을 숫자로 환산"하는 비교계산소 색깔의 콘텐츠로 차별화
- **목적**:
  1. 노조 요구안(순이익/영업이익 대비 %)을 1인당 환산액으로 계산해 체감 가능한 숫자 제공
  2. `기존 성과급 계산기 클러스터`(`bonus-after-tax-calculator`, `samsung-bonus`, `sk-hynix-bonus` 등)와 내부링크로 연결해 트래픽 분산
  3. "현대차 성과급", "삼성바이오로직스 파업", "카카오 노조" 등 시의성 키워드 신규 확보
- **타겟 사용자**: 해당 기업 재직자/관심 투자자, 노사 이슈에 관심 있는 일반 검색 유입

## 2. 콘텐츠 성격 및 데이터 신뢰성 원칙 (필수 확인)

- **본 기획안의 모든 수치(순이익 30%, 영업이익 20%, 직원 수 등)는 작성 시점 기준 미검증** — 실제 게시 전 아래를 반드시 확인:
  - 각사 노조 요구안 원문/보도자료 날짜와 수치
  - 직원 수(전체/생산직/사무직 등 기준 명확화)
  - 회사 제시안 vs 노조 요구안 차이
- 1인당 환산액은 "단순 산술 추정"이며 실제 배분 방식과 다를 수 있음 → 모든 계산 결과에 `추정` 배지 필수
- 카카오는 "확정 요구 비율"이 없으므로 금액 환산 콘텐츠로 만들지 않고 갈등/보상체계 설명형으로 작성 (사용자 지침대로)

## 4. 본문 구조 (공통 템플릿)

### 현대차 / 삼성바이오로직스 (계산형 리포트)
```
핵심 요약 → 노조 요구안 정리 → (순이익|영업이익) × 요구율 = 총 재원
→ 총 재원 ÷ 직원 수 = 1인당 환산액 → 기존 지급률과 비교 → 실제 지급 가능성
→ 시장/생산 영향 → FAQ
```

### 카카오 (정성형 리포트)
```
핵심 요약 → 노조 요구 내용 → 갈등 핵심 → 파업투표/행동 단계
→ 고용안정 요구가 같이 나온 이유 → IT업계 성과급 갈등 배경 → FAQ
```

### 3사 비교 허브
```
핵심 요약 → 3사 요구안 비교표 → 회사별 섹션(각 리포트 링크) → 행동 단계 비교
→ 1인당 환산액 비교(현대차·삼바만) → 요구 현실화 가능성 → 투자자/근로자 관점 → FAQ
```

# 품종별 반려동물 10년 총비용 비교 기획

## 1. 배경 및 목적

- **계기**: "말티즈 키우는 비용", "골든리트리버 비용", "렉돌 고양이 비용" 등 품종별 검색 수요가 높음. 단순 월 양육비보다 "10년 키우면 얼마?"라는 총비용 관점 콘텐츠는 희소하고 공유율이 높음
- **목적**:
  1. 품종명 키워드("말티즈 비용", "포메라니안 양육비") SEO 확보 → 검색 진입점 다변화
  2. 월 양육비 계산기(`pet-monthly-cost-calculator`)와 연계되는 허브 역할
  3. "생각보다 비싸다" 임팩트 → 소셜 공유 유도
- **타겟 사용자**: 특정 품종 입양 검토 중인 예비 반려인, 현재 반려인의 비교 참고

---

## 2. 콘텐츠 성격

- 리포트형 비교 콘텐츠 (`/reports/` 계열)
- 강아지 TOP 8종 + 고양이 TOP 6종 = 14종 비교
- 탭 전환: 강아지 / 고양이
- 입력: 기준 연봉(선택) → 연봉 대비 비율 추가 표시 가능

## 4. 데이터 스키마

```ts
export type PetType = "dog" | "cat";

export interface BreedCostEntry {
  id: string;
  name: string;           // "말티즈", "골든리트리버"
  petType: PetType;
  size: "소형" | "중형" | "대형" | "묘종";
  avgLifespan: number;    // 평균 수명 (년)
  monthlyCostMin: number; // 월 최소 비용
  monthlyCostMax: number; // 월 최대 비용
  monthlyCostAvg: number; // 월 평균 비용
  medicalRisk: "낮음" | "보통" | "높음"; // 유전질환 빈도

  purchasePrice: number;  // 분양가 평균
  note: string;           // 품종 특이사항 1줄
}
```

## 8. QA/리스크

- 스코티시폴드 의료 이슈(관절 질환)는 중립적으로 "의료 비용이 높을 수 있음" 수준 표현
- 분양가 편차가 너무 커서 범위로 표기 (최소~최대), 단순 평균값 사용 지양
- 지역·시기별 물가 차이 → "참고용 평균" 명시 필수
- 품종 누락 민원 방지 → "이 외 품종은 월 양육비 계산기에서 직접 입력" 안내

---

## 1. 배경 및 목적

돌잔치는 결혼식보다 규모는 작지만, 실제 준비 항목은 생각보다 많다. 연회장 식대, 보증 인원, 스튜디오 촬영, 성장 영상, 한복 대여, 답례품, 사회자, 돌상, 헤어메이크업까지 합치면 처음 예상한 금액보다 커지기 쉽다. 사용자는 "돌잔치 비용이 평균 얼마인지", "호텔 돌잔치와 일반 연회장 차이가 얼마나 나는지", "스냅·한복·답례품까지 넣으면 총액이 얼마인지"를 빠르게 알고 싶어 한다.

이 계산기는 실제 견적을 대체하지 않고, **돌잔치 준비 전 예산 범위를 잡는 참고용 추정 도구**로 설계한다. 결혼 준비 예산 계산기의 `지역 + 등급 + 인원 + 항목별 직접 수정` 패턴을 재사용해 개발 효율을 높이고, 육아·출산 카테고리의 후속 비용 콘텐츠로 확장한다.

### 핵심 컨셉

> "우리 아이 돌잔치, 하객 수와 장소 등급까지 넣으면 총 얼마가 들까?"

### 입력 UX 메모

- `하객 수`와 `보증 인원`을 분리한다. 실제 결제 식대는 `max(하객 수, 보증 인원) × 1인 식대`로 계산해야 한다.
- 사용자가 처음 들어와도 바로 결과를 볼 수 있도록 지역·유형별 기본값을 자동 입력한다.
- 세부 항목은 결혼 예산 계산기처럼 "기본값 채우기 → 직접 수정" 흐름을 재사용한다.
- 모바일에서는 입력 항목을 `장소`, `촬영·의상`, `진행·답례품`, `분담` 탭으로 나누면 피로도가 낮다.

## 8. 계산 로직

> 모든 금액은 지역·업체·계약 시점·패키지 구성에 따라 달라지는 **참고용 추정값**으로 표시한다.

### 8-1. 등급별 기본 프리셋

| 유형 | 하객 규모 | 1인 식대 | 장소·패키지 | 부대비용 기본값 | 총액 감 |
|---|---:|---:|---:|---:|---:|
| 가족 식사 | 10~25명 | 4만~8만원 | 0~30만원 | 50만~120만원 | 100만~300만원 |
| 일반 연회장 | 40~80명 | 5만~8만원 | 30만~100만원 | 120만~250만원 | 300만~700만원 |
| 호텔·프리미엄 | 50~120명 | 9만~18만원 | 100만~300만원 | 250만~500만원 | 700만~1,800만원 |
| 셀프 파티룸 | 10~30명 | 2만~5만원 | 20만~80만원 | 50만~150만원 | 100만~350만원 |

### 8-2. 항목별 기본 비용 범위

| 항목 | 가성비 | 일반 | 프리미엄 |
|---|---:|---:|---:|
| 스튜디오 촬영 | 30만~50만원 | 60만~100만원 | 120만~200만원 |
| 행사 스냅 | 20만~40만원 | 50만~80만원 | 100만~150만원 |
| 아이 한복 대여 | 5만~10만원 | 10만~20만원 | 25만~40만원 |
| 부모 한복 대여 | 20만~40만원 | 40만~80만원 | 100만원 이상 |
| 돌상 | 20만~40만원 | 50만~100만원 | 120만~250만원 |
| 답례품 | 1개 2천~4천원 | 4천~8천원 | 8천~1.5만원 |
| 성장 영상 | 0~10만원 | 10만~25만원 | 30만~50만원 |
| 사회자·이벤트 | 0~20만원 | 20만~50만원 | 60만~100만원 |
| 헤어메이크업 | 10만~20만원 | 20만~40만원 | 50만~80만원 |

### 8-3. 기본 수식

```text
식대 기준 인원 = max(하객 수, 보증 인원)
식대 합계 = 식대 기준 인원 × 1인 식대

답례품 합계 = 답례품 단가 × 답례품 수량

부대비용 합계 =
  스튜디오 촬영
  + 행사 스냅
  + 한복·의상
  + 돌상
  + 답례품
  + 사회자·이벤트
  + 성장 영상
  + 헤어메이크업
  + 기타 직접 입력

예비비 = (식대 합계 + 부대비용 합계) × 예비비율

총비용 = 식대 합계 + 장소 대관·패키지 + 부대비용 합계 + 예비비

순부담액 = 총비용 - 예상 축의금

1인당 실비 = 총비용 ÷ max(하객 수, 1)
```

### 8-4. 지역 보정

| 지역 | 보정 방향 |
|---|---|
| 서울 | 식대·장소 패키지 +10~25% |
| 수도권 | 기준값 |
| 광역시 | -5~+5% |
| 지방 | -10~0% |

## 9. 데이터 기준 및 표기 원칙

- 식대·촬영·한복·돌상 비용은 업체별 편차가 크므로 `추정` 또는 `시뮬레이션` 배지를 표시한다.
- 공식 통계가 아닌 가격 범위는 "평균"이라고 단정하지 않고 "예상 범위", "참고 견적", "프리셋"으로 표현한다.
- 특정 업체명을 직접 추천하지 않는다.
- 제휴 링크가 들어갈 경우 답례품, 돌상 소품, 성장 영상 제작, 한복 대여 등으로 제한하고 광고 고지를 명확히 표시한다.
- 실제 계약 전에는 포함 항목, 보증 인원, 취소 위약금, 부가세, 봉사료, 주차 지원을 확인하도록 안내한다.

## 10. UI/인터랙션 설계

### 10-1. 첫 화면

- H1: `돌잔치 비용 비교 계산기`
- 서브카피: `스튜디오 촬영, 한복 대여, 돌상, 답례품, 연회장 식대까지 한 번에 계산해보세요.`
- 상단 요약 카드 3개:
  - `일반 연회장 기준`
  - `보증 인원 반영`
  - `촬영·한복 옵션 포함`

### 10-2. 입력 영역

- `지역`과 `돌잔치 유형`은 세그먼트 버튼으로 제공한다.
- `하객 수`, `보증 인원`, `1인 식대`는 숫자 입력 + 빠른 선택 버튼을 제공한다.
- 옵션 항목은 토글을 켜면 금액 입력이 열리는 구조로 만든다.

### 10-3. 결과 영역

- 총비용 카드와 순부담액 카드를 가장 크게 배치한다.
- 비용 분해는 막대 그래프 또는 가로 비중 바를 사용한다.
- `소규모 / 일반 / 프리미엄` 비교 카드를 나란히 보여준다.
- "식대가 전체의 70% 이상"처럼 비중 기반 해석 문구를 자동 출력한다.

### 10-4. 공유 기능

- URL 파라미터로 입력값을 저장한다.
- 공유 문구 예시:

```text
우리 돌잔치 예상 비용은 약 420만원입니다. 식대 390만원, 부대비용 120만원 기준으로 계산했어요.
```

### intro 4단락 방향

1. 돌잔치 비용은 식대뿐 아니라 촬영, 의상, 돌상, 답례품이 함께 붙어 예상보다 커진다는 맥락을 설명한다.
2. 계산기는 보증 인원과 실제 하객 수 중 큰 값을 식대 기준으로 삼고, 선택 옵션을 더해 총예산을 추정한다고 설명한다.
3. 결과는 총액만 보지 말고 식대 비중, 부대비용 비중, 순부담액을 함께 봐야 한다고 안내한다.
4. 지역·업체·시즌·패키지 구성에 따라 실제 견적이 달라지므로 계약 전 세부 포함 항목 확인이 필요하다고 명시한다.

### inputPoints

- 하객 수와 보증 인원을 기준으로 식대 합계를 계산한다.
- 스튜디오 촬영, 한복 대여, 돌상, 답례품 등 선택 항목을 더한다.
- 소규모·일반·프리미엄 돌잔치 예산을 비교한다.

### criteria

- 모든 비용은 참고용 추정값이다.
- 식대는 보증 인원과 실제 하객 수 중 큰 값을 기준으로 계산한다.
- 답례품은 단가와 수량을 직접 조정할 수 있다.
- 실제 견적에는 부가세, 봉사료, 주차, 취소 위약금이 별도로 붙을 수 있다.

- 신생아~돌 육아비용 가이드 하단
- 분유비·기저귀비 계산기 하단
- 출산지원금/육아지원금 계산기 하단
- 결혼 준비 예산 계산기 하단의 가족 행사 확장 링크

### 나가는 링크

- 육아·출산 비용 비교 허브
- 출산지원금 계산기
- 기저귀값 계산기
- 분유값 계산기
- 결혼 준비 예산 계산기

### 카테고리 포지션

돌잔치 비용 비교는 `육아·출산 비용 비교` 카테고리에서 분유·기저귀·산후조리원 다음 단계의 이벤트성 지출 콘텐츠로 배치한다. 결혼 카테고리의 예산 계산기와 UI/계산 구조가 비슷하므로, 사이트 내부에서는 "가족 행사 예산 계산" 시리즈로 묶을 수 있다.

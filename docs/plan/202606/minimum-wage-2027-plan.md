---

## 3. 핵심 컨셉: "내 시급, 세계에서 얼마짜리?"

### 3가지 앵글

**① 글로벌 최저임금 순위**
- 주요국 최저임금을 달러 PPP 기준으로 환산해 순위 비교
- 한국이 OECD 몇 위인지 시각화

**② 빅맥 지수 (구매력 비교)**
- "시간당 빅맥 몇 개 살 수 있나?"
- 나라별 최저임금 / 해당국 빅맥 가격 = 구매력 직관화
- 빅맥 외: 버스 1회, 아메리카노 1잔, 스타벅스 톨사이즈 등 선택 가능

**③ 2027 실수령 계산기**
- 시급 입력 → 일급/주급/월급/연봉 자동 계산
- 4대보험·소득세 공제 후 실수령
- 주 15시간 미만(주휴수당 없음) vs 이상(주휴수당 포함) 분기

---

## 4. 입력값

### 계산기 섹션
| 입력 | 기본값 | 범위 |
|------|--------|------|
| 시급 | 2027년 발표값 (발표 전: 2026년 10,030원) | 9,000~30,000 |
| 하루 근무시간 | 8시간 | 1~12 |
| 주 근무일 | 5일 | 1~7 |
| 주휴수당 포함 여부 | 포함 (주 15시간 이상) | 토글 |

### 글로벌 비교 섹션
| 입력 | 기본값 |
|------|--------|
| 비교 기준 | 달러 환산 / PPP 환산 선택 |
| 비교 품목 | 빅맥 / 아메리카노 / 지하철 1회 선택 |

---

## 5. 출력값

### 계산기 결과
- 일급 / 주급 / 월급 (주휴수당 포함/제외)
- 세후 실수령 월급
- 연봉 환산

### 글로벌 비교 결과
- 한국 최저임금 OECD 순위 카드
- 나라별 "빅맥 구매 가능 개수" 막대 차트
- 주요국 최저임금 비교표 (달러/PPP 전환 가능)

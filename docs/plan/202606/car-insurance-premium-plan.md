# 자동차 보험료 계산기 + 보험사 비교 기획 문서

> 작성일: 2026-06-18  
> 연계 페이지: 자동차 3종 계산기 (신차vs중고차, 리스vs할부vs현금, 전기차vs내연기관)

---

## 1. 배경 및 목적

자동차 보험료는 매년 갱신 시 "왜 올랐지?" 또는 "할인받을 방법이 없나?" 를 검색하는 수요가 매우 높다. 단순 보험료 계산기가 아니라 **"내가 놓치고 있는 할인 항목을 찾아주는 도구"** 컨셉으로 차별화한다.

### 핵심 컨셉: "내 보험료, 이 할인 다 받고 있어?"

---

## 2. 타겟 사용자

- 갱신 시기가 다가온 차주 (연간 1회 확인)
- 신차 구매 후 첫 보험 가입자
- 아기·어린이 자녀가 생긴 부모 (자녀 할인 몰라서 못 받는 경우 많음)
- 블랙박스·마일리지 특약 미가입자
- 보험사를 매년 같은 곳에서 갱신하는 사람

---

## 4. 섹션1 — 기본 보험료 추정 계산기

### 입력값

| 입력 | 설명 | 기본값 |
|------|------|--------|
| 차량 연식 | 연식이 오래될수록 차량가 낮아져 자차 보험료 감소 | 3년 |
| 차량 가격대 | 1,000만원 단위 선택 | 3,000만원 |
| 운전자 나이 | 만 나이 | 35세 |
| 운전 경력 | 년 | 10년 |
| 전년도 사고 이력 | 무사고 / 1건 / 2건 이상 | 무사고 |
| 연간 주행거리 | km | 15,000km |
| 담보 선택 | 대인+대물 필수 / 자차 포함 | 자차 포함 |
| 운전자 범위 | 본인만 / 부부 / 가족 / 누구나 | 부부 |

### 출력값

- 예상 보험료 범위 (최저~최고 추정)
- 현재 적용 가능한 주요 할인 항목 리스트 자동 표시
- "이 조건이면 약 XX만원~XX만원 예상"

> **주의**: 실제 보험료는 보험사·차종·지역 등에 따라 다름. 추정 참고용.

## 5. 섹션2 — 놓치기 쉬운 할인 항목 체크리스트

사용자가 체크하면 할인 금액 자동 계산.

### 🔵 운전 습관·환경 할인

| 할인 항목 | 할인율 | 설명 |
|-----------|--------|------|
| 블랙박스 특약 | 3~5% | 대부분 보험사 제공, 미가입자 많음 |
| 마일리지 할인 | 최대 35% | 연 3,000km 이하 시 최대 할인 |
| 자동차 안전장치 | 1~3% | ABS·에어백·차선이탈경고 장착 시 |
| 심야운전 제한 특약 | 3~7% | 자정~새벽 5시 미운전 서약 |
| 대중교통 할인 | 5~10% | 대중교통 정기권 소지자 |

### 🟢 가입자 특성 할인

| 할인 항목 | 할인율 | 설명 |
|-----------|--------|------|
| 무사고 할인 | 최대 60% | 무사고 연수별 누적 적용 |
| 우수회원 할인 | 3~5% | 동일 보험사 장기 유지 시 |
| 단체계약 할인 | 5~10% | 직장·단체 경유 가입 시 |
| 인터넷 가입 할인 | 15~20% | 다이렉트 가입 시 설계사 수수료 제외 |
| 신규 가입 이벤트 | 5~10% | 보험사 전환 시 |

## 6. 섹션3 — 자녀·가족 특화 할인 (핵심 차별화)

**"아기 생기면 자동차 보험도 싸진다"** — 모르는 사람이 많음.

### 자녀 관련 할인 항목

| 할인 항목 | 할인율 | 조건 | 제공 보험사 |
|-----------|--------|------|------------|
| 영유아 자녀 할인 | 5~7% | 만 6세 이하 자녀 보유 | 삼성화재, DB손보, KB손보 등 |
| 다자녀 할인 | 7~10% | 자녀 2명 이상 | 대부분 보험사 |
| 출산 축하 할인 | 5% | 당해연도 출산 시 | 일부 보험사 |
| 어린이집 카시트 특약 | 보험료 포함 | 영유아 탑승 시 카시트 의무 관련 | 일부 보험사 |
| 임산부 할인 | 3~5% | 임신 중 가입 시 | 일부 보험사 |

### 계산 예시 (시각화)

```
조건: 만 35세, 무사고 5년, 연 15,000km, 차량가 3,000만원

기본 보험료:        약 85만원
블랙박스 할인 (-5%): -4.3만원
마일리지 할인 (-10%): -8.5만원
영유아 자녀 할인 (-6%): -5.1만원
인터넷 가입 (-15%): -12.8만원
─────────────────────
적용 후 예상:       약 54만원 (36% 절약)
```

## 7. 섹션4 — 주요 보험사 비교표

### 비교 기준

| 보험사 | 시장점유율 | 다이렉트 | 특화 강점 | 주요 특징 |
|--------|-----------|---------|----------|----------|
| 삼성화재 | 약 27% | ✅ | 브랜드 신뢰·네트워크 | 업계 1위, 긴급출동 전국망 최강, 합리적 보상 |
| DB손해보험 | 약 20% | ✅ | 합리적 가격 | 다이렉트 가격 경쟁력, 어플 편리 |
| KB손해보험 | 약 13% | ✅ | 자녀·가족 할인 | KB금융그룹 연계, 다자녀 할인 강점 |
| 현대해상 | 약 16% | ✅ | 운전자 특화 | Hi-Car 앱 편리, 마일리지 할인 강점 |
| 메리츠화재 | 약 11% | ✅ | 보상 빠름 | 보상 처리 속도 강점, 다이렉트 저렴 |
| 한화손해보험 | 약 7% | ✅ | 가격 경쟁력 | 보험료 저렴한 편, 중소형차 유리 |
| 롯데손해보험 | 약 4% | ✅ | 롯데 멤버십 연계 | 롯데포인트 적립, 롯데카드 할인 연계 |

### 선택 가이드

| 상황 | 추천 보험사 | 이유 |
|------|-----------|------|
| 무조건 저렴하게 | DB손보·메리츠·한화 다이렉트 | 다이렉트 보험료 낮음 |
| 아기·어린이 있는 가족 | KB손보·삼성화재 | 자녀 할인 체계적 |
| 보상 빠른 곳 | 메리츠화재 | 보상 처리 만족도 높음 |
| 긴급출동 중요 | 삼성화재 | 전국 네트워크 1위 |
| 마일리지 할인 극대화 | 현대해상 | Hi-Car 마일리지 정확도 높음 |
| 법인·사업자 차량 | 삼성화재·DB손보 | 법인 담당 네트워크 강점 |

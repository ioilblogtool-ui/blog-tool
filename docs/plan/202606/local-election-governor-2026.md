## 2. 기획 배경

2026년 6·3 지방선거에서 더불어민주당이 16개 광역단체장 중 13곳을 석권하며 압승했다. 선거 결과에 대한 관심은 투표 당일부터 이후 수일간 최고조에 달하며, "우리 지역 당선자가 누구인지", "어떤 공약을 내걸었는지"에 대한 검색 수요가 급증한다.

기존 선거 결과 콘텐츠는 뉴스 기사 형식(텍스트 나열)에 그쳐 한눈에 보기 어렵다. 비교계산소는 **인터랙티브 지도 + 당선자 프로필 + 공약 카드** 형식으로 차별화된 정보 소비 경험을 제공한다.

특히 이번 선거는 이재명 정부 1년 차 중간평가 성격이 강하고, 민주당의 지방권력 회복이라는 역사적 맥락이 있어 콘텐츠 수명이 길다. 공약 이행 여부 추적 업데이트를 통해 장기적 유입도 기대할 수 있다.

### C. 인터랙티브 지도 ← 핵심

**구현 방식**
- 한국 SVG 지도 (16개 시도 분리된 path)
- 정당별 색상 코딩

```
더불어민주당  →  #0064FF (진파랑)
국민의힘      →  #E61E2B (빨강)
무소속        →  #888888 (회색)
```

**인터랙션**
1. 지도 위 hover → 시도명 + 당선자명 툴팁 표시
2. 클릭 → 해당 시도 하이라이트 + 하단 패널 오픈
3. 패널 닫기 → 지도 초기 상태 복귀
4. 모바일 → 탭 형식으로 전환 (지도 → 리스트)

│  나이       68세 (1957년생)                │
│  득표율     54.2% (상대 45.8%)            │

├──────────────────────────────────────────┤
│  핵심 공약                                 │
│                                          │
│  🏠 주거  GTX 노선 확장, 청년주택 10만 호   │
│  💼 경제  경기북부 특구, 반도체 클러스터     │
│  🚌 교통  제2경인선, 광역버스 증편          │
│  📚 교육  AI교육 인프라, 학교폭력 근절      │

**공약 표시 원칙**
- 카테고리 5개 고정: 주거/경제/교통/교육/복지
- 카테고리별 공약 2~4개 (선관위 공약집 기준)
- 핵심 한 줄 + 상세 설명 접기/펼치기

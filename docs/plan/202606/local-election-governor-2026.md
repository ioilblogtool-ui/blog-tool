## 2. 기획 배경 및 차별화 포인트

### 2-1. 왜 이 콘텐츠인가

2026년 6·3 지방선거에서 더불어민주당이 16개 광역단체장 중 13곳을 차지하며 압승했다. 이번 선거는 이재명 정부 출범 약 1년 시점의 첫 번째 전국 단위 선거로, 사실상 중간평가 성격을 가진다.

선거 직후 유권자들이 가장 많이 검색하는 정보는:
1. **"우리 지역 당선자가 누구인지"** — 결과 확인
2. **"당선자가 어떤 공약을 내세웠는지"** — 공약 확인
3. **"득표 차이가 얼마였는지"** — 경쟁 강도 확인
4. **"당선자 이력이 어떻게 되는지"** — 인물 파악

현재 뉴스 미디어가 제공하는 정보는 기사형(텍스트 나열)이라 한눈에 보기 어렵다. 비교계산소는 **SVG 클릭 지도 + 당선자 프로필 카드 + 공약 카테고리 카드** 형식으로 기존 미디어와 완전히 다른 정보 소비 경험을 제공한다.

### 2-2. 콘텐츠 수명

- **단기 (D+0~7)**: 선거 결과 검색 최고조 — 빠른 오픈이 핵심
- **중기 (D+1개월~6개월)**: 공약 이행 초기 추적
- **장기 (임기 중)**: "○○ 지사 공약 이행" 검색 지속 유입

### 2-3. 경쟁 콘텐츠 분석

| 경쟁 채널 | 약점 | 비교계산소 강점 |
| --- | --- | --- |
| 뉴스 기사 | 지역별 기사 분산, 한눈에 못 봄 | 전국 16개 지역 한 페이지 통합 |
| 나무위키 | 공약 정리 없음, 디자인 없음 | 공약 카드 + 지도 시각화 |
| 선관위 공약마당 | UX 열악, 모바일 불편 | 인터랙티브 지도 + 모바일 최적화 |
| 정당 공식 사이트 | 당 편향, 상대 후보 정보 없음 | 전 당선자 중립 정리 |

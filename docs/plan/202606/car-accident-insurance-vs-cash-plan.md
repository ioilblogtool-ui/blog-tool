# 자동차 사고 보험처리 vs 현금처리 계산기 기획 문서

> 작성일: 2026-06-19  
> slug: `car-accident-insurance-vs-cash-calculator`  
> 연계 페이지: 자동차 보험료 계산기, 신차 vs 중고차, 리스 vs 할부 vs 현금

---

## 1. 배경 및 목적

자동차 보험료 계산기는 "갱신 시 한 번" 보는 도구다. 반면 **"사고 났을 때 보험처리 vs 현금처리 어느 게 유리해?"** 는 실제 사고 직후 즉각적인 검색 수요가 발생한다. 검색 의도(Intent)가 강하고 광고 단가가 높은 키워드 군이다.

핵심 컨셉: **"수리비 얼마부터 보험처리가 유리할까? 3년치 보험료 변화까지 계산해드립니다"**

기존 자동차 보험료 계산기와 완전히 다른 사용자 맥락:
- 보험료 계산기: 갱신 전 미리 비교
- 이 계산기: **사고 직후 즉각 판단** → 검색 의도 훨씬 강함

---

## 2. 타겟 사용자

- 경미한 접촉사고 후 "보험 써야 하나 말아야 하나" 고민 중인 운전자
- 수리비 견적 받고 현금처리 제안받은 사람
- 다음 갱신 할증이 두려워 현금처리 고민하는 사람
- 무사고 할인 유지가 중요한 장기 무사고 운전자
- 보험 등급 보호를 위해 현금처리가 유리한지 계산하고 싶은 사람

---

## 3. 핵심 계산 로직

### 현금처리 시 비용
```
현금처리 총비용 = 수리비 전액 (본인 부담)
할증 없음 → 향후 보험료 유지
```

### 보험처리 시 비용 (3년 기준)
```
보험처리 즉시 비용 = 자기부담금
향후 보험료 증가 = 현재 보험료 × 할증률 × 할증 유지 기간(기본 3년)
3년 총추가비용 = 자기부담금 + (연 할증금액 × 3)
```

### 판단 기준
```
보험처리 유리: 수리비 > 3년 총추가비용
현금처리 유리: 수리비 ≤ 3년 총추가비용
손익분기 수리비 = 자기부담금 + (연 할증금액 × 3)
```

### 사고점수 반영 (선택)
- 사고점수에 따라 할증률이 달라지므로 선택 입력
- 10점 미만 소손해: 할증 없는 경우도 있음 (소손해 처리 활용)
- 사고 이력 누적 시 갱신 거절 리스크 안내

## 4. 입력값

| 입력 항목 | 설명 | 기본값 | 범위 |
|----------|------|--------|------|
| 수리비 (견적) | 상대방 + 내 차량 합산 또는 내 차량만 | 150만원 | 10만원~2,000만원 |
| 자기부담금 | 내 보험 약관상 자기차량손해 자기부담금 | 20만원 | 0~50만원 |
| 현재 연 보험료 | 갱신 기준 | 80만원 | 30만원~300만원 |
| 예상 할증률 | 사고 후 보험료 인상률 | 10% | 0~60% |
| 할증 유지 기간 | 일반적으로 3년 적용 | 3년 | 1~5년 |
| 사고점수 (선택) | 사고 유형별 점수 참고 | - | 선택 |

### 사고점수 빠른 선택 (프리셋)
| 사고 유형 | 대략 할증률 | 설명 |
|----------|-----------|------|
| 소손해 (50만원 미만) | 0~5% | 소손해 할증 없는 경우도 있음 |
| 경미한 접촉 (50~150만원) | 5~15% | 가장 많은 고민 구간 |
| 중간 사고 (150~500만원) | 15~30% | 보험처리 유리 가능성 높음 |
| 대형 사고 (500만원+) | 30~60% | 거의 보험처리 유리 |

## 5. 출력값

### KPI 카드 (상단)
| 카드 | 내용 |
|------|------|
| **판단 결과** | "보험처리 유리" / "현금처리 유리" (큰 강조) |
| **3년 절약 금액** | 유리한 선택 시 절약 추정액 |
| **손익분기 수리비** | "수리비 XX만원 이상이면 보험처리 유리" |

### 비용 비교 상세표
| 항목 | 보험처리 | 현금처리 |
|------|---------|---------|
| 즉시 부담 | 자기부담금 | 수리비 전액 |
| 1년차 추가 보험료 | +XX만원 | 0 |
| 2년차 추가 보험료 | +XX만원 | 0 |
| 3년차 추가 보험료 | +XX만원 | 0 |
| **3년 총비용** | **XX만원** | **XX만원** |
| **차이** | colspan | **XX만원 절약** |

### 추가 고려사항 안내 패널
- 무사고 할인 유지 여부 (현재 몇 년 무사고인지)
- 보험 등급 보호 특약 가입 여부
- 상대방 과실 비율 (상대방 보험 청구 가능 시 계산 달라짐)
- 소손해 처리 제도 활용 안내

## 7. UI / UX 설계

### 레이아웃
- `SimpleToolShell` 사용
- 입력(좌/상) → 판단 결과 KPI → 비교 테이블(우/하)

### 판단 결과 강조
- "보험처리 유리" → 파란색 배경 배지
- "현금처리 유리" → 초록색 배경 배지
- 판단이 근소한 경우 (차이 10만원 미만) → "비슷함, 무사고 등급 고려 권장" 중립 안내

### 프리셋 버튼
- 상단에 사고 유형 버튼 4개
- 선택 시 수리비·할증률 자동 세팅 후 즉시 결과 표시

### 슬라이더
- 수리비: 10만원~2,000만원 (50만원 단위)
- 현재 보험료: 30만원~300만원 (10만원 단위)
- 할증률: 0~60% (1% 단위)

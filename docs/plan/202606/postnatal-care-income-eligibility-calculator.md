## 2. 기획 배경

현재 사이트에는 `산후도우미 비용 계산기`가 이미 있다. 이 페이지는 거주 지역, 출산 유형, 소득 구간, 이용 기간을 선택해 정부지원금과 본인부담금을 계산하는 구조다. 다만 사용자는 그 전에 더 앞선 질문을 한다.

1. 우리 집이 산후도우미 정부지원 대상인가?
2. 기준중위소득 150% 이하인지 어떻게 판단하나?
3. 건강보험료 기준은 직장가입자, 지역가입자, 맞벌이에 따라 어떻게 보나?
4. 150%를 넘으면 완전히 못 받는지, 지자체 예외지원이 있는지 궁금하다.

네이버 데이터에서도 `산후도우미 지원금 소득기준 계산`은 노출은 작지만 CTR이 높다. 이 키워드는 정보 탐색보다 "내 조건을 넣고 판정받고 싶다"는 계산기형 의도가 강하다. 따라서 기존 비용 계산기 안에 설명을 더 붙이는 것보다, 별도 진입 계산기로 만들고 결과 하단에서 비용 계산기로 넘기는 구조가 더 자연스럽다.

## 5. 데이터 기준과 표시 원칙

### 5-1. 2026년 기준 데이터

| 데이터 | 사용 방식 | 표시 |
| --- | --- | --- |
| 2026년 기준중위소득 | 가구원 수별 100% 기준액 저장 후 150% 계산 | 공식 확인 필요 |
| 2026년 건강보험료 본인부담금 기준표 | 직장가입자, 지역가입자, 혼합 기준 비교 | 공식 확인 필요 |
| 산모·신생아 건강관리 지원사업 지원 유형 | 자격확인, 통합, 라형 등 결과 라벨에 활용 | 공식/지자체 기준 |
| 2026년 서비스 가격·지원금 | 기존 `postnatalCareCost.ts`와 연결 | 기존 공개표 활용 |
| 지자체 예외지원 | 지역별 예외 여부 체크리스트 | 참고/확인 필요 |

### 5-2. 공식/추정 구분

이 주제는 복지 지원금과 건강보험료 기준이 걸려 있으므로 모든 결과에 다음 배지를 명확히 붙인다.

| 배지 | 의미 |
| --- | --- |
| 공식 | 공식표에 있는 기준 또는 공개표 금액 |
| 추정 | 사용자가 입력한 보험료·소득으로 단순 판정한 결과 |
| 확인 필요 | 지자체, 보건소, 복지로 최종 확인이 필요한 조건 |

### 5-3. 기준 문구

실제 지원 가능 여부와 정부지원금은 거주지 보건소, 복지로, 사회서비스 전자바우처, 지자체 공고 기준으로 최종 확인해야 합니다.

## 8. 계산 로직 설계

### 8-1. 건강보험료 기준 판정

```text
월 건강보험료 합계 = 본인 건강보험료 + 배우자 건강보험료
기준 보험료 = 가구원 수 + 가입 유형별 2026년 150% 건강보험료 기준

차이 = 기준 보험료 - 월 건강보험료 합계
```

| 조건 | 판정 |
| --- | --- |
| 입력 보험료 <= 기준 보험료 | 150% 이하 가능성 높음 |
| 입력 보험료가 기준 보험료의 95~105% | 경계 구간, 보건소 확인 필요 |
| 입력 보험료 > 기준 보험료 | 150% 초과 가능성 |
| 초과이지만 예외지원 사유 있음 | 예외지원 확인 필요 |

### 8-2. 월소득 기준 보조 판정

```text
월소득 기준액 = 2026년 기준중위소득 100% × 150%
차이 = 월소득 기준액 - 입력 월소득
```

월소득 기준은 사용자가 건강보험료를 모를 때만 보조로 제공한다. 실제 신청은 건강보험료, 자격확인, 지자체 기준이 반영될 수 있으므로 결과 카드에 `보조 판정` 배지를 붙인다.

### 8-3. 예외지원 판정

예외지원은 확정 계산하지 않는다. 사용자가 체크한 조건에 따라 "확인해야 할 사유"를 결과 카드로 노출한다.

| 예외 사유 | 결과 문구 |
| --- | --- |
| 둘째 이상 | 일부 지자체는 둘째 이상 출산가정에 확대지원이 있을 수 있습니다. |
| 쌍태아·삼태아 이상 | 다태아는 돌봄 강도와 지원 기간이 달라질 수 있어 보건소 확인이 필요합니다. |
| 미숙아·선천성이상아 | 입원·퇴원일 기준 신청 기간이 달라질 수 있습니다. |
| 희귀난치성질환 산모 | 예외지원 대상 여부를 보건소에서 확인하세요. |
| 장애 산모·신생아 | 예외지원 또는 추가지원 여부를 확인하세요. |
| 결혼이민·미혼모 | 지자체 기준에 따라 예외지원 확인이 필요합니다. |

## 11. FAQ 기획

### Q1. 산후도우미 지원금은 누구나 받을 수 있나요?

산모·신생아 건강관리 지원사업은 소득 구간, 출산 순위, 태아 유형, 예외지원 여부, 거주지 기준에 따라 지원 여부와 지원금이 달라질 수 있습니다. 기본적으로 기준중위소득 150% 이하 기준이 많이 쓰이지만, 지자체별 확대지원과 예외지원이 있으므로 최종 판정은 보건소 또는 복지로 신청 과정에서 확인해야 합니다.

### Q2. 산후도우미 소득기준은 월급으로 보나요, 건강보험료로 보나요?

실무에서는 건강보험료 본인부담금을 기준으로 판단하는 경우가 많습니다. 월급이나 연봉은 참고로 볼 수 있지만, 실제 신청에서는 직장가입자, 지역가입자, 혼합 세대별 건강보험료 기준표와 자격확인 여부가 중요합니다.

### Q3. 맞벌이 부부는 건강보험료를 합산하나요?

맞벌이이거나 부부가 각각 건강보험에 가입되어 있으면 보험료를 함께 보는 구조가 될 수 있습니다. 다만 직장·지역·혼합 가입 유형과 세대 구성에 따라 판정 방식이 달라질 수 있으므로 계산기에서는 합산 추정 결과와 함께 보건소 확인 안내를 제공합니다.

### Q4. 기준중위소득 150%를 넘으면 무조건 지원을 못 받나요?

그렇지 않을 수 있습니다. 둘째 이상, 다태아, 미숙아, 희귀난치성질환 산모, 장애 산모·신생아, 결혼이민 산모, 미혼모, 지자체 확대지원 등 예외지원 사유가 있으면 150% 초과라도 확인해볼 필요가 있습니다.

### Q5. 산후도우미 신청은 언제 해야 하나요?

일반적으로 출산 예정일 40일 전부터 출산 후 60일 이내 신청 안내가 많이 쓰입니다. 미숙아 또는 선천성 이상아 입원 등 특수한 경우에는 퇴원일 기준으로 달라질 수 있으므로, 출산 전 거주지 보건소 기준을 먼저 확인하는 것이 좋습니다.

### Q6. 지원 가능하면 실제 본인부담금은 얼마인가요?

본인부담금은 소득 구간뿐 아니라 태아 유형, 출산 순위, 서비스 기간, 지자체 추가지원, 제공기관 추가비에 따라 달라집니다. 이 페이지에서는 지원 가능성을 먼저 판단하고, 결과 하단에서 `산후도우미 비용 계산기`로 연결해 예상 본인부담금을 계산하게 합니다.

### Q7. 건강보험료가 기준 근처이면 어떻게 해야 하나요?

기준의 95~105% 안팎이면 경계 구간으로 표시합니다. 보험료 산정월, 장기요양보험료 포함 여부, 맞벌이 합산, 세대 구성에 따라 달라질 수 있으므로 신청 전 보건소에서 최신 기준표로 확인해야 합니다.

### Q8. 지자체마다 기준이 다른가요?

기본 사업 구조는 같아도 지자체별 예외지원, 추가지원, 150% 초과 지원 여부가 다를 수 있습니다. 그래서 결과에 지역 확인 CTA를 넣고, 서울·경기·인천 등 주요 지역은 추후 별도 리포트나 표로 확장할 수 있습니다.

## 17. 최종 기획 요약

`산후도우미 지원금 소득기준 계산기`는 기존 산후도우미 비용 계산기의 앞단 퍼널이다. 사용자는 처음부터 본인부담금을 알고 싶은 것이 아니라, 먼저 "내가 지원 대상인지"를 확인하고 싶어 한다. 특히 건강보험료 기준, 맞벌이 합산, 기준중위소득 150%, 예외지원은 검색 의도가 선명하지만 기존 비용 계산기 안에서는 묻히기 쉽다.

따라서 이 페이지는 2026년 기준중위소득과 건강보험료 기준을 바탕으로 지원 가능성을 `추정`하고, 예외지원 가능성을 체크해준 뒤, 곧바로 산후도우미 비용 계산기로 연결하는 구조가 가장 좋다. 검색 유입은 작게 시작하더라도 CTR이 높고 출산·복지 클러스터와 연결성이 강해, 신규 콘텐츠 우선순위 1번으로 구현할 가치가 있다.

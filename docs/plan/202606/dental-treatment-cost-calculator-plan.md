# 치과 치료비 비교 계산기 기획 문서

> 작성일: 2026-06-19  
> slug: `dental-treatment-cost-calculator`  
> 연계 페이지: 실손보험 계산기, 건강보험료 계산기

---

## 1. 배경 및 목적

치과 치료비는 병원마다 편차가 크고, 환자 입장에서 "내가 받은 견적이 비싼 건지 아닌지"를 판단할 기준이 없다. "임플란트 비용", "크라운 비용", "충치 치료 비용" 등은 검색량이 많고 의료·보험 광고 단가가 높은 키워드 군이다.

**핵심 컨셉**: "내 치과 견적, 평균 대비 높은 걸까? 낮은 걸까?"  
단순 비용 계산이 아닌 **"내 견적의 적정성 판단 도구"** 로 차별화한다.

---

## 2. 타겟 사용자

- 임플란트 시술 전 견적 받고 가격 비교하고 싶은 사람
- 크라운·인레이 치료 견적 받고 적정 여부 확인하고 싶은 사람
- 치아교정 비용 궁금한 사람
- 치과 치료 후 보험 청구 가능 여부 확인하고 싶은 사람
- 의료비 연말정산 공제 항목 확인하고 싶은 사람
- 노부모 치과 치료비 부담되는 자녀

---

## 3. 핵심 컨셉: 적정가 판단

치과 치료비는 지역·병원 규모·재료·술식에 따라 편차가 크다. 이 계산기는 **건강보험심사평가원 공개 데이터 + 시장 조사 기반 참고 범위**를 제공해 사용자가 자신의 견적이 어느 구간에 해당하는지 직관적으로 확인하게 한다.

### 판단 기준 (3단계)
| 구간 | 기준 | 배지 색상 |
|------|------|---------|
| 낮음 | 참고 범위 하한 미만 | 초록 |
| 보통 | 참고 범위 내 | 파랑 |
| 높음 | 참고 범위 상한 초과 | 주황 |

## 6. 출력값

### KPI 카드
| 카드 | 내용 |
|------|------|
| **적정성 판단** | 낮음 / 보통 / 높음 배지 |
| **참고 범위** | 하한~상한 (재료별) |
| **총 예상 비용** | 개수 × 단가 |
| **보험 적용 시 절감액** | 적용 가능 치료인 경우 표시 |

### 비교 시각화
- 수평 바 게이지: 내 견적이 참고 범위 어느 위치인지 시각화
- "하위 XX%" 또는 "상위 XX%" 표시 (범위 내 상대적 위치)

### 추가 정보 패널
- 해당 치료의 보험 적용 여부 및 조건
- 절감 팁 (예: "임플란트는 만 65세 이상 건강보험 적용 가능")
- 의료비 연말정산 공제 안내

## 8. UI / UX 설계

### 치료 종류 탭
- 상단에 치료 종류 탭 8개 (가로 스크롤 가능, 모바일 대응)
- 탭 클릭 시 세부 옵션 패널 동적 교체

### 게이지 바 (핵심 차별화 UI)
```
[낮음]━━━━━━━━━━━━━━━━━━━━━━━━━[높음]
   하한(80만)      ▲내견적      상한(130만)
                  (105만원)
                  보통 구간
```

### 적정성 판단 배지
- "낮음 (상위 80% 이하)" — 초록
- "보통 (일반적인 범위)" — 파랑
- "높음 (상위 20% 이상)" — 주황

### 보험 적용 알림
- 보험 적용 가능 치료 선택 시 → 상단 파란 배너로 "이 치료는 건강보험 적용 가능합니다" 표시

## 9. 데이터 설계

### 데이터 파일: `src/data/dentalTreatmentCost.ts`

```ts
export interface DentalTreatment {
  id: string;
  label: string;
  icon: string;
  options: DentalOption[];
  insuranceNote?: string;   // 보험 적용 안내
  perTooth: boolean;        // 개당 계산 여부
}

export interface DentalOption {
  id: string;
  label: string;
  low: number;    // 하한 (만원)
  avg: number;    // 평균 (만원)
  high: number;   // 상한 (만원)
  insuranceLow?: number;   // 보험 적용 시 하한
  insuranceHigh?: number;  // 보험 적용 시 상한
  insuranceCondition?: string; // 보험 조건
}
```

## 11. 보험·절세 안내 콘텐츠

### 건강보험 적용 치료 목록
| 치료 | 조건 | 본인부담률 |
|------|------|---------|
| 임플란트 | 만 65세 이상, 2개 한도 | 30% |
| 완전/부분틀니 | 만 65세 이상 | 30% |
| 스케일링 | 연 1회 | 20~30% |
| 충치 치료 (아말감·레진) | 나이 제한 없음 | 30% |
| 신경치료 | 나이 제한 없음 | 30% |
| 파노라마 X-ray | 나이 제한 없음 | 30% |

### 실손보험 청구 가능 여부
- 치과 실손은 기본 실손 적용 제한 — 별도 치과보험 필요
- 상해로 인한 치아 치료는 실손 적용 가능 (예외)

### 연말정산 의료비 공제
- 치과 치료비 전액 의료비 공제 대상 (비보험 포함)
- 공제 한도: 총급여 3% 초과분 × 15% (세액공제)
- 교정비도 포함 (저작·발음 기능 개선 목적 시)

# 골프 회원권 vs 퍼블릭 손익 비교 기획

## 1. 배경 및 목적

- **계기**: 골프 인구 증가와 함께 "회원권 사야 하나?", "회원권 본전은 몇 년?" 질문이 급증. 회원권은 수천만 원 투자로 고소득층 의사결정 콘텐츠 → 체류시간·공유율 우수, 고단가 광고 노출 적합
- **목적**:
  1. "골프 회원권 손익", "회원권 본전" 키워드 SEO 확보
  2. 고액 투자 결정 지원 도구 → 신뢰도 기반 체류시간 확보
  3. 회원권 거래소·골프장 예약 서비스 제휴 기회
- **타겟 사용자**: 회원권 구매 검토 중인 골퍼, 기존 회원권 보유자의 갱신/매각 판단

## 3. 입력값 구조

```
[이용 패턴 (공통)]
- 월 라운딩 횟수               기본값: 4회
- 라운딩 시 동반자 수           기본값: 3명 (4인 1팀)
- 연간 이용 개월                기본값: 10개월 (동절기 2개월 제외)

[회원권 시나리오]
- 회원권 매입가 (만원)          기본값: 3,000만원
- 예상 유지 기간 (년)           기본값: 5년
- 예상 매각가 (만원)            기본값: 2,500만원 (시세 하락 -500만원)
- 연 관리비 (만원)              기본값: 100만원
- 회원 그린피 (원/회)           기본값: 80,000원
- 캐디피 (원/인)               기본값: 30,000원
- 카트비 (원/인)               기본값: 15,000원

[퍼블릭 시나리오]
- 주중 그린피 (원/회)           기본값: 150,000원
- 주말 그린피 (원/회)           기본값: 200,000원
- 주말 라운딩 비율 (%)          기본값: 40%
- 캐디피 (원/인)               기본값: 30,000원
- 카트비 (원/인)               기본값: 15,000원
```

## 4. 계산 로직

```
// 회원권: 연간 라운딩 비용
memberAnnualRound = 월라운딩횟수 × 이용개월 × (회원그린피 + 캐디피 + 카트비)
// 회원권: 연간 총비용
memberAnnual = memberAnnualRound + 관리비
// 회원권: N년 누적 비용 (매각차손 포함)
memberCumulative[y] = memberAnnual × y + (매입가 - 매각가) / 유지기간 × y

// 퍼블릭: 연간 라운딩 비용
publicAnnualRound = 월라운딩횟수 × 이용개월 ×
  (주중그린피 × (1-주말비율) + 주말그린피 × 주말비율 + 캐디피 + 카트비)
// 퍼블릭: N년 누적 비용
publicCumulative[y] = publicAnnualRound × y

// 1회당 실질 단가 비교
memberCostPerRound = memberAnnual / (월라운딩횟수 × 이용개월)
  + (매입가 - 매각가) / 유지기간 / (월라운딩횟수 × 이용개월)
publicCostPerRound = publicAnnualRound / (월라운딩횟수 × 이용개월)

// 손익분기점
breakEvenYear = memberCumulative < publicCumulative 교차 시점
```

## 10. QA/리스크

- 회원권 매각가 예측은 시장 불확실성 있음 → "추정" 명시, 시나리오 탭으로 다변화
- 기회비용(투자 수익률) 기본 4% → 변경 불가 고정값으로 단순화
- 법인 회원권 세제 혜택은 계산기 범위 외 → "개인 회원권 기준" 고지
- 골프장별 회원 전용 혜택(레스토랑·숙박 할인 등) 미반영 → 안내 문구 추가

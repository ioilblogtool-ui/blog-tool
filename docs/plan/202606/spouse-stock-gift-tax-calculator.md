## 2. 콘텐츠 한 줄 정의

> 배우자에게 국내주식·해외주식·ETF를 증여할 때 10년 합산 6억 원 공제 한도와 증여 후 매도 시 양도소득세 차이를 함께 계산하고, 2025년 이후 증여분의 1년 이내 매도 주의 규정을 안내하는 계산기

---

## 3. 기획 배경

해외주식 수익이 큰 투자자는 매도 시 양도소득세 부담을 먼저 고민합니다. 이때 자주 언급되는 절세 전략이 **배우자에게 주식을 증여한 뒤, 배우자가 증여 당시 평가액을 취득가액으로 삼아 매도하는 방식**입니다.

하지만 2025년 이후 증여분부터는 배우자 또는 직계존비속에게 증여받은 주식을 1년 이내 양도하는 경우, 증여자의 취득가액을 기준으로 양도차익을 계산할 수 있는 규정이 중요해졌습니다. 따라서 이 콘텐츠는 단순히 "배우자 6억 공제"만 계산하면 부족합니다.

이 계산기의 핵심은 세 가지입니다.

1. 배우자 증여세 공제 6억 원 한도 계산
2. 증여하지 않고 직접 매도했을 때의 양도세 계산
3. 증여 후 배우자가 매도할 때, 1년 이내 매도인지 여부에 따른 경고와 절세액 비교

## 9. 계산 로직

### 9-1. 배우자 증여공제

```txt
배우자 공제한도 = 600,000,000원
잔여 공제한도 = max(0, 600,000,000 - 최근 10년 배우자 증여액)
이번 증여 공제액 = min(증여 주식 평가액, 잔여 공제한도)
증여세 과세표준 = max(0, 증여 주식 평가액 - 이번 증여 공제액)
```

### 9-2. 증여세 누진세율

| 과세표준 | 세율 | 누진공제 |
|---:|---:|---:|
| 1억 원 이하 | 10% | 0원 |
| 1억 원 초과~5억 원 이하 | 20% | 1,000만 원 |
| 5억 원 초과~10억 원 이하 | 30% | 6,000만 원 |
| 10억 원 초과~30억 원 이하 | 40% | 1억 6,000만 원 |
| 30억 원 초과 | 50% | 4억 6,000만 원 |

```txt
산출세액 = 과세표준 × 세율 - 누진공제
신고세액공제 = 산출세액 × 3% (기한 내 신고 가정)
예상 증여세 = 산출세액 - 신고세액공제
```

### 9-3. 증여하지 않고 직접 매도

```txt
직접 매도 양도차익 = 증여 후 예상 매도가액 - 증여자의 원래 취득가액
직접 매도 과세대상 = max(0, 직접 매도 양도차익 - 해외주식 기본공제)
직접 매도 양도세 = 직접 매도 과세대상 × 22%
```

해외주식 양도소득세는 기획·MVP 단계에서 지방소득세 포함 22% 단순 계산으로 처리합니다. 실제 세율과 손익통산은 사용자 상황에 따라 달라질 수 있음을 명시합니다.

### 9-4. 증여 후 배우자 매도

#### 1년 이상 보유

```txt
배우자 취득가액 = 증여 당시 평가액
증여 후 양도차익 = 증여 후 예상 매도가액 - 증여 당시 평가액
증여 후 과세대상 = max(0, 증여 후 양도차익 - 해외주식 기본공제)
증여 후 양도세 = 증여 후 과세대상 × 22%
```

#### 1년 이내 매도

```txt
배우자 취득가액 = 증여자의 원래 취득가액으로 볼 수 있음
증여 후 양도차익 = 증여 후 예상 매도가액 - 증여자의 원래 취득가액
증여 후 양도세 = 직접 매도와 유사하게 계산될 수 있음
절세액 = "주의 필요"로 표시
```

### 9-5. 예상 절세액

```txt
예상 절세액 = 직접 매도 양도세 - 증여 후 배우자 매도 양도세 - 예상 증여세
```

단, 보유기간이 1년 미만이면 절세액을 확정 표시하지 않고 아래 문구를 우선 노출합니다.

```md
1년 이내 매도라면 증여 당시 평가액을 취득가액으로 활용하는 절세 효과가 제한될 수 있습니다. 매도 시점을 다시 확인하세요.
```

## 12. UI/UX 설계

### 입력 UI

* 주식 종류: segmented control
* 증여자: 남편/아내 segmented control
* 금액 입력: 원 단위 input + 빠른 버튼 1억/3억/6억/10억
* 보유기간: 6개월 / 1년 이상 toggle
* 고급설정: 해외주식 기본공제, 양도세율, 신고세액공제 접힘 영역

### 결과 UI

* 핵심 결과 카드 5개
* 1년 이내 매도 시 결과 상단에 빨간 경고
* 1년 이상 보유 시 절세액 카드를 녹색 강조
* 비교 표는 `증여 안 함` vs `증여 후 매도` 2열 구조
* 모바일에서는 결과 카드를 1열로 쌓기

### 결과 문구 예시

#### 1년 이상 보유

```md
증여세는 0원으로 추정됩니다. 1년 이상 보유 후 매도하는 경우, 증여 당시 평가액을 취득가액으로 활용해 양도세 절세 효과를 기대할 수 있습니다.
```

#### 1년 이내 매도

```md
주의가 필요합니다. 증여 후 1년 이내 매도하면 증여자의 원래 취득가액을 기준으로 양도차익이 계산될 수 있어, 예상 절세 효과가 줄어들 수 있습니다.
```

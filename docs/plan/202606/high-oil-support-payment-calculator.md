## 2. 기획 배경

고유가 피해지원금은 2026년 4월 말부터 지급이 시작되었고, 2차 신청 기간이 2026년 7월 3일까지 이어진다. 6월에는 "아직 신청 가능한가", "나는 얼마 받을 수 있나", "비수도권 추가 5만 원이 붙나" 같은 검색 수요가 집중될 가능성이 높다.

기존 복지 콘텐츠가 기준 중위소득, 복지급여, 정부 지원금 허브 중심이라면 이 페이지는 **기간 한정 지원금의 실수령액 확인**에 초점을 둔다. 사용자는 제도 설명보다 "내 조건에서 수령액이 얼마인지"와 "어디서 신청해야 하는지"를 먼저 알고 싶어한다.

따라서 페이지는 긴 정책 해설보다 계산기와 신청 체크리스트를 상단에 배치하고, 본문에서 대상·금액·신청 기간·주의사항을 짧게 정리한다.

## 8. 계산 로직

## 8-1. 기본 지원금

```text
if 대상 유형 == "기초생활수급자":
  기본 지원금 = 550,000원
else if 대상 유형 == "차상위계층" or "한부모가족":
  기본 지원금 = 450,000원
else:
  기본 지원금 = 0원
```

## 8-2. 지역 추가금

```text
if 거주 권역 == "비수도권" and 인구감소지역 여부 == true:
  지역 추가금 = 50,000원
else:
  지역 추가금 = 0원
```

정책브리핑 문구는 "비수도권 및 인구감소지역 주민"으로 안내되어 있으므로, 계산기에서는 두 조건을 분리 입력하게 한다. 실제 해당 여부는 주소지 관할 지방정부 공지로 확인해야 한다.

## 8-3. 총 예상 수령액

```text
1인 예상 수령액 = 기본 지원금 + 지역 추가금
총 예상 수령액 = 1인 예상 수령액 x 대상 인원 수
```

결과에는 반드시 `모의계산` 배지를 붙인다. 지원 대상 확정과 지급 방식은 지자체 심사 결과가 우선이다.

## 8-4. 신청 기간 경고

```text
if 신청 예정일 > 2026-07-03:
  "2차 신청 기한이 지났을 수 있습니다. 관할 지자체 공지를 확인하세요."
else if 신청 예정일 >= 2026-06-24:
  "마감까지 10일 이내입니다. 주소지 관할 신청처를 먼저 확인하세요."
else:
  "2차 신청 기간 내 신청 가능성이 있습니다."
```

## 16. 최종 기획 요약

고유가 피해지원금 계산기는 2026년 6월에 가장 시의성이 강한 기간 한정 계산기다. 핵심은 정책 설명이 아니라 **대상 유형별 예상 수령액과 신청 마감 전 체크리스트**를 빠르게 보여주는 것이다.

MVP는 대상 유형, 대상 인원 수, 지역 조건만으로 총 예상 수령액을 계산하면 충분하다. 다만 실제 지급 여부는 지자체 심사 결과가 우선이므로 결과 카드와 FAQ에 `모의계산`, `확인 필요` 문구를 분명히 붙여야 한다.

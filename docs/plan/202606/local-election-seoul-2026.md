## 2. 기획 배경 및 차별화 포인트

### 2-1. 왜 서울 구청장인가

서울 구청장은 광역단체장(서울시장)보다 유권자 삶에 더 직접적인 영향을 미치는 경우가 많다:

| 구청장 권한 | 주민 삶 연관성 |
| --- | --- |
| 재건축·재개발 정비구역 지정 | 아파트 가격·이주 직결 |
| 도로·주차장 관리 | 일상 교통 직접 영향 |
| 구 복지 예산 편성 | 어린이집·노인시설 규모 |
| 지역 축제·문화 행사 | 지역 상권·관광 |
| 소상공인 지원금 | 자영업자 직접 수혜 |

서울 25개 구는 지역별 특성이 극명히 달라 "우리 동네 구청장" 검색 수요가 강하다.

### 2-2. 이번 선거 특징

- 민주당 20곳 vs 국민의힘 5곳 — 민주당 압도적 우세
- 강남3구(강남·서초·송파)에서의 결과가 특히 주목
- 서울시장(정원오 민주당)과 구청장 다수가 같은 당 → 서울 행정 일체화 예상
- 부동산 규제 완화 vs 공공임대 확대 정책 방향성이 향후 4년 서울 부동산 시장 결정

### 2-3. 경쟁 콘텐츠 분석

| 채널 | 약점 | 비교계산소 강점 |
| --- | --- | --- |
| 뉴스 기사 | 구별 기사 분산, 검색 불편 | 25개 구 통합 지도 |
| 서울시 홈페이지 | 단순 결과 표, 공약 없음 | 공약 + 이력 통합 |
| 부동산 커뮤니티 | 재건축 정보 편향 | 전 분야 공약 종합 |
| 나무위키 | 인물 기록 없음 (신규 당선자) | 즉시 업데이트 |

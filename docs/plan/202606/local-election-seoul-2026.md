## 2. 기획 배경

서울 25개 자치구청장 선거는 광역단체장 못지않게 유권자의 관심이 높다. 재건축·재개발, 교통, 생활 편의 등 일상에 직접 영향을 미치는 정책이 구청장에 의해 결정되기 때문이다.

이번 선거 결과 서울 25개 구 중 더불어민주당이 20곳, 국민의힘이 5곳을 차지한 것으로 집계됐다. 강남 3구(강남·서초·송파)에서의 결과가 특히 주목을 받고 있다.

기존 선거 결과 콘텐츠는 텍스트 나열에 그쳐 "우리 동네 구청장이 누구고 무슨 공약을 내세웠는지" 한눈에 보기 어렵다. 서울 25개 구 **클릭형 SVG 지도**로 차별화된 탐색 경험을 제공한다.

### C. 서울 25구 인터랙티브 지도 ← 핵심

**구현 방식**
- 서울 25개 자치구 SVG 지도
- 구별 개별 `<path>` 분리
- 정당별 색상 코딩 + 구청장 이름 표시

```
더불어민주당  →  #0064FF
국민의힘      →  #E61E2B
```

**인터랙션**
1. 호버 → 구 이름 + 당선자명 툴팁
2. 클릭 → 구 하이라이트 + 하단 패널 오픈
3. 상단 구 검색 입력창 (빠른 탐색)
4. 모바일 → 지도 + 자치구 드롭다운 선택

**예상 색상 배치 (확정 전)**

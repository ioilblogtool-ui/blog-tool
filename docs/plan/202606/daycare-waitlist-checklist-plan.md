# 어린이집 대기순번 입소 가능성 체크리스트 기획 문서

## 1. 개요

| 항목 | 내용 |
|------|------|
| 슬러그 | `daycare-waitlist-checklist` |
| URL | `/tools/daycare-waitlist-checklist` |
| 페이지 유형 | 체크리스트 + 결과 판정형 도구 |
| 레이아웃 쉘 | `SimpleToolShell.astro` |
| 타이틀 | `어린이집 대기순번 입소 가능성 체크리스트 2026 \| 대기 1번·2번이면 들어갈 수 있을까` |
| 디스크립션 | `대기순번·월령·맞벌이·형제 여부 입력하면 입소 가능성 바로 판정. 전화 문의 시 쓸 수 있는 멘트까지 제공.` |

---

## 2. 배경 및 목적

### 왜 이 콘텐츠인가

- 어린이집 대기순번 1~3번을 받은 부모는 "언제 들어갈 수 있는지"를 궁금해한다
- 대기순번 숫자만으로는 판단이 어렵고, 가중치 점수(맞벌이·형제·월령)에 따라 실제 입소 가능성이 크게 달라진다
- 기존 `daycare-kindergarten-cost-2026` 페이지는 비용 비교 중심이어서 **입소 가능성**을 다루지 않는다
- 부모 커뮤니티(맘카페·육아앱)에서 "대기 1번인데 들어갈 수 있나요?" 질문이 반복적으로 올라온다

### 타겟 사용자

- 자녀 출생 후 어린이집 대기 등록한 부모
- 복직을 앞두고 어린이집 입소 시점을 계획 중인 워킹맘·워킹대디
- 어린이집 대기순번을 받고 전화 문의 타이밍을 결정하려는 부모

### 핵심 검색 의도

- `어린이집 대기 1번` — 정보성 / 즉각 판단 니즈
- `어린이집 대기순번` — 정보 탐색
- `어린이집 입소 가능성` — 판단 니즈
- `어린이집 대기 몇 번이면 입소` — 직접 질문형

## 3. 어린이집 대기 시스템 이해

### 아이사랑 대기 등록 구조

국공립·직장·사회적기업 어린이집은 **아이사랑(childcare.go.kr)** 을 통해 대기 등록한다.
입소 우선순위는 아래 가중치 점수로 결정된다:

| 우선순위 | 항목 | 세부 기준 |
|---------|------|---------|
| 1순위 | 국민기초생활보장 수급자 자녀 | 법령 기준 |
| 1순위 | 한부모 가족 자녀 | 법령 기준 |
| 1순위 | 장애인 부모 자녀 | 법령 기준 |
| 1순위 | 다문화 가족 자녀 | 법령 기준 |
| 2순위 | 맞벌이 가정 자녀 | 부모 모두 취업 상태 |
| 2순위 | 다자녀 (형제 기존 재원) | 형제·자매 같은 어린이집 재원 |
| 3순위 | 일반 (영아·유아 구분) | 대기 등록일 순 |

### 반별 정원 기준 (2026)

| 반 이름 | 연령 | 정원 (명) |
|---------|------|----------|
| 영아 0세반 | 만 0세 (출생~12개월) | 3 |
| 영아 1세반 | 만 1세 (12~24개월) | 5 |
| 영아 2세반 | 만 2세 (24~36개월) | 7 |
| 유아 3세반 | 만 3세 | 15 |
| 유아 4세반 | 만 4세 | 20 |
| 유아 5세반 | 만 5세 | 20 |

### 입소 시점 결정 요인

1. **대기순번**: 낮을수록 유리하지만 절대적 기준이 아님
2. **자녀 월령**: 반이 나뉘므로 해당 반의 대기순번이 중요
3. **우선순위 점수**: 맞벌이·형제 여부로 순번이 역전될 수 있음
4. **반 자리 회전율**: 이사·전원 등으로 자리가 나는 빈도
5. **신청 어린이집 수**: 최대 3곳 동시 대기 가능 (아이사랑 기준)

## 4. 입력값 설계

### 입력 항목 (5개)

| 입력 항목 | 타입 | 범위/선택지 | 비고 |
|---------|------|---------|------|
| 현재 대기순번 | 숫자 슬라이더 | 1~30 | "순번 불명 시 10 이상 추정" 가이드 제공 |
| 자녀 월령 | 버튼 선택 | 0~5개월 / 6~11개월 / 12~17개월 / 18~23개월 / 24개월+ | 반 배정 기준 |
| 맞벌이 여부 | 토글 | 예 / 아니오 | 2순위 우선순위 해당 여부 |
| 형제·자매 재원 여부 | 토글 | 예 / 아니오 | 같은 어린이집 기준 |
| 희망 입소 시점 | 버튼 선택 | 1개월 내 / 3개월 내 / 6개월 내 / 언제든 | 시급성 판단 |

### 기본값

- 대기순번: 3
- 월령: 12~17개월 (1세반)
- 맞벌이: 예
- 형제재원: 아니오
- 입소 시점: 3개월 내

## 5. 판정 로직

### 점수 계산 방식

```
입소가능성 점수 = 기본점수 - 대기순번 패널티 + 우선순위 보너스 + 시급성 보정
```

#### 기본점수 (대기순번 기반)

| 대기순번 | 기본점수 |
|---------|---------|
| 1번 | 70 |
| 2번 | 55 |
| 3번 | 40 |
| 4~5번 | 25 |
| 6~10번 | 10 |
| 11번 이상 | 0 |

#### 우선순위 보너스

| 조건 | 보너스 |
|------|------|
| 맞벌이 (부모 모두 취업) | +20 |
| 형제·자매 해당 어린이집 재원 중 | +15 |
| 영아 0세반 (정원 3명, 회전율 높음) | +10 |
| 1세반 (정원 5명) | +5 |

#### 월령·반 특성 보정

| 월령 | 보정 이유 | 보정값 |
|------|---------|------|
| 0~5개월 | 0세반 정원 3명, 입소 후 1세반으로 빠른 전환 | +10 |
| 6~11개월 | 0세반 후반부, 연간 1~2자리 회전 | +5 |
| 12~17개월 | 1세반 대기 경쟁 심화 시기 | 0 |
| 18~23개월 | 연초 입소 또는 신규 어린이집 개원 기회 | +3 |
| 24개월+ | 2세반 이상, 정원 많아 상대적으로 빠름 | +8 |

#### 희망 시점 보정

| 희망 시점 | 보정 |
|---------|------|
| 1개월 내 | –10 (현실적으로 빠른 입소 어려움) |
| 3개월 내 | 0 |
| 6개월 내 | +5 |
| 언제든 | +10 |

### FAQ 항목 (6개)

1. **어린이집 대기 1번이면 무조건 입소되나요?**
   — 1번이어도 입소가 보장되지 않습니다. 해당 반에 자리가 생겨야 하고, 더 높은 우선순위(한부모·장애·기초수급)가 없어야 합니다. 맞벌이 여부와 형제 재원 여부에 따라 순번이 역전될 수 있습니다.

2. **맞벌이면 어린이집 대기 우선순위가 올라가나요?**
   — 아이사랑 시스템에서 맞벌이 가정은 2순위 우선순위를 받습니다. 일반 3순위보다 우선 배정되며, 부모 모두 건강보험료 납부 이력으로 취업 여부를 확인합니다.

3. **어린이집 대기 취소하면 순번이 사라지나요?**
   — 아이사랑 앱에서 대기를 취소하면 해당 어린이집 대기 목록에서 삭제됩니다. 재신청 시 신규 대기 등록 일자 기준으로 다시 순번이 배정됩니다.

4. **어린이집 대기 중 전화는 언제 하면 좋나요?**
   — 매월 초(1~5일)가 좋습니다. 전달 말일 기준으로 퇴소·전원이 집계되어 당월 초에 자리가 확정됩니다. 이 시기에 전화하면 당월 입소 가능 여부를 가장 정확히 확인할 수 있습니다.

5. **0세반과 1세반 대기순번이 다른가요?**
   — 반마다 별도 대기 순번이 있습니다. 0세반(정원 3명)과 1세반(정원 5명)은 정원이 달라 입소 속도가 다릅니다. 아이사랑 앱에서 반별로 대기 등록하고 순번을 각각 확인하세요.

6. **어린이집 대기 중 민간 어린이집 입소하면 국공립 대기가 취소되나요?**
   — 민간 어린이집 입소와 국공립 대기는 별도로 유지됩니다. 단, 국공립 입소 확정 통보를 받고 수락하지 않으면 대기 취소 처리될 수 있으니, 통보 후 즉시 의사결정이 필요합니다.

## 1. 배경 및 목적

### 기존 도구와의 차이

비교계산소에는 이미 `silson-insurance-refund-calculator`(실손보험 환급액 계산기)가 있다.  
그 도구는 **"가입 세대별로 자기부담금 구조가 다른데, 내가 실제로 얼마 받을 수 있나"** 가 핵심이다.

이 도구의 핵심 검색 의도는 다르다.

> **"청구하면 얼마 받는데, 청구 안 하는 게 나을 때도 있어?"**

즉, 이 계산기는 **청구 실익 판단 도구**다. 환급액만 계산하는 것이 아니라, **보험료 할인 손실 vs 환급액** 을 비교해서 "청구가 이득인지 손해인지"를 알려준다.

### 왜 이 수요가 크나

- 실손보험은 **3년 무청구 시 보험료 10% 할인** 조건이 있는 상품이 많음
- 소액 청구(1~5만원) 시 보험료 인상 손실이 환급액을 상회할 수 있음
- "청구하면 보험료 오르나요?" 라는 질문이 커뮤니티에 매우 잦음
- 기존 계산기로는 이 판단을 내릴 수 없음

## 3. 핵심 차별화: "청구 실익 판단" 로직

기존 환급액 계산기에 없는 로직:

```
무청구 할인액 = 현재 월 보험료 × 무청구 할인율 × 잔여 할인 기간(개월)
청구 실익 = 예상 환급액 - 무청구 할인액 상실분
```

이 수치가 **양수면 청구 권장**, **음수면 참는 게 이득** 이라고 결론을 내려준다.

## 5. 계산 로직

### 5-1. 급여 환급액

```
공단부담 제외 후 환자부담 = 급여 진료비 × (1 - 공단부담률)
  (공단부담률: 통원 70%, 입원 80%)

급여 환급액 = 환자부담 × (1 - 자기부담률)
```

단, **통원 자기부담금 최소 공제**: 1~2세대 1만원, 3세대 1만원, 4세대 2만원 공제 후 청구.

### 5-2. 비급여 환급액

```
비급여 환급액 = 비급여 진료비 × (1 - 비급여 자기부담률)
```

4세대 실손은 비급여 자기부담률 30%, 도수치료·MRI 등 특정 항목은 별도.

### 5-3. 약제비

통원의 경우 약제비 별도 청구 (자기부담금 공제 후).  
4세대 기준: 약제비 자기부담 8,000원 공제.

### 5-4. 합산 예상 환급액

```
총 예상 환급액 = 급여 환급액 + 비급여 환급액 + 약제비 환급액
```

### 5-5. 무청구 할인 손실 계산 (선택 로직)

```
무청구 할인 기준: 3년(36개월) 무청구 시 보험료 10% 할인

남은 무청구 기간 = 36 - 현재 경과 개월 (개월)
할인 손실액 = 월 보험료 × 할인율 × 남은 기간
  (단, 이미 할인 받고 있는 경우: 할인 손실 = 월 보험료 × 할인율 × 12개월로 추정)

청구 실익 = 총 예상 환급액 - 할인 손실액
```

### 5-6. 결론 판정

| 조건 | 결론 |
|------|------|
| 총 환급액 = 0 | 청구 불가 (비급여 치과 등) |
| 청구 실익 > 0 | **청구 권장** |
| 청구 실익 ≤ 0, 무청구 할인 없음 | **청구 권장** (할인 없으면 손해 없음) |
| 청구 실익 ≤ 0, 무청구 할인 있음 | **청구 비권장** — 할인 유지가 유리 |

## 10. FAQ (5개)

1. **소액 실손 청구하면 보험료 오르나요?**  
   보험료 자체는 청구 여부와 관계없이 갱신 시 오를 수 있으나, 무청구 할인 특약이 있는 경우 청구하면 할인 혜택을 잃을 수 있습니다. 계산기에 월 보험료와 무청구 기간을 입력하면 실익을 비교할 수 있습니다.

2. **치과 치료비도 실손보험으로 청구할 수 있나요?**  
   대부분 치과 치료는 실손보험 지급 대상 외입니다. 충치 치료, 임플란트, 교정 등은 제외되며, 구강 내 수술 등 일부 예외적 경우만 인정됩니다.

3. **3세대와 4세대 실손의 자기부담률 차이는 무엇인가요?**  
   3세대는 급여 20%, 비급여 30% 자기부담. 4세대는 급여 20%, 비급여 30%로 유사하나 비급여 항목별로 별도 특별약관이 적용되고 약제비 공제금액이 8,000원으로 높아집니다.

4. **비급여 MRI는 실손 청구 가능한가요?**  
   1~3세대 실손은 의사 지시에 따른 비급여 MRI는 대부분 청구 가능합니다. 4세대는 특별약관(비급여 의료비) 별도 가입 여부에 따라 다릅니다.

5. **입원과 통원의 청구 기준이 다른가요?**  
   네. 입원은 공단부담률 80%, 자기부담률 20%가 일반적이며 자기부담금 공제가 없습니다. 통원은 공단부담률 70%, 자기부담금(1~2만원) 공제 후 청구합니다.

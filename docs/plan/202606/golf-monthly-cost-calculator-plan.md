# 골프 월 유지비 계산기 기획

## 1. 배경 및 목적

- **계기**: 한국 골프 인구 600만 명 돌파. "골프 한 달에 얼마?", "골프 유지비 얼마나 드나" 검색 수요 높음. 골프 용품·레저 카테고리는 애드센스 CPC 단가가 높은 고소득층 타겟 카테고리
- **목적**:
  1. "골프 유지비", "골프 한달 비용" 키워드 SEO 확보
  2. 골프 고소득 타겟 → 금융·여행·용품 광고 단가 우위
  3. 회원권 손익 계산기 / 골프 vs 취미 비교와 내부링크 허브 구성
- **타겟 사용자**: 골프 입문 검토자, 현재 골퍼의 지출 점검, 부부·직장인 골퍼 예산 관리

```

- 월정액 시 금액               기본값: 150,000원
- 횟수권 시 (월 몇 회 × 단가)   기본값: 8회 × 10,000원

[장비 감가상각]
- 클럽 세트 구매가             기본값: 1,500,000원

[의류]
- 골프웨어 연간 지출           기본값: 300,000원

[기타]
- 골프 보험료 (선택)           기본값: 0원 (선택 OFF)

- 교통비 (라운딩 1회당)        기본값: 20,000원
```

## 4. 계산 로직

```
// 월 라운딩 비용
roundingMonthly = 라운딩횟수 × (그린피 + 캐디피ON×30000 + 카트비ON×15000 + 식사비)

// 월 연습장 비용
practiceMonthly = 월정액 OR (횟수 × 단가) OR 0

// 장비 월 감가상각
equipDepreciation = (클럽구매가 / 사용기간 / 12) + 소모품 + (골프화연간 / 12)

// 의류 월 환산
apparelMonthly = 골프웨어연간 / 12

// 기타 월 비용
otherMonthly = 보험료 + (스크린횟수 × 스크린단가) + (라운딩횟수 × 교통비)

// 월 총 유지비
totalMonthly = roundingMonthly + practiceMonthly + equipDepreciation + apparelMonthly + otherMonthly

// 연간 환산
totalAnnual = totalMonthly × 12
```

## 10. QA/리스크

- 캐디피·카트비는 골프장별 필수/선택 다름 → 토글로 사용자 선택
- 클럽 세트 가격 편차 매우 큼 (30만~수백만) → 슬라이더 범위 넓게 설정
- 지역별 그린피 차이 큼 (수도권 vs 지방 2~3배) → 직접 입력 방식으로
- "너무 비싸서 포기" 반응 방지 → 중립적 표현 유지, 절감 팁 섹션 없이 순수 계산기로

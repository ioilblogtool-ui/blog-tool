## 2. 업데이트 배경

청년미래적금은 2026년 6월 출시 예정 상품으로, 기존 청년도약계좌보다 만기가 짧고 정부기여금 구조가 단순해 검색 수요가 빠르게 커질 가능성이 높다. 비교계산소에는 이미 청년미래적금 단독 리포트와 청년 적금 비교 리포트가 있으므로, 신규 페이지를 추가하기보다 **기존 페이지를 6월 출시판으로 업데이트**하는 전략이 효율적이다.

특히 6월 검색 사용자는 "상품이 뭔가"보다 다음 질문에 가까울 가능성이 높다.

- 지금 신청할 수 있나?
- 나는 일반형인가 우대형인가?
- 월 50만 원 넣으면 만기 때 얼마인가?
- 청년도약계좌에서 갈아타는 게 유리한가?
- 은행별 금리와 우대조건은 어떻게 봐야 하나?

따라서 업데이트의 중심은 정책 설명이 아니라 **조건 체크 + 만기 수령액 시나리오 + 갈아타기 판단**이다.

## 6. 계산 시나리오 설계

## 6-1. 기본 입력값

| 입력 항목 | 타입 | 설명 | 기본값 |
| --- | --- | --- | --- |
| 월 납입액 | slider + number | 1만~50만 원 | 50만 원 |
| 가입 유형 | segmented control | 일반형 / 우대형 | 일반형 |
| 적용 금리 | number | 은행별 금리 확정 전 시나리오 입력 | 6% |
| 납입 기간 | fixed | 청년미래적금 36개월 | 36개월 |
| 비과세 적용 | toggle | 청년미래적금은 비과세로 기본 적용 | 켜짐 |

## 6-2. 출력값

| 출력 항목 | 설명 | 표시 |
| --- | --- | --- |
| 납입 원금 | 월 납입액 x 36개월 | 공식 구조 |
| 정부기여금 | 일반형 6%, 우대형 12% | 공식 비율 |
| 예상 이자 | 적용 금리 기반 단순 시나리오 | 추정 |
| 비과세 효과 | 이자소득세 면제 가정 | 추정 |
| 예상 만기 수령액 | 원금 + 정부기여금 + 예상 이자 | 추정 |
| 일반형·우대형 차이 | 같은 납입액 기준 수령액 차이 | 추정 |

## 6-3. 계산 로직 예시

```text
납입 원금 = 월 납입액 x 36
정부기여금 = 납입 원금 x 정부기여금 비율
예상 이자 = 자유적립식 월 납입 복리 또는 단리 근사 모델 적용
예상 만기 수령액 = 납입 원금 + 정부기여금 + 예상 이자
```

금리 확정 전에는 금융위원회 예시처럼 6% 가정 시나리오를 기본값으로 둘 수 있으나, UI에는 반드시 `금리 가정` 또는 `추정` 배지를 붙인다.

## 14. 최종 기획 요약

청년미래적금 6월 출시판 업데이트는 신규 페이지 추가보다 기존 자산을 빠르게 최신화하는 작업이 효율적이다. 단독 리포트는 조건·신청·만기 수령액을 중심으로, 비교 리포트는 청년도약계좌와의 차이를 중심으로, 계산기는 월 납입액별 수령액 시나리오를 중심으로 역할을 나눈다.

가장 중요한 품질 기준은 `공식 기준`과 `추정 계산`을 분리하는 것이다. 정부기여금 비율과 월 납입 한도는 공식값으로 표시하되, 금리와 만기 수령액은 은행별 확정 조건에 따라 달라질 수 있으므로 `추정`, `확인 필요` 배지를 반드시 붙인다.

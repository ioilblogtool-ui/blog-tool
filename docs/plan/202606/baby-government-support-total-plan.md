# 육아 정부지원금 통합 계산기 기획 문서

> 작성일: 2026-06-18  
> 연계 페이지: birth-support-money, birth-support-total, parental-leave-pay, single-parental-leave-total 등

---

## 1. 배경 및 목적

정부의 출산·육아 지원금은 종류가 많지만 **흩어져 있어 합산이 어렵다.**

| 지원금 종류 | 현재 우리 사이트 |
|------------|----------------|
| 출산장려금 (지자체) | `/tools/birth-support-money/` 있음 |
| 출산지원금 합계 | `/tools/birth-support-total/` 있음 |
| 육아휴직 급여 | `/tools/parental-leave-pay/` 있음 |
| 한부모 육아휴직 | `/tools/single-parental-leave-total/` 있음 |
| 기타 (아동수당·부모급여·보육료 등) | **없음 — 통합 페이지 없음** |

**"아이 낳으면 정부에서 얼마나 줘?"** 라는 질문에 한 번에 답해주는 페이지가 없다. 이 계산기 하나로 해결한다.

---

## 2. 타겟 사용자

- 임신 중인 예비 부모 (지원금 미리 파악)
- 출산 직후 부모 (지금 당장 뭘 받을 수 있는지 확인)
- 둘째·셋째 계획 중인 부모 (다자녀 추가 혜택 파악)
- 정부 지원을 잘 모르는 초보 부모
- 지역별 차이가 있다는 걸 모르는 부모

---

## 3. 핵심 컨셉: "아이 정보만 입력하면, 받을 수 있는 지원금 전부 계산"

사용자가 ①아이 수 ②아이 나이 ③거주 지역 ④소득 수준을 입력하면 — **월 지원금 총합 + 연간 총합 + 항목별 상세**를 한눈에 보여준다.

## 5. 입력값

| 입력 | 옵션 |
|------|------|
| 자녀 수 | 1명 / 2명 / 3명 / 4명+ |
| 자녀별 나이 | 0세 / 1세 / 2세 / 3~5세 / 6~7세 / 8세+ (각 자녀마다) |
| 거주 지역 | 17개 광역시도 선택 |
| 보육 형태 | 가정양육 / 어린이집 / 유치원 |
| 소득 구간 | 중위소득 150% 이하 / 이상 (산모 서비스 조건 분기) |

## 6. 출력값

### KPI 카드 (상단)
- **이번 달 받는 지원금 총합** (월 기준)
- **올해 받는 지원금 총합** (연 기준)
- **출생 후 1회성 지원금** (첫만남이용권 + 출산장려금)

### 지원금 항목별 상세 테이블

| 항목 | 월 지급액 | 연 지급액 | 지급 방식 |
|------|----------|----------|----------|
| 부모급여 | 100만원 | 1,200만원 | 현금 |
| 아동수당 | 10만원 | 120만원 | 현금 |
| 보육료 지원 | 54만원 | 648만원 | 바우처 |
| ... | ... | ... | ... |
| **합계** | **XXX만원** | **XXX만원** | |

### 1회성 지원금 별도 표시
- 첫만남이용권: 200만원
- 지자체 출산장려금: XXX만원
- 임신·출산 진료비: 100만원
- **1회성 합계: XXX만원**

### 자녀별 타임라인 (나이에 따른 지원금 변화)
- 0세 → 1세 → 2세 → 3세 → 8세로 갈수록 지원금이 어떻게 변하는지 꺾은선 그래프

# 자녀 생애 증여세 계산기 웹기획서 v1

---

## 1. 문서 개요

* **문서명**: 자녀 생애 증여세 계산기 웹기획서 v1
* **서비스명**: 비교계산소
* **페이지 유형**: 계산기형 콘텐츠 + 설명형 가이드 + 내부링크 허브
* **목적**: 자녀 나이(미성년/성년), 증여 시점(평소/혼인/출산), 최근 10년 증여 이력을 입력하면 기본공제·혼인·출산 추가공제를 반영한 과세표준과 예상 증여세를 계산하고, "0원 증여세" 한도를 생애 시점별로 보여준다.
* **핵심 가치**

  * 미성년 2,000만원 / 성년 5,000만원 기본공제를 나이로 자동 판단
  * 혼인·출산 추가공제(최대 1억원, 합산 한도)까지 반영한 정확한 한도 계산
  * "아기 2천만원 → 성년 5천만원 → 결혼 1억 5천만원"까지 생애 증여 플랜표 제공
  * 0원 신고 필요 여부 안내로 자금출처 리스크 절감
  * 결혼·출산·주택 관련 계산기로 자연스럽게 확장

## 2. 페이지 목표

### 사용자 목표

* 지금 자녀에게 증여하면 세금이 얼마나 나오는지 알고 싶다
* 미성년/성년 공제 한도가 정확히 얼마인지 알고 싶다
* 결혼할 때 부모가 얼마까지 줘도 세금이 없는지 알고 싶다
* 최근에 이미 증여한 적이 있다면 한도가 얼마나 남았는지 알고 싶다
* 증여세 신고를 꼭 해야 하는 경우인지 판단하고 싶다

### 서비스 목표

* 결혼·출산·부동산 시즌마다 반복 유입되는 검색 수요 확보
* 세금/금융 카테고리 체류시간 및 애드센스 단가 개선
* 결혼자금·주택자금·자녀 자산관리 콘텐츠 허브 역할 강화
* 공유 가능한 계산 결과 URL 제공

## 5. 계산 로직

### 5-1. 기본공제

```
나이 < 19세 → 기본공제 20,000,000원
나이 >= 19세 → 기본공제 50,000,000원
```

### 5-2. 혼인·출산 추가공제

* 혼인 요건(혼인신고일 전후 2년) 또는 출산 요건(출생일부터 2년 이내) 충족 시 적용
* 혼인공제와 출산공제는 **합산 한도 1억원**으로 처리 (중복 별도 한도 아님)

```
잔여 추가공제 = max(0, 100,000,000 - 기사용액)
혼인 또는 출산 요건 충족 시 → 잔여 추가공제 전액 적용 가능
```

### 5-3. 과세표준

```
과세표준 = max(0, 이번 증여금액 + 최근 10년 증여금액 - (기본공제 + 추가공제))
```

### 5-4. 증여세 (누진세율)

| 과세표준 | 세율 | 누진공제 |
|---|---|---|
| 1억원 이하 | 10% | 0원 |
| 1억원 초과 ~ 5억원 이하 | 20% | 1,000만원 |
| 5억원 초과 ~ 10억원 이하 | 30% | 6,000만원 |
| 10억원 초과 ~ 30억원 이하 | 40% | 1억 6,000만원 |
| 30억원 초과 | 50% | 4억 6,000만원 |

```
증여세 = 과세표준 × 세율 - 누진공제
```

### 5-5. 신고 권장 여부 안내 (참고용, 세액과 무관)

* 공제 한도 이내라도 1,000만원 이상 일괄 증여, 혼인·출산공제 적용, 주택자금 목적인 경우 "0원 신고 권장" 안내 노출
* 법적 의무가 아닌 **자금출처 증빙 목적의 권장 사항**임을 명시 (단정적 표현 금지, `참고` 라벨 필수)

## 6. 결과 화면 구성

* **요약 카드**: 이번 증여금액 / 적용 공제(기본+추가) / 과세표준 / 예상 증여세
* **공제 적용 내역**: 기본공제, 혼인·출산 추가공제(잔여 한도 표시) 분리 표기
* **결과 문구**: 케이스별 자동 생성 (예: "미성년 자녀는 10년 합산 2,000만원까지 공제되어 예상 증여세는 0원입니다")
* **신고 권장 안내**: `InfoNotice.astro`로 참고용 박스 처리, "추정/참고" 라벨 명시

## 7. 차별화 섹션 — 생애 증여 플랜표

| 시점 | 증여 가능 전략 | 비과세 가능액 |
|---|---|---|
| 출생~미성년 | 자녀 명의 계좌 증여 | 2,000만원 |
| 성년 이후 | 대학·사회초년생 자금 | 5,000만원 |
| 결혼 시 | 혼인 추가공제 활용 | 1억원 |
| 출산 시 | 출산 추가공제 검토 | 혼인·출산 합산 1억원 |
| 주택 구입 시 | 기존 증여 이력 합산 체크 | 초과분 과세 |

* 입력값과 무관하게 정적 가이드 표로 우선 제공 (계산 결과와 별도 섹션)
* 향후 2차 버전에서 "내 자녀의 생애 플랜" 형태로 입력값 기반 동적 표로 확장 가능

## 9. 데이터 정확성 / 법적 표현 주의사항

* 국세청 증여재산공제·세율 기준 인용 시 출처 명시 (국세청, 정책브리핑)
* "결혼할 때 1억 줘도 세금 없다" 식 단정 표현 대신 "요건 충족 시" 조건부 표현 사용
* 부모·조부모 등 증여자가 다를 경우 10년 합산 기준이 **수증자(자녀) 기준**이며 증여자별 무한 공제가 아님을 명확히 안내 — 잘못된 안내 시 사용자 세무 리스크 발생 가능
* 모든 결과는 `추정`/`참고` 라벨 필수 (실제 신고는 세무사·홈택스 확인 권장 문구 포함)
* `docs/GOOGLE_SEO_RULES.md` 기준에 따라 SeoContent 최소 분량 충족 필요

# 구축 아파트 인테리어 견적 계산기 — 기획 문서

## 1. 배경 및 목적

구축 아파트 매수 후 올수리를 검토하는 사용자가 가장 먼저 부딪히는 문제는 "얼마나 들까?"이다. 인테리어 업체 견적은 업체마다 편차가 크고, 카탈로그 가격과 실제 시공 후 금액 차이도 크다. 사용자는 "24평 아파트 도배·마루·샷시 다 하면 얼마냐"를 구글에 치지만, 정확한 계산기가 없어 커뮤니티 후기를 뒤진다.

비교계산소 포지션 — 숫자 입력 → 최소/보통/고급 3구간 견적 즉시 출력 → 항목별 비용 분해표 → 실전 의사결정 지원.

// 기본 구조
const BASE_PLATFORM = 24; // 기준 평수

function getAreaMultiplier(pyeong) {
  // 비선형 계수 테이블 보간
}

function calculateItem(itemKey, grade, pyeong) {
  const baseCost = UNIT_PRICES[itemKey][grade]; // min/mid/high
  const multiplier = getAreaMultiplier(pyeong);
  return Math.round(baseCost * multiplier / 10000) * 10000; // 만원 단위 반올림
}

function calculateTotal(selectedItems, grades, pyeong) {
  return selectedItems.reduce((sum, item) => {
    return sum + calculateItem(item, grades[item] || 'mid', pyeong);
  }, 0);
}

## 9. SeoContent 방향

1. 구축 아파트 올수리 비용 범위 소개 (24평 기준 1,000~3,000만 원대)
2. 비용이 크게 달라지는 3대 변수 — 샷시·욕실·마루 자재 등급
3. 업체 견적과 실제 비용 차이가 나는 이유 (추가 공사, 자재 업그레이드)
4. 우선순위 결정법 — 살면서 교체 어려운 항목(샷시·욕실·마루) 먼저

1. 24평 구축 아파트 올수리하면 보통 얼마나 드나요?

## 14. 우선순위 판단

**구현 우선 이유:**
- 부동산 카테고리 계산기가 취득세·보유세·대출 위주라 "시공 비용" 카테고리가 비어 있음
- 구축 아파트 매수 검색자는 인테리어 비용을 거의 동시에 검색 → 유입 시너지
- 광고 단가가 부동산·인테리어 카테고리는 상위권
- 구현 복잡도 대비 SEO 효과 좋음

**구현 후 확장:**
이 계산기가 트래픽 잡으면 "인테리어 업체 비교" 제휴 지면으로 발전 가능.

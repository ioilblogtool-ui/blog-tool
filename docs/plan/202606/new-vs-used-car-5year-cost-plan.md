# 신차 vs 중고차 5년 총비용 비교 계산기 기획

## 배경 및 목적
- 자동차 구매 결정 시 "신차 vs 중고차 어떤 게 더 저렴한가"는 매우 높은 검색량 키워드
- 단순 가격 비교가 아닌 5년 총보유비용(TCO) 개념으로 실질 비교
- 구매 결정형 콘텐츠 → 전환율 높고 체류시간 김

## 타겟
- 첫 차 구매를 고민하는 2030
- 신차 출고 대기 중 중고차 고민하는 직장인
- 가성비 차량 구매를 원하는 실용주의 소비자

## 페이지 유형
- `SimpleToolShell` (계산기)
- slug: `new-vs-used-car-5year-cost`

## 메타 타이틀·디스크립션
- title: `신차 vs 중고차 5년 총비용 계산기 2026 | 실제로 어떤 게 더 싸?`
- description: `신차·중고차 구매가격·보험·유지비·감가 입력하면 5년 총보유비용 바로 비교. 연도별 손익분기점 포함.`

## 입력값

### 신차
- 차량 가격 (만원)
- 취득세 자동 계산 (7%)
- 첫해 보험료 (만원)
- 월 유지비 (주유·주차·소모품, 만원)
- 예상 잔존가치 5년 후 (%, 기본 40%)

### 중고차
- 차량 가격 (만원)
- 연식 (년)
- 취득세 자동 계산 (지방세 포함)
- 첫해 보험료 (만원, 신차보다 낮음)
- 월 유지비 (만원, 신차보다 높음)
- 예상 잔존가치 5년 후 (%, 기본 25%)

## 출력값 (KPI 카드 4개)
1. 신차 5년 총비용
2. 중고차 5년 총비용
3. 차액 (어느 쪽이 유리한지)
4. 손익분기점 (몇 년 차에 역전되는지)

## 섹션 구성
1. **KPI 요약** — 총비용 비교 카드
2. **연도별 누적비용 라인 차트** — 신차/중고차 교차 시점 시각화
3. **비용 항목별 상세 테이블** — 구매가, 취득세, 보험, 유지비, 감가
4. **시나리오 프리셋** — 소형차(아반떼급) / 준중형(K5급) / SUV(투싼급)
5. **SeoContent + FAQ**

## FAQ 예시
- 신차 vs 중고차 5년 기준 어떤 게 더 저렴한가?
- 중고차 감가는 얼마나 빠른가?
- 취득세는 신차 중고차 다른가?
- 5년 후 되팔 때 차액은 얼마나 나는가?

## 데이터 출처
- 취득세율: 국토교통부 (승용차 7%, 경차 4%)
- 잔존가치: 카이즈유 데이터연구소 연식별 감가율 참고 (추정)
- 보험료: 금융감독원 자동차보험 평균 참고 (추정)

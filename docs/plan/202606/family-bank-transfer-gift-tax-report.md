## 2. 콘텐츠 한 줄 정의

> 부모·자녀·부부·형제 사이의 계좌이체가 언제 생활비·차용금·공동자금으로 볼 수 있고, 언제 증여로 의심받을 수 있는지 사례와 증빙 기준으로 정리하는 리포트

## 3. 기획 배경

가족 간 계좌이체는 검색 수요가 꾸준합니다. 사용자는 세법 조문 자체보다 아래 질문에 답을 원합니다.

* 부모님이 자녀에게 보낸 돈은 얼마부터 증여세가 나오나?
* 부부간 계좌이체도 증여세가 문제될 수 있나?
* 생활비·교육비·용돈은 증여가 아닌가?
* 전세자금·주택자금 지원은 왜 위험하다고 하나?
* 빌린 돈이라고 주장하려면 차용증만 있으면 되나?

이 콘텐츠의 핵심은 "가족 간 이체 = 무조건 증여"도 아니고, "가족이라 괜찮다"도 아니라는 점을 명확히 보여주는 것입니다. 계좌이체 자체는 자금 이동의 증거이고, 실제 판단은 **이체 목적, 금액 규모, 반복성, 사용처, 상환 여부, 증빙**을 함께 봐야 합니다.

## 5. 핵심 메시지

### 사용자가 가져가야 하는 결론

1. 가족 간 계좌이체에는 "이 금액 이하면 무조건 안전" 같은 단일 기준이 없다.
2. 생활비·교육비처럼 사회통념상 인정되는 지출은 비과세 증여재산으로 볼 수 있는 영역이 있다.
3. 전세자금·주택구입자금·투자금·목돈 이전은 증여로 의심받기 쉽다.
4. 빌린 돈이라면 차용증, 이자 지급, 원금 상환 기록, 계좌 메모가 함께 필요하다.
5. 증여로 처리할 돈이라면 관계별 10년 합산 공제한도를 먼저 확인해야 한다.

### 상단 요약 문구

```md
가족 간 계좌이체는 금액만으로 판단하지 않습니다. 생활비인지, 빌린 돈인지, 증여인지에 따라 세금 판단이 달라지고, 전세자금·주택자금·투자금처럼 목돈이 오간 경우에는 증빙이 특히 중요합니다.
```

## 7. 핵심 섹션 설계

### 7-1. 빠른 판정표

| 이체 유형 | 증여 리스크 | 리포트 해석 |
|---|---|---|
| 생활비·교육비 목적의 정기 이체 | 낮음~중간 | 실제 생활비로 사용됐는지 중요 |
| 명절 용돈·소액 축의금 | 낮음 | 사회통념상 인정 범위인지 확인 |
| 자녀 전세보증금 지원 | 높음 | 목돈 이전, 자금출처 조사 가능성 |
| 주택 구입자금 지원 | 매우 높음 | 증여 또는 차용 증빙 필요 |
| 주식·코인 투자금 지원 | 높음 | 수익 귀속과 자금 출처 이슈 |
| 부모 병원비 대납 | 낮음~중간 | 부양의무·사용처 증빙 중요 |
| 가족 간 돈 빌려줌 | 중간~높음 | 차용증, 이자, 상환 기록 필요 |
| 부부 생활비 이체 | 낮음 | 생활 공동체 비용인지 확인 |
| 부부 명의 자산 취득자금 이전 | 중간~높음 | 명의와 자금 출처 불일치 주의 |

### 7-2. 관계별 기준

#### 부모 → 자녀

핵심 포인트:

* 생활비·교육비와 자산 취득자금을 구분해야 한다.
* 성인 자녀에게 큰돈을 보내 전세보증금, 주택 구입, 투자에 사용하면 증여 이슈가 커진다.
* 증여로 처리한다면 성인 자녀는 10년 합산 5,000만 원, 미성년 자녀는 2,000만 원 공제를 검토한다.

대표 문장:

```md
부모가 자녀에게 보낸 돈이 실제 생활비로 사용됐다면 증여세 이슈가 작을 수 있지만, 자녀 명의의 전세보증금·주택·투자자산으로 남는다면 증여로 볼 가능성이 커집니다.
```

#### 자녀 → 부모

핵심 포인트:

* 부모 생활비·병원비 지원은 실제 부양 목적과 사용처가 중요하다.
* 부모 명의 자산 취득자금으로 쓰이면 증여 이슈가 생길 수 있다.
* 직계존속 공제는 10년 합산 5,000만 원 기준을 검토한다.

#### 부부 간

핵심 포인트:

* 부부 생활비·공동 생활자금은 일반적인 거래로 볼 여지가 있다.
* 배우자 명의 주택·주식·금융자산 취득자금으로 큰돈이 이동하면 증여 검토가 필요하다.
* 배우자 증여재산공제는 10년 합산 6억 원.

#### 형제·친족

핵심 포인트:

* 형제간 생활비 지원은 부모·자녀보다 설명이 더 필요할 수 있다.
* 기타 친족 공제는 10년 합산 1,000만 원 기준을 검토한다.
* 목돈 대여는 차용증과 상환 기록을 남겨야 한다.

## 10. 사례 구성

### 사례 1. 부모가 성인 자녀에게 매달 100만 원 생활비 이체

해석:

* 실제 월세, 식비, 교육비 등 생활비로 사용됐다면 리스크는 상대적으로 낮다.
* 다만 사용하지 않고 예금·투자자산으로 축적되면 증여로 볼 여지가 생긴다.

### 사례 2. 부모가 자녀 전세보증금 1억 원 지원

해석:

* 목돈이 자녀 명의 보증금으로 남기 때문에 증여 리스크가 높다.
* 증여로 신고할지, 차용으로 처리할지, 일부 증여+일부 차용으로 나눌지 검토가 필요하다.

### 사례 3. 부부간 생활비 계좌로 매달 300만 원 이체

해석:

* 공동 생활비 계좌이고 실제 생활비 지출로 사용된다면 일반적인 생활자금으로 설명 가능하다.
* 배우자 명의 투자계좌로 이동해 금융자산이 쌓이는 구조라면 별도 검토가 필요하다.

### 사례 4. 형이 동생에게 사업자금 5,000만 원 이체

해석:

* 기타 친족 공제 범위를 초과할 수 있어 증여 또는 차용 판단이 중요하다.
* 사업자금 대여라면 차용증, 이자, 상환 일정, 실제 상환 내역이 필요하다.

### 사례 5. 자녀가 부모 병원비 2,000만 원 대납

해석:

* 부양 목적과 실제 병원비 사용 내역이 명확하면 리스크는 낮아질 수 있다.
* 부모 명의 자산 취득이나 예금으로 남는 돈과는 구분해야 한다.

| 가족 간 계좌이체는 얼마부터 증여세가 나오나요? | 금액만으로 결정되지 않으며 관계별 공제한도와 이체 목적, 사용처, 증빙을 함께 봄 |
| 부모님이 자녀에게 생활비를 보내면 증여인가요? | 실제 생활비·교육비로 사용되는 통상 범위는 비과세 가능성이 있으나 축적되면 이슈 |
| 부부간 계좌이체도 증여세가 나오나요? | 생활비는 리스크 낮지만 배우자 명의 자산 취득자금은 증여 검토 |
| 자녀 전세자금을 부모가 지원하면 증여세가 나오나요? | 목돈 지원이라 증여 또는 차용 증빙 필요 |
| 차용증만 쓰면 증여세를 피할 수 있나요? | 차용증뿐 아니라 이자 지급과 원금 상환 기록이 중요 |
| 가족 간 무이자 대여도 괜찮나요? | 무이자·저리 대여는 이자 상당액 증여 이슈 검토 필요 |
| 계좌이체 메모는 도움이 되나요? | 단독 증빙은 아니지만 목적 설명 자료로 도움 |
| 현금으로 주면 더 안전한가요? | 오히려 자금 출처와 사용처 증빙이 어려울 수 있음 |
| 증여세 신고를 안 하면 언제 문제가 되나요? | 부동산 취득, 세무조사, 자금출처 확인 과정에서 이슈 가능 |
| 가족 간 돈 거래는 몇 년치 기록을 보관해야 하나요? | 최소 10년 합산 공제와 자금출처 소명을 고려해 장기 보관 권장 |

# 리스 vs 할부 vs 현금 구매 손익 계산기 기획

## 배경 및 목적
- "자동차 리스 할부 현금 뭐가 나아요?" 는 자동차 구매자들의 공통 고민
- 단순 비교 글이 많지만 개인 조건에 맞는 계산기는 거의 없음
- 소득공제, 기회비용까지 반영한 심화 계산기로 차별화

## 타겟
- 자동차 구매를 앞둔 직장인
- 리스 절세 효과가 궁금한 사업자/프리랜서
- 목돈 vs 월납 고민하는 2040

## 페이지 유형
- `CompareToolShell` (3방향 비교)
- slug: `car-purchase-method-comparison`

## 메타 타이틀·디스크립션
- title: `자동차 리스 vs 할부 vs 현금 비교 계산기 2026 | 내 조건엔 뭐가 유리?`
- description: `차량가격·금리·운용기간 입력하면 리스·할부·현금 구매 총비용 바로 비교. 직장인·사업자 절세 효과 포함.`

## 입력값 (공통)
- 차량 가격 (만원)
- 보유 기간 (년, 1~7)
- 현금 보유 여부 (있음/없음)
- 직장인 / 사업자 선택

### 리스
- 리스 기간 (만원)
- 보증금 비율 (%, 기본 20%)
- 월 리스료 (만원)
- 잔존가치 (%, 기본 30%)

### 할부
- 선납금 (만원)
- 할부 기간 (개월)
- 할부 금리 (%, 기본 5.5%)

### 현금
- 기회비용 금리 (%, 기본 4% — 예금 대신 차에 투자)

## 출력값 (KPI 카드)
1. 리스 총비용
2. 할부 총비용
3. 현금 총비용
4. 최저 비용 방식 강조

## 섹션 구성
1. **3방향 총비용 KPI 비교**
2. **연도별 누적비용 라인 차트**
3. **항목별 비용 breakdown 테이블**
4. **절세 효과 안내** (사업자: 리스료 전액 경비처리 vs 할부이자만 가능)
5. **추천 케이스 가이드** — 현금 있음+단기 보유 → 현금, 사업자 → 리스, 목돈 없음 → 할부
6. **SeoContent + FAQ**

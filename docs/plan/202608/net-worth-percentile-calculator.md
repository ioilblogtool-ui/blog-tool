## 1. 페이지 목적

부동산, 전세보증금, 예금·적금, 국내주식, 해외주식, 코인, 연금, 부채를 입력하면 **순자산(자산-부채)** 을 계산하고, 통계청 가계금융복지조사 순자산 분포를 기준으로 **전국 순자산 상위 몇 %에 해당하는지**를 추정해 보여주는 계산기다.

핵심 차별점은 단순 순위 표시가 아니라, 이미 서비스 중인 `korea-rich-top10-assets` 리포트와 연결해 "나는 상위 몇 %, 한국 부자 TOP10과는 이만큼 차이난다"는 **연속적인 스토리**를 만드는 것이다.

> 이 계산기는 통계청 가계금융복지조사 순자산 분포를 기준으로 한 참고용 추정이며, 표본 조사 특성상 개인별 정확한 순위를 보장하지 않는다.

### 3-2. 구현 전 반드시 재확인할 항목

- **10분위 경계값 전체 표**: 이번 조사에서 1~10분위 각각의 순자산 "평균"은 일부만 공개됐고, 분위 "경계값"(예: 상위 20% 진입에 필요한 순자산)은 국가데이터처 KOSIS 통계표 원자료에서 별도로 확인해야 한다. 이 문서의 수치만으로는 정밀한 백분위 함수를 만들 수 없으므로, 구현 착수 시 KOSIS(kosis.kr) "가계금융복지조사 > 순자산 10분위별 가구당 순자산" 표를 직접 조회해 경계값을 채운다.
- 2026년 가계금융복지조사는 2026년 3월 말 기준으로 진행 중이며 결과는 아직 미발표(통상 매년 12월 발표) — 발표되면 데이터를 갱신한다.
- 이미 유사한 계산기(`bebooja.com/calculator/korea-asset-2026`)가 존재하는 것으로 확인됨 — 차별화는 "부자 TOP10 리포트 연결"과 "자산 구성 분해"에 둔다.

### 3-3. MVP 백분위 산정 방식

정확한 10분위 경계값 확보 전까지는 아래 방식으로 MVP를 구성한다.

```text
1. 확보된 분위 지점(1분위 -771만 원, 전체 평균 4억 7,144만 원, 10분위 평균 21억 7,122만 원)을
   기준으로 로그정규분포 근사 곡선을 만든다.
2. 사용자 순자산을 이 근사 곡선에 대입해 백분위를 추정한다.
3. 결과 화면에는 "표본 조사 기반 추정치이며 정밀한 순위가 아님"을 명확히 배지로 표시한다.
```

구현 담당자는 KOSIS 원자료로 경계값을 확보하면 근사 곡선 대신 **구간별 실측 표**로 교체한다.

## 6. 계산 로직

```text
총자산 = 부동산 + 전세보증금 + 예금적금 + 국내주식 + 해외주식 + 코인 + 연금
순자산 = 총자산 - 부채
부채비율 = 부채 / 총자산
부동산비중 = (부동산 + 전세보증금) / 총자산
백분위 = 로그정규 근사 곡선(순자산) — 3-3 참고
```

### 결과 판정 배지

| 배지 | 조건 |
|---|---|
| 상위 1% 근접 | 상위 1~3% 구간 |

| 평균 이상 | 전체 평균(4.7억 원) 초과 ~ 상위 10% 미만 |
| 평균 이하 | 전체 평균 이하 |

## 11. FAQ 예시

### Q1. 이 계산기의 순위는 정확한가요?
아닙니다. 통계청 가계금융복지조사 표본 데이터를 기준으로 한 추정치이며, 실제 전 국민 대상 정밀 순위가 아닙니다.

### Q2. 전세보증금도 자산에 포함되나요?
네. 받을 전세보증금은 자산으로, 낸 전세자금대출은 부채로 계산합니다.

### Q3. 연금도 자산에 넣어야 하나요?
연금은 즉시 현금화가 어려운 자산이라 참고용으로만 포함했습니다. 결과 해석 시 이 점을 감안해야 합니다.

### Q4. 순자산이 마이너스면 어떻게 되나요?
2025년 조사 기준 하위 10% 가구의 평균 순자산도 -771만 원으로 마이너스입니다. 부채 초과는 드문 상황이 아닙니다.

### Q5. 부동산 시세는 어떻게 넣어야 하나요?
실거래가나 KB시세 등 최근 시세를 참고해 시가 기준으로 입력하는 것을 권장합니다.

## 14. 최종 기획 요약

이 계산기는 이미 서비스 중인 **한국/세계 부자 TOP10 리포트**의 자연스러운 다음 단계 콘텐츠다. 부자 이야기를 읽은 사용자가 바로 이어서 "그럼 나는?"을 확인할 수 있게 하여 리포트-계산기 간 내부 트래픽 흐름을 완성한다.

차별화 포인트는 세 가지다.

1. 통계청 공식 조사 데이터 기반의 신뢰도 있는 백분위 추정.

## 8. 디딤돌대출 vs 보금자리론 비교표

| 구분      |                                         디딤돌대출 |              보금자리론 |
| ------- | --------------------------------------------: | -----------------: |
| 주요 대상   |                                   무주택 서민·실수요자 |    무주택 또는 1주택 실수요자 |
| 주택가격 기준 |                    5억 원 이하, 신혼·2자녀 이상 6억 원 이하 |            6억 원 이하 |
| 소득 기준   | 부부합산 6천만 원 이하, 생애최초·2자녀 이상 7천만 원, 신혼 8,500만 원 |      부부합산 7천만 원 이하 |
| 생애최초 한도 |                                        2.4억 원 |             4.2억 원 |
| LTV     |                     기본 최대 70%, 생애최초 일부 80% 가능 |             최대 70% |
| DTI     |                                        60% 이내 |             60% 이내 |
| 만기      |                                  10·15·20·30년 | 10·15·20·30·40·50년 |
| 적합한 경우  |                  저가 주택, 소득 기준 충족, 금리 민감한 실수요자 |  대출 한도가 더 필요한 실수요자 |

디딤돌대출은 생애최초 주택구입자 한도 2.4억 원, 신혼·2자녀 이상 가구는 3.2억 원까지 가능하며, 보금자리론은 생애최초 한도 4.2억 원으로 안내되어 있습니다. ([한국주택금융공사][3])

## 9. 신생아 특례대출 연계 섹션

### 섹션 제목

**출산가구라면 생애최초보다 신생아 특례가 더 유리할 수도 있다**

| 항목      | 내용                               |
| ------- | -------------------------------- |
| 대상      | 대출접수일 기준 2년 내 출산한 무주택 세대주        |
| 소득 기준   | 부부합산 연소득 1.3억 원 이하, 맞벌이는 2억 원 이하 |
| 자산 기준   | 순자산가액 기준 적용                      |
| 전세자금 금리 | 연 1.3%~4.3%                      |
| 전세자금 한도 | 최대 2.4억 원 이내                     |

주택도시기금의 신생아 특례 버팀목대출은 대출접수일 기준 2년 내 출산 가구를 대상으로 하며, 부부합산 연소득 1.3억 원 이하, 맞벌이 2억 원 이하 기준이 제시되어 있습니다. ([마이홈][4])

> 리포트에서는 “생애최초 구입자”와 “출산가구”가 겹치는 경우, 디딤돌·보금자리론·신생아 특례 중 어떤 조합이 유리한지 별도 비교 박스로 보여주는 구성이 좋습니다.

## 10. 생애최초 특별공급 섹션

### 섹션 제목

**청약 가점이 낮아도 생애최초 특별공급은 기회가 될 수 있다**

| 항목     | 내용                             |
| ------ | ------------------------------ |
| 기본 요건  | 세대 구성원 전원이 과거 주택 소유 이력이 없어야 함  |
| 청약 자격  | 국민주택 일반공급 1순위 요건 등 충족 필요       |
| 저축 기준  | 국민주택 기준 선납금 포함 600만 원 이상 요건 존재 |
| 가족 요건  | 혼인 중이거나 자녀가 있는 사람 등            |
| 소득세 요건 | 근로자 또는 자영업자로 5년 이상 소득세 납부 요건   |
| 선정 방식  | 소득 구간별 우선공급·일반공급·추첨공급 구조       |

생애최초 특별공급은 청약 제도 내에서 별도 자격 요건을 요구하며, 국민주택 일반공급 1순위, 저축액, 혼인·자녀 여부, 5년 이상 소득세 납부 등 세부 조건을 확인해야 합니다. ([이지법률][5])

LH 분양가이드 기준으로 생애최초 특별공급은 소득 구간에 따라 우선공급 70%, 일반공급 20%, 추첨공급 10% 구조로 설명되어 있어, “소득이 낮을수록 우선공급에서 유리하고, 일부는 추첨 기회도 있다”는 메시지를 줄 수 있습니다. ([LH 신청 사이트][6])

## 16. FAQ 초안

### Q1. 생애최초 주택 구입이면 무조건 취득세가 면제되나요?

아닙니다. 생애최초 여부만으로 자동 면제되는 것이 아니라 주택가격, 소득, 세대원 주택 보유 이력, 취득 시점 등 세부 요건을 충족해야 합니다. 2026년 기준 생애최초 취득세 감면 운영기준은 행정안전부 고시로 확인해야 합니다. ([모바일 정부][1])

### Q2. 디딤돌대출과 보금자리론 중 뭐가 더 좋나요?

소득과 주택가격 기준을 충족한다면 디딤돌대출이 유리할 수 있지만, 대출 한도가 더 필요하면 보금자리론이 선택지가 될 수 있습니다. 디딤돌대출 생애최초 한도는 2.4억 원, 보금자리론 생애최초 한도는 4.2억 원으로 안내되어 있습니다. ([한국주택금융공사][3])

### Q3. 생애최초와 신생아 특례대출은 같이 볼 수 있나요?

네. 출산가구라면 생애최초 주택 구입 혜택과 신생아 특례대출 조건을 함께 비교해야 합니다. 신생아 특례는 대출접수일 기준 2년 내 출산 여부, 소득, 자산 기준이 핵심입니다. ([마이홈][4])

### Q4. 1인 가구도 생애최초 특별공급이 가능한가요?

일부 조건에서는 가능하지만, 공급 유형과 면적 제한 등 세부 요건이 달라질 수 있습니다. 생애최초 특별공급은 청약통장, 소득세 납부, 무주택 이력, 소득 기준 등을 함께 확인해야 합니다. ([이지법률][5])

### Q5. 지자체별 생애최초 지원금은 어디서 확인하나요?

시·군·구청 홈페이지, 복지로, 주거복지센터, 청년정책포털, 지자체 공고문을 확인해야 합니다. 지자체 지원금은 지역별 예산과 공고 시점에 따라 수시로 바뀌므로 리포트에서는 “대표 사례 + 확인 방법” 중심으로 구성하는 것이 안전합니다.

## 1. 페이지 목적

출산 전 예비 부모가 수유 방식(완전 모유수유 / 혼합 수유 / 완전 분유)에 따른 실질 비용 차이를 미리 계산해볼 수 있는 도구다.

단순한 분유값 비교가 아니라, 유축기·수유패드 등 모유수유 초기 용품 비용, 분유 브랜드별 월 지출, 그리고 손익분기점(몇 개월 시점에 모유수유 쪽이 저렴해지는지)까지 함께 보여준다.

> "유축기 사는 게 맞을까, 렌탈이 나을까?" "완모가 12개월이면 분유보다 얼마나 아낄 수 있을까?"

이 두 가지 질문에 명확하게 답해주는 것이 핵심이다.

## 5. 계산 로직

### 월령별 1일 분유 권장 소비량 (기준 데이터)

| 월령     | 1일 권장량(ml) | 1일 분유 사용량(g, 기준 1스쿱=30ml) |
| ------ | ---------- | ------------------------- |
| 0~1개월  | 약 480ml    | 약 48g                    |
| 2~3개월  | 약 600ml    | 약 60g                    |
| 4~5개월  | 약 720ml    | 약 72g                    |
| 6~7개월  | 약 600ml    | 약 60g (이유식 병행 시작)         |
| 8~9개월  | 약 480ml    | 약 48g                    |
| 10~12개월 | 약 360ml    | 약 36g                    |

### 분유 브랜드별 단가 기준 (추정치, `추정` 레이블 필수)

| 브랜드 등급  | 800g 기준 가격(추정) | 월 캔 소비 기준       |
| -------- | -------------- | --------------- |
| 일반       | 약 25,000~30,000원 | 월 1~2캔         |
| 유기농      | 약 38,000~45,000원 | 월 1~2캔         |
| 프리미엄     | 약 55,000~70,000원 | 월 1~2캔         |

```text
월 분유 비용 = (월령별 1일 분유 사용량 × 30일) ÷ 800g × 브랜드 단가

혼합 수유 분유 비용 = 월 분유 비용 × 혼합 비율(기본 50%)

모유수유 월 비용 = 소모품 비용 (수유패드·저장팩 등, 기본 15,000원~25,000원)
                 + 유축기 렌탈인 경우 월 렌탈비

유축기 손익분기점(개월) = 유축기 구매 비용 ÷ (월 분유 비용 - 월 소모품 비용)
```

## 14. FAQ 예시

### Q1. 모유수유는 무조건 분유보다 저렴한가요?

유축기 구매 비용, 수유 소모품, 모유수유 상담비 등을 포함하면 초반에는 분유보다 비쌀 수 있습니다. 일반적으로 3~4개월 이후부터 누적 비용이 분유보다 낮아지는 경향이 있습니다.

### Q2. 혼합 수유를 하면 비용이 절반으로 줄어드나요?

혼합 비율에 따라 다르지만, 50:50 혼합 기준으로 분유 소비량이 줄어 완분 대비 약 40~50% 절감 효과를 기대할 수 있습니다. 다만 소모품 비용은 계속 발생합니다.

### Q3. 이유식을 시작하면 분유량이 줄어드나요?

네. 일반적으로 6개월 이후 이유식을 시작하면 분유 1일 섭취량이 줄어들기 시작합니다. 이 계산기는 월령별 권장 분유량을 반영해 자동으로 조정합니다.

### Q4. 유축기는 얼마나 쓰면 구매가 렌탈보다 유리한가요?

전동 유축기 구매 비용(약 20만 원) 기준으로 월 렌탈비 약 2만 원이면 10개월이 손익분기점입니다. 12개월 완모를 계획한다면 구매가 유리합니다.

### Q5. 프리미엄 분유와 일반 분유 가격 차이가 얼마나 나나요?

800g 기준 일반 분유 약 25,000~30,000원, 프리미엄 약 55,000~70,000원으로 2배 이상 차이가 날 수 있습니다. 12개월 기준 총비용 차이는 약 50만 원 이상이 될 수 있습니다. (추정)

### 월령별 1일 분유 사용량 데이터

```ts
const dailyFormulaByAge: Record<number, number> = {
  0: 48, 1: 48,
  2: 60, 3: 60,
  4: 72, 5: 72,
  6: 60, 7: 60,
  8: 48, 9: 48,
  10: 36, 11: 36,
};
```

### 계산 함수 예시

```ts
function calcMonthlyFormulaCost(age: number, gradePrice: number): number {
  const dailyGrams = dailyFormulaByAge[age] ?? 48;
  const monthlyGrams = dailyGrams * 30;
  return (monthlyGrams / 800) * gradePrice;
}

function calcBreakEvenMonth(pumpCost: number, monthlyFormulaCost: number, monthlySuppliesCost: number): number | null {
  const monthlyDiff = monthlyFormulaCost - monthlySuppliesCost;
  if (monthlyDiff <= 0) return null;
  return Math.ceil(pumpCost / monthlyDiff);
}
```

## 19. 최종 기획 요약

모유수유 vs 분유 비용 계산기는 출산/임신 카테고리에서 반복 방문 가능성이 높은 도구다. 아기 월령이 바뀔 때마다 재계산을 유도할 수 있고, 수유 방식을 고민하는 예비 부모의 의사결정을 직접 돕는다는 점에서 체류 시간·공유율이 높을 것으로 예상된다.

특히 "유축기 구매가 렌탈보다 언제 유리한가"라는 질문은 검색 의도가 명확하고 정량적 답변을 원하는 사용자가 많아, 손익분기점 계산을 전면에 내세우는 것이 핵심 차별점이 된다.

MVP는 정적 데이터 기반으로 빠르게 구현하고, 이후 실시간 분유 가격 반영, 혼합 비율 세부 조정, 지역별 유축기 렌탈 가격 비교 등으로 확장하는 방향을 권장한다.

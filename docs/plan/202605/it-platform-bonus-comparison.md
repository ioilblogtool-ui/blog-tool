## 2. 콘텐츠 포지셔닝

IT 플랫폼 성과급은 반도체·자동차와 구조가 완전히 다르다. 현금 성과급 외에 RSU(양도제한조건부주식)·스톡옵션·이익분배금·특별격려금이 섞이고, 상장사와 비상장사(토스), 국내상장과 해외상장(쿠팡)이 혼재한다. 기존 비교 콘텐츠는 이 차이를 구분하지 않아 독자 혼란이 크다.

| 기존 비교 콘텐츠 문제점 | 이 도구의 방향 |
| --- | --- |
| "카카오 성과급 OO원" 단편 기사 | 현금 PI + RSU + 스톡옵션 + 계열사별 차이 분리 |
| 단순 연봉 순위 나열 | 총보상(Total Compensation) 구성비 시각화 |
| 상장·비상장 차이 무시 | 주가 연동 보상의 특수성과 리스크 명시 |
| 스톡옵션을 성과급처럼 표기 | 베스팅 일정·행사 조건·세금 구조 별도 안내 |
| 개인 연봉 적용 불가 | 입력 연봉 기준 회사별 예상 총보상 산출 |

- PI 지급 기준 (영업이익·목표 달성 연동 구조)
- RSU 부여 기준 및 베스팅 일정 (3년·4년 클리프)

- 네이버파이낸셜·웹툰·라인웍스 계열사별 차이
- 개발직 vs 기획·마케팅 직군 편차

- RSU·스톡옵션 부여 현황 (공시 기반)
- 계열사별 차이가 크므로 본사 기준 표기 원칙

### ⑥ 쿠팡 성과급 구조와 2026 현황
- 미국상장사로 SEC Filing 기반 공시 구조
- RSU가 총보상의 핵심: 부여 기준·베스팅·주가 연동 리스크
- 현금 성과급(연간 보너스) 규모와 지급 조건
- 쿠팡이츠·쿠팡플레이 등 사업 부문별 차이 가능성

### ⑦ 크래프톤 성과급 구조와 2026 현황
- 배틀그라운드 IP 중심 매출 연동 PI 구조
- RSU·스톡옵션 부여 현황 (IR·공시 기반)
- 게임 성적·글로벌 매출에 따른 성과급 변동성
- 개발 직군 집중형 인재 확보 전략과 보상 수준

### ⑧ 라인플러스 성과급 구조와 2026 현황
- LY Corp(구 라인야후) 자회사로 글로벌 인사 정책 적용
- 일본 본사 실적·엔/원 환율 연동 요소
- 재직자 후기 기반 성과급 수준 (추정값)
- 네이버와의 비교: 같은 뿌리지만 다른 보상 체계

### ⑨ 토스(비바리퍼블리카) 성과급 구조와 2026 현황
- 비상장사 특성: 현금 성과급보다 스톡옵션(ESOP)이 핵심
- 스톡옵션 행사 시 기대 수익 vs 미상장 유동성 리스크
- 채용공고·재직자 커뮤니티 기반 성과급 수준 (추정)
- 상장 기대감과 보상 전략 연결

### ⑩ RSU·스톡옵션·현금 PI: 세금과 리스크가 다르다
- 현금 PI: 근로소득 과세, 즉시 수령
- RSU: 베스팅 시점에 근로소득 과세, 주가 연동 리스크
- 스톡옵션: 행사 시 근로소득(비상장) or 양도소득(상장 후) 과세
- 각 유형의 세전/세후 체감 비교 예시 (시뮬레이션)

- 주가 상승/하락 시나리오별 RSU 실수령 차이
- "현금만 보면 언더카운트, 주식만 보면 오버카운트" 메시지

### ⑫ 취준생·이직자 관점의 IT 플랫폼 보상 해석법

## 12. FAQ 초안

| IT 플랫폼 성과급은 반도체보다 적은가요? | 현금 PI는 반도체 대기업 대비 적을 수 있으나, RSU·스톡옵션 포함 총보상(TC)은 회사·직군·경력에 따라 다름 |

| RSU를 받으면 세금은 어떻게 내나요? | 베스팅 시점에 시가 기준 근로소득으로 과세되며, 이후 매도 시 양도소득세 별도 |
| 이직할 때 RSU를 포기하면 손해가 큰가요? | 미베스팅 RSU의 잔여 금액과 이직 패키지를 비교해야 하며, 클리프 직전 이직은 특히 손실 클 수 있음 |

## 16. 최종 콘텐츠 방향

이 도구는 IT 플랫폼 재직자와 이직 준비자가 **"성과급 포함 총보상을 제대로 비교하는 법"**을 배우는 허브 페이지다.

검색 유입은 `카카오 성과급`, `네이버 성과급 2026`, `쿠팡 RSU 수준`, `크래프톤 성과급`, `토스 스톡옵션`, `IT 기업 성과급 비교` 키워드로 받는다.

반도체·자동차 성과급 리포트와 달리, IT 플랫폼은 **현금보다 주식 보상의 비중과 리스크**가 핵심이다. 이 차이를 독자가 이해하도록 돕고, 개인 연봉 기준 시뮬레이션으로 전환하며, 세금·이직 타이밍·베스팅 리스크까지 연결하는 깊이가 경쟁 페이지와의 차별점이다.

성과급 수치를 크게 보여주되, 반드시 `배지`, `보상 유형 구분`, `리스크 안내`를 함께 제공하는 방향을 유지한다.

## 2) 기획 배경

연금 관련 콘텐츠는 꾸준히 높은 검색 수요를 갖는다. 특히 "공무원 연금이 국민연금보다 얼마나 유리한가"는 취준생·이직자 커뮤니티에서 반복적으로 등장하는 질문이다.

하지만 기존 콘텐츠는 대부분:
1. 단일 연금 제도 소개 (각각 따로 설명)
2. 제도 구조 중심 (납입률·수급 연령 등 나열)
3. 같은 조건으로 비교한 수령액 시뮬레이션 없음

비교계산소에서는 이를 다음과 같이 차별화한다.

- **같은 기준선**: 월 기준소득 300만 원, 20년 납입이라는 동일 조건
- **수령액 직접 비교**: 4개 직종 연금의 예상 월 수령액을 같은 표에 나열
- **납입 부담 비교**: 본인 부담 납입률이 달라 실제 '순이익'이 다름
- **20년 vs 30년**: 납입 기간별 수령액 격차 시각화
- **손익분기점**: 몇 세까지 살아야 본전인지 직종별 비교

이 콘텐츠는 연금 제도 설명이 아니라 **"어떤 직종에서 일하면 연금을 얼마나 받나"를 숫자로 직접 보는 비교 리포트**로 포지셔닝한다.

## 3) 기획 목표

### 1차 목표

- 4개 직종 연금을 동일 조건으로 비교해 "공무원 연금이 정말 얼마나 다른지" 숫자로 보여준다.
- `공무원연금 국민연금 비교`, `군인연금 수령액`, `직종별 연금` 키워드 상위 노출을 목표로 한다.

### 2차 목표

- SNS·커뮤니티(블라인드·인사이트·공시생 카페) 공유 가능한 비교표 콘텐츠로 활용한다.
- 기존 국민연금 세대별 비교 리포트와 시리즈로 연결해 연금 콘텐츠 허브 구축한다.

### 3차 목표

- 직종 선택 → 연금 → 노후 계획 흐름으로 사이트 체류 시간 확장
- "연금·노후 비교 사이트" 포지셔닝 강화

## 5) 핵심 콘텐츠 구성

### 섹션 1. 히어로 요약
- 4개 직종 연금 이름 + 한 줄 핵심 차이
- KPI 카드: 20년 납입 기준 최고 수령액 직종 / 최저 수령액 직종 / 수령액 격차

### 섹션 2. 4개 연금 제도 핵심 비교표
- 연금 이름 / 가입 대상 / 본인 납입률 / 지급 개시 나이 / 수급 방식 / 현재 상태(개혁 여부)

### 섹션 3. 같은 월급 기준 납입 부담 비교
- 월 기준소득 300만 원 기준 본인 부담 납입액 비교
- 국가(고용주) 부담분 포함 총 납입액 비교

### 섹션 4. 20년 납입 기준 예상 월 수령액 비교 (핵심)
- 4개 직종 동일 조건(월 300만 원, 20년 납입) 기준 예상 월 수령액
- CSS bar row 시각화
- 30년 납입 시 비교도 추가

### 섹션 5. 납입 기간별 수령액 격차 변화
- 10년 / 20년 / 30년 납입 기준 직종별 수령액 변화
- Chart.js 꺾은선 또는 묶음 막대 차트

### 섹션 6. 손익분기점 비교
- "몇 세까지 살아야 납입 총액을 회수하나"
- 직종별 손익분기 연령 비교

### 섹션 7. 연금개혁 영향 요약
- 2025~2026 공무원·국민연금 개혁 주요 변경 사항
- 개혁 전·후 수령액 변화 포인트 간단 정리

### 섹션 8. 데이터 기준 및 면책
- 인사혁신처·국민연금공단·군인연금공단·사학연금공단 공개 자료 기준
- 개인 소득·재직기간·물가 반영 방식에 따라 실제 수령액 다를 수 있음

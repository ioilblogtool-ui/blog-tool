## 3. LG전자 성과급 구조 정리

### 핵심 용어

| 용어 | 의미 | 지급 주기 |
| --- | --- | --- |
| **PI** (Performance Incentive) | 사업부 목표 달성률 기반 인센티브 | 상반기(7월경) / 하반기(1월경) |
| **OB** (Outperformance Bonus) | 목표 초과 달성 시 추가 지급 | 비정기 (실적 연계) |
| **임금협상 타결금** | 임협 타결 시 지급되는 일시금 | 연 1회 |
| **기본급** | PI 계산 기준이 되는 월 기본급 | 매월 지급 |

### 사업부 구조

| 코드 | 사업부명 | 주요 제품 | 2026 실적 전망 |
| --- | --- | --- | --- |
| **H&A** | Home Appliance & Air Solution | 냉장고·세탁기·에어컨 | 꾸준한 실적, PI 100~200% 구간 |
| **HE** | Home Entertainment | TV·모니터·사이니지 | TV 시장 부진, PI 변동 폭 큼 |
| **VS** | Vehicle component Solutions | 전기차 부품·인포테인먼트 | 흑자 전환 과도기, PI 불확실 |
| **BS** | Business Solutions | B2B·상업용 디스플레이 | 상대적 안정, PI 50~150% 구간 |

> 모든 지급률은 보도 및 추정 기반 시뮬레이션 값이며 확정 지급률과 다를 수 있습니다.

### PI 산식 (참고)

```
PI 지급액 = 월 기본급 × PI 지급률(%)
연간 PI = 상반기 PI + 하반기 PI
```

- 지급률은 사업부 목표 달성률에 따라 0~200% 범위에서 결정
- 개인 성과 반영 여부는 사업부·직급별로 상이
- 기본급은 연봉 구성 항목 중 순수 기본급 기준 (각종 수당 제외)

| 하단 CTA 그리드 | 직장인 재테크, ETF·예금 콘텐츠 |

광고와 계산 결과는 명확히 분리한다. 특정 사업부 입사를 권유하거나 확정 성과급을 보장하는 문구는 사용하지 않는다.

## 13. 품질 체크리스트

- [ ] 추정·시뮬레이션 배지가 계산기 상단과 결과 카드에 표시된다.
- [ ] 확정 지급률처럼 오해할 수 있는 표현이 없다.
- [ ] 사업부 4개(H&A·HE·VS·BS) 모두 선택 가능하다.
- [ ] 직급별 기본급 preset이 적용되고 직접 수정도 가능하다.
- [ ] 세후 추정값은 참고값으로만 안내한다.
- [ ] 삼성전자·SK하이닉스·현대차 계산기 내부 링크가 연결된다.
- [ ] 대기업 성과급 시뮬레이터(`/tools/bonus-simulator/`)에서 LG전자가 추가되거나 CTA로 연결된다.
- [ ] 모바일에서 사업부 셀렉트와 슬라이더가 사용하기 편하다.
- [ ] FAQ가 계산기 하단에 노출된다.
- [ ] `npm run build` 성공, 빌드 에러 없음.

`SimpleToolShell.astro` — 입력 + 결과 카드 구조

`/tools/bonus-simulator/` 에 LG전자 항목을 추가하거나, 계산기 하단 CTA로 시뮬레이터를 연결한다. 기획 확정 후 bonus-simulator 데이터 파일 업데이트 필요.

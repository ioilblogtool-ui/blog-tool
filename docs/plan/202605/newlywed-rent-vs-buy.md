## 1. 페이지 목적

신혼부부가 가장 현실적으로 부딪히는 의사결정 — **"지금 전세로 살아야 하나, 무리해서 매매할까?"** — 를 숫자로 풀어주는 계산기.

단순히 월 부담액 비교에 그치지 않고, **기회비용(보증금 운용 수익)**, **집값 상승 시나리오**, **손익분기 전환 시점(몇 년째에 매매가 유리해지는지)** 까지 한 화면에서 확인할 수 있게 구성한다.

신혼부부 특례대출·생애최초 대출 조건 입력란을 포함해 실제 대출 연계 전환을 자연스럽게 유도한다.

> "보증금 3억 운용 vs 매매 3억 대출이자, 5년 뒤 어느 쪽이 더 이득이지?"
> "집값이 연 3% 오르면 언제부터 매매가 유리해지나?"

### 기간별 누적 비용 비교표
- 1년 / 3년 / 5년 / 10년 / 입력기간 누적 순비용 (전세 vs 매매)
- 각 연도별 전세 총비용 = 대출이자 + 기회비용
- 각 연도별 매매 총비용 = 대출이자 + 취득세 초기비용 – 집값 상승분

### 시나리오별 라인 차트
- X축: 거주 연수 (1~20년)
- Y축: 누적 순비용
- 전세 곡선 vs 매매 곡선 → 교차점 = 손익분기 전환 시점

### 자연어 결과 메시지
```
보증금 3억 / 매매가 5억 기준,
현재 설정값에서는 전세가 5년간 약 X만 원 저렴합니다.
집값이 연 2% 오른다고 가정하면 X년째부터 매매가 더 유리해집니다.
신혼부부 특례대출(3.0% → 적용 시 2.1%) 이용 시 매매 전환 시점이 X년 단축됩니다.
```

## 6. 계산 로직

### 전세 시나리오 연간 순비용
```
전세 대출 이자 = 전세 대출금 × 전세 대출 금리
보증금 기회비용 = (전세 보증금 – 전세 대출금) × 보증금 운용 수익률
전세 연간 총비용 = 전세 대출 이자 + 보증금 기회비용
```
- 보증금은 만기 회수 → 거주 종료 시점에 전액 복귀 (순자산에 반영)

### 매매 시나리오 연간 순비용
```
매매 대출 이자 = 대출 원금 × 금리 (원리금균등 기준 월 상환액 × 12에서 원금 상환분 제외)
취득세·등기비 (초기 1회) = 매매가 × 취득세율 + 등기비 추정
집값 상승분 = 매매가 × (1 + 연 상승률)^N – 매매가
매매 연간 순비용 = 대출 이자 – 집값 상승분/N
```

### 손익분기 전환 시점
```
N년차 전세 누적 총비용 > N년차 매매 누적 총비용(대출이자) – N년차 집값 상승 누적분
→ 처음으로 매매가 유리해지는 N을 계산
```

### 취득세율 자동 적용
```
6억 이하: 1.1%
6억 초과 ~ 9억 이하: 비례 (1~3%)
9억 초과: 3.3%
생애최초·신혼부부 감면: 최대 200만원 한도 별도 안내
```

## 2. 콘텐츠 핵심 컨셉

**부동산 취득세 계산기**는 매매가, 주택 수, 조정대상지역 여부, 생애최초 여부 등을 입력하면 **취득세·지방교육세·농어촌특별세를 합산한 예상 납부세액**을 계산해주는 도구입니다.

주택 유상거래 취득세는 기본적으로 취득가액 기준으로 산정되며, 6억 원 이하 주택은 1%, 6억 초과~9억 이하 구간은 산식 적용, 9억 초과 주택은 3% 세율이 적용됩니다. 다만 다주택자·조정대상지역 여부에 따라 중과세율이 적용될 수 있습니다. ([이지법률][1])

## 5. 세율 로직 설계

### 5-1. 기본 주택 취득세율

| 구간                |                     기본 취득세율 |
| ----------------- | --------------------------: |
| 6억 원 이하           |                          1% |
| 6억 원 초과 ~ 9억 원 이하 | `(취득가액 × 2/3억 - 3) × 1/100` |
| 9억 원 초과           |                          3% |

주택 유상거래의 경우 위 구간별 세율이 적용되며, 6억~9억 구간은 단일 2%가 아니라 가격에 따라 1~3% 사이에서 산식으로 계산됩니다. ([이지법률][1])

### 5-2. 다주택자 중과 분기

| 보유 상황     | 조정대상지역         | 계산 방향    |
| --------- | -------------- | -------- |
| 무주택 → 1주택 | 무관             | 기본세율     |
| 1주택 → 2주택 | 비조정지역          | 기본세율 가능  |
| 1주택 → 2주택 | 조정대상지역         | 중과 가능    |
| 2주택 → 3주택 | 조정대상지역 / 비조정지역 | 중과 가능    |
| 3주택 이상    | 무관             | 고율 중과 가능 |

1세대 2주택이 조정대상지역 주택을 취득하거나, 1세대 3주택 이상에 해당하는 경우에는 지방세법상 중과세율 적용 대상이 될 수 있습니다. ([이지법률][1])

## 11. 개발 구현 로직 예시

```ts
type PropertyType = 'apartment' | 'house' | 'officetel' | 'commercial' | 'land';

interface AcquisitionTaxInput {
  propertyType: PropertyType;
  price: number;
  areaM2?: number;
  currentHouseCount: 0 | 1 | 2 | 3;
  isRegulatedArea: boolean;
  isFirstHomeBuyer: boolean;
}

interface AcquisitionTaxResult {
  acquisitionTaxRate: number;
  acquisitionTax: number;
  localEducationTax: number;
  ruralSpecialTax: number;
  discountAmount: number;
  totalTax: number;
  totalCashNeeded: number;
  messages: string[];
}
```

```ts
function getBasicHouseTaxRate(price: number): number {
  const eok = price / 100_000_000;

  if (eok <= 6) return 0.01;
  if (eok <= 9) {
    return ((eok * 2 / 3) - 3) / 100;
  }
  return 0.03;
}
```

```ts
function calculateAcquisitionTax(input: AcquisitionTaxInput): AcquisitionTaxResult {
  const baseRate = getBasicHouseTaxRate(input.price);

  let acquisitionTaxRate = baseRate;
  const messages: string[] = [];

  if (input.currentHouseCount >= 1 && input.isRegulatedArea) {
    messages.push('조정대상지역 다주택 취득은 중과세율 적용 가능성이 있습니다.');
  }

  const acquisitionTax = input.price * acquisitionTaxRate;

  const localEducationTax = acquisitionTax * 0.1;

  const ruralSpecialTax =
    input.areaM2 && input.areaM2 > 85
      ? input.price * 0.002
      : 0;

  const discountAmount = input.isFirstHomeBuyer
    ? Math.min(acquisitionTax, 2_000_000)
    : 0;

  const totalTax =
    acquisitionTax + localEducationTax + ruralSpecialTax - discountAmount;

  return {
    acquisitionTaxRate,
    acquisitionTax,
    localEducationTax,
    ruralSpecialTax,
    discountAmount,
    totalTax,
    totalCashNeeded: input.price + totalTax,
    messages,
  };
}
```

## 13. FAQ 초안

### Q1. 부동산 취득세는 언제 내나요?

부동산을 취득한 날부터 60일 이내에 신고·납부해야 합니다. 위택스를 통한 신고·납부도 가능합니다. ([이지법률][1])

### Q2. 6억~9억 주택은 취득세율이 2%인가요?

아닙니다. 6억 초과~9억 이하 주택은 가격에 따라 1~3% 사이에서 산식으로 계산됩니다. ([이지법률][1])

### Q3. 생애최초 주택 구입자는 취득세가 감면되나요?

2026년에도 생애최초 주택 구입 취득세 감면 운영기준이 고시되어 있으며, 세부 요건은 행정안전부 고시와 지방세특례제한법 기준을 확인해야 합니다. ([모바일 정부][2])

### Q4. 조정대상지역이면 무조건 취득세가 비싼가요?

무조건은 아닙니다. 다만 1세대 2주택 이상에서 조정대상지역 주택을 취득하는 경우 중과세율 적용 가능성이 커집니다. ([이지법률][1])

## 2. 콘텐츠 목적

이 페이지는 회사별 성과급을 공식 지급액처럼 단정하는 문서가 아니라, 사용자가 입력한 연봉·월급·성과급률을 기준으로 자동차 기업의 예상 성과급을 같은 기준에서 비교하는 시뮬레이터다.

기존 현대자동차 성과급 계산기는 현대차 단일 회사의 상세 계산에 집중한다. 이 페이지는 그 상위 허브로, 현대차·기아·현대모비스·한국GM·르노코리아를 한 화면에서 비교하고 상세 계산기로 이동시키는 역할을 한다.

핵심 질문:

- 같은 연봉이라면 현대차와 기아 성과급 차이는 얼마나 날까?
- 현대차그룹 계열사와 외국계(한국GM·르노코리아) 성과급은 어떻게 다를까?
- 자동차 업계 성과급을 세후 기준으로 월평균 환산하면 실제 차이는 어느 정도일까?
- 임단협 요구안과 확정 합의안을 구분해서 봐야 하는 이유는 무엇인가?

## 7. 계산 로직

### 7-1. 세전 성과급

```text
월급 n개월 모드:
세전 성과급 = 월 기본급 * 개월 수

연봉 대비 % 모드:
세전 성과급 = 기준 연봉 * 성과급률

고정 금액 모드:
세전 성과급 = 입력한 고정 금액
```

### 7-2. 자사주·정액 현금 포함 총보상

```text
현금 성과급 총계 = 세전 성과급 + 정액 현금 (격려금 등)
자사주 환산액 = 자사주 주수 * 현재 주가 (입력 기준)
세전 총보상 성과급 = 현금 성과급 총계 + 자사주 환산액
```

자사주는 취득 시점·처분 조건·세금 처리 방식이 달라 현금 성과급과 직접 비교하지 않고 참고용으로만 표시한다.

### 7-3. 간편 세후 추정

```text
예상 공제액 = 세전 성과급 * 추정 공제율
예상 세후 성과급 = 세전 성과급 - 예상 공제액
실수령률 = 예상 세후 성과급 / 세전 성과급
```

권장 추정 공제율:

| 연봉 구간 | 기본 추정 공제율 |
| --- | ---: |
| 5,000만 원 이하 | 12% |
| 5,000만 원 초과 ~ 8,000만 원 이하 | 18% |
| 8,000만 원 초과 ~ 1.2억 원 이하 | 24% |
| 1.2억 원 초과 ~ 2억 원 이하 | 30% |
| 2억 원 초과 | 36% |

주의:

- 실제 원천징수액은 지급월 급여, 부양가족, 비과세 항목, 4대보험, 연말정산 결과에 따라 달라진다.
- 이 페이지는 세무 계산기가 아니라 회사별 성과급 비교 목적의 간편 시뮬레이터다.
- 상세 세후 계산은 향후 `/tools/bonus-after-tax-calculator/`로 연결한다.

### 7-4. 비교 결과

```text
월평균 환산액 = 예상 세후 성과급 / 12
성과급 포함 총보상 = 기준 연봉 + 세전 성과급
세후 기준 최고 회사와 차이 = 최고 예상 세후 성과급 - 해당 회사 예상 세후 성과급
현대차 대비 차이 = 해당 회사 예상 세후 성과급 - 현대차 예상 세후 성과급
```

### 9-3. 결과 비교

- KPI 카드
- 회사별 비교 표
- 세후 성과급 막대 차트
- 월평균 환산액 비교

### 9-4. 자동차 성과급 용어 정리

| 용어 | 주로 쓰이는 회사 | 의미 | 주의사항 |
| --- | --- | --- | --- |
| 격려금 | 현대차, 기아 등 | 임단협 결과 지급하는 일시금 | 정액·주식·상품권 혼합 가능 |
| 성과급 | 여러 회사 | 회사 실적 연동 보상 | 산식이 회사별로 다름 |
| 자사주 | 현대차, 기아 등 | 임단협 패키지의 자사주 지급 | 처분 조건·취득 세금 별도 확인 필요 |
| 임단협 | 자동차 업계 전반 | 임금 및 단체협약 교섭 | 요구안·제시안·잠정합의·확정안 구분 필수 |
| 요구안 | 노조 | 교섭 전·중 노조 측 요구 | 확정된 지급액 아님 |

### 9-5. 비교할 때 주의할 점

- 성과급률이 높아도 기준이 연봉인지 기본급인지에 따라 체감 금액이 달라진다.
- 세전 성과급과 통장 입금액은 다르다.
- 자사주는 주가 변동에 따라 가치가 달라지므로 현금 성과급과 단순 합산하지 않는다.
- 같은 회사라도 직군, 직급, 평가, 사업부에 따라 결과가 달라질 수 있다.
- 노사 협의안, 요구안, 보도된 추정치를 확정 지급액처럼 보면 안 된다.
- 이 페이지는 사용자 입력값 기반 비교 도구이며 실제 지급액 보장을 의미하지 않는다.

## 10. FAQ

### 현대차와 기아 성과급은 어떻게 비교해야 하나요?

같은 현대차그룹 계열이지만 임단협은 별도로 진행된다. 격려금·자사주·정액 현금 패키지 구성이 다를 수 있으므로 동일 기준 연봉으로 세전·세후·월평균을 함께 비교해야 한다.

### 현대모비스 성과급은 현대차와 얼마나 다른가요?

현대차그룹 계열사이지만 완성차 대비 성과급 패키지 구성이 다를 수 있다. 공개된 확정 지급액이 없는 경우 성과급률 직접 입력 방식으로 비교한다.

### 한국GM이나 르노코리아 성과급도 계산할 수 있나요?

가능하다. 외국계 자동차사는 국내 대기업과 노사 구조가 달라 직접 비교가 어려우므로 사용자가 성과급률, 월급 개월 수, 고정 금액을 직접 입력해 비교하는 방식으로 제공한다.

### 자사주는 어떻게 계산에 반영하나요?

주가 기준 환산액을 참고용으로 표시하되, 처분 조건과 취득 세금이 현금 성과급과 다르므로 현금 비교와 분리해서 보여준다.

### 자동차 성과급 계산 결과는 실제 지급액과 같은가요?

아니다. 결과는 사용자가 입력한 조건을 바탕으로 한 시뮬레이션이다. 실제 지급액은 회사, 직급, 평가, 임단협 합의 결과, 세금 및 공제 방식에 따라 달라질 수 있다.

### 임단협 요구안을 기준으로 계산하면 안 되나요?

가능은 하지만 요구안은 확정 지급액이 아니므로 `요구안` 배지를 붙이고 시나리오로만 사용한다. 확정 합의안과 요구안을 혼동하지 않도록 선택 시 배지가 자동 표시된다.

## 12. 데이터·표현 안전 기준

반드시 지켜야 할 기준:

- 추정값에는 `추정` 또는 `시뮬레이션` 배지를 표시한다.
- 노조 요구안, 언론 보도, 커뮤니티 추정치를 확정 지급액처럼 표현하지 않는다.
- 임단협 요구안은 반드시 `요구안` 배지를 붙이고 확정 합의안과 분리한다.
- 회사별 성과급률 기본값을 둘 경우 출처와 기준일을 함께 표시한다.
- 자사주는 현금 성과급과 단순 합산하지 않고 참고 영역으로 분리한다.

```text
이 계산기는 사용자가 입력한 연봉과 성과급률을 기준으로 한 비교용 시뮬레이션입니다. 실제 지급액은 회사, 직급, 평가, 임단협 합의 결과, 세금 및 공제 방식에 따라 달라질 수 있습니다.
```

피해야 할 문구:

- `2026 현대차 성과급 확정`
- `기아 실제 지급액`
- `현대모비스 공식 성과급률`
- `임단협 요구안대로 지급`
- 출처 없는 `평균 지급률`

## 14. 1차 MVP 범위

1차는 검색 반응을 빠르게 확인할 수 있도록 아래 범위로 제한한다.

- 현대차, 기아, 현대모비스, 한국GM, 르노코리아 비교
- 기준 연봉 입력
- 회사별 성과급 방식 선택 (월급 n개월, 연봉 대비 %, 고정 금액)
- 자사주·정액 현금 추가 입력 (현대차·기아)
- 세전 성과급, 예상 세후, 월평균 환산, 총보상 출력
- 현대차 상세 계산기 내부 링크
- 용어 정리, 주의사항, FAQ 포함

2차 확장:

- 현대차그룹 추가 계열사 (현대트랜시스, 현대위아 등)
- 자동차 부품 독립 계열사 (만도, 한온시스템 등)
- 세후 계산기와 입력값 연동
- 업종별 성과급 비교 허브와 연결
- 검색 유입 데이터 기반 회사 카드 재정렬

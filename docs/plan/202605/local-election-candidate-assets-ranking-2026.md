## 2. 콘텐츠 포지셔닝

이 리포트는 특정 후보를 지지하거나 공격하는 정치 기사형 콘텐츠가 아니라, **중앙선거관리위원회 공개자료를 기준으로 후보자 재산 신고액을 구조화해 보여주는 데이터형 비교 콘텐츠**다.

| 일반 선거 기사 | 비교계산소 리포트 방향 |
| --- | --- |
| 특정 후보 중심 이슈 보도 | 전체 후보 중 재산 신고액 상위 50명 랭킹 |
| “부자 후보” headline | 총재산, 부동산, 예금, 증권, 채무 분리 |
| 의혹·평가성 문장 중심 | 신고자료 기준의 중립적 숫자 비교 |
| 지역별 파편화 | 광역·기초·교육감·재보궐 후보를 필터로 탐색 |
| 원문 접근 어려움 | 선관위 공개자료와 확인 경로를 함께 안내 |

### 핵심 차별점

1. **총재산 TOP 50**을 빠르게 훑고, 후보별 자산 구성을 함께 보여준다.
2. **부동산 보유액 TOP**, **채무 TOP**, **순자산 추정 TOP** 등 보조 랭킹으로 검색면을 넓힌다.
3. “재산 많음 = 문제”처럼 단정하지 않고, **공개 신고액의 구조와 기준**을 설명한다.
4. 선거 이후에도 `지방선거 후보 재산`, `후보 재산 조회`, `후보자 공개자료 보는 법` 에버그린 유입을 유지한다.

## ⑦ 채무가 큰 후보와 순자산 해석

### 핵심 내용

재산 신고자료에서는 총재산과 채무를 함께 봐야 한다. 고가 부동산을 보유했지만 채무도 큰 후보는 총액만으로 해석하면 왜곡될 수 있다.

### 산식

```text
순자산 추정 = 총재산 신고액 - 채무 신고액
부동산 비중 = 부동산 신고액 / 총재산 신고액
금융자산 비중 = (예금 + 증권) / 총재산 신고액
```

### 배지 예시

- `부동산중심`
- `금융자산중심`
- `채무동반`
- `순자산주의`
- `부분공개`

## 17. 최종 콘텐츠 방향

이 페이지는 선거 시즌의 짧은 트래픽을 노리는 단발성 기사보다, **공식 후보자 공개자료를 비교계산소식으로 정리한 데이터 허브**로 만드는 것이 맞다.

검색 유입은 `지방선거 후보 재산`, `후보 재산 순위`, `후보자 재산 조회`, `후보 부동산`으로 받고, 본문에서는 총재산 랭킹뿐 아니라 부동산·금융자산·채무·지역 분포까지 분해한다. 숫자는 크게 보여주되, 표현은 최대한 차분하게 유지한다. 선거 콘텐츠의 핵심은 “누가 나쁘다”가 아니라 “공개된 숫자를 유권자가 이해하기 쉽게 정리한다”는 방향이다.

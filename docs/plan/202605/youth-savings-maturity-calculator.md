## 1. 페이지 목적

`청년 적금 만기 수령액 계산기`는 월 납입액, 금리, 만기, 정부기여금, 비과세 여부를 입력해 **청년미래적금·청년도약계좌·일반 적금의 예상 만기 수령액**을 한 번에 비교하는 계산기다.

청년미래적금 리포트는 조건과 제도 이해에 초점을 둔다. 이 계산기는 그 다음 질문인 **"그래서 내 월 납입액이면 어떤 적금이 가장 유리한가"**를 해결한다.

정책금융 상품은 가입 자격과 은행별 우대조건이 달라질 수 있으므로, 결과는 `추정`으로 표시하고 공식 공고·상품설명서 확인을 필수 고지한다.

---

## 2. 콘텐츠 한 줄 정의

> 월 납입액과 금리를 입력하면 청년미래적금, 청년도약계좌, 일반 적금의 원금·이자·정부기여금·세후 만기 수령액을 비교해주는 청년 정책적금 계산기

## 4. 콘텐츠 포지셔닝

이 계산기는 단순 예금이자 계산기가 아니라 **정책지원금 포함 비교 계산기**로 포지셔닝한다.

| 일반 적금 계산기 | 청년 적금 만기 수령액 계산기 |
| --- | --- |
| 원금 + 이자만 계산 | 원금 + 이자 + 정부기여금 + 비과세 효과 계산 |
| 상품 1개 계산 | 청년미래적금·청년도약계좌·일반 적금 동시 비교 |
| 금리만 중요 | 가입유형, 소득구간, 우대금리, 만기 유지가 함께 중요 |
| 결과 금액만 표시 | 왜 차이가 나는지 항목별 분해 |
| 세전 기준이 많음 | 세전·세후·비과세 차이 표시 |

### 핵심 차별화

1. 정책 적금 2개와 일반 적금을 한 화면에서 비교한다.
2. 정부기여금을 별도 항목으로 분해해 보여준다.
3. “최고금리 기준”과 “기본금리 기준” 결과 차이를 같이 보여준다.
4. 청년도약계좌 기존 가입자에게 유지·전환 판단 체크리스트를 제공한다.
5. 계산 결과마다 `추정`, `공식`, `공시`, `확인 필요` 배지를 붙인다.

## 8. 계산 로직

### 8-1. 공통 적금 이자

월 납입식 적금의 단순 이자 추정:

```text
월 이율 = 연 금리 ÷ 12
납입회차별 이자 = 월 납입액 × 월 이율 × 남은 개월 수
예상 이자 = 모든 납입회차별 이자의 합
```

단순화 공식:

```text
예상 이자 = 월 납입액 × (개월 수 × (개월 수 + 1) ÷ 2) × (연 금리 ÷ 12)
```

세후 이자:

```text
세후 이자 = 비과세 적용 ? 예상 이자 : 예상 이자 × (1 - 15.4%)
```

### 8-2. 청년미래적금

기본값:

| 항목 | 값 |
| --- | ---: |
| 만기 | 36개월 |
| 월 납입 한도 | 500,000원 |
| 일반형 정부기여금 | 납입 원금의 6% |
| 우대형 정부기여금 | 납입 원금의 12% |
| 기본금리 | 5.0% |
| 우대금리 | 2.0~3.0% |
| 과세 | 비과세 가정 |

계산식:

```text
적용 월 납입액 = min(입력 월 납입액, 500,000)
원금 = 적용 월 납입액 × 36
정부기여금 = 원금 × 기여금율
금리 = 기본금리 + 우대금리
세후 이자 = 적금 이자(금리, 36개월) // 비과세 가정
만기 수령액 = 원금 + 정부기여금 + 세후 이자
```

주의:

- 우대형은 중소기업 재직, 신규 취업, 소상공인 매출 등 세부 조건 확인 필요.
- 은행별 우대금리 충족 여부에 따라 결과가 달라진다.
- 결과에는 `추정` 배지 사용.

### 8-3. 청년도약계좌

기본값:

| 항목 | 값 |
| --- | ---: |
| 만기 | 60개월 |
| 월 납입 한도 | 700,000원 |
| 금리 | 6.0% 기본값 |
| 정부기여금 | 소득구간별 월 최대 기여금 |
| 과세 | 비과세 가정 |

소득구간 기본 테이블:

| 개인소득 구간 | 월 기여금 산정 기준 | 지급률 | 월 최대 기여금 |
| --- | ---: | ---: | ---: |
| 총급여 2,400만 원 이하 | 40만 원 | 6.0% | 24,000원 |
| 총급여 3,600만 원 이하 | 50만 원 | 4.6% | 23,000원 |
| 총급여 4,800만 원 이하 | 60만 원 | 3.7% | 22,000원 |
| 총급여 6,000만 원 이하 | 70만 원 | 3.0% | 21,000원 |
| 총급여 7,500만 원 이하 | 없음 | 0% | 0원 |

계산식:

```text
적용 월 납입액 = min(입력 월 납입액, 700,000)
원금 = 적용 월 납입액 × 60
월 정부기여금 = min(적용 월 납입액, 기여금 산정 기준) × 지급률
정부기여금 = min(월 정부기여금, 월 최대 기여금) × 60
세후 이자 = 적금 이자(금리, 60개월) // 비과세 가정
만기 수령액 = 원금 + 정부기여금 + 세후 이자
```

주의:

- 청년도약계좌는 가입 시점, 은행, 금리 구조, 중도해지 조건에 따라 실제 결과가 달라질 수 있다.
- 2026년 기준 신규 가입 종료·전환 안내는 공식 공고 기준으로 업데이트한다.

### 8-4. 일반 적금

기본값:

| 항목 | 값 |
| --- | ---: |
| 만기 | 36개월 |
| 월 납입액 | 입력값 전체 |
| 금리 | 3.5% |
| 과세 | 15.4% |

계산식:

```text
원금 = 월 납입액 × 만기개월
세전 이자 = 적금 이자(금리, 만기개월)
세금 = 세전 이자 × 15.4%
세후 이자 = 세전 이자 - 세금
만기 수령액 = 원금 + 세후 이자
```

## ⑦ FAQ

### Q1. 계산 결과가 실제 만기 수령액과 같나요?

아닙니다. 금리, 우대조건, 납입일, 중도해지 여부, 정부기여금 지급 방식에 따라 달라지는 추정값입니다.

### Q2. 청년미래적금 우대형은 누구나 선택해도 되나요?

아닙니다. 중소기업 재직, 신규 취업, 소상공인 매출, 소득 기준 등 세부 조건을 충족해야 합니다.

### Q3. 청년도약계좌는 2026년에도 계산해야 하나요?

기존 가입자나 전환을 고민하는 사용자는 유지했을 때의 예상 만기 수령액을 비교할 필요가 있습니다.

### Q4. 일반 적금 금리는 세전으로 입력하나요?

기본 입력은 세전 연 금리입니다. 세금 반영을 켜면 이자소득세 15.4%를 단순 차감합니다.

### Q5. 월 납입액이 한도를 넘으면 어떻게 계산하나요?

정책 적금은 상품별 월 납입 한도까지만 반영하고, 초과분은 계산에서 제외하거나 일반 적금 보조 납입으로 분리할 수 있게 설계합니다.

## 2. 지금 사이트에서의 추천 방향

청년미래적금은 현재 사이트에서 **리포트 먼저, 계산기 확장** 흐름이 가장 적합하다.

| 방식 | 장점 | 주의점 |
| --- | --- | --- |
| 리포트 단독 | 출시 전 검색 수요를 빠르게 잡을 수 있음 | 만기 수령액 계산 니즈를 충분히 흡수하기 어려움 |
| 계산기 단독 | 비교계산소 브랜드와 가장 잘 맞음 | 정책 세부 조건 변경 시 유지보수 부담 큼 |
| 리포트 + 계산기 확장 | 정보성 검색과 계산 의도를 모두 흡수 | 초기에는 `발표 기준`, `은행별 확인 필요` 배지 필요 |

### 추천 결론

1. 1차로 `/reports/youth-future-savings-2026/` 리포트를 만든다.
2. 은행별 금리와 우대조건이 확정·공시되면 `/tools/youth-future-savings-calculator/`를 만든다.
3. 리포트 안에서는 계산기 영역을 “간단 시뮬레이션” 수준으로 먼저 제공하고, 이후 독립 계산기로 분리한다.

## 3. 콘텐츠 포지셔닝

일반 블로그 글처럼 “청년미래적금 좋다/나쁘다”로 쓰지 말고, 비교계산소답게 **내 조건에서 가입 가능성 → 받을 수 있는 기여금 → 만기 수령액 → 청년도약계좌와 비교** 순서로 설계한다.

| 일반 안내 콘텐츠 | 비교계산소 콘텐츠 방향 |
| --- | --- |
| 가입 조건 나열 | 나이·소득·가구소득·직장 유형별 체크리스트 |
| 최대 2,200만 원 강조 | 일반형·우대형·월 납입액별 만기 수령액 비교 |
| 은행 금리 표 복사 | 기본금리, 공통 우대금리, 은행별 우대조건 분리 |
| 청년도약계좌와 단순 비교 | 갈아타기 가능 여부와 유지/전환 판단 포인트 정리 |
| 신청 방법 안내 | 신청 전 확인해야 할 서류·소득·중도해지 리스크 안내 |

### 차별화 포인트

- `나는 일반형인가, 우대형인가?`를 카드형 체크로 보여준다.
- `월 10만/30만/50만 원` 납입 시나리오별 만기 수령액을 표로 제공한다.
- 청년도약계좌 가입자에게는 `그대로 유지`, `청년미래적금 전환 검토`, `중도해지 주의`를 나눠 설명한다.
- 확정되지 않았거나 은행별로 다른 조건은 `은행별 확인 필요` 배지로 처리한다.

## ⑧ FAQ

### Q1. 청년미래적금은 누구나 가입할 수 있나요?

아닙니다. 나이, 개인소득, 가구소득, 금융소득 종합과세 여부, 직장 또는 사업 유형 등 조건을 확인해야 합니다.

### Q2. 월 50만 원을 꼭 넣어야 하나요?

월 최대 50만 원 한도 안에서 자유롭게 납입하는 구조로 설명하는 것이 좋습니다. 다만 만기 수령액을 최대로 보려면 납입액이 중요합니다.

### Q3. 일반형과 우대형의 가장 큰 차이는 무엇인가요?

정부기여금 비율 차이가 핵심입니다. 일반형은 납입액의 6%, 우대형은 12% 기준으로 설명하되, 우대형은 세부 조건 확인이 필요합니다.

### Q4. 청년도약계좌에서 갈아타는 게 무조건 유리한가요?

무조건 유리하다고 단정하면 안 됩니다. 기존 납입 기간, 적용 혜택, 전환 가능 기간, 본인의 청년미래적금 유형을 함께 봐야 합니다.

### Q5. 이자소득세는 내야 하나요?

비과세 혜택이 안내되어 있으나 실제 적용은 가입 조건과 상품 약관을 확인해야 합니다.

## 2) 기획 배경

청년 대상 정부 지원 저축 상품은 청년희망적금 → 청년도약계좌 → 청년미래적금 순으로 계속 변화해왔다.
문제는 상품이 바뀔 때마다 조건이 달라지고, 기존 상품과의 차이를 한눈에 파악할 수 있는 콘텐츠가 없다는 점이다.

대부분의 관련 정보는:
1. 금융위원회·서민금융진흥원 공식 페이지 (복잡하고 읽기 어려움)
2. 단건 기사 (하나씩 나와 비교 불가)
3. 블로그 (오래된 정보, 가입 조건 오류 많음)

비교계산소에서는 이를 다음과 같이 차별화한다.

- 3개 상품 조건을 동일 기준 테이블로 나란히 비교
- 소득 구간별(연 2400만 / 3600만 / 5000만 원) 유불리 분석
- 월 70만 원 기준 만기 수령액 시뮬레이션
- "지금 청년도약계좌 가입 중인데 미래적금으로 바꿔야 하나?" 같은 실질 Q&A

## 3) 기획 목표

### 1차 목표

- 청년미래적금 신상품 출시 시의성을 활용해 검색 유입을 확보한다.
- 3개 상품을 동일 기준으로 비교해 "어느 상품이 낫나" 질문에 직접 답한다.
- `청년미래적금 비교`, `청년미래적금 청년도약계좌 차이` 키워드 상위 노출을 목표로 한다.

### 2차 목표

- SNS/커뮤니티(블라인드·에브리타임·재테크 카페) 공유 가능한 비교표 콘텐츠로 활용한다.
- 향후 청년 재테크 콘텐츠 시리즈(ISA·IRP·연금저축 비교) 확장 기반을 만든다.

### 3차 목표

- 청년 타겟 방문자 유입 → 연봉·투자 계산기 체류 연결
- "청년 재테크 비교 사이트"로 비교계산소 인지도 강화

## 5) 핵심 콘텐츠 구성

### 섹션 1. 히어로 요약
- 3개 상품 이름 + 한 줄 핵심 차이
- KPI 요약 카드: 최대 정부 지원금 / 월 납입 한도 / 만기 기간

### 섹션 2. 3개 상품 핵심 비교표 (가로 나열)
- 상품명 / 출시연도 / 가입 대상 / 소득 요건 / 월 납입 한도 / 금리 / 정부 기여금 / 비과세 여부 / 만기 / 현재 가입 가능 여부

### 섹션 3. 소득 구간별 유불리 분석
- 연 2,400만 원 이하 / 3,600만 원 이하 / 5,000만 원 이하 3개 구간
- 각 구간별 추천 상품 + 이유 한 줄

### 섹션 4. 월 70만 원 기준 만기 수령액 시뮬레이션
- 3개 상품 동일 납입 조건 기준 만기 시 예상 수령액 비교
- 정부 기여금 포함 / 이자 포함 / 세후 기준

### 섹션 5. 중복 가입 가능 여부
- 청년미래적금 + 청년도약계좌 동시 가입 가능?
- ISA·IRP와 중복 가입 여부

### 섹션 6. 자주 하는 실수 / 주의사항
- 소득 요건 착각 (개인소득 vs 가구소득)
- 중도 해지 시 불이익
- 가입 가능 기간 (종료 전 서두르기 필요한 경우)

### 섹션 7. 데이터 기준 및 면책
- 금융위원회·서민금융진흥원 공개 자료 기준
- 금리·지원금은 변동 가능하므로 가입 전 재확인 필요

## 1. 페이지 목적

코인 DCA 수익 계산기는 사용자가 비트코인, 이더리움, XRP 등 주요 코인을 매월 일정 금액씩 적립식으로 매수했을 때의 누적 투자금, 추정 보유 수량, 평균 매수 단가, 현재 평가금액, 수익률을 계산해주는 도구다.

핵심은 단순한 수익률 계산이 아니라, 사용자가 가장 궁금해하는 질문에 바로 답하는 것이다.

## 5. 계산 로직

### 기본 계산식

```text
월별 매수 수량 = 월 투자금 ÷ 해당 월 코인 가격

총 보유 수량 = 월별 매수 수량의 합계

누적 투자금 = 월 투자금 × 투자 개월 수

평균 매수 단가 = 누적 투자금 ÷ 총 보유 수량

현재 평가금액 = 총 보유 수량 × 현재 코인 가격

총 손익 = 현재 평가금액 - 누적 투자금

수익률 = 총 손익 ÷ 누적 투자금 × 100
```

## 14. FAQ 예시

### Q1. 코인 DCA는 무조건 수익이 나나요?

아닙니다. DCA는 매수 시점을 분산해 평균 단가를 낮추는 전략일 뿐, 코인 가격이 장기간 하락하면 손실이 발생할 수 있습니다.

### Q2. 매일 사는 것과 매월 사는 것 중 뭐가 좋나요?

가격 변동성이 큰 코인은 매일 또는 매주 매수 방식이 평균 단가를 더 부드럽게 만들 수 있습니다. 다만 일반 투자자는 월급일 기준 월 1회 방식이 관리하기 쉽습니다.

### Q3. BTC와 ETH 중 DCA에 더 적합한 코인은 무엇인가요?

안정성을 중시하면 BTC, 성장성과 변동성을 감수할 수 있다면 ETH도 선택지가 될 수 있습니다.

### Q4. 알트코인도 DCA가 괜찮나요?

시가총액이 낮거나 생존 가능성이 불확실한 알트코인은 장기 DCA에 적합하지 않을 수 있습니다. DCA는 장기간 살아남을 가능성이 높은 자산일수록 효과가 큽니다.

## 15. 개발 구현 포인트

### 데이터 구조 예시

```ts
type CoinDcaInput = {
  coinSymbol: 'BTC' | 'ETH' | 'XRP' | 'SOL';
  monthlyAmount: number;
  startMonth: string;
  endMonth: string;
  currentPrice: number;
  feeRate?: number;
};

type CoinMonthlyPrice = {
  month: string;
  price: number;
};

type CoinDcaResult = {
  totalInvestment: number;
  totalQuantity: number;
  averageBuyPrice: number;
  currentValue: number;
  profitAmount: number;
  profitRate: number;
  breakEvenPrice: number;
};
```

### 계산 함수 예시

```ts
function calculateCoinDca(
  input: CoinDcaInput,
  monthlyPrices: CoinMonthlyPrice[]
): CoinDcaResult {
  const feeRate = input.feeRate ?? 0;

  let totalInvestment = 0;
  let totalQuantity = 0;

  monthlyPrices.forEach(({ price }) => {
    const fee = input.monthlyAmount * feeRate;
    const netAmount = input.monthlyAmount - fee;
    const quantity = netAmount / price;

    totalInvestment += input.monthlyAmount;
    totalQuantity += quantity;
  });

  const averageBuyPrice = totalInvestment / totalQuantity;
  const currentValue = totalQuantity * input.currentPrice;
  const profitAmount = currentValue - totalInvestment;
  const profitRate = (profitAmount / totalInvestment) * 100;

  return {
    totalInvestment,
    totalQuantity,
    averageBuyPrice,
    currentValue,
    profitAmount,
    profitRate,
    breakEvenPrice: averageBuyPrice,
  };
}
```

## 2) 기획 배경

공직선거법에 따라 광역단체장 후보는 재산 내역을 중앙선거관리위원회에 신고·공개한다.
하지만 이 데이터는 선관위 PDF 또는 복잡한 표 형태로 공개되어, 일반 유권자가 한눈에 비교하기 어렵다.

비교계산소에서는 이를 다음과 같이 차별화한다.

- 17개 광역자치단체(특별시·광역시·도·특별자치도) 후보자 재산 한 번에 비교
- 정당별 평균 재산 비교 (국민의힘 vs 더불어민주당 vs 기타)
- 지역별 재산 격차 시각화
- 재산 구성 항목(부동산·금융자산·채무 등) 분석
- "재산이 가장 많은 후보", "재산이 가장 적은 후보" 같은 직관적 랭킹

즉, 이 콘텐츠는 선거 뉴스 기사가 아니라
**숫자로 비교하는 광역단체장 후보 재산 리포트**로 포지셔닝한다.

## 3) 기획 목표

### 1차 목표

- 2026년 지방선거 광역단체장 후보 재산을 한 화면에서 비교할 수 있게 한다.
- 중앙선관위 공개 데이터를 시각화해 유권자의 정보 접근성을 높인다.
- 선거 시즌 검색 트래픽 집중 키워드("시도지사 후보 재산", "광역단체장 재산")를 타겟으로 검색 유입을 확보한다.

### 2차 목표

- 정당별·지역별 비교 리포트 확장 기반을 만든다.
- SNS/커뮤니티에서 공유 가능한 비교형 콘텐츠로 활용한다.
- "공직자 재산 투명성" 관련 콘텐츠 카테고리를 비교계산소에 추가한다.

### 3차 목표

- 선거 관련 리포트 시리즈(국회의원 재산, 기초단체장 재산 등) 확장 토대 마련.
- 비교계산소의 "데이터 기반 공익 리포트" 정체성을 강화한다.

## 5) 핵심 콘텐츠 구성

### 섹션 1. 히어로 요약
- 전체 후보 평균 재산 (1인 기준)
- 최고 재산 후보 / 최저 재산 후보 하이라이트
- 정당별 평균 재산 요약 카드

### 섹션 2. 전체 후보 재산 랭킹 테이블
- 후보명 / 소속 정당 / 출마 지역 / 신고 재산 총액 / 전년 대비 증감
- 정렬 기능: 재산 높은 순 / 낮은 순 / 지역별 / 정당별
- 데이터 출처: 중앙선거관리위원회 재산 신고 공개 자료

### 섹션 3. 정당별 평균 재산 비교
- 국민의힘 후보 평균 vs 더불어민주당 후보 평균 vs 기타 정당 평균
- 막대 차트 시각화

### 섹션 4. 지역별 재산 분포
- 17개 시도별 후보 재산 분포 시각화 (수도권 vs 비수도권 비교 포함)

### 섹션 5. 재산 구성 항목 분석
- 부동산 / 금융자산 / 채무 항목별 분류
- 재산 상위 후보의 구성 패턴 분석

### 섹션 6. 데이터 노트 및 면책 고지
- 데이터 기준일 명시 (선관위 공개 기준)
- 신고 재산과 실제 재산의 차이 가능성 설명
- 추정값 없이 공개 데이터만 사용

# 1. 페이지 목적

여행 환전 손익 계산기는 사용자가 해외여행 전 원화를 외화로 바꿀 때 **은행 창구, 공항 환전, 트래블카드, 현지 ATM 인출** 중 어떤 방법이 가장 유리한지 계산해주는 도구다.

단순 환율 계산기가 아니라, 실제 사용자가 궁금해하는 질문에 답하는 구조로 설계한다.

> “100만 원을 일본 엔화로 바꾸면 어디서 환전하는 게 제일 이득일까?”

# 5. 핵심 계산 로직

## 5-1. 기본 환전 계산

```text
실수령 외화 = 환전 원화 금액 ÷ 적용 환율
```

예시:

```text
1,000,000원 ÷ 910원 × 100 = 약 109,890엔
```

JPY처럼 100엔 단위로 고시되는 통화는 계산 단위를 별도로 처리해야 한다.

---

## 5-2. 환율 우대 적용 계산

```text
환전 수수료율 = 기본 스프레드 × (1 - 우대율)

적용 환율 = 매매기준율 × (1 + 환전 수수료율)
```

예시:

| 항목      |        값 |
| ------- | -------: |
| 매매기준율   |     900원 |
| 기본 스프레드 |    1.75% |
| 우대율     |      90% |
| 실제 수수료율 |   0.175% |
| 적용 환율   | 901.575원 |

---

## 5-3. 방법별 비교 계산

| 환전 방법   | 계산 방식                    |
| ------- | ------------------------ |
| 은행 창구   | 기준 환율 + 수수료 - 우대율        |
| 공항 환전   | 높은 스프레드, 낮은 우대율 적용       |
| 트래블카드   | 낮은 환전 수수료 또는 무료 환전 가정    |
| 현지 ATM  | 환율 + ATM 수수료 + 해외 이용 수수료 |
| 신용카드 결제 | 국제 브랜드 수수료 + 카드사 수수료 반영  |

# 17. FAQ 구성

## Q1. 여행 환전은 은행이 제일 싼가요?

항상 그렇지는 않다. 은행 앱 환전은 우대율이 높으면 유리하지만, 공항 환전은 수수료가 불리할 수 있다. 최근에는 트래블카드나 현지 ATM 인출이 더 유리한 경우도 있다.

## Q2. 공항 환전은 왜 비싼가요?

공항 환전은 접근성이 좋고 즉시 환전할 수 있다는 장점이 있지만, 일반적으로 환율 우대율이 낮거나 적용 환율이 불리할 수 있다.

## Q3. 트래블카드는 무조건 이득인가요?

아니다. 통화별 환전 수수료, ATM 출금 수수료, 충전 한도, 환불 조건에 따라 달라진다. 카드 결제가 많은 국가에서는 유리하지만, 현금 사용이 많은 국가는 일부 현금 환전이 필요하다.

## Q4. 현지 ATM 인출은 괜찮나요?

장기 여행이나 동남아 여행에서는 편리할 수 있다. 다만 현지 ATM 수수료, 카드사 해외 이용 수수료, 1회 출금 한도를 반드시 확인해야 한다.

## Q5. 일본 여행은 얼마나 환전하는 게 좋나요?

카드 결제가 가능한 곳이 많아졌지만, 소규모 식당·교통·관광지에서는 현금이 필요할 수 있다. 여행 기간과 동선에 따라 일부 현금 + 트래블카드 조합이 적합하다.

## 계산 함수 예시

```ts
function calculateExchangeResult(
  input: TravelExchangeInput,
  methodConfig: ExchangeMethodConfig
): ExchangeResult {
  const unit = input.unit ?? 1;

  const effectiveSpreadRate =
    methodConfig.spreadRate * (1 - methodConfig.preferentialRate / 100);

  const appliedRate =
    input.baseRate * (1 + effectiveSpreadRate / 100);

  const variableFee =
    input.krwAmount * (methodConfig.cardFeeRate / 100);

  const availableKrw =
    input.krwAmount - methodConfig.fixedFee - variableFee;

  const foreignAmount =
    (availableKrw / appliedRate) * unit;

  const idealForeignAmount =
    (input.krwAmount / input.baseRate) * unit;

  const totalFeeKrw =
    input.krwAmount - (foreignAmount / unit) * input.baseRate;

  return {
    method: methodConfig.method,
    label: methodConfig.label,
    appliedRate,
    foreignAmount,
    totalFeeKrw,
    savingVsWorst: 0,
    savingVsBankBranch: 0,
    rank: 0,
  };
}
```

# 23. 최종 기획 요약

여행 환전 손익 계산기는 비교계산소의 여행/항공/숙박비 카테고리에서 검색 수요와 반복 사용성이 모두 높은 계산기다.

특히 **일본 여행 환전**, **엔화 환전**, **트래블카드 환전**, **공항 환전 수수료** 같은 키워드와 궁합이 좋다. 사용자는 여행 전뿐만 아니라 환율이 바뀔 때마다 다시 계산할 가능성이 높다.

MVP는 다음 순서로 구현하는 것이 좋다.

| 우선순위 | 구현 내용                      |
| ---- | -------------------------- |
| 1순위  | 국가/통화 선택 + 환전 금액 입력        |
| 2순위  | 은행 창구·은행 앱·공항·트래블카드·ATM 비교 |
| 3순위  | 실수령 외화·수수료·절약액 결과 카드       |
| 4순위  | 일본/미국/동남아 국가별 환전 팁         |
| 5순위  | 여행 경비 계산기·여행 적금 계산기 내부 링크  |

초기에는 사용자가 기준 환율을 직접 입력하게 하고, 이후 실시간 환율 API와 트래블카드별 조건 데이터를 붙이면 충분히 확장성 있는 여행 계산기 콘텐츠가 된다.

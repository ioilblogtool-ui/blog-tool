## 1. 문서 개요

### 1-1. 목표
- 기획 문서를 실제 구현 단위로 해석한 설계 문서
- 결과물 제목: `소방관 계급별 연봉·수당 비교 2026`
- 콘텐츠 유형: 인터랙티브 비교 리포트 (`/reports/` 계열)

### 1-2. 비교 리포트 포지션
- 경찰관 연봉 리포트와 같은 직업·연봉 시리즈 2탄
- 경찰과 동일 봉급표를 쓰지만, 소방은 교대근무와 현장 수당 구조가 더 중요하다는 점을 강조
- 사용자가 `계급 → 호봉 → 체감 연봉 → 수당 구조 → 입직 경로` 순서로 자연스럽게 읽도록 설계

### 1-3. 핵심 메시지
- 기본급은 공무원보수규정 별표 10 기준으로 공식 수치를 사용한다.
- 체감 연봉은 교대근무, 야간, 출동 관련 수당 반영 여부에 따라 달라진다.
- 경찰과 기본 봉급표는 같지만 실제 보수 구조는 다르게 느껴질 수 있다.
- 소방사~소방정감까지 계급별 차이를 한 화면에서 이해하도록 만든다.

### 1-4. 권장 slug
- `firefighter-salary-2026`

## 3. 데이터 설계

### 3-1. 계급 데이터
`FireRank`는 아래 정보를 포함한다.
- `id`: 내부 식별자
- `name`: 한글 계급명
- `nameEn`: 영문 명칭
- `equiv`: 급수 상당 표현
- `description`: 역할 설명
- `officialBase1`: 1호봉 공식 기본급
- `representativeStep`: 대표 호봉
- `steps`: 호봉 배열
- `isShiftWork`: 교대근무 직군 여부
- `shiftNote`: 교대 형태 설명
- `estimatedMonthlyRange`: 기본 추정 월 보수 범위
- `estimatedMonthlyShiftRange`: 교대 포함 추정 월 보수 범위
- `estimatedAnnual`: 추정 연봉
- `colorHex`: 카드/차트 포인트 컬러

### 3-2. 수당 데이터
`FireAllowance`
- `type`: `fixed | variable | annual`
- `firefighterOnly`: 소방 전용 수당 여부

구분 기준:
- 소방 전용 수당
- 공통 고정/변동 수당
- 연간 특별 지급 항목

### 3-3. 교대근무 패턴 데이터
`ShiftPattern`
- `three-shift`
- `two-shift`
- `admin`

각 항목은 근무 사이클, 월 추가 보수 추정, 설명 문구를 가진다.

### 3-4. 입직 방법 데이터
`FireEntryMethod`
- 소방사 공채
- 소방간부후보생
- 경력경쟁채용
- 의무소방 또는 별도 경력 인정형 케이스

### 3-5. 비교 배너 데이터
`ComparePoint`
- 봉급표 기준
- 위험근무수당
- 현장 출동 관련 수당
- 주요 수당 구조
- 체감 연봉 차이 포인트

### 3-6. 생애 소득 데이터
`FireLifetimePath`
- 소방사 입직 · 3교대 · 보통 승진
- 소방사 입직 · 3교대 · 빠른 승진
- 간부 입직 · 보통 승진

## 4. 화면 구조

### 4-1. Hero
- eyebrow: `직업·연봉 리포트`
- title: `소방관 계급별 연봉·수당 비교 2026`
- description: `2026년 공식 봉급표 기준 · 소방사~소방정감 기본급·호봉·수당 구조 한눈에 비교`

### 4-2. 데이터 기준 안내
반드시 포함할 내용:
- 1호봉 기본급은 공식 수치
- 이후 호봉과 체감 연봉은 추정치
- 교대근무 수당은 개인차가 큼
- 수당 항목은 법령 및 인사혁신처 보도자료 기준만 표기

### 4-3. KPI 카드 4개
- 계급 수
- 2026 보수 인상률
- 저연차 초임 인상률
- 소방사 1호봉 기본급

### 4-4. 경찰 vs 소방 비교 배너
왼쪽:
- 경찰관 요약
- 경찰 리포트로 이동 링크

오른쪽:
- 소방관 요약
- 현재 페이지 배지
- 교대·출동수당 비중이 더 높다는 메시지

### 4-5. 계급 카드 그리드
카드 공통 요소:
- 계급명
- 공식 배지
- 교대근무 배지(해당 시)
- 1호봉 기본급
- 예상 체감 월 보수
- 교대 포함 추정 보수(해당 시)
- `호봉표 보기` 버튼

확장 패널 요소:
- 설명 문구
- 교대근무 설명 박스
- 호봉표 테이블
- 예상 실수령 / 예상 연봉 요약

### 4-6. 예상 연봉 차트
- 가로 막대형 비교
- 최대 연봉 기준 상대 폭 계산
- 각 막대에 계급명과 추정 연봉 표시

### 4-7. 교대근무 & 수당 구조
1. 교대근무 패턴 3종 설명 카드
2. 수당 탭 3개
- 소방 전용 수당
- 공통 고정 수당
- 연간 특별 지급
3. 실수령 계산기로 이어지는 CTA

### 4-8. 승진 경로
표 형식:
- 계급
- 일반적 근속 기간
- 비고

### 4-9. 30년 생애 소득 시뮬레이션
- 경로 선택 버튼
- 라인 차트
- 경로별 누적 보수 요약 카드
- 하단 주의 문구

### 4-10. 입직 방법 비교
카드 형식:
- 입직명
- 시작 계급
- 급여 메모
- 설명

### 4-11. SEO / FAQ
`SeoContent`에 아래 내용을 포함
- 소방공무원 급여 구조 설명
- 2026년 인상 포인트
- 경찰과 비교 시 차이
- 공식/추정 기준 명시

## 5. 구현 메모

### 5-1. JS 요구사항
- `.ff-rank-card` accordion 동작
- `.ff-allowance-tab` 탭 전환
- `Chart.js` 기반 생애 소득 라인 차트 렌더
- 첫 경로 기본 선택 상태 유지

### 5-2. 접근성
- 카드에 `role="button"`, `tabindex="0"`, `aria-expanded` 사용
- 탭에는 `role="tab"`, `aria-selected` 사용
- 시뮬레이션 버튼에는 `aria-pressed` 사용

### 5-3. 카피 원칙
- `공식`, `추정`, `참고용`을 명확히 구분
- 실제 실수령으로 오해될 수 있는 표현은 피하고 `추정 월 보수`, `추정 연봉` 표현 사용
- 교대근무 수당은 근무지와 출동 빈도에 따라 달라진다는 주석 유지

## 6. QA 체크리스트
- 빌드 통과 여부
- 계급 카드 열고 닫기 동작
- URL `?rank=` 파라미터 동작
- 수당 탭 전환 정상 여부
- 생애 소득 경로 버튼 전환 시 차트 업데이트 여부
- 모바일에서 카드/표 overflow 여부
- 경찰 리포트 링크, 연봉 계산기 링크 동작 여부
- `/reports/` 목록 등록 여부

## 7. 현재 구현 상태 기준 메모
- A~K 주요 섹션은 구현 완료 상태를 목표로 한다.
- 경찰 리포트와 비교 시 누락 없이 구조를 맞추되, 소방 고유 메시지는 별도 유지한다.
- 이후 수정 시 우선순위는 `데이터 정확성 > 공식/추정 라벨 명확성 > 상호작용 안정성 > 시각 완성도` 순서로 둔다.

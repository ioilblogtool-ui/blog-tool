## 4. 데이터 모델

```ts
export type ExchangeType = "domestic" | "global";

export interface CoinExchange {
  id: string;
  name: string;                      // 거래소명 (한국어)
  nameEn: string;                    // 영문명
  type: ExchangeType;                // 국내 / 해외
  makerFee: string;                  // 지정가 수수료 (%, 문자열로 표시)
  takerFee: string;                  // 시장가 수수료 (%)
  wonDeposit: boolean;               // 원화 입금 가능
  wonWithdrawal: boolean;            // 원화 출금 가능
  coinCount: string;                 // 상장 코인 수 (약 N종)
  kyc: boolean;                      // KYC 필수 여부
  twoFa: boolean;                    // 2FA 지원
  mobile: boolean;                   // 모바일 앱 지원
  koreanSupport: boolean;            // 한국어 고객센터
  regulated: boolean;                // 국내 규제 준수 (VASP 신고)
  pros: string[];                    // 강점 3개
  cons: string[];                    // 약점 2개
  bestFor: string;                   // 추천 대상 한 줄
  officialUrl: string;               // 공식 수수료 페이지 URL
  updatedAt: string;                 // 확인 기준일
}

export interface ExchangeRecommendation {
  situation: string;                 // 상황 설명
  exchange: string;                  // 추천 거래소명
  reason: string;                    // 추천 이유 2~3문장
}
```

## 5. 비교 데이터 상세

### 국내 거래소 (4개)

#### 업비트 (Upbit)
- 수수료: 지정가/시장가 모두 0.05% (단일 수수료)
- 원화 입출금: 가능 (케이뱅크 제휴)
- 상장 코인: 약 200종 이상
- KYC: 필수, 2FA: 지원, 앱: 지원, 한국어 CS: 지원, VASP 신고: 완료
- 강점: 국내 최대 거래량·유동성, 직관적인 UI·앱, 다양한 코인 상장
- 약점: 수수료가 업계 최저 수준은 아님, 레버리지·선물 미지원
- 추천 대상: 코인 처음 시작하는 초보자, 안정적인 원화 입출금이 필요한 투자자

#### 빗썸 (Bithumb)
- 수수료: 0.04% (VIP 등급별 추가 할인)
- 원화 입출금: 가능 (NH농협 제휴)
- 상장 코인: 약 200종
- KYC: 필수, 2FA: 지원, 앱: 지원, 한국어 CS: 지원, VASP 신고: 완료
- 강점: 업비트보다 낮은 수수료, 이벤트·에어드롭 활발, 대형 거래소 신뢰도
- 약점: UI가 업비트보다 복잡하다는 평가, 과거 해킹 이력
- 추천 대상: 수수료 절약을 중시하는 중급 투자자, 이벤트 활용 원하는 투자자

#### 코인원 (Coinone)
- 수수료: 지정가 0.02%, 시장가 0.02% (국내 최저 수준)
- 원화 입출금: 가능 (우리은행 제휴)
- 상장 코인: 약 200종
- KYC: 필수, 2FA: 지원, 앱: 지원, 한국어 CS: 지원, VASP 신고: 완료
- 강점: 국내 최저 수준 수수료, 안정적인 운영, 깔끔한 UI
- 약점: 업비트 대비 유동성 낮음, 상장 코인 수 적음
- 추천 대상: 수수료 절약 최우선인 적립식 투자자, 장기 홀더

#### 코빗 (Korbit)
- 수수료: 지정가/시장가 0.05%
- 원화 입출금: 가능 (신한은행 제휴)
- 상장 코인: 약 50종 (주요 코인 위주)
- KYC: 필수, 2FA: 지원, 앱: 지원, 한국어 CS: 지원, VASP 신고: 완료
- 강점: 국내 최초 거래소로 신뢰도, 주요 코인 안정적 지원
- 약점: 상장 코인 수 제한적, 거래량 상대적으로 적음
- 추천 대상: BTC·ETH 등 주요 코인만 거래하는 보수적 투자자

### 해외 거래소 (3개)

#### 바이낸스 (Binance)
- 수수료: 지정가 0.10%, 시장가 0.10% (BNB 사용 시 25% 할인)
- 원화 입출금: 불가 (USDT·USDC 등 스테이블코인 기반)
- 상장 코인: 350종 이상 (세계 최다 수준)
- KYC: 필수, 2FA: 지원, 앱: 지원, 한국어 CS: 제한적, 한국 서비스 제한 운영
- 강점: 세계 최대 거래량, 알트코인 최다 상장, 선물·레버리지 지원, DeFi 연동
- 약점: 원화 직접 입출금 불가, 한국 규제 환경에서 사용 제한, 한국어 CS 불편
- 추천 대상: 알트코인 다양하게 거래하고 싶은 중급 이상 투자자

#### 코인베이스 (Coinbase)
- 수수료: 지정가 0.40%, 시장가 0.60% (Pro 플랜 시 낮아짐)
- 원화 입출금: 불가 (USD 기반)
- 상장 코인: 약 200종
- KYC: 필수, 2FA: 지원, 앱: 지원, 한국어 CS: 불가, 미국 나스닥 상장사
- 강점: 미국 규제 준수 최강, 기관 투자자 신뢰, 상장 코인 안전성 높음
- 약점: 수수료 가장 높음, 원화 입출금 불가, 한국어 미지원
- 추천 대상: 해외 송금·달러 자산 연동이 필요한 투자자, 기관 수준 보안 원하는 투자자

#### OKX
- 수수료: 지정가 0.08%, 시장가 0.10% (OKB 토큰 사용 시 할인)
- 원화 입출금: 불가 (USDT 기반)
- 상장 코인: 약 300종
- KYC: 필수, 2FA: 지원, 앱: 지원, 한국어 CS: 제한적
- 강점: 바이낸스 대안으로 수수료 저렴, 선물·옵션 지원, Web3 지갑 내장
- 약점: 국내 규제 환경에서 사용 제한, 한국어 CS 불편
- 추천 대상: 바이낸스 외 대형 해외 거래소를 원하는 투자자

### intro (5문단)

1. 2026년 국내에서 가상자산을 거래할 수 있는 주요 거래소는 업비트, 빗썸, 코인원, 코빗 4개이며, 해외 거래소로는 바이낸스, 코인베이스, OKX가 대표적입니다. 거래소마다 수수료, 상장 코인 수, 원화 입출금 여부, 보안 수준, 고객센터 지원이 다르기 때문에 본인의 투자 목적에 맞는 거래소를 선택하는 것이 중요합니다.

2. 수수료는 장기 투자자에게 특히 중요합니다. 국내 거래소 중 코인원이 지정가·시장가 모두 0.02%로 가장 낮고, 빗썸 0.04%, 업비트·코빗 0.05% 순입니다. 월 100만 원씩 거래할 경우 코인원과 업비트의 연간 수수료 차이는 3,600원에 불과하지만, 월 1,000만 원 이상 거래한다면 연 수십만 원의 차이가 납니다. 본인의 거래 금액과 빈도를 먼저 파악하고 수수료를 계산해 보세요.

3. 원화 입출금은 국내 4개 거래소에서만 가능합니다. 바이낸스, 코인베이스, OKX 등 해외 거래소는 원화를 직접 입금할 수 없고 USDT 등 스테이블코인을 먼저 구해야 합니다. 따라서 코인 투자를 처음 시작하거나 원화 자금을 직접 운용하고 싶다면 국내 거래소가 필수적입니다. 해외 거래소는 국내에 미상장된 알트코인을 거래하거나 선물·레버리지가 필요한 경우에 주로 활용합니다.

4. 보안과 규제 준수 면에서는 국내 거래소 4개가 모두 금융정보분석원(FIU)에 VASP(가상자산 사업자) 신고를 완료하여 국내 규제 체계 내에서 운영됩니다. 해외 거래소는 각 국가의 규제를 따르지만 한국 규제 체계 외에 있어 분쟁 발생 시 법적 보호가 제한될 수 있습니다. 어떤 거래소를 선택하든 2단계 인증(2FA) 설정과 피싱 사이트 주의는 필수입니다.

5. 이 리포트의 수수료와 기능 정보는 2026년 5월 기준이며, 거래소 정책 변경에 따라 달라질 수 있습니다. 실제 가입이나 거래 전에 각 거래소 공식 홈페이지에서 최신 수수료와 약관을 반드시 재확인하세요. 이 리포트는 특정 거래소를 권유하지 않으며, 가상자산 투자에는 원금 손실 위험이 있습니다.

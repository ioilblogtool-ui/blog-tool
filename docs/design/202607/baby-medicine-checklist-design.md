## 2. 콘텐츠 포지셔닝

### 2-1. 한 줄 정의

```text
아기 해열제, 체온계, ORS, 생리식염수, 복약 기록표, 병원 상담 신호까지 한 번에 점검하는 보호자용 상비약 체크리스트
```

### 2-2. 비교계산소와 잘 맞는 이유

| 이유 | 설명 |
|---|---|
| 기존 독자와 맞음 | 분유 가격, 출산지원금, 산후조리원, 실손보험 등 육아/의료/생활비 콘텐츠와 자연스럽게 연결됨 |
| 검색 의도가 강함 | “아기 열남”, “아기 해열제”, “아기 상비약”은 불안이 큰 상황에서 바로 검색하는 키워드 |
| 도구화가 가능함 | 단순 글보다 체크리스트, 기록표, 위험 신호 체크로 체류 시간을 만들 수 있음 |
| 광고 친화성 있음 | 육아용품, 보험, 병원비, 건강관리 광고와 문맥이 맞음 |
| 확장성이 높음 | 이후 “아기 열날 때 병원 기준”, “아기 해열제 기록표”, “어린이집 준비물”로 클러스터 확장 가능 |

### 2-3. 독자가 얻어야 하는 결과

페이지를 본 뒤 사용자는 다음 4가지를 얻어야 한다.

1. 우리 집에 기본적으로 준비된 아기 상비약/상비용품이 무엇인지 알 수 있다.
2. 빠진 항목이 무엇인지 우선순위대로 확인할 수 있다.
3. 아기가 열이 나거나 설사/구토가 있을 때 무엇을 기록해야 하는지 알 수 있다.
4. 집에서 지켜볼 문제가 아니라 병원, 약국, 응급 상담이 필요한 신호를 확인할 수 있다.

### 2-4. 페이지가 하지 말아야 하는 것

| 하지 말 것 | 이유 |
|---|---|
| “몇 mL 먹이세요” 자동 계산 | 제품별 농도, 체중, 연령, 기저질환, 복용 이력에 따라 달라 의료 리스크가 큼 |
| “응급실 가야 합니다/안 가도 됩니다” 확정 | 실제 상태 판단은 의료진이 해야 함 |
| 특정 브랜드 약 추천 | 광고/의료 조언 오해 가능성 |
| 해열제 교차복용 방법 상세 가이드 | 보호자가 오남용할 수 있으므로 기록과 상담 유도 중심 |
| 민간요법 중심 설명 | 검색 품질과 안전성 모두에 불리함 |

### 3-5. 네이버 검색 최적화 포인트

네이버는 제목과 본문 초반의 키워드 일치, 사용자 체류, 문답형 구조, 표/리스트 가독성이 중요하다. 특히 의료/육아 키워드는 사용자가 빠르게 답을 찾고 이탈하기 쉬우므로, 첫 화면에서 바로 체크 기능을 제공해야 한다.

| 영역 | 설계 |
|---|---|
| 제목 | “아기 상비약 체크리스트”를 제목 맨 앞에 배치 |
| 본문 초반 | 첫 300자 안에 “아기 상비약, 아기 해열제, 아기 열날 때, 병원 상담 신호” 포함 |
| 표 구성 | 상비약 리스트, 해열제 비교, 위험 신호를 표로 제공 |
| FAQ | 실제 검색어 형태로 질문 작성 |
| 체류 시간 | 준비율 계산, 복약 기록, 위험 신호 체크를 순서대로 배치 |
| 내부 링크 | 분유 가격, 첫해 육아비, 실손보험, 출산지원금 콘텐츠로 연결 |
| 신뢰도 | 식품의약품안전처, 응급의료포털, 의약품안전나라 링크 명시 |

### 3-6. 구글 검색 최적화 포인트

구글에서는 YMYL 주제의 안전성과 신뢰 신호가 중요하다. 이 페이지는 의료 조언처럼 보이지 않도록 “체크리스트 도구”로 포지셔닝하고, 공식 출처를 반복해서 연결한다.

| 영역 | 설계 |
|---|---|
| E-E-A-T | 공식 출처 링크, 업데이트 날짜, 의료 면책 문구, 진단/처방 금지 명시 |
| 구조화 데이터 | `WebApplication`, `FAQPage`, `BreadcrumbList` 권장 |
| 본문 깊이 | 단순 리스트가 아니라 사용 상황, 주의점, 기록 방법까지 포함 |
| 검색 스니펫 | FAQ 질문을 검색어와 비슷하게 작성 |
| 중복 방지 | 해열제 용량 계산 페이지처럼 보이지 않게 안전 체크 중심으로 구성 |
| 내부 링크 | 같은 육아/의료비 클러스터와 연결해 주제 권위 강화 |

## 5. 핵심 기능 설계

### 5-1. 준비율 체크리스트

사용자가 보유한 항목을 체크하면 준비율 점수, 부족한 항목, 우선 준비 항목을 보여준다. 점수는 의료 판단이 아니라 “준비물 정리 점수”임을 명확히 표시한다.

| 카테고리 | 항목 | 점수 | 필수 | 안내 |
|---|---|---:|---|---|
| 체온 확인 | 디지털 체온계 | 12 | 예 | 같은 방식으로 반복 측정해 변화를 기록 |
| 체온 확인 | 체온 기록 메모 또는 앱 | 8 | 예 | 시간, 체온, 증상, 복용 내용을 함께 기록 |
| 해열/통증 | 아세트아미노펜 계열 해열제 | 12 | 예 | 성분명과 제품 농도 확인 |
| 해열/통증 | 이부프로펜 또는 덱시부프로펜 계열 | 8 | 아니오 | 연령, 탈수, 구토, 신장질환 등 상황에 따라 주의 |
| 코막힘 | 생리식염수 스프레이/앰플 | 8 | 아니오 | 약 성분이 없는 제품 위주로 확인 |
| 설사/구토 | 경구수분보충액 ORS | 10 | 예 | 탈수 신호가 있으면 진료 상담 우선 |
| 피부 | 보습제와 기저귀 발진 보호크림 | 8 | 아니오 | 스테로이드 성분은 상담 후 사용 |
| 상처 | 멸균 거즈, 밴드, 소독용품 | 8 | 예 | 깊은 상처, 출혈 지속 시 진료 |
| 투약 도구 | 약 먹이는 주사기, 계량컵, 스푼 | 10 | 예 | 일반 숟가락으로 임의 계량하지 않기 |
| 기록 | 복약 기록표 | 12 | 예 | 성분, 시간, 체온, 용량을 남김 |
| 응급 정보 | 119, 소아청소년과, 달빛어린이병원 메모 | 12 | 예 | 야간/휴일에 바로 확인 가능해야 함 |
| 보관 | 유효기간 확인 라벨 | 6 | 아니오 | 개봉일과 폐기 예정일 기록 |

|---:|---|---|
| 80~100 | 준비 양호 | 기본 상비약과 기록 도구가 대부분 준비되어 있습니다. 유효기간과 성분 중복만 다시 확인하세요. |
| 50~79 | 보완 필요 | 기본 항목은 일부 준비되어 있지만 복약 기록, ORS, 응급 연락처 같은 핵심 항목을 보완하세요. |
| 0~49 | 우선 준비 | 체온계, 해열제 성분 확인, 투약 도구, 응급 연락처부터 먼저 준비하는 것이 좋습니다. |

결과 카드에는 다음을 함께 보여준다.

| 요소 | 내용 |
|---|---|
| 준비율 | `72점 / 100점` 형식 |
| 부족 항목 | 체크하지 않은 필수 항목 우선 표시 |
| 다음 행동 | “약국에서 성분 확인”, “유효기간 확인”, “응급 연락처 저장” |
| 안전 문구 | “점수는 준비 상태이며 의료 판단이 아닙니다” |

### 5-3. 복약 기록표

복약 기록표는 페이지의 핵심 체류 기능이다. 단, 개인 민감 정보가 될 수 있으므로 기본적으로 브라우저에 저장하지 않는다. 사용자가 원할 때만 “기록 복사” 또는 “인쇄”를 제공한다.

| 입력값 | 타입 | 예시 | 안내 |
|---|---|---|---|
| 아이 이름 또는 별칭 | text | 지우 | 선택 입력 |
| 체온 | number | 38.5 | 측정 시간과 함께 기록 |
| 증상 | checkbox/text | 열, 콧물, 기침, 설사 | 복수 선택 가능 |
| 복용 성분 | select | 아세트아미노펜 | 제품명보다 성분명 확인 유도 |
| 제품명 | text | 보호자 입력 | 선택 입력 |
| 복용 시간 | time | 21:30 | 중복 복용 방지 목적 |
| 복용량 | text | 제품 설명서 기준 입력 | 자동 계산하지 않음 |
| 메모 | textarea | 물을 잘 못 마심 | 진료 시 전달할 내용 |

복약 기록 출력 예시:

```text
[아기 복약 기록]
이름/별칭: 지우
체온: 38.5도
증상: 열, 콧물
성분: 아세트아미노펜
제품명: 보호자 입력
복용 시간: 21:30
복용량: 제품 설명서/약사 안내 기준 입력
메모: 물을 잘 못 마심

※ 이 기록은 진료 또는 약국 상담 시 전달하기 위한 참고용입니다.
```

### 5-4. 병원 상담 신호 체크

위험 신호 체크는 “결론을 확정”하지 않고, 신호가 하나라도 체크되면 즉시 상담 권고를 보여준다.

| 문항 | 심각도 | 결과 안내 |
|---|---|---|
| 아이가 축 처져 있거나 깨우기 어렵나요? | urgent | 즉시 진료 또는 응급 상담 권고 |
| 숨쉬기 힘들어 보이나요? | urgent | 즉시 진료 또는 응급 상담 권고 |
| 경련이 있었나요? | urgent | 즉시 진료 또는 응급 상담 권고 |
| 입술이나 얼굴색이 푸르거나 창백한가요? | urgent | 즉시 진료 또는 응급 상담 권고 |
| 반복 구토로 물도 마시기 어렵나요? | consult | 진료 상담 권고 |
| 소변량이 줄고 탈수가 의심되나요? | consult | 진료 상담 권고 |
| 생후 3개월 미만인데 고열이 있나요? | urgent | 즉시 진료 상담 권고 |
| 2세 미만 감기 증상에서 임의로 감기약을 먹이려 하나요? | consult | 의사 진료 상담 권고 |

결과 문구:

```text
체크한 항목 중 진료 상담이 필요한 신호가 있습니다.
이 도구의 결과로 집에서 지켜볼지 결정하지 말고, 소아청소년과·응급의료 상담·119 안내를 우선하세요.
```

### 5-5. 해열제 비교 표

목적은 “무엇을 먹여라”가 아니라 성분명과 주의점을 구분하게 돕는 것이다.

| 성분 | 일반적 용도 | 확인할 점 | 주의 문구 |
|---|---|---|---|
| 아세트아미노펜 | 발열, 통증 | 제품 농도, 성분 중복 | 감기약 등 복합제에 같은 성분이 들어 있을 수 있음 |
| 이부프로펜 | 발열, 통증, 염증 | 연령, 탈수 여부, 구토 여부 | 탈수가 의심되거나 잘 먹지 못하면 상담 우선 |
| 덱시부프로펜 | 발열, 통증 | 이부프로펜 계열 여부 | 같은 계열 성분 중복 주의 |
| 아스피린 | 성인 진통/해열 등 | 소아 사용 제한 | 의사 처방 없이 어린이에게 먹이지 않도록 안내 |

### 5-6. 상비약 보관 체크

| 체크 항목 | 설명 |
|---|---|
| 아이 손이 닿지 않는 곳에 보관 | 잠금장치가 있거나 높은 장소 권장 |
| 원래 용기와 설명서 보관 | 성분, 용량, 유효기간 확인을 위해 필요 |
| 개봉일 기록 | 시럽, 연고 등은 개봉 후 보관 기간을 따로 확인 |
| 직사광선과 고온 피하기 | 제품별 보관 방법 우선 |
| 유효기간 지난 약 폐기 | 임의 사용하지 않고 약국 수거함 등 활용 |
| 가족 약과 분리 | 성인용 약과 혼동 방지 |

## 19. 최종 설계 요약

이 페이지는 “아기 상비약 리스트” 글이 아니라 “보호자가 바로 체크하고 기록할 수 있는 안전형 도구”로 만들어야 한다. 검색 노출은 `아기 상비약 체크리스트`를 중심으로 잡고, 본문에서는 `아기 해열제`, `아기 열날 때`, `아기 응급실 기준`, `아기 설사 구토` 키워드를 자연스럽게 확장한다.

핵심 성공 조건은 세 가지다.

1. 첫 화면에서 바로 체크할 수 있어야 한다.
2. 복용량 계산과 진단을 하지 않는다는 안전 경계가 명확해야 한다.
3. FAQ와 표 구조가 풍부해 네이버와 구글 모두에서 긴 꼬리 검색어를 흡수해야 한다.

후속 콘텐츠까지 고려하면 이 페이지는 비교계산소의 육아/의료비 클러스터를 넓히는 첫 번째 건강 안전 도구로 적합하다.

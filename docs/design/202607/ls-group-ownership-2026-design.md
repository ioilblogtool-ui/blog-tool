## 1. 이 그룹의 핵심 특징 — 3개 지주사 "사촌경영", 트리의 다중 루트

LS그룹은 ㈜LS·E1·인베니(구 예스코홀딩스) **3개의 별도 지주회사**로 구성되며, 이 셋은 지분으로 연결되지 않고 **구씨 일가 3개 방계가 동일인(총수)만 공유**하는 독특한 구조다. 삼성 설계 13장에서 미리 정해둔 대로, `OwnershipTree`의 `buildLayout`은 `roots` 배열을 지원하므로 **`primaryParentId: null`인 노드 3개를 그대로 넣으면 컴포넌트 수정 없이 3그루의 나무가 나란히 렌더링**된다 — 이번 설계에서 이 다중 루트 케이스를 처음 실제로 검증한다.

## 4. 데이터

### 4-1. 메타
- `isLegalHoldingCompany: true` (단, 3개 지주사 병존 — `holdingCompanyNote`에서 설명)
- `holdingCompanyNote`: "㈜LS·E1·인베니(구 예스코홀딩스) 3개의 독립 지주회사가 있습니다. 세 회사는 서로 지분 관계가 없고, 구씨 일가 3개 방계(구태회·구평회·구두회 가문)가 동일인만 공유하는 '사촌경영' 구조입니다."
- `affiliateCount: 72` (비금융 70·금융 2, 2026-05-01, HIGH)
- `listedAffiliateCount: 10` (HIGH — 이번 조사에서 종목코드·시장 전부 교차검증)
- `fairTradeRank: 14` (14위·15위 두 표기가 혼재 — 산정 기준 차이로 추정, 페이지에 명시)

### 4-2. 트리 노드 (7개, 3개 루트 — 최초의 다중 루트 케이스)

```
[루트 1] 구자열 외 특수관계인 → ㈜LS(006260) [33.11%] → LS일렉트릭(010120) [48.47%]
[루트 2] 구자열 외 특수관계인(E1) → E1(017940) [45.36%]
[루트 3] 구자은 외 특수관계인 → 인베니(015360) [68.99%]
```

세 그루가 지분으로 안 이어진다는 점 자체가 이 그룹 콘텐츠의 핵심이므로, 트리에 억지로 연결선을 만들지 않고 나란히 배치되는 `buildLayout`의 기본 동작을 그대로 활용한다.

# 비교계산소 월 100만 원 목표 문서

## 문서 목적
- 비교계산소의 장기 목표를 `월 100만 원 수익` 기준으로 고정한다.
- 이후 기능 추가, 페이지 기획, SEO, 광고, 콘텐츠 확장 판단은 이 목표를 기준으로 우선순위를 잡는다.
- 목표 달성 전까지 계속 참고하는 기준 문서로 사용한다.

---

## 최종 목표
**월 100만 원 수익 달성**

이 목표는 단순히 계산기 몇 개를 운영하는 수준이 아니라,
비교계산소를 **검색형 계산 콘텐츠 플랫폼**으로 확장했을 때 현실적으로 도달 가능한 목표로 본다.

---

## 핵심 전제
월 100만 원을 만들기 위해서는 아래 구조가 같이 필요하다.

- 계산기
- 비교 페이지
- 설명형 가이드
- 회사 / 직급 / 상황별 랜딩 페이지
- 급여 / 성과급 / 초봉 / 실수령 / 총보상 확장

즉, 비교계산소는 단순 도구 사이트가 아니라
**숫자를 비교해서 더 잘 판단하게 만드는 계산 플랫폼**으로 가야 한다.

---

## 목표 해석
광고 수익은 대략 아래 식으로 본다.

```text
월 수익 = 월 페이지뷰 / 1000 × RPM
```

보수적으로 보면 월 100만 원은 아래 범위에서 가능성이 생긴다.

- 월 15만 ~ 25만 PV
- 또는 검색 유입 + 비교적 단가 괜찮은 주제 조합

### 단계별 수치 목표
- 1차 목표: 월 10만 PV
- 2차 목표: 월 20만 PV
- 3차 목표: 월 100만 원

## 가장 중요한 카테고리 우선순위
월 100만 원 기준으로 가장 가능성 높은 순서는 아래다.

### 1순위
**급여 / 연봉 실수령 / 성과급 / 총보상**

### 2순위
**대기업 초봉 / 직급별 연봉 / 회사 비교**

### 3순위
**출산 / 육아휴직 / 부모급여**

### 판단 이유
- 육아/복지는 시작점으로 좋고 안정 유입이 있다.
- 하지만 월 100만 원 목표는 결국 더 넓은 검색 수요가 있는
  **급여·실수령·보상 카테고리 확장**이 핵심이다.

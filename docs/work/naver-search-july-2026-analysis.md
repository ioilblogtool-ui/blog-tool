## 1. 요약

7월 네이버 검색 유입은 목표선에 거의 도달했다. 기록된 12개 일자 기준 일평균 클릭은 652.5회로, 7월 목표 일평균 645회를 소폭 상회한다. 이 평균을 31일로 단순 환산하면 월 20,228클릭으로, 월 목표 20,000클릭을 약 1.1% 웃돈다.

다만 CTR은 목표 4.2%와 거리가 있다. 스냅샷 합산 기준 가중 CTR은 약 2.18%로, 6월 평균 3.5%보다도 낮다. 원인은 검색 노출이 늘어난 만큼 클릭 의도가 약한 키워드가 함께 들어왔기 때문이다. 특히 이강인·손흥민·K리그·한성숙 프로필·일부 성과급 광범위 키워드에서 대량 노출과 저CTR이 반복됐다.

현재 7월의 핵심 구조는 다음과 같다.

| 구분 | 판단 |
|---|---|
| 클릭 규모 | 목표권 진입 |
| CTR | 목표 미달, 저CTR 대량 노출 개선 필요 |
| 강한 축 | 교사·교수 연봉, 삼성 성과급, 국회의원 재산순위 |
| 성장 축 | 재산세·보유세, 최저임금, 복지 계산기, 국제학교 |
| 병목 축 | K리그/월드컵 일부 페이지, 한성숙 프로필, 광범위 성과급 허브 |

## 6. 클러스터별 판단

### 성과급·연봉

가장 큰 트래픽 축이다. 삼성전자, 삼성전기, 성과급 계산기, 성과급 세금 계산기, 건설사·공기업·조선사 성과급이 모두 반복 진입했다. 단, `/compare/bonus/`는 허브 페이지 특성상 노출은 큰데 CTR이 낮아, 검색 결과에서 “어떤 회사 성과급을 비교해주는지”가 더 빨리 보여야 한다.

추천 액션:
- `/compare/bonus/` title과 description에 `삼성·SK하이닉스·현대차·공기업`처럼 대표 비교 대상을 명시
- 삼성전자 TAI, 하이닉스 PS/PI 키워드용 내부 앵커와 FAQ 강화
- 성과급 세금 계산기와 보너스 비교 허브 간 내부링크 강화

### 교사·교수·공무원

7월 후반의 가장 안정적인 클릭 축이다. `teacher-salary-2026`은 클릭 합계 1위, `professor-salary-2026`은 고CTR 페이지로 성장했다. 교수 연봉은 노출이 아직 크지 않은데도 클릭 효율이 높아 추가 확장 가치가 크다.

추천 액션:
- 교수 연봉 페이지에 `국립대 교수 연봉`, `사립대 교수 연봉`, `정교수 부교수 조교수 연봉` 섹션 강화
- 교사 월급 계산기와 교사 연봉 리포트 상호 링크를 상단 CTA로 강화
- 공무원·군인·경찰 연봉 페이지를 “직업별 월급 비교” 허브로 묶기

### 부동산·재산세

7월 시즌 키워드와 잘 맞는다. `아파트 보유세 계산기`, `재산세 납부`, `취득세 계산기`, `수원/영통/서울 구별 아파트`가 다수 진입했다. 재산세는 7월 말 납부 기한 전까지 추가 상승 여지가 있다.

추천 액션:
- 재산세 납부기간 페이지의 title에 `7월 31일`과 `아파트 재산세`를 전면 배치
- 보유세 계산기에서 재산세 납부 리포트로 연결
- 지역 아파트 리포트는 `대장아파트`, `실거래가`, `84㎡` 키워드별 소제목 강화

### 스포츠 연봉

이슈성 노출은 강하지만 CTR 품질이 낮다. 이강인·손흥민·K리그 관련 키워드는 노출을 크게 만들었지만 클릭률이 0%대인 경우가 많았다. 검색 결과에서 사용자의 즉시 질문에 답한다는 신호가 부족했을 가능성이 높다.

추천 액션:
- 이강인/손흥민 페이지 title을 `연봉 얼마`, `주급`, `세후`, `재산` 중심으로 구체화
- K리그 페이지는 `구단순위` 검색 노출을 받는 만큼, 연봉뿐 아니라 순위/선수명/구단별 총액 섹션이 검색 결과에 드러나게 보강
- 스포츠 클러스터는 신규 확장보다 기존 저CTR 페이지의 제목·요약 보강을 먼저 진행

### 정치인·인물 재산

국회의원 재산순위는 CTR이 매우 높다. 반면 한성숙 프로필은 노출은 크지만 클릭이 약하다. “프로필” 단독 의도보다 재산·경력·연봉·관련 정책처럼 숫자형 비교 의도로 좁히는 편이 사이트 성격과 맞다.

추천 액션:
- 한성숙 페이지 title/description을 `프로필`보다 `경력·재산·연봉` 중심으로 조정
- 국회의원 재산순위에서 고CTR 인물명별 내부 링크 확장
- 정치인 재산 페이지는 `순위`, `얼마`, `재산 공개`, `연봉` 키워드를 우선 반영

## 8. 7월 말까지 운영 기준

7월 남은 기간은 새 페이지를 무리하게 늘리기보다, 이미 노출을 받고 있는 저CTR 페이지를 고치는 편이 효율적이다. 특히 노출이 수천 단위인데 CTR이 1% 미만인 페이지는 title·description·H1·첫 문단을 검색어와 더 직접적으로 맞춰야 한다.

남은 기간 KPI는 다음처럼 잡는다.

| KPI | 목표 |
|---|---:|
| 일 클릭 | 650 이상 유지 |
| CTR | 2.7% 이상으로 회복 |
| 저CTR URL | 노출 1,000 이상·CTR 1% 미만 페이지 3개 이하로 축소 |
| 신규 고CTR 키워드 | 교수·재산세·국회의원 재산·복지 계산기 계열에서 추가 발굴 |

결론적으로 7월은 “클릭 규모 목표는 달성권, CTR 품질은 개선 필요” 상태다. 성과급과 교사/교수 연봉이 기본 체력을 만들었고, 재산세·정치인 재산·복지 계산기가 보조 성장축으로 붙었다. 남은 작업은 트래픽을 더 만드는 것보다, 이미 받은 노출을 더 많은 클릭으로 바꾸는 쪽에 집중하는 것이 좋다.

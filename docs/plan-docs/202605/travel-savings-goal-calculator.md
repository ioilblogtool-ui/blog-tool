# 4. 핵심 계산 로직

## 4-1. 기본 계산식

```text
목표 여행 총예산 = 항공비 + 숙박비 + 식비 + 교통비 + 쇼핑/기타비

부족 금액 = 목표 여행 총예산 - 현재 보유 여행 자금

월 적립 필요액 = 부족 금액 ÷ 출발까지 남은 개월 수
```

## 4-2. 적금 이자 반영 계산식

단순 월 적립액 계산 후, 연이율 2%, 3%, 4% 시나리오를 함께 제공.

```text
월 이율 = 연이율 ÷ 12

미래가치 = 월 적립액 × {((1 + 월이율)^개월수 - 1) ÷ 월이율}

목표 달성 가능 여부 = 현재 보유 자금 + 미래가치 ≥ 목표 여행 총예산
```

# 15. 개발 구현 메모

## 계산 함수 예시

```ts
type TravelBudgetInput = {
  flightCost: number;
  lodgingCost: number;
  foodCost: number;
  transportCost: number;
  activityCost: number;
  shoppingCost: number;
  currentSavings: number;
  monthsLeft: number;
  annualInterestRate: number;
};

type TravelSavingsResult = {
  totalBudget: number;
  shortageAmount: number;
  monthlySavingRequired: number;
  weeklySavingRequired: number;
  dailySavingRequired: number;
  estimatedFinalAmount: number;
  isGoalAchievable: boolean;
};

export function calculateTravelSavingsGoal(
  input: TravelBudgetInput
): TravelSavingsResult {
  const totalBudget =
    input.flightCost +
    input.lodgingCost +
    input.foodCost +
    input.transportCost +
    input.activityCost +
    input.shoppingCost;

  const shortageAmount = Math.max(totalBudget - input.currentSavings, 0);

  const monthlyRate = input.annualInterestRate / 100 / 12;

  let monthlySavingRequired = 0;

  if (input.monthsLeft <= 0) {
    monthlySavingRequired = shortageAmount;
  } else if (monthlyRate === 0) {
    monthlySavingRequired = shortageAmount / input.monthsLeft;
  } else {
    monthlySavingRequired =
      shortageAmount /
      (((1 + monthlyRate) ** input.monthsLeft - 1) / monthlyRate);
  }

  const estimatedFinalAmount =
    input.currentSavings +
    monthlySavingRequired *
      (((1 + monthlyRate) ** input.monthsLeft - 1) / (monthlyRate || 1));

  return {
    totalBudget: Math.round(totalBudget),
    shortageAmount: Math.round(shortageAmount),
    monthlySavingRequired: Math.ceil(monthlySavingRequired),
    weeklySavingRequired: Math.ceil(monthlySavingRequired / 4.345),
    dailySavingRequired: Math.ceil(monthlySavingRequired / 30),
    estimatedFinalAmount: Math.round(estimatedFinalAmount),
    isGoalAchievable: estimatedFinalAmount >= totalBudget,
  };
}
```

# 17. FAQ

## Q1. 여행 적금은 몇 개월 전부터 드는 게 좋나요?

일본·제주처럼 가까운 여행은 보통 3~6개월 전부터 준비해도 충분합니다. 유럽·미주처럼 항공권과 숙박비가 큰 여행은 12개월 전부터 월 적립액을 나눠 준비하는 것이 부담이 적습니다.

## Q2. 여행비 예산에서 항공권 비중은 어느 정도가 적당한가요?

단거리 해외여행은 항공권 비중이 전체 예산의 30~40% 수준이 되는 경우가 많습니다. 장거리 여행은 항공권 비중이 더 커질 수 있어 숙박비와 함께 가장 먼저 예산을 잡는 것이 좋습니다.

## Q3. 적금 이자를 꼭 반영해야 하나요?

6개월 이내 단기 여행이라면 이자 효과는 크지 않습니다. 하지만 1년 이상 준비하는 여행이라면 연 2~4% 적금 이자도 일부 도움이 될 수 있습니다.

## Q4. 현재 여행 자금이 하나도 없으면 어떻게 계산하나요?

현재 보유 여행 자금에 0원을 입력하면 됩니다. 계산기는 전체 목표 예산을 남은 개월 수로 나눠 월 적립 필요액을 계산합니다.

## Q5. 2인 여행도 계산할 수 있나요?

가능합니다. 총 여행 예산을 2인 기준으로 입력하거나, 동행 인원을 입력해 1인당 부담액을 함께 보여주는 방식으로 확장할 수 있습니다.

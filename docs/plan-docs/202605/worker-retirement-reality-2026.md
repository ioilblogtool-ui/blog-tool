## 3. 핵심 메시지

```md
직장인의 노후 준비는 “국민연금만 있으면 되겠지” 수준으로는 부족할 가능성이 높습니다.

특히 은퇴 후 최소 생활비, 의료비, 주거비, 물가상승률을 함께 고려하면
국민연금 + 퇴직연금 + 개인연금의 3층 구조를 갖추지 않은 사람은
은퇴 후 월 현금흐름 갭이 발생할 수 있습니다.
```

2025년 사회조사에서는 19세 이상 국민 중 노후 준비를 하고 있거나 준비가 되어 있다고 응답한 비중이 71.5%로 집계됐고, 주된 노후 준비 방법은 국민연금이 58.5%로 가장 높게 나타났습니다. 즉, 준비율은 높아졌지만 상당수는 여전히 국민연금 의존도가 높은 구조입니다. ([마켓인][1])

## ② 연령대별 평균 연금 예상 수령액

### 섹션 목적

| 연령대 | 국민연금 예상 수령 구조       | 핵심 체크            |
| --- | ------------------- | ---------------- |
| 30대 | 납부 기간이 길게 남아 있음     | 소득 상승, 납부 지속성 중요 |
| 40대 | 가입 기간과 소득 수준이 본격 반영 | 퇴직연금·IRP 병행 필요   |

국민연금공단은 2025년 7월 기준 기준소득월액 상한액 637만 원, 하한액 40만 원을 반영한 노령연금 예상연금월액표를 제공하고 있습니다. 국민연금 수령액은 가입기간과 기준소득월액에 따라 달라지므로, 리포트에서는 “평균 수령액”과 “가입기간별 예상액”을 분리해 보여주는 것이 좋습니다. ([국민연금공단][3])

## ③ 국민연금 단독으로 생활 가능한가?

### 핵심 질문

```md
국민연금 월 수령액만으로 은퇴 후 최소 생활비를 감당할 수 있을까?
```

### 비교표 템플릿

| 구분          |   월 금액 | 해석            |
| ----------- | -----: | ------------- |
| 국민연금 평균 수령액 | 데이터 입력 | 기본 노후소득       |
| 1인 최소 생활비   | 데이터 입력 | 최소 생계 기준      |
| 부부 최소 생활비   | 데이터 입력 | 부부 기준 생활비     |
| 부족액         |  자동 계산 | 매월 추가 필요 현금흐름 |

### 산식

```text
월 부족액 = 은퇴 후 월 생활비 - 국민연금 월 수령액
연 부족액 = 월 부족액 × 12
20년 부족액 = 연 부족액 × 20
30년 부족액 = 연 부족액 × 30
```

### 본문 예시

```md
국민연금은 노후 소득의 기초 역할을 하지만, 생활비 전체를 대체하기에는 부족할 수 있습니다.
특히 주거비, 의료비, 보험료, 부모·자녀 지원비가 남아 있다면 국민연금 단독 생활은 현실적으로 빠듯해질 가능성이 큽니다.
```

언론이 인용한 국민연금공단 2025년 7월 기준 공표통계에 따르면 전체 노령연금 수급자의 월평균 수령액은 약 67만 9,924원으로 보도됐고, 20년 이상 가입자의 평균은 월 112만 원 수준으로 소개됐습니다. 이는 가입기간에 따라 수령액 차이가 크다는 점을 보여줍니다. ([연합뉴스][4])

## ④ 퇴직연금 적립 현황 분포

### 핵심 메시지

```md
퇴직연금은 직장인의 두 번째 노후 안전망이지만,
적립금 규모와 운용 방식에 따라 은퇴 후 체감 효과가 크게 달라집니다.
```

2024년 말 기준 퇴직연금 적립금은 431.7조 원으로 처음 400조 원을 돌파했고, 실적배당형 상품 투자금은 75.2조 원으로 전년 대비 53.3% 증가했습니다. 퇴직연금이 단순 예금형에서 투자형으로 이동하는 흐름이 나타나고 있습니다. ([고용노동부][5])

### 구성표

| 구분    | 설명                           |
| ----- | ---------------------------- |
| DB형   | 회사가 운용 책임, 퇴직급여 안정성 높음       |
| DC형   | 근로자가 운용 책임, 수익률에 따라 격차 발생    |
| IRP   | 개인형 퇴직연금, 이직·퇴직금·추가납입 관리     |
| 디폴트옵션 | 운용 지시가 없을 때 사전 지정 상품으로 자동 운용 |

## ⑤ 연금저축·IRP 가입률 현실

### 섹션 목적

국민연금 외 개인연금 준비의 현실을 보여줌.

### 핵심 포인트

| 항목     | 설명                          |
| ------ | --------------------------- |
| 연금저축   | 세액공제와 장기 적립 목적              |
| IRP    | 퇴직금 관리 + 추가납입 + 세액공제        |
| 연금저축펀드 | ETF 장기투자 수요 증가              |
| 연금저축보험 | 기존 가입자 많지만 신규 관심은 상대적으로 낮아짐 |

보험연구원 자료에 따르면 연금저축 적립금은 2024년 180.1조 원 규모로 제시됐고, 연금저축펀드 계약 건수는 2019년 94만 건에서 2024년 430만 건으로 크게 늘었습니다. 반면 연금저축보험 계약 건수는 같은 기간 482만 건에서 411.9만 건으로 감소했습니다. ([Kiri][6])

## ⑥ 노후 자금 부족 갭 시뮬레이션

### 섹션 목적

가장 중요한 체류시간 유도 구간. 30대·40대·50대별로 “월 부족액 → 필요 자산”을 보여줌.

### 기본 산식

```text
은퇴 후 월 부족액 = 목표 월 생활비 - 예상 월 연금수령액

필요 추가자산 =
은퇴 후 연 부족액 ÷ 안전 인출률
```

### 예시표

| 연령대 | 목표 은퇴 생활비 | 예상 연금 월수령액 |  월 부족액 | 4% 룰 기준 필요 추가자산 |
| --- | --------: | ---------: | -----: | --------------: |
| 30대 |    300만 원 |     100만 원 | 200만 원 |            6억 원 |
| 40대 |    300만 원 |     120만 원 | 180만 원 |          5.4억 원 |
| 50대 |    280만 원 |     130만 원 | 150만 원 |          4.5억 원 |

### 해석 문구

```md
월 150만 원이 부족해 보이면 작게 느껴질 수 있지만,
이를 4% 룰 기준으로 환산하면 약 4억 5,000만 원의 추가 은퇴자산이 필요합니다.
노후 준비의 핵심은 “목돈 얼마”가 아니라 “매월 부족한 현금흐름을 어떻게 메울 것인가”입니다.
```

## ⑧ 실제 은퇴자 인터뷰 기반 생활비 체감 데이터

### 운영 방식

실제 인터뷰가 없다면 처음에는 “가상 케이스 기반”으로 작성하고, 이후 독자 설문을 받아 업데이트하는 구조 추천.

| 유형        |       월 생활비 | 체감 포인트          |

| 1인 은퇴자    | 180만~250만 원 | 주거비 유무에 따라 차이 큼 |
| 부부 은퇴자    | 280만~400만 원 | 의료비·식비·경조사비 변수  |

| 월세 거주 은퇴자 |   300만 원 이상 | 주거비 부담 큼        |

### 문구 예시

```md
은퇴 후 생활비는 단순 식비와 공과금만의 문제가 아닙니다.
건강보험료, 병원비, 경조사비, 주거 수선비, 자녀 지원비까지 포함하면 체감 생활비는 예상보다 커질 수 있습니다.
```

## ⑨ 노후 준비 우수 사례: 월 50만 원으로 준비한 40대

### 시나리오

| 항목      |         값 |
| ------- | --------: |
| 시작 나이   |       40세 |
| 월 납입액   |     50만 원 |
| 투자 기간   |       20년 |
| 연평균 수익률 |        5% |
| 예상 적립금  | 약 2.05억 원 |

### 산식

# 2. 콘텐츠 한 줄 정의

> 현재 임신 주수, 쌍둥이 여부, 고령임신 여부를 입력하면 출산 전까지 남은 산부인과 검사 항목과 예상 검사비를 계산해주는 임신 검사비 계산기

---

# 3. 핵심 포인트

임신·출산 진료비 지원은 건강보험 가입자 또는 피부양자 대상으로 제공되며, 임신 1회당 기본 100만 원, 다태아는 140만 원이 지원됩니다. 분만취약지의 경우 20만 원이 추가될 수 있고, 2024년 1월 1일 이후 임신 20주 이상 다태아 임신을 유지 중이거나 출산한 임산부는 태아당 100만 원이 되도록 추가 지급 대상이 될 수 있습니다. ([보건복지부 대표홈페이지][1])

이 계산기는 **병원별 실제 비용을 확정하는 도구가 아니라**, 앞으로 남은 검사비를 대략적으로 예산화하는 도구로 설계하는 것이 안전합니다.

# 7. 임신 주수별 검사 데이터 구조

임신 중 검사는 병원마다 차이가 있지만, 일반적으로 초기 검사, 11~13주 기형아 선별검사, 15~18주 중기 기형아 검사, 20~24주 정밀초음파, 24~28주 임신성 당뇨 검사, 35~37주 GBS 검사 등이 주요 타임라인으로 활용됩니다. NIPT는 보통 임신 10주 이후부터 22주 사이에 시행 가능하며, 양수검사는 고령 산모이거나 선별검사 이상 소견이 있을 때 16~20주 전후에 고려될 수 있습니다. ([강남차병원][2])

|     주수 | 검사 항목             | 기본/선택   |           예상 비용 범위 | 비고              |
| -----: | ----------------- | ------- | -----------------: | --------------- |
|   5~8주 | 임신 확인 초음파, 기본 진료  | 기본      |     20,000~80,000원 | 병원별 차이 큼        |
|  8~10주 | 산전 기본검사, 혈액·소변검사  | 기본      |    50,000~150,000원 | 보건소 무료 검사 여부 확인 |
| 10~22주 | NIPT / 니프티 검사     | 선택      |   400,000~700,000원 | 고령임신 시 관심 높음    |
| 11~13주 | 1차 기형아 검사, NT 초음파 | 기본/권장   |    50,000~150,000원 | 병원 패키지 여부 차이    |
| 15~18주 | 2차 기형아 검사         | 기본/권장   |    30,000~100,000원 | 통합검사 연계         |
| 16~20주 | 양수검사              | 선택/소견 시 | 600,000~1,200,000원 | 검사 이상·고령임신 등    |
| 20~24주 | 정밀초음파             | 기본/권장   |   100,000~250,000원 | 태아 구조 확인        |
| 24~28주 | 임신성 당뇨 검사         | 기본      |     20,000~70,000원 | 당부하 검사          |
| 28~32주 | 빈혈검사, 초음파         | 기본      |    30,000~100,000원 | 병원별 상이          |
| 32~34주 | 후기 초음파, 태아 성장 확인  | 기본      |    50,000~150,000원 | 다태아는 빈도 증가 가능   |
| 35~37주 | GBS 검사            | 기본/권장   |     20,000~70,000원 | 분만 전 검사         |
| 36~40주 | 막달 진료, 태동검사, 초음파  | 기본      |   100,000~300,000원 | 진료 횟수 증가        |

> 비용은 콘텐츠용 예시 범위입니다. 실제 금액은 병원, 지역, 급여·비급여 여부, 검사 패키지, 태아 수, 의료진 소견에 따라 달라질 수 있습니다.

# 8. 계산 로직

## 8-1. 기본 계산식

```text
남은 검사비 합계 = 현재 주수 이후 검사 항목별 예상 비용 합산

바우처 잔액 = 임신·출산 진료비 지원금 - 현재까지 사용한 바우처 금액

예상 본인부담액 = max(남은 검사비 합계 - 바우처 잔액, 0)
```

---

## 8-2. 단태아/다태아 지원금 기준

| 구분       |               기본 지원금 | 계산 반영       |
| -------- | -------------------: | ----------- |
| 단태아      |           1,000,000원 | 기본값         |
| 다태아      |           1,400,000원 | 기본값         |
| 분만취약지    |            +200,000원 | 지역 선택 시 반영  |
| 다태아 추가지원 | 태아당 1,000,000원 기준 가능 | 별도 안내 문구 표시 |

보건복지부 기준으로 건강보험 임신·출산 진료비 지원은 임신 1회당 100만 원, 다태아는 140만 원이며, 분만취약지는 20만 원 추가 지원이 가능합니다. 다태아 추가지원은 태아당 100만 원이 되도록 추가 지급하는 구조로 안내되어 있습니다. ([보건복지부 대표홈페이지][1])

## 17-2. 계산 함수 예시

```ts
type PregnancyCostInput = {
  currentWeek: number;
  isMultiplePregnancy: boolean;
  isAdvancedMaternalAge: boolean;
  usedVoucherAmount: number;
  includeNipt: boolean;
  includeAmniocentesis: boolean;
  isUnderservedArea: boolean;
};

type PregnancyCostResult = {
  totalMinCost: number;
  totalMaxCost: number;
  voucherAmount: number;
  remainingVoucherAmount: number;
  estimatedOutOfPocketMin: number;
  estimatedOutOfPocketMax: number;
  remainingTests: PregnancyTestItem[];
};

export function calculatePregnancyCheckupCost(
  input: PregnancyCostInput,
  testItems: PregnancyTestItem[]
): PregnancyCostResult {
  const baseVoucherAmount = input.isMultiplePregnancy ? 1400000 : 1000000;
  const areaBonus = input.isUnderservedArea ? 200000 : 0;
  const voucherAmount = baseVoucherAmount + areaBonus;

  const remainingTests = testItems.filter((item) => {
    if (item.endWeek < input.currentWeek) return false;

    if (item.id === 'nipt' && !input.includeNipt) return false;
    if (item.id === 'amniocentesis' && !input.includeAmniocentesis) return false;

    if (input.isMultiplePregnancy && !item.appliesTo.multiple) return false;
    if (!input.isMultiplePregnancy && !item.appliesTo.singleton) return false;

    return true;
  });

  const totalMinCost = remainingTests.reduce((sum, item) => sum + item.minCost, 0);
  const totalMaxCost = remainingTests.reduce((sum, item) => sum + item.maxCost, 0);

  const remainingVoucherAmount = Math.max(voucherAmount - input.usedVoucherAmount, 0);

  return {
    totalMinCost,
    totalMaxCost,
    voucherAmount,
    remainingVoucherAmount,
    estimatedOutOfPocketMin: Math.max(totalMinCost - remainingVoucherAmount, 0),
    estimatedOutOfPocketMax: Math.max(totalMaxCost - remainingVoucherAmount, 0),
    remainingTests,
  };
}
```

# 19. FAQ

## Q1. 임신 검사비는 총 얼마 정도 나오나요?

임신 검사비는 병원, 검사 선택 여부, 급여·비급여 여부에 따라 차이가 큽니다. 일반적인 산전검사만 진행하면 바우처 안에서 상당 부분 해결되는 경우도 있지만, NIPT, 양수검사, 정밀초음파 등 선택 검사를 포함하면 추가 본인부담이 생길 수 있습니다.

## Q2. 니프티 검사는 필수인가요?

NIPT는 일반적으로 필수 검사가 아니라 선택 검사입니다. 다만 만 35세 이상 고령임신, 기형아 선별검사 이상 소견, 염색체 이상 위험 평가가 필요한 경우 관심이 높습니다. NIPT는 임신 10주 이후 시행 가능한 검사로 안내되는 경우가 많습니다. ([강남차병원][2])

## Q3. 고령임신이면 검사비가 무조건 더 많이 드나요?

무조건은 아닙니다. 다만 NIPT, 양수검사, 추가 정밀초음파 등을 선택하거나 의료진이 권유하는 경우 검사비가 늘어날 수 있습니다.

## Q4. 쌍둥이 임신이면 지원금도 늘어나나요?

네. 임신·출산 진료비 지원은 단태아 100만 원, 다태아 140만 원이 기본입니다. 또한 2024년 1월 1일 이후 임신 20주 이상 다태아 임신을 유지 중이거나 출산한 경우 태아당 100만 원이 되도록 추가 지급 대상이 될 수 있습니다. ([보건복지부 대표홈페이지][1])

## Q5. 보건소 검사를 이용하면 비용을 줄일 수 있나요?

지역별로 임신 초기 혈액검사, 소변검사, 엽산제·철분제 지원 등이 제공될 수 있습니다. 거주지 보건소 지원 항목을 확인하면 초기 검사비를 줄이는 데 도움이 됩니다.

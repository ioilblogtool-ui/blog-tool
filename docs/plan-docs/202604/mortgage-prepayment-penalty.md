## 2. 콘텐츠 핵심 컨셉

> **중도상환수수료 계산기는 ‘수수료가 얼마냐’보다 ‘지금 갚는 게 이득이냐’를 판단하게 해주는 계산기다.**

대출을 조기 상환할 때는 단순히 수수료만 계산하면 부족합니다.

중요한 판단 기준은 다음 4가지입니다.

| 판단 항목   | 설명                       |
| ------- | ------------------------ |
| 중도상환수수료 | 중도상환금액 × 수수료율 × 잔여기간 비율  |
| 절감 이자   | 조기 상환으로 앞으로 내지 않아도 되는 이자 |
| 순이익/손해  | 절감 이자 - 중도상환수수료          |
| 면제 도달일  | 대출 실행 후 3년 등 수수료 면제 시점   |

2025년 1월 13일부터 신규 대출은 중도상환수수료를 실제 발생한 비용 범위 안에서만 부과하도록 제도가 개편됐고, 금융위원회는 소비자가 대출일부터 3년 이내에 상환하는 경우 예외적으로 중도상환수수료 부과가 가능하다고 설명했습니다. ([금융위원회][1])

또한 2026년 1월 1일부터는 농협·수협 등 상호금융권 신규 취급 대출에도 같은 취지의 중도상환수수료 부과체계 개선이 적용될 예정입니다. ([금융위원회][2])

# 6. 계산 로직

## 6.1 기본 중도상환수수료 공식

```txt
중도상환수수료 = 중도상환금액 × 중도상환수수료율 × 잔여 부과기간 비율
```

잔여 부과기간 비율:

```txt
잔여 부과기간 비율 = 남은 수수료 부과일수 ÷ 전체 수수료 부과일수
```

예시:

| 항목         |           값 |
| ---------- | ----------: |
| 중도상환금액     | 50,000,000원 |
| 수수료율       |        1.2% |
| 수수료 부과기간   |          3년 |
| 대출 경과기간    |          1년 |
| 남은 부과기간    |          2년 |
| 잔여 부과기간 비율 |         2/3 |
| 예상 수수료     |    400,000원 |

계산:

```txt
50,000,000 × 1.2% × 2/3 = 400,000원
```

---

## 6.2 절감 이자 계산

단순 계산 버전:

```txt
절감 이자 = 중도상환금액 × 대출금리 × 남은 기간
```

예시:

| 항목          |           값 |
| ----------- | ----------: |
| 중도상환금액      | 50,000,000원 |
| 대출금리        |      연 4.2% |
| 남은 수수료 부과기간 |          2년 |
| 예상 절감 이자    |  4,200,000원 |

계산:

```txt
50,000,000 × 4.2% × 2년 = 4,200,000원
```

---

## 6.3 순이익/손해 계산

```txt
순이익 = 절감 이자 - 중도상환수수료
```

예시:

| 항목      |          값 |
| ------- | ---------: |
| 절감 이자   | 4,200,000원 |
| 중도상환수수료 |   400,000원 |
| 순이익     | 3,800,000원 |

출력 문구:

```txt
지금 5,000만 원을 상환하면 예상 중도상환수수료는 약 40만 원입니다.
하지만 남은 기간 동안 약 420만 원의 이자를 줄일 수 있어,
단순 계산 기준 약 380만 원 이득입니다.
```

# 10. 은행별 수수료율 프리셋 설계

실제 수수료율은 은행·상품·금리 유형·대출 실행일에 따라 다르므로, 계산기에서는 **프리셋 + 직접 입력** 방식을 추천합니다.

| 은행/기관     |   기본 프리셋 | 설명                               |
| --------- | -------: | -------------------------------- |
| 시중은행 주담대  | 0.6~1.4% | 2025년 이후 신규 대출은 기존보다 낮아진 수수료율 가능 |
| 인터넷은행 주담대 | 0.6~1.4% | 상품별 확인 필요                        |
| 전세자금대출    | 0.5~1.0% | 보증기관·은행별 차이                      |
| 신용대출      | 0.0~0.8% | 일부 상품은 면제 가능                     |
| 상호금융 주담대  | 0.6~1.5% | 2026년 신규 대출부터 실비용 기준 개선 적용 예정    |

금융위원회는 2025년 1월 13일부터 은행·저축은행·보험사 등 신규 대출의 중도상환수수료율이 실비용 기준으로 개편된다고 밝혔고, 정부 설명자료에서는 중도상환수수료율이 절반 수준으로 인하된다고 안내했습니다. ([금융위원회][1])

# 12. 고급 기능: 손익분기 시점 계산

## 12.1 손익분기 개념

손익분기 시점은 다음 조건이 같아지는 시점입니다.

```txt
절감 이자 = 중도상환수수료
```

즉, 조기 상환으로 아끼는 이자가 수수료보다 커지는 순간입니다.

---

# 13. 추가 비교 기능: 대환대출 판단

중도상환수수료 계산기는 대출 갈아타기 계산기와 연결하면 체류시간이 길어집니다.

## 대환대출 비교 공식

```txt
대환 순이익 = 기존 대출 이자 - 신규 대출 이자 - 중도상환수수료 - 신규 대출 부대비용
```

| 입력값        |           예시 |
| ---------- | -----------: |
| 기존 대출금리    |         4.8% |

# 17. 개발 구현 포인트

## 17.1 계산 함수 예시

```ts
type PrepaymentPenaltyInput = {
  loanPrincipal: number;
  prepaymentAmount: number;
  loanStartDate: string;
  prepaymentDate: string;
  penaltyRate: number;
  penaltyPeriodYears: number;
  annualInterestRate: number;
};

type PrepaymentPenaltyResult = {
  elapsedDays: number;
  totalPenaltyDays: number;
  remainingPenaltyDays: number;
  remainingPenaltyRatio: number;
  penaltyAmount: number;
  savedInterest: number;
  netBenefit: number;
  penaltyFreeDate: string;
  decision: "REPAY_NOW" | "WAIT" | "NEUTRAL";
};

export function calculatePrepaymentPenalty(
  input: PrepaymentPenaltyInput
): PrepaymentPenaltyResult {
  const startDate = new Date(input.loanStartDate);
  const repayDate = new Date(input.prepaymentDate);

  const msPerDay = 1000 * 60 * 60 * 24;
  const elapsedDays = Math.max(
    0,
    Math.floor((repayDate.getTime() - startDate.getTime()) / msPerDay)
  );

  const totalPenaltyDays = input.penaltyPeriodYears * 365;
  const remainingPenaltyDays = Math.max(0, totalPenaltyDays - elapsedDays);
  const remainingPenaltyRatio =
    totalPenaltyDays === 0 ? 0 : remainingPenaltyDays / totalPenaltyDays;

  const penaltyAmount =
    input.prepaymentAmount *
    (input.penaltyRate / 100) *
    remainingPenaltyRatio;

  const remainingYears = remainingPenaltyDays / 365;

  const savedInterest =
    input.prepaymentAmount *
    (input.annualInterestRate / 100) *
    remainingYears;

  const netBenefit = savedInterest - penaltyAmount;

  const penaltyFreeDate = new Date(startDate);
  penaltyFreeDate.setFullYear(startDate.getFullYear() + input.penaltyPeriodYears);

  const decision =
    netBenefit > 0 ? "REPAY_NOW" : netBenefit < 0 ? "WAIT" : "NEUTRAL";

  return {
    elapsedDays,
    totalPenaltyDays,
    remainingPenaltyDays,
    remainingPenaltyRatio,
    penaltyAmount: Math.round(penaltyAmount),
    savedInterest: Math.round(savedInterest),
    netBenefit: Math.round(netBenefit),
    penaltyFreeDate: penaltyFreeDate.toISOString().slice(0, 10),
    decision,
  };
}
```

# 중도상환수수료 계산기

주택담보대출이나 전세자금대출을 미리 갚으려 할 때 가장 먼저 확인해야 할 것은 중도상환수수료입니다.

하지만 수수료만 보면 판단이 어렵습니다.  
조기 상환으로 줄어드는 이자와 중도상환수수료를 함께 비교해야 “지금 갚는 게 이득인지” 알 수 있습니다.

## 입력값

- 대출 원금
- 대출 실행일
- 중도상환 예정일
- 중도상환 금액
- 대출 금리
- 중도상환수수료율
- 수수료 부과 기간
- 대출 종류

## 계산 결과

- 예상 중도상환수수료
- 예상 절감 이자
- 순이익 또는 손해
- 수수료 면제 도달일
- 면제일까지 남은 기간
- 지금 상환 유리 여부

## 중도상환수수료 계산 공식

중도상환수수료는 일반적으로 다음 방식으로 계산합니다.

중도상환수수료 = 중도상환금액 × 수수료율 × 잔여 부과기간 비율

예를 들어 5,000만 원을 중도상환하고 수수료율이 1.2%, 수수료 부과기간 3년 중 1년이 지났다면 예상 수수료는 다음과 같습니다.

5,000만 원 × 1.2% × 2/3 = 40만 원

## 지금 갚는 게 이득인지 계산하기

중도상환 여부는 수수료만 보고 판단하면 안 됩니다.

절감 이자 = 중도상환금액 × 대출금리 × 남은 기간  
순이익 = 절감 이자 - 중도상환수수료

절감 이자가 수수료보다 크면 지금 상환하는 것이 유리할 수 있습니다.

## 중도상환수수료 면제일

많은 대출 상품은 대출 실행 후 3년이 지나면 중도상환수수료가 면제됩니다.

다만 대출 증액, 대환, 기한연장, 상품 변경 이력이 있으면 기준일이 달라질 수 있습니다.  
실제 상환 전에는 은행 앱이나 영업점에서 최종 수수료를 확인해야 합니다.

## 이런 분들에게 유용합니다

- 주택담보대출 일부 상환을 고민하는 분
- 전세자금대출을 조기 상환하려는 분
- 대출 갈아타기 전에 수수료를 계산하려는 분
- 여유자금을 투자할지 대출 상환할지 고민하는 분
- 중도상환수수료 면제일까지 얼마나 남았는지 알고 싶은 분

## 주의사항

이 계산기는 입력값을 기준으로 한 예상 결과입니다.  
실제 수수료는 은행, 대출 상품, 대출 실행일, 금리 유형, 대출 변경 이력에 따라 달라질 수 있습니다.

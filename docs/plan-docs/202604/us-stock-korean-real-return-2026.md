# 6. 본문 상세 섹션

## Section 1. 왜 달러 수익률과 원화 수익률이 다른가

### 핵심 설명

한국 투자자가 미국주식을 살 때는 보통 다음 과정을 거칩니다.

| 단계 | 설명                    |
| -: | --------------------- |
|  1 | 원화를 달러로 환전            |
|  2 | 달러로 미국주식 또는 ETF 매수    |
|  3 | 주가 상승·하락 발생           |
|  4 | 매도 후 달러 확보            |
|  5 | 달러를 다시 원화로 환전         |
|  6 | 세금·수수료 차감 후 최종 실수익 확정 |

따라서 실제 수익률은 아래 구조로 계산됩니다.

```text
원화 실수익률 ≒ 주가 수익률 + 환율 수익률 - 환전수수료 - 거래수수료 - 세금
```

### 예시

| 구분        |   사례 A |   사례 B |
| --------- | -----: | -----: |
| 매수 환율     | 1,200원 | 1,450원 |
| 매도 환율     | 1,450원 | 1,200원 |
| 주가 수익률    |   +20% |   +20% |
| 환율 효과     |    플러스 |   마이너스 |
| 원화 체감 수익률 |  크게 증가 |  크게 감소 |

## Section 2. 2016~2026년 S&P500 달러 수익 vs 원화 환산 수익 비교

## 데이터 설계

S&P 500 수익률은 Yahoo Finance, FRED, Slickcharts 등에서 연도별 데이터를 수집할 수 있습니다. FRED의 S&P 500 데이터는 가격지수이며 배당 재투자는 포함하지 않는다고 명시되어 있으므로, 본문에서는 **가격지수 기준**과 **Total Return 기준**을 분리하는 것이 좋습니다. ([FRED][2])

원·달러 환율은 한국은행 ECOS 또는 Federal Reserve H.10 데이터로 검증할 수 있습니다. Federal Reserve H.10은 한국 원화의 달러당 환율 히스토리 데이터를 제공합니다. ([한국은행 경제통계시스템][3])

### 표 구조

| -------: | ------------: | ---------: | -----: | --------: | ----: |
|     2016 |            입력 |         입력 |     입력 |     자동 계산 | 자동 계산 |

### 계산식

```text

(1 + 달러 주가 수익률) × (1 + 환율 변동률) - 1
```

## Section 8. 배당금도 환차손 적용되는가

### 핵심 답변

미국주식 배당금은 달러로 지급됩니다.
따라서 배당 투자자도 최종적으로 원화 환산을 하면 환율 영향을 받습니다.

| 항목        | 환율 영향     |
| --------- | --------- |
| 달러 배당금 수령 | 있음        |
| 원화 환전 시점  | 중요        |
| 배당소득세     | 원천징수 영향   |
| 재투자       | 재투자 환율 영향 |
| 생활비 인출    | 원화 환율 중요  |

### 본문 예시

```text
배당금이 매달 100달러씩 들어와도, 환율이 1,500원일 때는 15만 원 수준이고 환율이 1,200원일 때는 12만 원 수준입니다.  
배당금 자체는 같아도 원화 생활비로 환산하면 체감 현금흐름이 달라집니다.
```

## Section 10. 미국주식 양도소득세 22% 포함 세후 실수익률

국내 투자자가 해외주식 매매차익을 실현하면 연간 기본공제 250만 원을 제외한 과세표준에 대해 22% 세율이 적용되는 구조로 안내됩니다. 최근 보도에서도 해외주식 양도차익에 대해 기본공제 250만 원과 22% 세율이 언급됩니다. ([조선비즈][5])

### 계산 구조

```text
과세대상 양도차익 = 연간 해외주식 양도차익 - 250만 원
양도소득세 = 과세대상 양도차익 × 22%
세후 수익 = 양도차익 - 양도소득세 - 수수료
```

### 예시

| 항목     |       금액 |
| ------ | -------: |
| 매수 원금  | 1,000만 원 |
| 매도 평가금 | 1,500만 원 |
| 양도차익   |   500만 원 |
| 기본공제   |   250만 원 |
| 과세대상   |   250만 원 |
| 세율     |      22% |
| 예상 세금  |    55만 원 |
| 세후 차익  |   445만 원 |

### 주의

| 구분          | 설명                            |
| ----------- | ----------------------------- |
| 손익통산        | 같은 해 해외주식 손익끼리 통산 가능          |
| 기본공제        | 연 250만 원                      |
| 신고 시기       | 통상 다음 해 5월 종합소득세 신고 기간과 함께 확인 |
| 환율 적용       | 매수·매도·비용 환산 기준 확인 필요          |
| 국내상장 해외 ETF | 과세 체계가 다를 수 있어 별도 설명 필요       |

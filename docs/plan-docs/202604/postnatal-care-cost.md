## 1. 콘텐츠 핵심 컨셉

> **거주 지역, 이용 기간, 이용 시간, 바우처 적용 여부를 입력하면 산후도우미 예상 본인부담금을 계산해주는 도구**

산후도우미 비용은 단순히 “몇 주 이용하느냐”만으로 결정되지 않고, 아래 요소에 따라 크게 달라진다.

| 비용 결정 요소 | 설명                              |
| -------- | ------------------------------- |
| 출산 유형    | 단태아 / 쌍태아 / 삼태아 이상              |
| 출산 순위    | 첫째 / 둘째 / 셋째 이상                 |
| 소득구간     | 자격확인 / 기준중위소득 150% 이하 / 150% 초과 |
| 이용 기간    | 단축 / 표준 / 연장                    |
| 지역 추가지원  | 서울·경기·인천 등 지자체별 본인부담금 추가지원 여부   |
| 민간 추가비용  | 주말, 야간, 입주형, 쌍둥이, 프리미엄 관리사 등    |

정부 산모·신생아 건강관리 지원사업은 기본적으로 기준중위소득 150% 이하 출산가정을 대상으로 하며, 쌍생아 이상·둘째아 이상·희귀난치성질환 산모·장애인 산모 등은 예외지원 대상에 포함될 수 있다. 서울시 안내 기준으로 신청 기간은 출산예정일 40일 전부터 출산일 이후 30일까지, 바우처 유효기간은 출산일로부터 60일 이내로 안내되어 있다. ([서울시 임신·출산 정보센터][1])

## 7. 계산 함수 예시

```ts
export interface PostnatalCareInput {
  region: RegionCode;
  birthType: BirthType;
  childOrder: 1 | 2 | 3;
  incomeType: IncomeType;
  servicePeriod: ServicePeriod;
  useVoucher: boolean;
  careType: 'DAY' | 'NIGHT' | 'LIVE_IN' | 'FULL_TIME_24H';
  privateExtraCost: number;
}

export interface PostnatalCareResult {
  totalPrice: number;
  governmentSubsidy: number;
  localSubsidy: number;
  baseUserPayment: number;
  privateExtraCost: number;
  careTypeExtraCost: number;
  finalUserPayment: number;
}

export function calculatePostnatalCareCost(
  input: PostnatalCareInput,
  rates: PostnatalCareRate[],
): PostnatalCareResult {
  const rate = rates.find(
    (item) =>
      item.region === input.region &&
      item.birthType === input.birthType &&
      item.childOrder === input.childOrder &&
      item.incomeType === input.incomeType &&
      item.servicePeriod === input.servicePeriod,
  );

  if (!rate) {
    throw new Error('조건에 맞는 산후도우미 비용 기준을 찾을 수 없습니다.');
  }

  const careTypeExtraCostMap = {
    DAY: 0,
    NIGHT: 150000,
    LIVE_IN: 300000,
    FULL_TIME_24H: 500000,
  } as const;

  const governmentSubsidy = input.useVoucher ? rate.governmentSubsidy : 0;
  const baseUserPayment = rate.totalPrice - governmentSubsidy;

  const localSubsidy = 0;
  const careTypeExtraCost = careTypeExtraCostMap[input.careType];

  const finalUserPayment =
    baseUserPayment -
    localSubsidy +
    input.privateExtraCost +
    careTypeExtraCost;

  return {
    totalPrice: rate.totalPrice,
    governmentSubsidy,
    localSubsidy,
    baseUserPayment,
    privateExtraCost: input.privateExtraCost,
    careTypeExtraCost,
    finalUserPayment: Math.max(finalUserPayment, 0),
  };
}
```

## 17. FAQ 섹션

### Q1. 산후도우미 비용은 평균 얼마인가요?

정부지원 여부에 따라 다릅니다. 정부 바우처를 적용하면 전체 서비스 가격에서 정부지원금을 제외한 본인부담금만 납부하는 구조입니다. 단태아 첫째아 표준형 기준으로 공개된 2026년 일부 지자체 가격표에서는 총 서비스 가격이 약 146만 원 수준으로 안내되어 있습니다. ([서초구청][2])

### Q2. 산후도우미 정부지원은 누구나 받을 수 있나요?

기본적으로 기준중위소득 150% 이하 출산가정이 기본 지원대상이며, 둘째아 이상·쌍생아 이상·희귀난치성질환 산모·장애인 산모 등은 예외지원 대상에 포함될 수 있습니다. ([서울시 임신·출산 정보센터][1])

### Q3. 출산 후 언제까지 신청해야 하나요?

지역별로 안내가 조금 다를 수 있으나, 대체로 출산예정일 40일 전부터 출산 후 일정 기간 내 신청해야 합니다. 서울시 안내에서는 출산일로부터 60일 이내 바우처를 사용해야 한다고 안내하고 있습니다. ([서울시 임신·출산 정보센터][1])

### Q4. 쌍둥이는 비용이 더 비싼가요?

네. 쌍태아 이상은 관리 난이도와 서비스 기준이 달라져 총 서비스 가격과 지원금 구조가 단태아와 다르게 적용됩니다.

### Q5. 산후조리원과 산후도우미 중 뭐가 더 저렴한가요?

일반적으로 산후도우미가 산후조리원보다 총지출은 낮을 가능성이 큽니다. 다만 식사, 신생아 케어, 산모 휴식, 가족 도움 여부까지 고려해야 하므로 단순 비용만으로 결정하기는 어렵습니다.

## 2. 콘텐츠 핵심 컨셉

> 여행 인원과 항목별 지출을 입력하면
> **1인당 부담액, 개인별 최종 정산액, 누가 누구에게 얼마 보내야 하는지**를 자동 계산해주는 실전형 여행 정산 계산기

여행 경비는 단순히 총액을 인원수로 나누면 끝나는 경우보다 복잡한 경우가 많습니다.

예를 들면:

```text
A가 숙소 40만 원 결제
B가 식비 12만 원 결제
C는 놀이공원 안 감
D는 항공권을 따로 결제
커플 2명은 한 명이 대표 결제
가족 여행은 부모가 60%, 자녀가 40% 부담
```

이런 상황을 계산기에서 한 번에 정리하는 것이 핵심입니다.

## 5. 계산 모드

## 5.1 더치페이 모드

가장 기본적인 정산 방식입니다.

```text
항목별 1인 부담액 = 해당 항목 비용 ÷ 해당 항목 참여 인원 수
```

예시:

| 항목  |       금액 | 참여자 |    1인 부담 |
| --- | -------: | --- | -------: |
| 숙박  | 400,000원 | 4명  | 100,000원 |
| 입장료 |  90,000원 | 3명  |  30,000원 |
| 렌터카 | 120,000원 | 4명  |  30,000원 |

---

## 5.2 비율 분담 모드

소득, 나이, 가족 관계, 커플 합의 기준으로 분담률을 지정하는 방식입니다.

| 참여자 | 분담 비율 |
| --- | ----: |
| 아빠  |   50% |
| 엄마  |   30% |
| 자녀1 |   10% |
| 자녀2 |   10% |

계산식:

```text
개인 부담액 = 전체 비용 × 개인 분담 비율
```

활용 예시:

| 상황             | 추천 분담 방식         |
| -------------- | ---------------- |
| 가족 여행          | 부모 부담률 높게 설정     |
| 커플 여행          | 5:5, 6:4, 7:3 선택 |
| 소득 차이 있는 친구 여행 | 소득 기반 비율 분담      |
| 부모님 모시고 가는 여행  | 자녀들이 비용 분담       |

## 11. 계산 로직 상세

### 11.1 항목별 부담액

```ts
type Participant = {
  id: string;
  name: string;
  ratio?: number;
};

type ExpenseItem = {
  id: string;
  category: 'flight' | 'hotel' | 'food' | 'transport' | 'ticket' | 'etc';
  title: string;
  amount: number;
  paidBy: string;
  includedParticipantIds: string[];
};
```

---

### 11.2 더치페이 계산

```ts
function calculateEqualShare(
  expenses: ExpenseItem[],
  participants: Participant[]
) {
  const balanceMap = new Map<string, number>();

  participants.forEach((p) => balanceMap.set(p.id, 0));

  expenses.forEach((expense) => {
    const includedCount = expense.includedParticipantIds.length;
    const share = expense.amount / includedCount;

    // 선결제자는 낸 금액만큼 플러스
    balanceMap.set(
      expense.paidBy,
      (balanceMap.get(expense.paidBy) || 0) + expense.amount
    );

    // 참여자는 부담액만큼 마이너스
    expense.includedParticipantIds.forEach((participantId) => {
      balanceMap.set(
        participantId,
        (balanceMap.get(participantId) || 0) - share
      );
    });
  });

  return balanceMap;
}
```

---

### 11.3 송금 내역 생성 로직

```ts
type Settlement = {
  from: string;
  to: string;
  amount: number;
};

function generateSettlements(balanceMap: Map<string, number>): Settlement[] {
  const debtors = Array.from(balanceMap.entries())
    .filter(([, amount]) => amount < 0)
    .map(([id, amount]) => ({ id, amount: Math.abs(amount) }));

  const creditors = Array.from(balanceMap.entries())
    .filter(([, amount]) => amount > 0)
    .map(([id, amount]) => ({ id, amount }));

  const settlements: Settlement[] = [];

  let i = 0;
  let j = 0;

  while (i < debtors.length && j < creditors.length) {
    const debtor = debtors[i];
    const creditor = creditors[j];
    const amount = Math.min(debtor.amount, creditor.amount);

    settlements.push({
      from: debtor.id,
      to: creditor.id,
      amount: Math.round(amount),
    });

    debtor.amount -= amount;
    creditor.amount -= amount;

    if (debtor.amount === 0) i++;
    if (creditor.amount === 0) j++;
  }

  return settlements;
}
```

## 17. FAQ 초안

### Q1. 여행 경비는 총액을 인원수로 나누면 되나요?

가장 단순한 방식은 총액을 인원수로 나누는 것입니다. 다만 일부 인원만 참여한 액티비티, 선결제자, 항공권 개별 결제, 가족 분담률 등이 있으면 항목별로 따로 계산하는 것이 정확합니다.

### Q2. 선결제자가 여러 명이면 어떻게 계산하나요?

각 비용 항목마다 선결제자를 지정하면 됩니다. 계산기는 각 사람이 먼저 낸 금액과 실제 부담해야 할 금액을 비교해 최종적으로 누가 누구에게 얼마를 보내야 하는지 정리합니다.

### Q3. 특정 사람이 어떤 일정에 참여하지 않았으면 어떻게 하나요?

해당 비용 항목에서 그 사람을 제외하면 됩니다. 예를 들어 4명 여행 중 3명만 놀이공원에 갔다면 입장료는 3명에게만 나눠 계산합니다.

### Q4. 커플 여행은 무조건 5:5로 나누는 게 좋나요?

정답은 없습니다. 둘이 합의했다면 5:5, 6:4, 7:3처럼 비율 분담도 가능합니다. 계산기에서는 더치페이와 비율 분담을 모두 지원하는 것이 좋습니다.

### Q5. 해외여행 경비도 계산할 수 있나요?

가능합니다. 원화 기준 입력을 기본으로 하되, 엔화·달러·유로 입력과 환율 변환 기능을 추가하면 해외여행 정산에도 사용할 수 있습니다.

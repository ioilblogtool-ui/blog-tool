# AdSense 재심사 체크리스트 및 진행표 (2026-05-19)

## 문서 목적
- Google AdSense 재심사 전 `사이트 소유권 확인`과 `가치가 별로 없는 콘텐츠` 이슈를 한 번에 관리한다.
- 작업자는 이 문서를 기준으로 기술 상태, 정책 페이지, 얇은 콘텐츠, 대표 페이지 보강, 최종 QA를 순서대로 처리한다.
- 재심사 요청 전에는 이 문서의 `최종 재심사 게이트`가 모두 통과되어야 한다.

## 현재 판정

| 항목 | 내용 |
|---|---|
| 대상 도메인 | `bigyocalc.com` |
| AdSense 상태 | 광고 게재가 준비되지 않은 사이트 |
| 노출 메시지 | 사이트 소유권 확인 필요, 정책 위반 발견 |
| 주요 정책 사유 | 가치가 별로 없는 콘텐츠 |
| 이번 라운드 목표 | 소유권/기술 신호 정리 + 신뢰 페이지 보강 + 미완성/얇은 페이지 정리 + 대표 콘텐츠 강화 |

## 상태값 규칙

| 상태 | 의미 | 다음 액션 |
|---|---|---|
| `대기` | 아직 확인 또는 수정 전 | 파일/URL/현재 노출 상태 확인 |
| `진행중` | 수정 또는 점검 중 | 체크 항목을 순서대로 처리 |
| `검수필요` | 수정 완료, 빌드/브라우저/프로덕션 확인 전 | `npm run build`, 4321 또는 프로덕션 확인 |
| `완료` | 완료 기준 통과 | 다음 단계 진행 |
| `보류` | 계정 정보, 외부 콘솔, 자료 부족 등으로 중단 | 보류 사유와 필요한 결정을 기록 |

완료 처리 기준:
- 사용자-facing 문구가 모두 자연스러운 한국어다.
- 공식/참고/추정 데이터가 구분되어 있다.
- 미완성 문구, 빈 링크, 준비 중 문구가 없다.
- `npm run build`가 성공한다.
- 배포 후 프로덕션 URL에서 핵심 페이지가 200으로 접근된다.

## Phase 1. 정책/신뢰 페이지 정비

목표: 사이트가 누가, 어떤 기준으로, 어떤 책임 범위에서 운영되는지 명확히 보이게 한다.

| ID | URL | 작업 | 상태 | 완료 기준 | 메모 |
|---|---|---|---|---|---|
| P1-1 | `/about/` | 운영 목적/작성 기준/검수 기준 보강 | `검수필요` | 사이트 운영 기준과 데이터 분류가 10초 안에 이해됨 | 영어 `About` 등 노출 레이블 제거 |
| P1-2 | `/privacy/` | 개인정보처리방침 한국어화 및 광고/쿠키 안내 | `검수필요` | Google 광고, 쿠키, 외부 서비스 안내 포함 | 시행일/운영자/문의 정보 보강, 영어 레이블 제거 |
| P1-3 | `/terms/` | 이용약관 점검 | `검수필요` | 서비스 이용 범위와 금지행위 명확 | 영어 `Terms of Service` 등 노출 레이블 제거 |
| P1-4 | `/disclaimer/` | 면책 안내 강화 | `검수필요` | 계산 결과가 참고용 추정치임을 명확히 표시 | 금융/의료/법률 오해 방지, 영어 레이블 제거 |
| P1-5 | `/contact/` | 문의 안내 강화 | `검수필요` | 실제 연락 가능한 이메일과 문의 범위 표시 | 오류 제보/제휴/정정 요청 구분, title의 `Contact` 제거 |

### 정책 페이지 필수 문구
- 비교계산소는 공식 기관이 아니며, 공개 기준과 입력값을 바탕으로 계산/비교를 제공한다.
- 결과는 참고용이며 실제 지급액, 세금, 보험료, 계약 조건은 달라질 수 있다.
- 추정값은 `추정`, 공식 출처는 `공식`, 참고 링크는 `참고`로 구분한다.
- 오류 제보와 정정 요청을 받을 수 있는 문의 경로를 제공한다.

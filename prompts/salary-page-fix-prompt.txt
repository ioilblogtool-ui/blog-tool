## 수정 1 — [P1] URL 파라미터 자동 계산 (salary.js)

현재 문제:
URL에 ?sal=50000000&bon=3000000&r1=2000000 등 파라미터가 있어도
페이지 로드 시 결과가 "-" 상태로 표시됨.
사용자가 버튼을 직접 눌러야만 계산이 실행됨.

수정 방법:
salary.js 초기화 코드 맨 아래에 아래 로직 추가:

// URL 파라미터 있으면 자동 계산
const urlParams = new URLSearchParams(window.location.search);
const hasParams = urlParams.has('sal') || urlParams.has('r1');
if (hasParams) {
  // 짧은 딜레이 후 실행 (DOM 렌더링 완료 보장)
  setTimeout(() => {
    const calcBtn = document.getElementById('calcBtn');
    if (calcBtn) calcBtn.click();
  }, 100);
}

주의: 기존 URL 파라미터 파싱 로직이 이미 있으면 중복 추가 금지.
있는 경우 해당 위치에 if (hasParams) calcBtn.click() 만 추가.
